## 解题思路

令：

$$
t=\left\lfloor\frac{2m}{\pi}\right\rfloor,k=4(m-t)-2
$$


则：

$$
\frac{t(t+1)\times n(n+1)(2n+1)}{6}\pi+\frac{mn(n+1)(k+4m(n-1)+2kn+6\times[m>1])}{6}
$$

至此，已成艺术。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll=long long;
const double pi=acos(-1);
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	ll n,m;
	cin>>n>>m;
	ll t=m*2/pi,k=(m-t)*4-2;
	ll p=t*(t+1)*n*(n+1)*(n*2+1)/6;
	ll q=m*n*(n+1)*(m*(n-1)*4+k*(n*2+1)+(m>1)*6)/6;
	cout<<fixed<<setprecision(12)<<p*pi+q<<'\n';
	return 0;
}
```
