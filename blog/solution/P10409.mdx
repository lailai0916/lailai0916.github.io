## 解题思路

读入小时数字 $h$ 和剩余部分字符串 $s$，若 $h\ge 6$ 则直接输出原小时，否则将小时加 $24$。

利用 `setfill('0')` 与 `setw(2)` 保证小时数字始终为两位补零，然后拼接输出 $s$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int h;
	string s;
	cin>>h>>s;
	cout<<setfill('0')<<setw(2)<<(h>=6?h:h+24)<<s<<'\n';
	return 0;
}
```
