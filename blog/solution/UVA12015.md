## 题意简述

给定 $T$ 组数据，每组包含 $10$ 个网址及其相关性值，要求找出相关性最高的网站。若有多个满足要求，应该全部输出。

## 解题思路

首先读入 $10$ 个网址及其对应的相关性值，找到其中的最大相关性值。

然后再次遍历这些网址，输出所有相关性等于最大值的网站。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<pair<string,int>> a;
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int T;
	cin>>T;
	for(int j=1;j<=T;j++)
	{
		a.clear();
		int mx=0;
		for(int i=1;i<=10;i++)
		{
			string s;
			int v;
			cin>>s>>v;
			a.push_back({s,v});
			mx=max(mx,v);
		}
		cout<<"Case #"<<j<<":"<<'\n';
		for(auto [s,v]:a)
		{
			if(v==mx)cout<<s<<'\n';
		}
	}
	return 0;
}
```
