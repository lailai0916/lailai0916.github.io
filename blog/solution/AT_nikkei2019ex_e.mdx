## 解题思路

设 $p=f(x)$，则有：

$$
p-1=
\begin{cases}
  f\left(\frac{x}{2}\right) & x\bmod 2=0 \\
  f(3x+1) & x\bmod 2=1
\end{cases}
$$

样例 #2 给了 $p=1000$ 的一个解 $x=1789997546303$，根据结论倒推即可。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll=long long;
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int p;
	cin>>p;
	ll f=1789997546303;
	for(int i=1000;i>p;i--)f=f&1?f*3+1:f>>1;
	cout<<f<<'\n';
	return 0;
}
```
