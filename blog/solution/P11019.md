## 题意简述

给定字符串 $s$，从中截取 `[` 到 `]]` 之间的驼峰拼音表情名，将每个音节的大写首字母转为小写后拼接，并在最前方加上 `/`。

## 解题思路

先输出 `/`，顺序扫描字符串 $s$，在遇到 `]` 之前，将所有大写字母转为小写输出。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	string s;
	cin>>s;
	cout<<'/';
	for(auto c:s)
	{
		if(c==']')break;
		if(c>='A'&&c<='Z')cout<<char(c+32);
	}
	return 0;
}
```
