## 数组下标

> `i[a]` 等价于 `a[i]`。

`a[i]` 等价于 `*(a+i)`，而 `i[a]` 等价于 `*(i+a)`。

两者只是交换了加法顺序，结果是一样的。

## 字符串拼接

> 字符串 `"lailai""0916"` 等价于 `"lailai0916"`。

## 符号替换

> `<::>` 等价于 `[]`，`<%%>` 等价于 `{}`。

早期 IBM 在计算机产业中的地位非常重要，但是它的键盘没有中括号和大括号按键。

所以在 C 语言中可以使用其他的组合符号代替，并且一直保留到现在。

## 趋近运算符

```cpp
int x=10;
do
{
	cout<<x<<'\n';
}while(x-->0);
```

## INT_MAX

方程 $-x=x$ 有两个解：

- `-0=0`；
- `-(-2147483648)==(-2147483648)`。
