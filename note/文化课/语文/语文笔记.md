# 1. 记叙文答题模板

判断题型——匹配方法——联系原文作答

## 1.1 概括题

判断题型：表格、短横线、长横线、填空

答题方法：“审定概仿”

- 审题：主人公、线索、范围

- 定位：找到原文当中具体的段落，通过已知推未知

- 概括：谁 + 什么情况（背景、环境）+ 做了什么 + 结果怎么样+ 情感（精神、影响）
- 仿写：字数相仿、格式相同

## 1.2 赏析题

判断题型：某段、划线句、某种手法咋好、怎么说出情感

答题方法：“法事情人”

- 手法：运用了……手法（修辞、表达、描写）

	修辞：比喻、排比、设问、反问、反复、对偶、夸张、拟人

	描写：心理、动作、外貌、语言、神态、环境、正面、侧面

- 事件：写了……事

- 情感：表达、讴歌、抒发了……情感

- 人物：刻画了……人物形象

## 1.3 作用题

### 1.3.1 段落作用题

判断题型：第几段、划线句、描写有啥作用

答题方法：结构 + 内容

- 结构
	- 开头：点题、总领、引出、铺垫、伏笔、渲染、烘托、悬念、兴趣
	- 中间：承上启下、过渡
	- 结尾：深思、启示、回味、总结、首尾呼应、照应
- 内容：概括段意、概括人物形象、概括情感

### 1.3.2 标题作用

判断题型：围绕标题

答题方法：概括文章主要内容、贯穿全文线索、揭露主要情感、点明主要人物、吸引读者

- 因为原文写了……，起到了……作用，所以……

### 1.3.3 环境描写作用题

判断题型：段、句、环境描写

答题方法：“铺事情推气人深”

- 铺垫：为下文作铺垫
- 事件：交代故事的发生背景（时间、地点、天气）
- 情感：烘托……人物心情
- 推动：推动故事情节发展
- 气氛：渲染……氛围
- 人物：刻画人物性格、品质
- 深化：升华主题、深化中心

## 1.4 含义题

判断题型：含义、内涵、表达、分析

答题方法：“表深情”

- 表面：概括、换句话 + 原文
- 深层：隐晦表达
- 情感：抒发……情感

## 1.5 人物形象分析题

判断题型：刻画、形象、塑造

答题方法：“法事人”

- 手法：同赏析题手法（描写）
- 事件：……事情
- 人物：刻画人物形象

## 1.6 理解启示题

答题方法：联系原文 + 分析 + 感悟 + 实际生活

## 1.7 创新题

- 自己提问 + 自己作答：手法、概括
- 自己提问 + 说明理由：主旨、情感
- 自己提问 + 无需回答（围绕……内容）

# 2. 非连续性文本答题模板

从不同的角度提供材料，相互独立，但中心一致

所有答案出自原文：概括能力

材料：文字、数据、图片、广告、地图、时间表……

作用：实用性、综合性、直观性、关联性

## 2.1 常见考查内容

1. 梳理材料间的关系：并列、补充、因果、总分
2. 梳理图文间的关系：图文解析
3. 多则材料共同说明：相同、不同
4. 概括图表分析图片：表头、图例、最值、趋势
5. 词语辨析或选用：逻辑
6. 发表观点、看法：联系原文 + 合理分析

## 2.2 基本答题策略

1. 分层、比较、归纳（整体）
	- 有中心句：总起、过度、结尾
	- 无中心句：重点例子、整合观点
2. 细致的剖析（句子）
	- 本句作用：本身意义、逻辑关系、是否符合中心、文中的影响

## 2.3 考点一：理解文意、提取整合

- 问题形式：概括主要内容、细节、主要信息、选择文章分析、某种情况出现的原因、行动、意思、理解
- 答题方法：定位、取舍、完善、表述
- 答题模板：
	- 简答题：答题范围 + 提取关键词句 + 标题 / 序号 / 表头 + 图文互补
	- 选择题：找到每个选项在文章的出处，对比原文差异、偷换概念、以偏概全、无中生有、强加因果
	
## 2.4 考点二：分析归纳材料

- 问题形式：相同点、不同点、是什么、为什么、怎么办
- 答题模板：分析异同提炼 + 抓关键（表象、实质、结果、意义）+ 紧扣原文（慎加修饰词）+整理结果 + 对照检查

## 2.5 考点三：图文转换

- 问题形式：表格、图片转成文字；表述图标信息；推断
- 答题模板：表头 + 分类、图例 + 数据、注释性文字（字数要求）

## 2.6 考点四：解决具体问题、感悟

- 问题形式：结合现象谈理解、结合生活体验、给建议
- 答题模板：总分 + 应该怎么做 + 提见解、措施
