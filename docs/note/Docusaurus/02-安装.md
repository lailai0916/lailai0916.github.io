# 安装

```bash
npx create-docusaurus@latest my-website classic --typescript
```

## 本地运行

- 进入项目目录，运行以下命令启动本地服务器：

```bash
cd my-website
npm start
```

- 打开浏览器，访问 `http://localhost:3000`，你将看到 Docusaurus 网站的默认主页。

## 项目结构

- `my-website` 目录结构如下：

```plaintext
my-website
├── blog
├── docs
├── src
├── docusaurus.config.js
├── package.json
└── README.md
```

- `blog`：博客文章目录。
- `docs`：文档目录。
- `src`：网站源代码目录。
- `docusaurus.config.js`：Docusaurus 配置文件。
- `package.json`：项目依赖和脚本配置文件。
- `README.md`：项目说明文件。
