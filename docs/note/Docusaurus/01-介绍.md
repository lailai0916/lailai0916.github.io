# 介绍

## 前言

我以前使用过 [CSDN](https://www.csdn.net)、[博客园](https://www.cnblogs.com) 等现成的博客平台，但它们的自由度较低，功能受限，而且经常有广告。

我也尝试过 [WordPress](https://wordpress.org)、[Hexo](https://hexo.io) 等博客框架，但效果始终都不是很满意。

最终我发现了 Docusaurus，这是由 Facebook 开发的开源的静态网站生成器，专门用于构建文档网站。

但 Docusaurus 在国内比较冷门，所以相关的资料教程比较少。

另外 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 也是一个不错的选择，例如 [OI Wiki](https://oi-wiki.org) 就是基于它搭建的。

经过对比后，我最终还是选择了简洁美观、易于使用、且扩展性更强的 Docusaurus。

## 说明

此教程写于 2025 年 1 月，当前最新版本为 Docusaurus v3.7。

使用 macOS Sequoia 15.2 操作系统的 MacBook Pro 进行演示。
