## 高中

### 学期安排

|    学期    |                        考试安排                        |
| :--------: | :----------------------------------------------------: |
| 高一上学期 |                           —                            |
| 高一下学期 |     **历史**、**地理**、**化学**、**生物学** 学考      |
| 高二上学期 |              **思想政治**、**物理** 学考               |
| 高二下学期 |           **语文**、**数学**、**技术** 学考            |
| 高三上学期 |        **外语** 高考兼学考、**七选三科目** 选考        |
| 高三下学期 | **语文**、**数学**、**外语** 高考、**七选三科目** 选考 |
