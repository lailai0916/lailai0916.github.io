## 第一章 原子结构与性质

### 第一节 原子结构

#### 能层与能级

- **能层**（Energy Level）：核外电子按能量差异分层排布，由内到外记作 K、L、M、N、O、P、Q，能量依次升高（$E(\text{K})<E(\text{L})<E(\text{M})<\dots$）。记忆口诀「起于国王，终于皇后」（K king 到 Q queen）；
- **能级**（Energy Sublevel）：同一能层中电子能量的进一步差异，用 s、p、d、f 等表示；
- 第 $n$ 能层含 $n$ 个能级、$n^2$ 个原子轨道，最多容纳 $2n^2$ 个电子。

各能层所含能级与最多容纳电子数：

|      能层      |  K   |    L     |      M       |        N         |
| :------------: | :--: | :------: | :----------: | :--------------: |
|      能级      | $1s$ | $2s\,2p$ | $3s\,3p\,3d$ | $4s\,4p\,4d\,4f$ |
|  各能级电子数  | $2$  |  $2,6$   |   $2,6,10$   |   $2,6,10,14$    |
| 能层最多电子数 | $2$  |   $8$    |     $18$     |       $32$       |

s、p、d、f 能级最多容纳的电子数依次为 $2$、$6$、$10$、$14$，正好是序数 $1$、$3$、$5$、$7$ 的两倍。

量子力学用四个 **量子数** 完整描述一个电子的运动状态：

|      量子数      |    物理意义    |                            取值                             |
| :--------------: | :------------: | :---------------------------------------------------------: |
|   主量子数 $n$   | 能层，离核远近 |            $1,2,3,\dots$，对应 K、L、M、$\dots$             |
|   角量子数 $l$   | 能级，轨道形状 | $0\sim n-1$：$0$ 为 s（球形），$1$ 为 p（哑铃形），$2$ 为 d |
|  磁量子数 $m_l$  | 轨道的空间取向 |      $-l\sim l$，如 p 能级 $3$ 个取向（$p_x,p_y,p_z$）      |
| 自旋量子数 $m_s$ |  电子自旋方向  |              $+\frac{1}{2}$ 或 $-\frac{1}{2}$               |

$n$、$l$、$m_l$ 共同确定一个原子轨道，$n$、$l$ 决定轨道能量。同一原子中不存在四个量子数完全相同的两个电子，这正是泡利原理的另一种表述。

#### 构造原理与电子排布式

多电子原子基态时，电子填入能级的顺序遵循 **构造原理**（Aufbau Principle）：按能级能量由低到高填充，顺序为

$$1s\,2s\,2p\,3s\,3p\,4s\,3d\,4p\,5s\,4d\,5p\,6s\,4f\,5d\,6p\,7s\dots$$

注意 $4s$ 先于 $3d$ 填充，这一 **能级交错** 是理解过渡元素排布的关键。能级交错的根源是 **屏蔽效应** 与 **钻穿效应**：内层电子抵消部分核电荷对外层电子的吸引（屏蔽），使外层能量升高；而 s 电子有较大概率钻穿到近核区（钻穿），感受到更大的有效核电荷，能量降低。二者竞争的结果，使 $4s$ 能量反而低于 $3d$。

**电子排布式** 把电子按能级填入并注明数目，例如：

- $\text{Na}$：$1s^2 2s^2 2p^6 3s^1$；
- $\text{Fe}$：$1s^2 2s^2 2p^6 3s^2 3p^6 3d^6 4s^2$；
- $\text{Cu}$：$1s^2 2s^2 2p^6 3s^2 3p^6 3d^{10} 4s^1$。

书写时按能层顺序（$3d$ 写在 $4s$ 前），而非按填充顺序。为简化，可用相应稀有气体的元素符号加方括号表示内层，即 **简化电子排布式**：$\text{Fe}$ 写作 $[\text{Ar}]3d^6 4s^2$。

**价电子**（Valence Electron）是参与化学反应的电子，主族元素为最外层电子，过渡元素还包括次外层的 $d$ 电子（甚至倒数第三层的 $f$ 电子）。只写价电子的排布式称 **价层电子排布式**，如 $\text{Fe}$ 为 $3d^6 4s^2$。

#### 泡利原理与洪特规则

**电子排布的三条规则**：

- **能量最低原理**：整个原子处于能量最低状态，电子优先占据能量低的能级；
- **泡利原理**（Pauli Exclusion Principle）：一个原子轨道最多容纳 $2$ 个电子，且自旋方向相反；
- **洪特规则**（Hund's Rule）：电子分占能量相同的轨道时，优先单独占据且自旋方向相同（先独占，后成对）。

**洪特规则特例**：能量相同的轨道处于 **全充满**（$p^6$、$d^{10}$、$f^{14}$）、**半充满**（$p^3$、$d^5$、$f^7$）或 **全空**（$p^0$、$d^0$、$f^0$）时，能量较低、更稳定。这解释了 $\text{Cr}$ 为 $[\text{Ar}]3d^5 4s^1$（而非 $3d^4 4s^2$）、$\text{Cu}$ 为 $[\text{Ar}]3d^{10} 4s^1$（而非 $3d^9 4s^2$）。同族的 $\text{Mo}$（$[\text{Kr}]4d^5 5s^1$）、$\text{Ag}$（$[\text{Kr}]4d^{10} 5s^1$）、$\text{Au}$（$[\text{Xe}]4f^{14}5d^{10}6s^1$）同理。

轨道表示式用一个方框（或圆圈）表示一个原子轨道，框内用向上、向下的箭头表示两种自旋方向的电子。如 $\text{N}$ 的价电子排布为 $2s^2 2p^3$，$2p$ 能级三个轨道各占一个自旋相同的电子（洪特规则），而非两个配对、一个空。

#### 电子云与原子轨道

电子在核外高速运动，没有确定的轨道。据不确定性原理，无法同时准确测定电子的位置与速度，只能用统计的方式描述它在核外各处出现的机会。

- **概率密度**：电子在某处单位体积内出现的概率；
- **电子云**：用疏密不同的小点形象地描绘概率密度分布，点密处电子出现机会多；
- **电子云轮廓图**：把电子出现概率约为 $90\%$ 的空间区域勾画出来的图形。

原子轨道的形状与取向：s 轨道呈 **球形**，只有 $1$ 种空间取向；p 轨道呈 **哑铃形**（纺锤形），有 $p_x$、$p_y$、$p_z$ 三种相互垂直的取向。

#### 基态与激发态、光谱

- **基态**（Ground State）：处于最低能量状态的原子；
- **激发态**（Excited State）：基态原子吸收能量，电子跃迁到较高能级后的状态；
- 电子从较高能级跃迁回较低能级时，以光的形式释放能量，产生 **发射光谱**；吸收特定波长的光则产生 **吸收光谱**。二者统称 **原子光谱**，每种元素都有特征谱线，可用于元素的鉴定与含量测定（光谱分析），灵敏度极高。

**焰色反应**：某些金属元素（游离态或化合态）灼烧时使火焰呈现特殊颜色，本质是电子的跃迁与回落，属物理变化。常用铂丝或铁丝蘸取，每次用稀盐酸洗净。

|    元素     |               颜色               |
| :---------: | :------------------------------: |
| $\text{Li}$ |              紫红色              |
| $\text{Na}$ |               黄色               |
| $\text{K}$  | 紫色（需透过蓝色钴玻璃滤去黄光） |
| $\text{Ca}$ |              砖红色              |
| $\text{Sr}$ |              洋红色              |
| $\text{Ba}$ |              黄绿色              |
| $\text{Cu}$ |               绿色               |

焰色试验、霓虹灯发光、激光都与电子在能级间的跃迁有关。

### 第二节 原子结构与元素的性质

#### 原子结构与元素周期表

- 周期序数 $=$ 电子层数（能层数）；
- 主族序数 $=$ 最外层电子数 $=$ 价电子数；
- 一个能级组最多容纳的电子数，等于对应周期所含的元素种数。

按价电子排布，周期表分为五个 **区**：

|    区     |               价电子排布                |    包含族     |
| :-------: | :-------------------------------------: | :-----------: |
| **s 区**  |             $ns^{1\sim 2}$              |    ⅠA、ⅡA     |
| **p 区**  |           $ns^2 np^{1\sim 6}$           | ⅢA – ⅦA、0 族 |
| **d 区**  |     $(n-1)d^{1\sim 9}ns^{1\sim 2}$      |  ⅢB – ⅦB、Ⅷ   |
| **ds 区** |        $(n-1)d^{10}ns^{1\sim 2}$        |    ⅠB、ⅡB     |
| **f 区**  | $(n-2)f^{0\sim 14}(n-1)d^{0\sim 2}ns^2$ |  镧系、锕系   |

判断依据是最后一个电子填入的能级：填入 s、p 为主族，填入 d 为过渡元素，填入 f 为镧系、锕系。除 s 区外，最外层电子数不一定等于族序数。s 区（除 H）、d 区、ds 区都是金属元素。d 区元素反应时优先失去 $ns$ 电子，再失去 $(n-1)d$ 电子。

#### 电离能

**电离能**（Ionization Energy，$I$）：气态基态原子失去一个电子转化为气态基态阳离子所需的最低能量，第一电离能记作 $I_1$，单位 $\text{kJ/mol}$。

- **同周期**：从左到右，$I_1$ 呈 **增大** 趋势（核电荷增大，原子半径减小，失电子渐难）；
- **同主族**：从上到下，$I_1$ **减小**（原子半径增大，失电子渐易）；
- **反常**：ⅡA 族（$ns^2$ 全充满）$I_1$ 大于同周期ⅢA 族；ⅤA 族（$np^3$ 半充满）$I_1$ 大于同周期ⅥA 族。如 $I_1(\text{Mg})>I_1(\text{Al})$、$I_1(\text{P})>I_1(\text{S})$。

同一原子逐级电离能 **依次增大**（$I_1<I_2<I_3<\dots$）。当电离能出现 **突跃**（骤然增大）时，说明前面已失去全部价电子，再失就要动内层电子，故电离能突跃处的电子数即最外层电子数，可据此判断元素所在族。例如 $\text{Na}$ 的 $I_2\gg I_1$，说明它只有 $1$ 个价电子，为ⅠA 族。

电离能小的元素一般金属活动性强，但二者不完全对应（还与反应环境有关），如 $I_1(\text{Zn})>I_1(\text{Cu})$ 而活动性 Zn 强于 Cu。

**电子亲和能**：气态原子获得一个电子成为气态阴离子时放出的能量，反映元素结合电子的难易。它不像电离能那样有简单的周期性变化规律，数据也不易精确测定，应用远不如电离能广泛。

#### 电负性

**电负性**（Electronegativity）：元素的原子在化合物中吸引 **键合电子** 能力的标度。以氟的电负性 $4.0$ 为标准，值越大，吸引电子能力越强。

- **同周期**：从左到右，电负性 **增大**；
- **同主族**：从上到下，电负性 **减小**；
- 金属元素电负性一般小于 $1.8$，非金属元素一般大于 $1.8$；
- 电负性最大的是 F，非金属性最强；金属性最强的是 Cs（放射性元素除外）；
- 与电离能不同，金属元素的电负性通常与其活动性 **正相关**。

**电负性的应用**：

- 判断元素金属性、非金属性强弱；
- 判断化学键类型：电负性差值大（一般 $>1.7$）成 **离子键**，差值小成 **共价键**（反例 $\text{HF}$ 差值约 $1.9$，仍为共价键）；
- 判断化合价正负：电负性大的显负价，小的显正价（如 $\text{HCl}$ 中 H 为 $+1$、Cl 为 $-1$）；
- **对角线规则**：处于周期表左上到右下对角线位置的元素（如 Li 与 Mg、Be 与 Al、B 与 Si）电负性相近，性质相似。如 Be、Al 均两性，都有 $\text{Na}[\text{Al}(\text{OH})_4]$ 型的含氧酸盐。

#### 微粒半径比较

- **电子层数** 越多，半径越大（同主族从上到下半径增大）；
- 电子层数相同时，**核电荷数** 越大，半径越小（同周期从左到右半径减小，稀有气体除外）；
- 核外电子数相同（**等电子体**）时，核电荷数越大，半径越小，如 $r(\text{O}^{2-})>r(\text{F}^-)>r(\text{Na}^+)>r(\text{Mg}^{2+})>r(\text{Al}^{3+})$；
- 同种元素：阳离子半径小于原子，阴离子半径大于原子（$r(\text{Na}^+)<r(\text{Na})$，$r(\text{Cl}^-)>r(\text{Cl})$）；价态越高，半径越小。

比较半径时依次看 **电子层数 → 核电荷数 → 核外电子数**。注意电子层数多不一定半径大，如 $r(\text{At})<r(\text{K})$。

## 第二章 分子结构与性质

### 第一节 共价键

#### 共价键的本质与特征

- **共价键**（Covalent Bond）：原子间通过 **共用电子对** 形成的化学键，本质是高概率地出现在两核之间的电子与两原子核之间的电磁作用；
- 成键的两个条件：两原子各提供一个能量相近、自旋相反的单电子配对（电子配对原理）；轨道重叠程度越大，核间电子概率密度越大，键越牢固（最大重叠原理）；
- **特征**：具有 **饱和性**（原子形成共价键的数目一定）和 **方向性**（沿轨道最大重叠方向成键，决定分子空间结构）。

只含共价键的化合物是共价化合物；含离子键的即为离子化合物。离子化合物中可以含共价键（如 $\text{NaOH}$），但共价化合物中不含离子键。

#### σ 键与 π 键

按原子轨道重叠方式，共价键分为两类：

|            |           $\sigma$ 键            |       $\pi$ 键       |
| :--------: | :------------------------------: | :------------------: |
|  重叠方式  |       「头碰头」，轴向重叠       | 「肩并肩」，平行重叠 |
| 电子云形状 |         轴对称（圆柱形）         |       镜面对称       |
|  重叠程度  |          大，键强、稳定          |    小，键弱、活泼    |
|  牢固程度  |         强，可绕键轴旋转         |     弱，不能旋转     |
|    存在    | 单键即 $\sigma$ 键；共价键中必有 | 只存在于双键、三键中 |

- 单键：$1$ 个 $\sigma$ 键；
- 双键：$1$ 个 $\sigma$ 键 $+1$ 个 $\pi$ 键；
- 三键：$1$ 个 $\sigma$ 键 $+2$ 个 $\pi$ 键。

两原子成键时先形成 $\sigma$ 键，再形成 $\pi$ 键。$\sigma$ 键重叠程度大、化学性质稳定，$\pi$ 键则较活泼。如 $\text{N}_2$ 分子中 $\text{N}\equiv\text{N}$ 为三键，含 $1$ 个 $\sigma$ 键、$2$ 个 $\pi$ 键，键能大、极稳定。$\pi$ 键与环状结构一样限制单键旋转，这正是顺反异构存在的原因。

#### 键参数

- **键能**（Bond Energy）：气态分子中 $1\ \text{mol}$ 化学键断裂成气态原子所吸收的能量。键能越大，共价键越牢固、分子越稳定。手册中的键能是对大量分子统计得到的 **平均值**；
- **键长**（Bond Length）：成键两原子核间的平均距离，键长越短，键能越大，键越牢固。可由晶体 X 射线衍射实验测定；
- **键角**（Bond Angle）：分子中两个相邻共价键之间的夹角，决定分子的空间结构。

一般原子半径越小、共用电子对越多、元素电负性越大，键长越短、键能越大。**键能、键长决定分子的稳定性，键长、键角决定分子的空间结构。**

如 $\text{CO}_2$ 键角 $180^\circ$（直线形），$\text{H}_2\text{O}$ 键角约 $105^\circ$（V 形），$\text{CH}_4$ 键角 $109^\circ28'$（正四面体形）。

#### 配位键

**配位键**（Coordinate Bond）是一种特殊的共价键，共用电子对由 **一方原子单独提供**，另一方提供空轨道。

- 表示为 $\text{A}\to\text{B}$，A 为提供孤对电子的一方，B 为提供空轨道的一方；
- 如 $\text{NH}_4^+$ 中，$\text{N}$ 提供孤对电子、$\text{H}^+$ 提供空轨道，形成 $1$ 个配位键，但 $4$ 个 $\text{N—H}$ 键完全等同；
- $\text{H}_3\text{O}^+$、$\text{Cu}(\text{NH}_3)_4^{2+}$ 中也含配位键。

#### 等电子体

**等电子体**（Isoelectronic Species）：**原子总数相同、价电子总数也相同** 的微粒，具有相似的结构与性质。

常见等电子体：

- $\text{CO}$ 与 $\text{N}_2$、$\text{CN}^-$、$\text{NO}^+$（$2$ 原子，$10$ 价电子，直线形）；
- $\text{CO}_2$、$\text{CS}_2$、$\text{N}_2\text{O}$ 与 $\text{SCN}^-$、$\text{N}_3^-$、$\text{BeCl}_2$（$3$ 原子，$16$ 价电子，直线形）；
- $\text{SO}_2$、$\text{O}_3$、$\text{NO}_2^-$（$3$ 原子，$18$ 价电子，V 形）；
- $\text{CO}_3^{2-}$、$\text{NO}_3^-$、$\text{SO}_3$、$\text{BF}_3$（$4$ 原子，$24$ 价电子，平面三角形）；
- $\text{SO}_4^{2-}$、$\text{PO}_4^{3-}$、$\text{ClO}_4^-$、$\text{SiF}_4$（$5$ 原子，$32$ 价电子，正四面体形）；
- $\text{CH}_4$、$\text{NH}_4^+$（$5$ 原子，$8$ 价电子）。

利用等电子原理可推测简单微粒的空间结构：把一个原子换成原子序数相邻的原子、或增减电荷使价电子总数不变，即得等电子体。如把 $\text{CO}_2$ 中一个 O 换成 N 并加一个负电荷，得 $\text{N}_3^-$，同为直线形。

### 第二节 分子的空间结构

#### 价层电子对互斥模型

**价层电子对互斥模型**（Valence Shell Electron Pair Repulsion，VSEPR）：分子中中心原子周围的价层电子对（成键电子对 $+$ 孤对电子对）尽可能相互远离，使斥力最小，从而决定分子的空间结构。

**价层电子对数** 计算（以 $\text{AB}_n$ 型分子或离子为例）：

$$\text{价层电子对数}=\text{成键电子对数}+\text{孤对电子对数}$$

其中孤对电子对数

$$\text{孤对电子对数}=\frac{1}{2}\left(a-xb\right)$$

$a$ 为中心原子的价电子数（阳离子减去所带电荷、阴离子加上所带电荷），$x$ 为与中心原子成键的原子数，$b$ 为每个配位原子接受的电子数（H、卤素为 $1$，O、S 作配位原子记 $0$；其余非金属原子为 $8$ 减去其最外层电子数）。

价层电子对数与 VSEPR 模型、分子空间结构的对应：

| 电子对数 | VSEPR 模型 | 孤对电子对 | 分子空间结构 |              示例              |      键角      |
| :------: | :--------: | :--------: | :----------: | :----------------------------: | :------------: |
|   $2$    |   直线形   |    $0$     |    直线形    | $\text{CO}_2$、$\text{BeCl}_2$ |  $180^\circ$   |
|   $3$    | 平面三角形 |    $0$     |  平面三角形  |  $\text{BF}_3$、$\text{SO}_3$  |  $120^\circ$   |
|   $3$    | 平面三角形 |    $1$     |     V 形     |         $\text{SO}_2$          |  $<120^\circ$  |
|   $4$    | 正四面体形 |    $0$     |  正四面体形  | $\text{CH}_4$、$\text{NH}_4^+$ | $109^\circ28'$ |
|   $4$    | 正四面体形 |    $1$     |   三角锥形   |         $\text{NH}_3$          |  $107^\circ$   |
|   $4$    | 正四面体形 |    $2$     |     V 形     |      $\text{H}_2\text{O}$      |  $105^\circ$   |

孤对电子对不出现在分子构型里，但它占据空间、且斥力更大。斥力大小顺序为

$$\text{lp-lp}\gg\text{lp-bp}>\text{bp-bp}$$

（lp 为孤对电子对，bp 为成键电子对）。孤对电子越多，对成键电子对的排斥越强，键角越小：$\text{CH}_4$（$109^\circ28'$）$>\text{NH}_3$（$107^\circ$）$>\text{H}_2\text{O}$（$105^\circ$）。此外，多重键斥力大于单键，配位原子不对称时键角也会相应调整。

#### 杂化轨道理论

**杂化轨道理论**（Hybrid Orbital Theory）：成键时，中心原子能量相近的原子轨道 **重新组合** 成一组能量相同、方向不同的新轨道，即 **杂化轨道**，用于形成 $\sigma$ 键或容纳孤对电子。

- 杂化前后轨道数目不变（$n$ 个原子轨道杂化成 $n$ 个杂化轨道）；
- 杂化的实质是把「小头向内、大头向外」，增大轨道重叠、提升成键能力；
- 只有 $\sigma$ 键和孤对电子占用杂化轨道，$\pi$ 键由未参与杂化的 p 轨道形成。

三种常见杂化类型：

| 杂化类型 |        参与轨道        | 杂化轨道数 |  空间取向  |      键角      |                        示例                        |
| :------: | :--------------------: | :--------: | :--------: | :------------: | :------------------------------------------------: |
|   $sp$   | $1$ 个 $s$、$1$ 个 $p$ |    $2$     |   直线形   |  $180^\circ$   |       $\text{CO}_2$、$\text{C}_2\text{H}_2$        |
|  $sp^2$  | $1$ 个 $s$、$2$ 个 $p$ |    $3$     | 平面三角形 |  $120^\circ$   |       $\text{BF}_3$、$\text{C}_2\text{H}_4$        |
|  $sp^3$  | $1$ 个 $s$、$3$ 个 $p$ |    $4$     | 正四面体形 | $109^\circ28'$ | $\text{CH}_4$、$\text{NH}_3$、$\text{H}_2\text{O}$ |

以甲烷为例：碳的价电子排布本为 $2s^2 2p^2$，成键时一个 $2s$ 电子激发到 $2p$ 空轨道，$1$ 个 $2s$ 与 $3$ 个 $2p$ 轨道重新组合成 $4$ 个能量相同的 $sp^3$ 杂化轨道，各与一个 H 的 $1s$ 轨道重叠成 $\sigma$ 键，故 $\text{CH}_4$ 为正四面体形。乙烯的碳采用 $sp^2$ 杂化，$3$ 个 $sp^2$ 轨道成 $\sigma$ 键、余下一个未杂化 p 轨道成 $\pi$ 键。乙炔的碳采用 $sp$ 杂化，两碳间为 $1$ 个 $\sigma$ 键加 $2$ 个 $\pi$ 键。

**判断杂化类型**：杂化轨道数 $=$ 价层电子对数（成键电子对 $+$ 孤对电子对）$=\sigma$ 键数 $+$ 孤对电子对数。如 $\text{NH}_3$ 有 $3$ 个 $\sigma$ 键加 $1$ 个孤对电子对，共 $4$，为 $sp^3$ 杂化，空间结构为三角锥形（孤对电子占一个杂化轨道，不计入分子构型）。

**苯的结构**：每个碳采用 $sp^2$ 杂化，$2$ 个 $sp^2$ 轨道与相邻碳成 $\sigma$ 键构成正六边形碳环、$1$ 个与 H 成 $\sigma$ 键；每个碳还有一个垂直于环平面的未杂化 $2p$ 轨道，六个 $2p$ 轨道「肩并肩」重叠形成 **离域大 $\pi$ 键**。故苯为平面正六边形、键角 $120^\circ$，且格外稳定。

### 第三节 分子结构与物质的性质

#### 键的极性与分子的极性

- **非极性键**：同种元素原子间的共价键，共用电子对不偏移（如 $\text{H—H}$、$\text{Cl—Cl}$）；
- **极性键**：不同种元素原子间的共价键，共用电子对偏向电负性大的一方，使其带部分负电荷 $\delta^-$、另一方带 $\delta^+$（如 $\text{H—Cl}$）；
- **非极性分子**：正、负电荷中心重合的分子；
- **极性分子**：正、负电荷中心不重合的分子。

**判断分子极性**：

- 只含非极性键的分子一般是非极性分子（$\text{H}_2$、$\text{O}_2$）；
- 含极性键的分子，看各键极性的 **矢量和**：矢量和为零（结构对称）则为非极性分子，否则为极性分子；
- 以 $\text{AB}_n$ 型分子为例，中心原子无孤对电子、结构对称的（$\text{CO}_2$、$\text{BF}_3$、$\text{CH}_4$）为非极性分子；有孤对电子、结构不对称的（$\text{H}_2\text{O}$、$\text{NH}_3$）为极性分子。

|         分子         |  空间结构  | 键的极性 |  分子极性  |
| :------------------: | :--------: | :------: | :--------: |
|    $\text{CO}_2$     |   直线形   |  极性键  | 非极性分子 |
|    $\text{BF}_3$     | 平面三角形 |  极性键  | 非极性分子 |
|    $\text{CH}_4$     | 正四面体形 |  极性键  | 非极性分子 |
| $\text{H}_2\text{O}$ |    V 形    |  极性键  |  极性分子  |
|    $\text{NH}_3$     |  三角锥形  |  极性键  |  极性分子  |

一个易错的特例：臭氧 $\text{O}_3$ 是 **含极性键的极性分子**，因为中心 O 与端基 O 所处环境不同，键有微弱极性，且分子呈 V 形不对称。

**相似相溶原理**：极性溶质易溶于极性溶剂，非极性溶质易溶于非极性溶剂。故 $\text{NH}_3$、$\text{HCl}$ 易溶于水（极性），$\text{I}_2$ 易溶于 $\text{CCl}_4$（非极性）。若溶质与溶剂间能形成氢键，溶解度增大。

#### 分子的手性

具有完全相同的组成和原子排列的一对分子，互为镜像却在三维空间里不能重合，称为 **手性异构体**，含有这种性质的分子称为 **手性分子**。最典型的是连有四个不同原子或基团的碳原子（手性碳）。

手性分子在制药中意义重大：一对手性异构体往往只有一种有药效，另一种可能无效甚至有害。用手性催化剂选择性地合成单一手性分子，称为手性合成，2001 年诺贝尔化学奖即授予这一领域。

#### 范德华力

**范德华力**（van der Waals Force）：分子间普遍存在的一种较弱的相互作用，比化学键弱 $1\sim 2$ 个数量级，无方向性、无饱和性。

- 影响因素：组成和结构相似时，**相对分子质量越大，范德华力越大**，熔沸点越高；相对分子质量相近时，分子极性越大，范德华力越大；
- 如卤素单质、稀有气体的熔沸点随相对分子质量增大而升高（$\text{F}_2<\text{Cl}_2<\text{Br}_2<\text{I}_2$）；
- 同分异构体中支链越多，分子越「圆」，接触面积越小，范德华力越小、熔沸点越低（正戊烷 $>$ 异戊烷 $>$ 新戊烷）；
- 主要影响物质的熔点、沸点、溶解度等物理性质。

#### 氢键

**氢键**（Hydrogen Bond）：已经与电负性很大的原子（**N、O、F**）相连的氢原子，与另一个电负性很大的原子之间的作用力，表示为 $\text{X—H}\cdots\text{Y}$（X、Y 为 N、O、F）。

- 氢键 **比范德华力强，比化学键弱**，也有一定的方向性（$\text{X—H}\cdots\text{Y}$ 成直线时最强）和饱和性（一个 H 一般只形成一个氢键）；
- 分为分子间氢键和分子内氢键；
- **对性质的影响**：
  - 使物质熔沸点 **反常升高**，如 $\text{H}_2\text{O}$、$\text{HF}$、$\text{NH}_3$ 的沸点高于同族其他氢化物；
  - 冰中水分子间形成氢键，排列疏松，使 **冰的密度小于水**；
  - 分子间氢键增大溶解度，分子内氢键则减小溶解度。

氢键的强弱主要取决于分子能形成的氢键数目（由孤对电子数与氢原子数中的较小者决定），也与 X、Y 的电负性（正相关）、半径（负相关）有关：

|            | $\text{NH}_3$ | $\text{N}_2\text{H}_4$ | $\text{H}_2\text{O}$ | $\text{HF}$ |
| :--------: | :-----------: | :--------------------: | :------------------: | :---------: |
| 孤对电子数 |      $1$      |          $2$           |         $2$          |     $3$     |
|  氢原子数  |      $3$      |          $4$           |         $2$          |     $1$     |
|   氢键数   |      $1$      |          $2$           |         $2$          |     $1$     |

**分子间作用力对熔沸点的影响顺序**：化学键（决定分解温度）$>$ 氢键 $>$ 范德华力。判断分子晶体熔沸点：先看有无氢键，再比范德华力（相对分子质量）；但相对分子质量相差极大时，范德华力也可能反客为主。

## 第三章 晶体结构与性质

### 第一节 物质的聚集状态与晶体的常识

#### 晶体与非晶体

- **晶体**（Crystal）：内部微粒在空间按一定规律 **周期性重复排列** 构成的固体，有规则的几何外形、固定的熔点、**自范性** 与 **各向异性**；
- **非晶体**（Amorphous）：内部微粒排列相对无序，无固定熔点、各向同性（如玻璃、松香、橡胶）；
- **区分方法**：间接看是否有 **固定熔点**；最科学的方法是 **X 射线衍射实验**——单一波长的 X 射线通过晶体时会产生分立的衍射斑点或明锐的衍射峰，据此可得晶胞的形状大小、原子的种类数目与位置等信息。

得到晶体的途径：熔融态物质凝固、气态物质冷却不经液态直接凝华、溶质从溶液中过饱和析出。

#### 晶胞

- **晶胞**（Unit Cell）：描述晶体结构的 **基本重复单元**，晶体可看作大量晶胞「无隙并置」而成（不能把需要旋转才能拼接的单元当作晶胞）；
- 晶胞一般是平行六面体，可由三组棱长 $a$、$b$、$c$ 和三组夹角 $\alpha$、$\beta$、$\gamma$ 共六个 **晶胞参数** 描述。同一晶体中所有晶胞完全相同；
- **原子坐标**：以晶胞参数为单位长度，用 $(x,y,z)$（$0\le x,y,z\le 1$）标出原子在晶胞中的位置。

**均摊法**：晶胞中一个微粒对该晶胞的贡献，取决于它被几个晶胞共用。

- 顶点上的微粒：被 $8$ 个晶胞共用，贡献 $\frac{1}{8}$；
- 棱上的微粒：被 $4$ 个晶胞共用，贡献 $\frac{1}{4}$；
- 面上的微粒：被 $2$ 个晶胞共用，贡献 $\frac{1}{2}$；
- 体内的微粒：完全属于该晶胞，贡献 $1$。

**晶胞密度计算**：设晶胞棱长为 $a\ \text{cm}$，晶胞中含 $Z$ 个化学式单位，摩尔质量为 $M\ \text{g/mol}$，阿伏加德罗常数为 $N_\text{A}$，则

$$\rho=\frac{ZM}{N_\text{A}a^3}$$

分子中 $ZM$ 为一个晶胞的质量，分母 $a^3$ 为晶胞体积。已知密度也可反求晶胞参数或 $N_\text{A}$。此外，空间利用率 $=\frac{\text{晶胞内球体积}}{\text{晶胞体积}}\times100\%$。

### 第二节 分子晶体与共价晶体

#### 分子晶体

- **分子晶体**（Molecular Crystal）：**分子** 之间以 **分子间作用力**（范德华力、氢键）结合而成的晶体；
- **特点**：熔沸点低、硬度小、易升华，固态和熔融态都不导电（部分溶于水电离后导电）；
- **常见分子晶体**：所有非金属氢化物（$\text{H}_2\text{O}$、$\text{HCl}$、$\text{NH}_3$）、多数非金属单质（$\text{I}_2$、$\text{S}$、稀有气体）、多数非金属氧化物（$\text{CO}_2$、$\text{SO}_2$）、几乎所有的酸、绝大多数有机物。

**干冰**（固态 $\text{CO}_2$）是典型分子晶体，属面心立方结构：$\text{CO}_2$ 分子占据立方体的 $8$ 个顶点和 $6$ 个面心，每个晶胞含 $\text{CO}_2$ 分子数为

$$8\times\frac{1}{8}+6\times\frac{1}{2}=4$$

每个 $\text{CO}_2$ 分子周围紧邻 $12$ 个 $\text{CO}_2$ 分子（配位数 $12$），熔化时只破坏分子间作用力、不破坏化学键。

**碘晶体** 也是面心立方，顶点和面心各有一个 $\text{I}_2$ 分子，每个晶胞含 $4$ 个 $\text{I}_2$，分子间以范德华力结合。**冰** 中水分子间主要靠氢键结合，氢键的方向性迫使每个水分子与四面体顶角的 $4$ 个水分子相吸引，排列疏松，故密度反常小于液态水。

#### 共价晶体

- **共价晶体**（Covalent Crystal，又称原子晶体）：相邻原子间以 **共价键** 相结合形成空间网状结构的晶体，可看作一个巨型分子，没有分子式，化学式表示原子个数比；
- **特点**：熔沸点很高、硬度很大，一般不导电、难溶于任何溶剂；
- **常见共价晶体**：金刚石（C）、硅（Si）、$\text{SiO}_2$、SiC、$\text{Si}_3\text{N}_4$、BN 等。

**金刚石**：每个 C 原子以 $sp^3$ 杂化与相邻 $4$ 个 C 原子形成共价键，构成正四面体空间网状结构，最小碳环含 $6$ 个 C 原子，C 原子与 $\text{C—C}$ 键数之比为 $1:2$（每个 C 参与 $4$ 个键，每键为 $2$ 个 C 共用）。

**$\text{SiO}_2$**：每个 Si 原子与 $4$ 个 O 原子成键、每个 O 原子与 $2$ 个 Si 原子成键，Si 与 O 个数之比为 $1:2$，$1\ \text{mol}\ \text{SiO}_2$ 含 $4\ \text{mol}$ Si—O 键，最小环含 $12$ 个原子（Si、O 各 $6$ 个）。**SiC** 结构类似金刚石，Si、C 交替排列，个数之比 $1:1$。

计算共价键数目：无机物中「相同原子看一半、不同原子都要看」；有机物按「碳四、氢一、氧二」记价键，成键数加和后再减半，如 $\text{C}_2\text{H}_5\text{OH}$ 有 $\frac{2\times4+6+2}{2}=8$ 个键。

#### 熔沸点比较

- 共价晶体的熔沸点由 **键长（键能）** 决定：键长越短、键能越大，熔沸点越高，如金刚石 $>$ SiC $>$ 硅（原子半径 C $<$ Si，即 $\text{C—C}>\text{C—Si}>\text{Si—Si}$）；
- 分子晶体的熔沸点由 **分子间作用力** 决定：先看氢键，再比范德华力（相对分子质量）。

### 第三节 金属晶体与离子晶体

#### 金属键与金属晶体

- **金属键**（Metallic Bond）：金属阳离子与 **自由电子** 之间的强烈相互作用，可用 **电子气理论** 描述（金属阳离子「浸泡」在自由电子的「海洋」中），既无方向性也无饱和性，是遍布整个晶体的离域键；
- **金属晶体**（Metallic Crystal）：通过金属键结合的晶体，其中没有单个分子；
- **通性**（均由自由电子解释）：
  - 不透明、有金属光泽（自由电子吸收并放出各种频率的光）；
  - 良导电性（自由电子在电场下定向移动；温度越高导电性越差）；
  - 良导热性（自由电子与金属阳离子碰撞传递能量）；
  - 良延展性（受力时原子层滑动而金属键不断裂）；
- 金属键强弱：**离子半径越小、价电子数越多，金属键越强**，熔沸点越高、硬度越大。同周期从左到右金属键增强（熔点 $\text{Na}<\text{Mg}<\text{Al}$），同主族从上到下减弱（碱金属熔沸点递减）。

金属熔沸点整体无简单规律，最低的是铯、最高的是钨。

#### 离子晶体

- **离子晶体**（Ionic Crystal）：由 **阳离子和阴离子** 通过 **离子键** 结合而成的晶体。离子键是阴阳离子间的静电作用，既无方向性也无饱和性（离子电荷球形对称，可吸引各方向的异号离子，且尽可能多地吸引以降低能量）；
- **特点**：熔沸点较高、硬度较大、硬而脆，固态不导电，熔融态或溶于水后能导电；
- **常见离子晶体**：强碱（$\text{NaOH}$）、活泼金属氧化物（$\text{Na}_2\text{O}$、$\text{MgO}$）、绝大多数盐（$\text{NaCl}$、$\text{Na}_2\text{CO}_3$，$\text{AlCl}_3$ 为特例）。

**晶格能**（Lattice Energy）：气态离子形成 $1\ \text{mol}$ 离子晶体所释放的能量。**离子半径越小、离子所带电荷越多，晶格能越大**，离子键越强，熔沸点越高、硬度越大。如晶格能 $\text{MgO}>\text{NaCl}$（Mg、O 电荷高、半径小）、$\text{LiF}>\text{KBr}$（半径小）。晶格能无法直接测量，可用 **玻恩–哈伯循环**（Born–Haber Cycle）间接求得：把 $\text{NaCl}$ 的生成拆成升华、电离、解离、亲和、成键几步，由盖斯定律令各步焓变之和等于生成焓，其中只有晶格能未知，即可解出。

#### 典型离子晶体的晶胞

**$\text{NaCl}$ 型**（面心立方）：$\text{Cl}^-$ 占顶点和面心，$\text{Na}^+$ 占棱心和体心。

- $\text{Cl}^-$ 数：$8\times\frac{1}{8}+6\times\frac{1}{2}=4$；
- $\text{Na}^+$ 数：$12\times\frac{1}{4}+1=4$；
- $\text{Na}^+$ 与 $\text{Cl}^-$ 个数之比 $4:4=1:1$，符合化学式；
- 配位数均为 $6$（每个 $\text{Na}^+$ 周围有 $6$ 个 $\text{Cl}^-$，反之亦然）。

**$\text{CsCl}$ 型**（体心立方）：$\text{Cl}^-$ 占顶点，$\text{Cs}^+$ 占体心。

- $\text{Cl}^-$ 数：$8\times\frac{1}{8}=1$；$\text{Cs}^+$ 数：$1$；比为 $1:1$；
- 配位数均为 $8$。

**配位数**（Coordination Number）：晶体中一个微粒周围最紧邻的异种（或同种）微粒数目。同一晶体中两种微粒配位数之比与其个数之比成反比。

#### 四类晶体的比较

|   晶体类型   |         微粒         |    作用力    | 熔沸点 |  硬度  |     导电性     |          典型代表           |
| :----------: | :------------------: | :----------: | :----: | :----: | :------------: | :-------------------------: |
| **分子晶体** |         分子         | 分子间作用力 |   低   |   小   |   一般不导电   |   干冰、冰、$\text{I}_2$    |
| **共价晶体** |         原子         |    共价键    |  很高  |  很大  |   一般不导电   |   金刚石、$\text{SiO}_2$    |
| **金属晶体** | 金属阳离子、自由电子 |    金属键    | 差异大 | 差异大 |     良导体     |         Na、Fe、Al          |
| **离子晶体** |      阴、阳离子      |    离子键    |  较高  |  较大  | 熔融或溶解导电 | $\text{NaCl}$、$\text{MgO}$ |

熔沸点整体规律：共价晶体 $>$ 离子晶体 $\gg$ 分子晶体。比较同类晶体时只写支持结论的原因即可：同为共价晶体比键长，同为离子晶体比晶格能。

**石墨** 是 **混合型晶体**：层内 C 以 $sp^2$ 杂化形成共价键（平面六元环），每个 C 还有一个垂直于平面的未杂化 $2p$ 轨道，六个一组形成遍布整层的 **离域大 $\pi$ 键**，电子可自由移动，故石墨能导电、有各向异性；层间以范德华力结合，很容易滑开，故质软、有滑腻感，可作铅笔芯。层内 C 与键数之比为 $1:1.5$（每个 C 参与 $3$ 个键）。化学键往往介于典型模型之间，这正是石墨等混合晶体的成因。

### 第四节 配合物与超分子

#### 配合物

**配位化合物**（Coordination Compound，简称配合物）：由 **中心原子（离子）** 与 **配体** 通过配位键结合形成的化合物。

以 $[\text{Cu}(\text{NH}_3)_4]\text{SO}_4$ 为例，其组成：

- **中心离子（原子）**：提供空轨道，如 $\text{Cu}^{2+}$（多为过渡金属离子）；
- **配体**：提供孤对电子的分子或离子，如 $\text{NH}_3$（配位原子为 N），常见配体还有 $\text{H}_2\text{O}$、$\text{CO}$、$\text{CN}^-$、$\text{Cl}^-$；
- **配位数**：与中心离子直接成键的配位原子数目，此例为 $4$（单齿配体时配位数等于配体数）；
- **内界与外界**：方括号内 $[\text{Cu}(\text{NH}_3)_4]^{2+}$ 为内界，$\text{SO}_4^{2-}$ 为外界。

配位键的形成需中心原子（离子）有 **空轨道**、配体有 **孤对电子**。

**常见配合物与现象**：

- $\text{Fe}^{3+}$ 遇 $\text{SCN}^-$ 生成血红色 $\text{Fe}(\text{SCN})_3$，是检验 $\text{Fe}^{3+}$ 最灵敏的方法；
- $\text{Cu}^{2+}$ 遇过量氨水先生成蓝色 $\text{Cu}(\text{OH})_2$ 沉淀，继续加氨水沉淀溶解，得深蓝色 $[\text{Cu}(\text{NH}_3)_4]^{2+}$；
- $\text{Ag}^+$ 遇过量氨水生成 $[\text{Ag}(\text{NH}_3)_2]^+$（银氨溶液，银镜反应的基础）。

配合物的形成常带来三种「反常」：**沉淀意外溶解**（如 $\text{AgCl}$ 溶于氨水、$\text{Al}(\text{OH})_3$ 溶于强碱生成 $[\text{Al}(\text{OH})_4]^-$）、**颜色突变**（如浓 $\text{CuCl}_2$ 呈黄绿色、稀释后变蓝）、**离子被掩蔽**（如 $\text{F}^-$ 与 $\text{Fe}^{3+}$ 生成无色 $[\text{FeF}_6]^{3-}$，使 $\text{Fe}^{3+}$ 失去反应活性）。做工艺流程题时，遇到「加过量氨水/氰化物」、「加氟化物」、「难溶物遇特定盐反而溶解」或「颜色特征性突变」，都应联想到配位。

#### 超分子

**超分子**（Supramolecule）：由两种或两种以上的分子通过 **分子间作用力**（氢键、范德华力、配位键等 **非共价键**）相互结合而成的、具有特定结构和功能的聚集体。

- **两个重要特征**：**分子识别** 和 **自组装**；
- 典型例子：冠醚识别碱金属离子（不同冠醚空腔匹配不同大小的离子）、杯酚通过尺寸匹配分离 $\text{C}_{60}$ 与 $\text{C}_{70}$、$\text{C}_{60}$ 与 $(\text{HF})_n$ 的组装等；
- 超分子的形成靠非共价键作用，这也是其能可逆识别、自组装的原因。
