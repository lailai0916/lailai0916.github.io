## 说明

这份笔记对照人教版教材，按知识板块归类，重概念、结论、对比与方程式，省略大段过程推导与实验细节。加粗的多是关键概念、标志性结论和必记数据。

浙江高中化学学考考必修两册；选考在此基础上加试三册选择性必修。下面按知识对象与分析方法重新编排，不再沿用教材册次。

## 板块

- [基本概念与计算](/docs/note/chemistry/fundamentals)：物质分类、离子反应、氧化还原与物质的量；
- [无机化学](/docs/note/chemistry/inorganic)：金属、非金属、无机材料与资源利用；
- [物质结构](/docs/note/chemistry/structure)：原子结构、周期律、化学键、分子与晶体；
- [化学反应原理](/docs/note/chemistry/reaction)：反应热、反应速率、化学平衡与反应方向；
- [水溶液与电化学](/docs/note/chemistry/solution-electrochemistry)：离子平衡、酸碱、沉淀与电极过程；
- [有机化学](/docs/note/chemistry/organic)：烃、官能团、生物大分子、高分子与有机合成。
