## 说明

这份笔记对照人教版教材，按专题归类，重考点：重结论、重概念、重对比、重方程式，省略大段过程推导与实验细节；加粗的多是关键概念、标志性结论和必记数据。

浙江高中化学用的是 **人教版**。**学考** 考 **必修** 两册（《必修第一册》《必修第二册》）；**选考** 在必修的基础上，加试 **选择性必修** 三册（《化学反应原理》《物质结构与性质》《有机化学基础》）。下面列出这五册的教材目录。

化学式与方程式用 KaTeX 排版，如 $\text{CO}_2$、$\text{Ca}^{2+}$、$\text{Fe}(\text{OH})_3\downarrow$，反应写作 $2\text{Na}+2\text{H}_2\text{O}\to 2\text{NaOH}+\text{H}_2\uparrow$。
