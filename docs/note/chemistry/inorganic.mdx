## 钠及其化合物

### 钠单质

- **物理性质**：银白色、质软（可用小刀切）、密度小（比水小、比煤油大）、熔点低（约 $98\ ^\circ\text{C}$）、导电导热；
- 保存在 **煤油** 中，隔绝空气和水；取用用镊子，不可用手接触。

**化学性质**（钠是强还原剂，很活泼）

- 与 $\text{O}_2$ 反应，条件不同产物不同：

$$4\text{Na}+\text{O}_2\to 2\text{Na}_2\text{O}\text{（常温，白色）}$$

$$2\text{Na}+\text{O}_2\xrightarrow{\Delta}\text{Na}_2\text{O}_2\text{（点燃 / 加热，淡黄色）}$$

- 与水反应，**浮、熔、游、响、红**（浮在水面、熔成小球、四处游动、发出嘶响、酚酞变红），分别对应密度比水小、熔点低且放热、生成气体、反应剧烈、生成 $\text{NaOH}$：

$$2\text{Na}+2\text{H}_2\text{O}\to 2\text{NaOH}+\text{H}_2\uparrow$$

$$2\text{Na}+2\text{H}_2\text{O}\to 2\text{Na}^++2\text{OH}^-+\text{H}_2\uparrow$$

- 与酸溶液反应比与水更剧烈，钠直接与 $\text{H}^+$ 反应：$2\text{Na}+2\text{H}^+\to 2\text{Na}^++\text{H}_2\uparrow$；
- 与盐溶液反应，**先与水反应** 生成 $\text{NaOH}$，再由 $\text{NaOH}$ 与盐复分解，**不能** 置换出盐中的金属。投入 $\text{CuSO}_4$ 溶液出现蓝色沉淀而非红色铜：

$$2\text{Na}+2\text{H}_2\text{O}+\text{CuSO}_4\to\text{Cu}(\text{OH})_2\downarrow+\text{Na}_2\text{SO}_4+\text{H}_2\uparrow$$

钠着火不能用水或泡沫灭火器（会与水反应放 $\text{H}_2$、在 $\text{CO}_2$ 中也能燃烧），要用干燥沙土盖灭。

### 氧化钠与过氧化钠

|            |                  $\text{Na}_2\text{O}$                   |                             $\text{Na}_2\text{O}_2$                             |
| :--------: | :------------------------------------------------------: | :-----------------------------------------------------------------------------: |
|    颜色    |                         白色固体                         |                                   淡黄色固体                                    |
| 氧的化合价 |                           $-2$                           |                                      $-1$                                       |
|    与水    | $\text{Na}_2\text{O}+\text{H}_2\text{O}\to 2\text{NaOH}$ | $2\text{Na}_2\text{O}_2+2\text{H}_2\text{O}\to 4\text{NaOH}+\text{O}_2\uparrow$ |
|    类别    |                        碱性氧化物                        |                            过氧化物（非碱性氧化物）                             |

$\text{Na}_2\text{O}_2$ 中含离子键和 $\text{O}\text{—}\text{O}$ 非极性共价键，阴阳离子（$\text{O}_2^{2-}:\text{Na}^+$）个数比为 $1:2$。它与 $\text{CO}_2$ 反应可用于呼吸面具和潜水艇供氧：

$$2\text{Na}_2\text{O}_2+2\text{CO}_2\to 2\text{Na}_2\text{CO}_3+\text{O}_2$$

该反应中 $\text{Na}_2\text{O}_2$ 既是氧化剂又是还原剂（氧由 $-1$ 歧化为 $-2$ 和 $0$）。$\text{Na}_2\text{O}_2$ 强氧化性还能把 $\text{Fe}^{2+}$ 氧化成 $\text{Fe}^{3+}$、使滴入水中的酚酞 **先变红后褪色**（中间产物 $\text{H}_2\text{O}_2$ 漂白）。

三条定量规律很好用：

- **气体减半**：$\text{CO}_2$ 或 $\text{H}_2\text{O}(\text{g})$ 通过足量 $\text{Na}_2\text{O}_2$，反应气体与生成 $\text{O}_2$ 的物质的量之比恒为 $2:1$，气体体积减小一半；
- **电子转移**：每生成 $1\ \text{mol}\ \text{O}_2$ 转移 $2\ \text{mol}$ 电子（$1\ \text{mol}\ \text{Na}_2\text{O}_2$ 转移 $1\ \text{mol}$ 电子）；
- **固体增重**：$\text{Na}_2\text{O}_2$ 吸收 $\text{CO}_2$ 增重相当于等物质的量的 $\text{CO}$，吸收 $\text{H}_2\text{O}$ 增重相当于等物质的量的 $\text{H}_2$。凡组成符合 $(\text{CO})_m(\text{H}_2)_n$ 的物质（如 $\text{HCHO}$、$\text{CH}_3\text{COOH}$、葡萄糖）完全燃烧的产物通过足量 $\text{Na}_2\text{O}_2$，固体增重等于该物质的原质量。

### 碳酸钠与碳酸氢钠

|          | $\text{Na}_2\text{CO}_3$（纯碱、苏打） | $\text{NaHCO}_3$（小苏打） |
| :------: | :------------------------------------: | :------------------------: |
|   俗名   |               纯碱、苏打               |           小苏打           |
|  溶解性  |        易溶（同温下溶解度更大）        |        易溶（较小）        |
| 热稳定性 |                  稳定                  |          受热分解          |
|   与酸   |                  较慢                  |          剧烈、快          |
|   用途   |            玻璃、造纸、洗涤            |     发酵、灭火、制胃药     |

$\text{NaHCO}_3$ 受热分解：

$$2\text{NaHCO}_3\xrightarrow{\Delta}\text{Na}_2\text{CO}_3+\text{H}_2\text{O}+\text{CO}_2\uparrow$$

**鉴别与除杂**：

- 加热法——只有 $\text{NaHCO}_3$ 分解放出使石灰水变浑的 $\text{CO}_2$。「套管实验」把 $\text{NaHCO}_3$ 放内管、$\text{Na}_2\text{CO}_3$ 放外管，温度较低的内管先分解，更有说服力；
- 滴加相同浓度盐酸，$\text{NaHCO}_3$ 产气更快；加 $\text{BaCl}_2$（或 $\text{CaCl}_2$）只有 $\text{Na}_2\text{CO}_3$ 产生白色沉淀（不能用 $\text{Ba}(\text{OH})_2$、$\text{Ca}(\text{OH})_2$，两者都产生沉淀）；
- 除杂：$\text{Na}_2\text{CO}_3$ 固体中混 $\text{NaHCO}_3$ 直接加热即可；$\text{Na}_2\text{CO}_3$ 溶液中混 $\text{NaHCO}_3$ 加适量 $\text{NaOH}$，**不能** 用加热（会带出 $\text{Na}_2\text{CO}_3$ 且溶液中 $\text{NaHCO}_3$ 分解不彻底）。

二者可相互转化，通入足量 $\text{CO}_2$ 或加 $\text{NaOH}$：

$$\text{Na}_2\text{CO}_3+\text{CO}_2+\text{H}_2\text{O}\to 2\text{NaHCO}_3$$

$$\text{NaHCO}_3+\text{NaOH}\to\text{Na}_2\text{CO}_3+\text{H}_2\text{O}$$

**侯氏制碱法**（联合制碱法）由侯德榜创立，向饱和食盐水中先通 $\text{NH}_3$ 再通 $\text{CO}_2$，析出溶解度较小的 $\text{NaHCO}_3$，灼烧得纯碱，母液冷却盐析出副产品氯化铵（氮肥）：

$$\text{NaCl}+\text{NH}_3+\text{CO}_2+\text{H}_2\text{O}\to\text{NaHCO}_3\downarrow+\text{NH}_4\text{Cl}$$

先通溶解度大的 $\text{NH}_3$ 使溶液显碱性，才能吸收更多 $\text{CO}_2$。

### 焰色试验

- 焰色试验是 **物理变化**，属元素的性质，是原子核外电子跃迁放出特定波长的光；
- 用铂丝（或洁净铁丝）蘸取样品，先用 **稀盐酸** 洗净灼烧至无色（不用稀硫酸，其盐熔沸点高、难挥发）；
- **钠** 焰色为 **黄色**；**钾** 焰色为 **紫色**，需 **透过蓝色钴玻璃** 观察（滤去钠的黄光）。

## 氯及其化合物

### 氯气

- **物理性质**：黄绿色、有刺激性气味、有毒、密度比空气大、能溶于水（$1$ 体积水溶约 $2$ 体积氯气）；
- **氯气是强氧化剂**，化学性质活泼。

**与金属反应**（把金属氧化到高价）：

$$2\text{Na}+\text{Cl}_2\xrightarrow{\text{点燃}}2\text{NaCl}$$

$$2\text{Fe}+3\text{Cl}_2\xrightarrow{\text{点燃}}2\text{FeCl}_3$$

**与非金属反应**（$\text{H}_2$ 在 $\text{Cl}_2$ 中安静燃烧，苍白色火焰）：

$$\text{H}_2+\text{Cl}_2\xrightarrow{\text{点燃}}2\text{HCl}$$

**与水反应**（部分反应，是可逆反应）：

$$\text{Cl}_2+\text{H}_2\text{O}\rightleftharpoons\text{HCl}+\text{HClO}$$

**次氯酸**（$\text{HClO}$）是弱酸（比碳酸还弱）、强氧化性，有漂白和杀菌作用，见光分解：

$$2\text{HClO}\xrightarrow{\text{光照}}2\text{HCl}+\text{O}_2\uparrow$$

新制氯水成分复杂，含 $\text{Cl}_2$、$\text{H}_2\text{O}$、$\text{HClO}$ 三分子和 $\text{H}^+$、$\text{Cl}^-$、$\text{ClO}^-$、$\text{OH}^-$ 四离子。久置后 $\text{HClO}$ 分解、$\text{Cl}_2$ 耗尽，最终变为稀盐酸。据此氯水既显酸性（$\text{H}^+$）、又有强氧化漂白性（$\text{HClO}$）、还能与 $\text{AgNO}_3$ 生成沉淀（$\text{Cl}^-$）。

**与碱反应**（工业制漂白粉，有效成分 $\text{Ca}(\text{ClO})_2$）：

$$\text{Cl}_2+2\text{NaOH}\to\text{NaCl}+\text{NaClO}+\text{H}_2\text{O}$$

$$2\text{Cl}_2+2\text{Ca}(\text{OH})_2\to\text{Ca}(\text{ClO})_2+\text{CaCl}_2+2\text{H}_2\text{O}$$

**漂白原理对比**：$\text{HClO}$、$\text{Na}_2\text{O}_2$、$\text{O}_3$ 靠强氧化性漂白，把有色物质氧化为无色，褪色 **不可逆**；$\text{SO}_2$ 靠与有色物质结合成不稳定无色物，加热又复原，是 **可逆** 漂白；活性炭靠 **吸附** 脱色，是物理过程。

### 氯离子的检验

取待测液，先滴加稀硝酸酸化，再加 $\text{AgNO}_3$ 溶液，生成 **不溶于稀硝酸的白色沉淀**，即含 $\text{Cl}^-$：

$$\text{Ag}^++\text{Cl}^-\to\text{AgCl}\downarrow$$

加稀硝酸是为了排除 $\text{CO}_3^{2-}$ 等离子的干扰（$\text{Ag}_2\text{CO}_3$ 溶于硝酸而 $\text{AgCl}$ 不溶）。$\text{Ag}^+$ 与 $\text{Cl}^-$、$\text{Br}^-$、$\text{I}^-$ 分别生成白、淡黄、黄色沉淀，均不溶于稀硝酸，可用于区分卤离子。

## 铁 金属材料

### 铁及其化合物

#### 铁单质

- 铁是较活泼金属，通常显 $+2$、$+3$ 价；纯铁抗腐蚀，含杂质易生锈；
- 与非金属：$3\text{Fe}+2\text{O}_2\xrightarrow{\text{点燃}}\text{Fe}_3\text{O}_4$（黑色），$2\text{Fe}+3\text{Cl}_2\xrightarrow{\text{点燃}}2\text{FeCl}_3$（生成 $+3$ 价）；
- 与水蒸气（高温）：

$$3\text{Fe}+4\text{H}_2\text{O}(\text{g})\xrightarrow{\text{高温}}\text{Fe}_3\text{O}_4+4\text{H}_2$$

- 与非氧化性酸（盐酸、稀硫酸）生成 **$+2$ 价**：$\text{Fe}+2\text{H}^+\to\text{Fe}^{2+}+\text{H}_2\uparrow$；
- 与盐溶液（置换较不活泼金属）：$\text{Fe}+\text{Cu}^{2+}\to\text{Fe}^{2+}+\text{Cu}$。

价态口诀「遇强变三、遇弱变二」：与足量 $\text{Cl}_2$、稀 $\text{HNO}_3$ 等强氧化剂生成 $\text{Fe}^{3+}$，与盐酸、稀硫酸、$\text{S}$、$\text{Cu}^{2+}$ 等生成 $\text{Fe}^{2+}$。铁与稀硝酸的产物还跟用量有关：

$$\text{Fe}+4\text{H}^++\text{NO}_3^-\to\text{Fe}^{3+}+\text{NO}\uparrow+2\text{H}_2\text{O}\text{（硝酸足量）}$$

$$3\text{Fe}+8\text{H}^++2\text{NO}_3^-\to 3\text{Fe}^{2+}+2\text{NO}\uparrow+4\text{H}_2\text{O}\text{（铁足量）}$$

常温下铁、铝遇 **冷的浓硫酸或浓硝酸** 会 **钝化**（表面生成致密氧化膜），故可用铁、铝容器盛装冷浓硫酸 / 浓硝酸。

#### 铁的氧化物与氢氧化物

|            | $\text{FeO}$ | $\text{Fe}_2\text{O}_3$ | $\text{Fe}_3\text{O}_4$ |
| :--------: | :----------: | :---------------------: | :---------------------: |
| 铁的化合价 |     $+2$     |          $+3$           |       $+2$、$+3$        |
|    颜色    |     黑色     |         红棕色          |          黑色           |
|    俗名    |      —       |          铁红           |       磁性氧化铁        |

**氢氧化亚铁与氢氧化铁**

- $\text{Fe}(\text{OH})_2$ 是 **白色** 沉淀，在空气中迅速变灰绿，最终变 **红褐色**：

$$4\text{Fe}(\text{OH})_2+\text{O}_2+2\text{H}_2\text{O}\to 4\text{Fe}(\text{OH})_3$$

- $\text{Fe}(\text{OH})_3$ 是 **红褐色** 沉淀，受热分解：$2\text{Fe}(\text{OH})_3\xrightarrow{\Delta}\text{Fe}_2\text{O}_3+3\text{H}_2\text{O}$；
- 制备 $\text{Fe}(\text{OH})_2$ 要 **隔绝空气**：先煮沸溶液赶尽溶解氧，把吸取 $\text{NaOH}$ 的滴管 **伸入 $\text{FeSO}_4$ 液面下** 再挤出，必要时在液面覆盖一层煤油，否则很快被氧化。

#### 铁盐与亚铁盐的相互转化

- $\text{Fe}^{2+}$ 遇氧化剂（$\text{Cl}_2$、$\text{HNO}_3$、$\text{H}_2\text{O}_2$、酸性 $\text{KMnO}_4$ 等）被氧化为 $\text{Fe}^{3+}$：$2\text{Fe}^{2+}+\text{Cl}_2\to 2\text{Fe}^{3+}+2\text{Cl}^-$；
- $\text{Fe}^{3+}$ 遇还原剂（$\text{Fe}$、$\text{Cu}$ 等）被还原为 $\text{Fe}^{2+}$：$2\text{Fe}^{3+}+\text{Fe}\to 3\text{Fe}^{2+}$、$2\text{Fe}^{3+}+\text{Cu}\to 2\text{Fe}^{2+}+\text{Cu}^{2+}$（工业用 $\text{FeCl}_3$ 腐蚀铜制电路板）。

保存 $\text{Fe}^{2+}$ 溶液常加少量 **铁粉** 防氧化、加少量对应酸抑制水解。

|                  |      $\text{Fe}^{2+}$      | $\text{Fe}^{3+}$ |
| :--------------: | :------------------------: | :--------------: |
|     溶液颜色     |           浅绿色           |       黄色       |
| 加 $\text{NaOH}$ | 白色 $\to$ 灰绿 $\to$ 红褐 |    红褐色沉淀    |
| 加 $\text{KSCN}$ |           不变色           |    **变红色**    |

**离子检验**：

- 检验 $\text{Fe}^{3+}$：滴加 $\text{KSCN}$ 溶液变 **血红色**（$\text{Fe}^{3+}+3\text{SCN}^-\to\text{Fe}(\text{SCN})_3$），最灵敏；
- 检验 $\text{Fe}^{2+}$：先滴 $\text{KSCN}$ **不变色**，再滴氯水（或双氧水）**变红**，证明原溶液是 $\text{Fe}^{2+}$（被氧化成 $\text{Fe}^{3+}$）。顺序不能反，否则无法排除原溶液已含 $\text{Fe}^{3+}$。

#### 铁三角

$\text{Fe}$、$\text{Fe}^{2+}$、$\text{Fe}^{3+}$ 三者可相互转化，抓住 **化合价升降** 就能判断该加氧化剂还是还原剂：

- $\text{Fe}\to\text{Fe}^{2+}$：与非氧化性酸、$\text{Cu}^{2+}$ 或适量 $\text{Fe}^{3+}$ 反应；
- $\text{Fe}\to\text{Fe}^{3+}$：与足量 $\text{Cl}_2$、足量稀 $\text{HNO}_3$ 反应；
- $\text{Fe}^{2+}\to\text{Fe}^{3+}$：加氧化剂（$\text{Cl}_2$、$\text{H}_2\text{O}_2$……）；
- $\text{Fe}^{3+}\to\text{Fe}^{2+}$：加还原剂（$\text{Fe}$、$\text{Cu}$）；
- $\text{Fe}^{3+}\to\text{Fe}$、$\text{Fe}^{2+}\to\text{Fe}$：更强还原剂或电解。

其中 $\text{Fe}$ 与 $\text{Fe}^{3+}$ 归中生成 $\text{Fe}^{2+}$（$\text{Fe}+2\text{Fe}^{3+}\to 3\text{Fe}^{2+}$）是最典型的一步。

### 金属材料

#### 铝及其化合物

铝是地壳中含量最高的金属元素，性质活泼但表面易生成致密 $\text{Al}_2\text{O}_3$ 膜而耐腐蚀。铝、$\text{Al}_2\text{O}_3$、$\text{Al}(\text{OH})_3$ 都 **既能与强酸反应、又能与强碱反应**，是「两性」的典型。

- 铝与酸、碱都放出 $\text{H}_2$：

$$2\text{Al}+6\text{H}^+\to 2\text{Al}^{3+}+3\text{H}_2\uparrow$$

$$2\text{Al}+2\text{OH}^-+2\text{H}_2\text{O}\to 2\text{AlO}_2^-+3\text{H}_2\uparrow$$

- 两性氧化物 $\text{Al}_2\text{O}_3$：$\text{Al}_2\text{O}_3+6\text{H}^+\to 2\text{Al}^{3+}+3\text{H}_2\text{O}$，$\text{Al}_2\text{O}_3+2\text{OH}^-\to 2\text{AlO}_2^-+\text{H}_2\text{O}$；
- 两性氢氧化物 $\text{Al}(\text{OH})_3$：$\text{Al}(\text{OH})_3+3\text{H}^+\to\text{Al}^{3+}+3\text{H}_2\text{O}$，$\text{Al}(\text{OH})_3+\text{OH}^-\to\text{AlO}_2^-+2\text{H}_2\text{O}$。$\text{Al}(\text{OH})_3$ 只溶于强酸强碱，**不溶于弱酸弱碱**，故实验室用可溶铝盐加 **氨水**（而非 $\text{NaOH}$，以免过量溶解）制取。

**铝三角** 是 $\text{Al}^{3+}$、$\text{Al}(\text{OH})_3$、$\text{AlO}_2^-$ 三者的相互转化，$\text{Al}(\text{OH})_3$ 居中，加酸得 $\text{Al}^{3+}$、加碱得 $\text{AlO}_2^-$：

$$
\begin{aligned}
  \text{Al}^{3+}+3\text{OH}^- & \to\text{Al}(\text{OH})_3\downarrow \\
  \text{Al}(\text{OH})_3+\text{OH}^- & \to\text{AlO}_2^-+2\text{H}_2\text{O}
\end{aligned}
$$

$$
\begin{aligned}
  \text{AlO}_2^-+\text{H}^++\text{H}_2\text{O} & \to\text{Al}(\text{OH})_3\downarrow \\
  \text{Al}(\text{OH})_3+3\text{H}^+ & \to\text{Al}^{3+}+3\text{H}_2\text{O}
\end{aligned}
$$

**滴加顺序影响现象**（易错）：

- 向 $\text{AlCl}_3$ 中 **滴入 $\text{NaOH}$**：先见白色沉淀，$\text{NaOH}$ 过量后沉淀溶解；
- 向 $\text{NaOH}$ 中 **滴入 $\text{AlCl}_3$**：开始无沉淀（$\text{Al}^{3+}$ 立即被过量碱溶为 $\text{AlO}_2^-$），后期才出现沉淀且不再溶解。

明矾 $\text{KAl}(\text{SO}_4)_2\cdot 12\text{H}_2\text{O}$ 溶于水，$\text{Al}^{3+}$ 水解生成 $\text{Al}(\text{OH})_3$ 胶体，吸附水中悬浮杂质而 **净水**。

#### 合金

- **合金**：由两种或两种以上的金属（或金属与非金属）熔合而成的、具有金属特性的物质；
- 合金一般 **硬度比成分金属大、熔点比成分金属低**（原子大小不同扰乱了规则排列），性能更优；
- 常见合金：生铁与钢（铁碳合金，靠含碳量区分）、铝合金、铜合金、储氢合金、钛合金。

生铁含碳量 $2\% - 4.3\%$，硬而脆；钢含碳量 $0.03\% - 2\%$，硬而韧。不锈钢因含铬、镍而抗腐蚀。

#### 正确使用金属材料

- 金属的活泼性与用途、抗腐蚀性相关；铝表面易形成致密氧化膜（$\text{Al}_2\text{O}_3$）而耐腐蚀；
- 金属回收利用可节约资源、减少污染；
- 物质的性质决定用途，选材要综合考虑性能、成本与环境。

## 化工生产中的重要非金属元素

### 硫及其化合物

#### 硫单质

- 俗称硫黄，**淡黄色固体**，质地脆、易研成粉末；不溶于水、微溶于酒精、易溶于二硫化碳（$\text{CS}_2$）。粘在试管壁上的硫，可用 $\text{CS}_2$ 或热 $\text{NaOH}$ 溶液洗去；
- 最外层 $6$ 个电子，既有氧化性又有还原性，常见化合价为 $-2$、$0$、$+4$、$+6$；
- **氧化性**——与氢气化合生成有臭鸡蛋气味、剧毒的硫化氢：$\text{H}_2+\text{S}\xrightarrow{\Delta}\text{H}_2\text{S}$；与变价金属反应只生成低价硫化物：$\text{Fe}+\text{S}\xrightarrow{\Delta}\text{FeS}$，$2\text{Cu}+\text{S}\xrightarrow{\Delta}\text{Cu}_2\text{S}$（产物是亚铁、亚铜，证明硫氧化性比氯气弱）；与汞在常温下即反应：$\text{Hg}+\text{S}\to\text{HgS}$——洒落的汞可撒硫粉处理，防止汞中毒（银器久置变黑同理，生成 $\text{Ag}_2\text{S}$）；
- **还原性**——在空气中燃烧发出 **淡蓝色火焰**，在纯氧中为 **蓝紫色火焰**：$\text{S}+\text{O}_2\xrightarrow{\text{点燃}}\text{SO}_2$（产物只有 $\text{SO}_2$，不生成 $\text{SO}_3$）。

#### 二氧化硫

**物理性质**：无色、有 **刺激性气味** 的有毒气体，密度比空气大，易液化，易溶于水（$1$ 体积水溶约 $40$ 体积 $\text{SO}_2$）。

**化学性质**

- 属 **酸性氧化物**，与水化合生成亚硫酸（弱酸）：$\text{SO}_2+\text{H}_2\text{O}\rightleftharpoons\text{H}_2\text{SO}_3$；
- 与碱反应，$\text{SO}_2$ 少量生成正盐、过量生成酸式盐：

$$\text{SO}_2+2\text{NaOH}\to\text{Na}_2\text{SO}_3+\text{H}_2\text{O}$$

$$\text{SO}_2+\text{NaOH}\to\text{NaHSO}_3$$

- **还原性**（主要性质，硫为 $+4$ 价，可被氧化到 $+6$）：$2\text{SO}_2+\text{O}_2\xrightarrow[\Delta]{\text{催化剂}}2\text{SO}_3$，也能被氯水、溴水、酸性 $\text{KMnO}_4$、$\text{H}_2\text{O}_2$、$\text{Fe}^{3+}$ 等氧化而褪色；
- **氧化性**（遇更强还原剂 $\text{H}_2\text{S}$）：$\text{SO}_2+2\text{H}_2\text{S}\to 3\text{S}\downarrow+2\text{H}_2\text{O}$；
- **漂白性**：能使 **品红溶液褪色**，与有色物质化合成不稳定的无色物质，**加热后恢复红色**（这是 $\text{SO}_2$ 漂白的特征，与氯水的氧化漂白不同），但不能漂白石蕊等指示剂；
- **毒性与用途**：可作食品防腐剂、杀菌剂，如葡萄酒中添加少量 $\text{SO}_2$ 消毒，但过量有害。

**$\text{SO}_2$ 与 $\text{Cl}_2$ 漂白对比**

|        |                                                     $\text{SO}_2$                                                     | $\text{Cl}_2$（氯水）  |
| :----: | :-------------------------------------------------------------------------------------------------------------------: | :--------------------: |
|  原理  |                                                    与有色物质化合                                                     | 强氧化性，氧化有色物质 |
| 稳定性 |                                                 不稳定，加热恢复原色                                                  |      稳定，不恢复      |
|  范围  |                                               漂白品红等，不能漂白石蕊                                                |  能使石蕊先变红后褪色  |
|  混合  | 等物质的量混合于水中漂白性几乎消失：$\text{SO}_2+\text{Cl}_2+2\text{H}_2\text{O}\to\text{H}_2\text{SO}_4+2\text{HCl}$ |                        |

#### 亚硫酸与亚硫酸盐

- 亚硫酸（$\text{H}_2\text{SO}_3$）只存在于水溶液中，极不稳定，易分解为 $\text{SO}_2$ 和水；是二元中强酸，酸性 $\text{H}_2\text{SO}_3>\text{H}_2\text{CO}_3>\text{H}_2\text{S}$；
- 有明显 **还原性**，露置空气中被氧气缓慢氧化成硫酸：$2\text{H}_2\text{SO}_3+\text{O}_2\to 2\text{H}_2\text{SO}_4$；
- 亚硫酸盐中碱金属盐易溶，$\text{BaSO}_3$、$\text{CaSO}_3$ 难溶。

#### 硫酸

**稀硫酸**：具有酸的通性——与活泼金属反应放氢气、与碱及碱性氧化物反应、与盐反应。

**浓硫酸的特性**

- **吸水性**：能吸收现成的水，可作干燥剂（不能干燥 $\text{NH}_3$、$\text{H}_2\text{S}$、$\text{HBr}$、$\text{HI}$ 等碱性或还原性气体）；
- **脱水性**：按 $2:1$ 夺取有机物中氢、氧元素，使蔗糖等炭化变黑（黑面包实验，生成的碳会继续与浓硫酸反应放出 $\text{SO}_2$、$\text{CO}_2$）；
- **强氧化性**（体现 $+6$ 价硫）：
  - 与铜反应：$\text{Cu}+2\text{H}_2\text{SO}_4(\text{浓})\xrightarrow{\Delta}\text{CuSO}_4+\text{SO}_2\uparrow+2\text{H}_2\text{O}$；
  - 与碳反应：$\text{C}+2\text{H}_2\text{SO}_4(\text{浓})\xrightarrow{\Delta}\text{CO}_2\uparrow+2\text{SO}_2\uparrow+2\text{H}_2\text{O}$；
  - **常温下使铁、铝钝化**（表面生成致密氧化膜），故可用铁、铝罐车装运冷的浓硫酸。
- 浓硫酸难挥发，可用于制取挥发性酸，如实验室用食盐固体与浓硫酸微热制氯化氢。

浓硫酸与金属反应生成 $\text{SO}_2$ 而非 $\text{H}_2$；随反应进行浓度变稀，稀硫酸不再与铜反应。稀释浓硫酸时，须将 **浓硫酸沿器壁缓缓注入水中** 并搅拌，切不可反过来。

#### 硫酸根离子的检验

- 先加 **足量稀盐酸** 排除 $\text{CO}_3^{2-}$、$\text{SO}_3^{2-}$、$\text{Ag}^+$ 干扰，再滴加 $\text{BaCl}_2$ 溶液；
- 产生 **不溶于稀盐酸的白色沉淀**，即证明含 $\text{SO}_4^{2-}$：$\text{Ba}^{2+}+\text{SO}_4^{2-}\to\text{BaSO}_4\downarrow$；
- 酸化不能用稀硝酸，也不能引入 $\text{NO}_3^-$——酸性条件下 $\text{NO}_3^-$ 会把 $\text{SO}_3^{2-}$ 氧化成 $\text{SO}_4^{2-}$ 造成误判。

常见硫酸盐俗名：石膏（$\text{CaSO}_4\cdot 2\text{H}_2\text{O}$）、芒硝（$\text{Na}_2\text{SO}_4\cdot 10\text{H}_2\text{O}$）、胆矾（$\text{CuSO}_4\cdot 5\text{H}_2\text{O}$）、绿矾（$\text{FeSO}_4\cdot 7\text{H}_2\text{O}$）、重晶石（$\text{BaSO}_4$）。

#### 硫酸的工业制备

工业上用 **接触法** 制硫酸，分三步。

- **造气**——燃烧硫黄或焙烧黄铁矿得 $\text{SO}_2$：$4\text{FeS}_2+11\text{O}_2\xrightarrow{\text{高温}}2\text{Fe}_2\text{O}_3+8\text{SO}_2$。设备是 **沸腾炉**，把矿石粉碎、增大接触面积使反应充分。炉气须除尘、洗涤、干燥净化，**防止杂质使催化剂中毒**；
- **催化氧化**——在 **接触室** 内经催化剂氧化：$2\text{SO}_2+\text{O}_2\xrightarrow[\Delta]{\text{催化剂}}2\text{SO}_3$。催化剂是 **五氧化二钒**（$\text{V}_2\text{O}_5$），温度控制在 $400\sim 500\ ^\circ\text{C}$（此区间催化剂活性最高，兼顾速率与转化率），常压下转化率已很高；
- **吸收**——用 **$98.3\%$ 的浓硫酸** 吸收 $\text{SO}_3$，得发烟硫酸后再稀释。不直接用水吸收，是因为 $\text{SO}_3$ 遇水放热剧烈、生成大量硫酸酸雾，吸收率反而低。

尾气中残余 $\text{SO}_2$ 须回收，常用氨水吸收或钙基固硫，防止 **酸雨**（$\text{pH}<5.6$）。

### 氮及其化合物

#### 氮气与氮的固定

- $\text{N}_2$ 分子中含 **氮氮三键**，键能大，**性质稳定**；
- **氮的固定**：将游离态氮转化为氮的化合物，分自然固氮（雷电、根瘤菌等生物固氮）和人工固氮（合成氨）；
- 与氢气合成氨：$\text{N}_2+3\text{H}_2\xrightarrow[\text{高温高压}]{\text{催化剂}}2\text{NH}_3$；
- 高温或放电下与氧气化合：$\text{N}_2+\text{O}_2\xrightarrow{\text{放电}}2\text{NO}$；
- 与镁点燃生成氮化镁：$3\text{Mg}+\text{N}_2\xrightarrow{\text{点燃}}\text{Mg}_3\text{N}_2$。

#### 氮的氧化物

- **$\text{NO}$**：无色气体，**不溶于水**，有毒，常温下遇氧气立即被氧化：$2\text{NO}+\text{O}_2\to 2\text{NO}_2$；
- **$\text{NO}_2$**：**红棕色**、有刺激性气味的有毒气体，与水反应：

$$3\text{NO}_2+\text{H}_2\text{O}\to 2\text{HNO}_3+\text{NO}$$

- 工业上 $\text{NO}_2$ 与 $\text{O}_2$ 通入水中可被完全吸收：$4\text{NO}_2+\text{O}_2+2\text{H}_2\text{O}\to 4\text{HNO}_3$，$4\text{NO}+3\text{O}_2+2\text{H}_2\text{O}\to 4\text{HNO}_3$；
- $\text{NO}_2$ 存在可逆聚合：$2\text{NO}_2\rightleftharpoons\text{N}_2\text{O}_4$，故标准状况下 $\text{NO}_2$ 并非气态，$2\ \text{mol}\ \text{NO}$ 与 $1\ \text{mol}\ \text{O}_2$ 充分反应后产物小于 $2\ \text{mol}$；
- $\text{NO}$、$\text{NO}_2$ 是形成 **光化学烟雾、酸雨** 的主要污染物。

|        | $\text{NO}$ |      $\text{NO}_2$      |
| :----: | :---------: | :---------------------: |
|  颜色  |    无色     |         红棕色          |
| 溶于水 |    不溶     | 反应生成 $\text{HNO}_3$ |
|  毒性  |    有毒     |          有毒           |
|  收集  | 排水法收集  |      向上排空气法       |

#### 氨

**物理性质**：无色、有 **刺激性气味** 的气体，密度比空气小，**极易溶于水**（$1$ 体积水溶约 $700$ 体积氨），易液化（液氨可作制冷剂）。

**喷泉实验**：氨极易溶于水，使烧瓶内气压骤减，形成喷泉；溶液显碱性（滴入石蕊变蓝）。**只要气体极易溶于吸收液即可形成喷泉**——极易溶于水的 $\text{HCl}$、$\text{SO}_2$（配 $\text{NaOH}$）等同理。

**化学性质**

- 与水反应，一水合氨是弱碱：$\text{NH}_3+\text{H}_2\text{O}\rightleftharpoons\text{NH}_3\cdot\text{H}_2\text{O}\rightleftharpoons\text{NH}_4^++\text{OH}^-$；
- 与酸反应生成铵盐，与挥发性酸相遇产生白烟：$\text{NH}_3+\text{HCl}\to\text{NH}_4\text{Cl}$（**白烟**，可检验氨或氯化氢）；
- **还原性**——催化氧化是工业制硝酸的基础：

$$4\text{NH}_3+5\text{O}_2\xrightarrow[\Delta]{\text{催化剂}}4\text{NO}+6\text{H}_2\text{O}$$

**实验室制氨**：$2\text{NH}_4\text{Cl}+\text{Ca}(\text{OH})_2\xrightarrow{\Delta}\text{CaCl}_2+2\text{NH}_3\uparrow+2\text{H}_2\text{O}$，向下排空气法收集，用湿润的红色石蕊试纸检验（变蓝）。几个易错点：不能用 $\text{NaOH}$ 代替熟石灰（$\text{NaOH}$ 高温腐蚀玻璃、易吸潮结块），不能只加热 $\text{NH}_4\text{Cl}$（分解产物遇冷又重新化合），不能用 $\text{NH}_4\text{NO}_3$（受热可能爆炸）。干燥氨气只能用 **碱石灰**，不能用浓硫酸、无水氯化钙（$\text{CaCl}_2$ 会与氨结合）。

#### 铵盐

- 都是易溶于水的白色晶体，常用作氮肥；
- **受热易分解**：$\text{NH}_4\text{Cl}\xrightarrow{\Delta}\text{NH}_3\uparrow+\text{HCl}\uparrow$；
- **与碱反应放出氨气**：$\text{NH}_4^++\text{OH}^-\xrightarrow{\Delta}\text{NH}_3\uparrow+\text{H}_2\text{O}$——**铵根离子的检验** 即据此：加浓 $\text{NaOH}$ 溶液并加热，放出使湿润红色石蕊试纸变蓝的气体；
- 铵态氮肥不能与碱性物质（如草木灰、熟石灰）混用，否则放出氨气而失效。

#### 硝酸

**物理性质**：无色、易挥发、有刺激性气味的液体，发烟硝酸暴露在空气中冒白雾；浓硝酸常因溶有分解产生的 $\text{NO}_2$ 而呈黄色。

**化学性质**

- 不稳定，见光或受热分解：$4\text{HNO}_3\xrightarrow{\text{光照}}4\text{NO}_2\uparrow+\text{O}_2\uparrow+2\text{H}_2\text{O}$（故保存在棕色瓶、置阴凉处）；
- **强氧化性**——与金属反应 **不生成氢气**，浓硝酸放 $\text{NO}_2$、稀硝酸放 $\text{NO}$：

$$\text{Cu}+4\text{HNO}_3(\text{浓})\to\text{Cu}(\text{NO}_3)_2+2\text{NO}_2\uparrow+2\text{H}_2\text{O}$$

$$3\text{Cu}+8\text{HNO}_3(\text{稀})\to 3\text{Cu}(\text{NO}_3)_2+2\text{NO}\uparrow+4\text{H}_2\text{O}$$

- **常温下使铁、铝钝化**，可用铁、铝容器盛装冷的浓硝酸；
- 与碳反应：$\text{C}+4\text{HNO}_3(\text{浓})\xrightarrow{\Delta}\text{CO}_2\uparrow+4\text{NO}_2\uparrow+2\text{H}_2\text{O}$；
- 浓硝酸与浓盐酸按体积比 $1:3$ 配成 **王水**，能溶解金、铂。

**浓、稀硝酸氧化性对比**：浓硝酸氧化性更强，但与金属反应时得电子数少（$+5\to+4$），稀硝酸每分子得电子多（$+5\to+2$）。同一金属与浓硝酸反应，转移相同电子放出的气体体积更大。

**工业制硝酸**（氨催化氧化法）分三步：氨催化氧化 $4\text{NH}_3+5\text{O}_2\xrightarrow[\Delta]{\text{催化剂}}4\text{NO}+6\text{H}_2\text{O}$，$\text{NO}$ 被氧化 $2\text{NO}+\text{O}_2\to 2\text{NO}_2$，$\text{NO}_2$ 被水吸收 $3\text{NO}_2+\text{H}_2\text{O}\to 2\text{HNO}_3+\text{NO}$。

:::tip[一处易错]

把红热的木炭插入浓硝酸中放出红棕色气体，**不能** 据此说明浓硝酸能氧化碳——高温下浓硝酸自身也分解生成 $\text{NO}_2$。

:::

#### 含氮物质与环境

- **酸雨**（$\text{pH}\le 5.6$）：$\text{SO}_2$ 形成硫酸型酸雨，$\text{NO}_x$ 形成硝酸型酸雨；
- **光化学烟雾**：汽车尾气中的氮氧化物与碳氢化合物经紫外线照射生成有毒烟雾（尾气基本不含 $\text{SO}_2$）；
- **水体富营养化**：氮、磷等含量过高致水质恶化，出现水华、赤潮。

### 无机非金属材料

#### 硅

- 自然界中硅 **只以化合态存在**（$\text{SiO}_2$ 及硅酸盐），是地壳中含量第二的元素；
- 晶体硅是 **灰黑色、有金属光泽** 的固体，硬而脆，是良好的 **半导体材料**；
- 工业制硅：$\text{SiO}_2+2\text{C}\xrightarrow{\text{高温}}\text{Si}+2\text{CO}\uparrow$。

#### 二氧化硅

- **原子晶体**，熔点高、硬度大（水晶、石英、玛瑙的主要成分）；
- 属 **酸性氧化物**，但 **不溶于水**，对应水化物是硅酸；
- **不与一般酸反应，唯独与氢氟酸反应**（故 $\text{HF}$ 刻蚀玻璃、不能用玻璃瓶盛装）：$\text{SiO}_2+4\text{HF}\to\text{SiF}_4\uparrow+2\text{H}_2\text{O}$；
- 与碱、碱性氧化物反应：$\text{SiO}_2+2\text{NaOH}\to\text{Na}_2\text{SiO}_3+\text{H}_2\text{O}$（故盛碱液的试剂瓶用橡胶塞而非玻璃塞），$\text{SiO}_2+\text{CaO}\xrightarrow{\text{高温}}\text{CaSiO}_3$。

#### 硅酸与硅酸盐

- 硅酸（$\text{H}_2\text{SiO}_3$）是 **难溶弱酸**，酸性比碳酸弱，故不能由 $\text{SiO}_2$ 与水直接制得，用可溶性硅酸盐与酸反应：$\text{Na}_2\text{SiO}_3+2\text{HCl}\to 2\text{NaCl}+\text{H}_2\text{SiO}_3\downarrow$；
- $\text{Na}_2\text{SiO}_3$ 水溶液俗称 **水玻璃**，是矿物胶、木材防火剂；
- 硅酸盐组成常用氧化物形式表示，如钠长石写作 $\text{Na}_2\text{O}\cdot\text{Al}_2\text{O}_3\cdot 6\text{SiO}_2$。

#### 传统与新型无机非金属材料

|   材料   |       主要原料       |                                 说明                                 |
| :------: | :------------------: | :------------------------------------------------------------------: |
| **水泥** |     石灰石、黏土     |                            水硬性胶凝材料                            |
| **玻璃** | 纯碱、石灰石、石英砂 | 主要成分 $\text{Na}_2\text{SiO}_3$、$\text{CaSiO}_3$、$\text{SiO}_2$ |
| **陶瓷** |         黏土         |                            传统硅酸盐材料                            |

新型无机非金属材料：光导纤维（成分 $\text{SiO}_2$）、碳化硅、氮化硅、硅太阳能电池等，具有耐高温、耐腐蚀、强度高等特点。

## 化学与可持续发展

### 自然资源的开发利用

#### 金属矿物的开发利用

- 金属冶炼的实质是把金属 **从化合态还原为游离态**，据金属活动性选择冶炼方法：
  - **活泼金属**（$\text{K}$、$\text{Ca}$、$\text{Na}$、$\text{Mg}$、$\text{Al}$）：**电解法**（如电解熔融 $\text{Al}_2\text{O}_3$、$\text{NaCl}$、$\text{MgCl}_2$）；
  - **中等活泼金属**（$\text{Zn}$、$\text{Fe}$、$\text{Sn}$、$\text{Pb}$、$\text{Cu}$）：**热还原法**（用 $\text{C}$、$\text{CO}$、$\text{H}_2$、$\text{Al}$ 还原），如 $3\text{CO}+\text{Fe}_2\text{O}_3\xrightarrow{\text{高温}}2\text{Fe}+3\text{CO}_2$；
  - **不活泼金属**（$\text{Hg}$、$\text{Ag}$）：**热分解法**；金、铂等以游离态存在，直接开采。
- 电解铝要加冰晶石助熔以降低能耗；电解 $\text{Al}_2\text{O}_3$ 而不电解 $\text{AlCl}_3$，因为 $\text{AlCl}_3$ 熔融态不导电。

#### 海水资源的开发利用

- **海水淡化**：蒸馏法、电渗析法、离子交换法等；
- **海水提溴**：用氯气氧化苦卤中的 $\text{Br}^-$，$\text{Cl}_2+2\text{Br}^-\to\text{Br}_2+2\text{Cl}^-$，再经吹出、富集提纯；
- **海带提碘**：海水中碘浓度太低，工业上以富集了碘的海带为原料，用氧化剂将 $\text{I}^-$ 氧化为 $\text{I}_2$（$2\text{I}^-+\text{H}_2\text{O}_2+2\text{H}^+\to\text{I}_2+2\text{H}_2\text{O}$），再用四氯化碳萃取；
- **海水提镁**：用石灰乳沉淀 $\text{Mg}^{2+}$ 得 $\text{Mg}(\text{OH})_2$，酸化、脱水得无水 $\text{MgCl}_2$，再电解熔融 $\text{MgCl}_2$ 得镁。

#### 煤、石油和天然气的综合利用

- **煤的综合利用**：干馏（隔绝空气加强热，得焦炭、煤焦油、焦炉气；煤焦油分馏可得苯、甲苯等大量芳香烃）、气化、液化，均属化学变化；
- **石油的加工**：分馏（利用沸点不同分离，物理变化）、裂化和裂解（把长链烃断成短链、获得乙烯等气态烃，化学变化）、催化重整（把直链烃转化为芳香烃）；
- 天然气主要成分是甲烷，是较清洁的化石燃料，也是合成氨、制甲醇的重要原料。

### 化学品的合理使用

#### 化肥与农药

- 化肥合理施用能增产，过量施用会造成土壤板结、水体富营养化；
- 农药能防治病虫害，但滥用会污染环境、危害健康，应合理、安全使用。

#### 合成药物与食品添加剂

- 合成药物为防治疾病提供保障，须遵医嘱、按剂量使用，滥用抗生素会使病菌产生抗药性；
- 食品添加剂（防腐剂、着色剂、营养强化剂等）在规定范围内使用是安全的，但不得滥用。

#### 绿色化学与可持续发展

- **绿色化学** 的核心是从源头上 **减少或消除污染**，理想目标是反应物中的原子全部转化为期望的产物，即 **原子利用率达到 $100\%$**；
- 提倡使用无毒无害的原料、催化剂和溶剂，开发环境友好的化工工艺；
- 合理开发利用资源、防治污染、发展循环经济，是实现可持续发展的必由之路。
