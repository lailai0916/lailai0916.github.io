## 第一章 物质及其变化

### 第一节 物质的分类及转化

#### 物质的分类

- 按组成和性质，物质先分为 **混合物** 与 **纯净物**；
- 纯净物分为 **单质**（同种元素组成，如 $\text{O}_2$、$\text{Fe}$）与 **化合物**（不同元素组成）；
- 化合物按是否含碳分为无机物与有机物，无机物又分为 **氧化物、酸、碱、盐**。

|  类别  |                         定义                         |                                     举例                                      |
| :----: | :--------------------------------------------------: | :---------------------------------------------------------------------------: |
| **酸** |        电离时生成的阳离子全部是 $\text{H}^+$         |             $\text{HCl}$、$\text{H}_2\text{SO}_4$、$\text{HNO}_3$             |
| **碱** |        电离时生成的阴离子全部是 $\text{OH}^-$        | $\text{NaOH}$、$\text{Ca}(\text{OH})_2$、$\text{NH}_3\cdot\text{H}_2\text{O}$ |
| **盐** | 电离时生成金属阳离子（或 $\text{NH}_4^+$）和酸根离子 |           $\text{NaCl}$、$\text{Na}_2\text{CO}_3$、$\text{CuSO}_4$            |

酸按电离出的 $\text{H}^+$ 个数分一元酸、二元酸、多元酸；按酸根是否含氧分含氧酸、无氧酸。盐按酸中 $\text{H}^+$ 是否全部被中和，分正盐（$\text{Na}_2\text{CO}_3$）、酸式盐（$\text{NaHCO}_3$）、碱式盐（碱式碳酸铜 $\text{Cu}_2(\text{OH})_2\text{CO}_3$）。

**氧化物** 分类：

- 按元素分：金属氧化物（$\text{Na}_2\text{O}$、$\text{Fe}_2\text{O}_3$）、非金属氧化物（$\text{CO}_2$、$\text{SO}_2$）；
- 按性质分：
  - **酸性氧化物**：与碱反应只生成一种盐和水（如 $\text{CO}_2$、$\text{SO}_3$，多为非金属氧化物）；
  - **碱性氧化物**：与酸反应只生成盐和水（如 $\text{Na}_2\text{O}$、$\text{CaO}$、$\text{Fe}_2\text{O}_3$，一定是金属氧化物）；
  - 两性氧化物（$\text{Al}_2\text{O}_3$）、不成盐氧化物（$\text{CO}$、$\text{NO}$）。

四条辨析里只有一条「一定」成立，其余都是「主要」，别记反：

- 碱性氧化物 **一定是** 金属氧化物（唯一的「一定」）；
- 酸性氧化物 **主要是** 非金属氧化物，反例 $\text{Mn}_2\text{O}_7$（酸性）是金属氧化物；
- 金属氧化物 **主要是** 碱性氧化物，反例 $\text{Al}_2\text{O}_3$（两性）、$\text{Mn}_2\text{O}_7$（酸性）；
- $\text{Na}_2\text{O}_2$ 与酸、$\text{CO}_2$ 反应都放出 $\text{O}_2$，不只生成盐和水，**不是** 碱性氧化物（属过氧化物）。

#### 分散系及胶体

- **分散系**：把一种（或多种）物质分散到另一种（或多种）物质中形成的混合物；被分散的是分散质，容纳分散质的是分散剂；
- 按分散质粒子直径大小分为三类。

|  分散系  |  粒子直径   |          特征          |                举例                 |
| :------: | :---------: | :--------------------: | :---------------------------------: |
| **溶液** |  小于 1 nm  |    均一、稳定、透明    |         盐水、糖水、稀硫酸          |
| **胶体** | 1 – 100 nm  |  均一、较稳定、能透光  | $\text{Fe}(\text{OH})_3$ 胶体、豆浆 |
| **浊液** | 大于 100 nm | 不均一、不稳定、易沉降 |          泥水、油水混合物           |

三者的本质区别是 **分散质粒子直径**，不是外观是否均一。按分散剂状态胶体又分气溶胶（云、雾、烟）、液溶胶（蛋白质溶液、$\text{Fe}(\text{OH})_3$ 胶体）、固溶胶（有色玻璃、烟水晶）。

**胶体的性质**

- **丁达尔效应**：光束通过胶体时出现一条光亮的通路，源于胶粒对光的散射，是区分溶液与胶体的常用方法（但不是本质区别）；
- **电泳**：胶粒吸附离子而带电，在电场中定向移动。$\text{Fe}(\text{OH})_3$ 胶粒带正电——注意带电的是胶粒，胶体整体仍电中性。工业静电除尘就利用这一点；
- **聚沉**：加电解质、加带相反电荷的胶体或加热，都能破坏介稳状态使胶体聚沉，如盐卤点豆腐、三角洲的形成；
- **渗析**：胶体能透过滤纸、不能透过半透膜，据此可提纯胶体。

$\text{Fe}(\text{OH})_3$ 胶体的制备：向沸水中滴加饱和 $\text{FeCl}_3$ 溶液，继续煮沸至液体呈红褐色，停止加热。

$$\text{FeCl}_3+3\text{H}_2\text{O}\xrightarrow{\Delta}\text{Fe}(\text{OH})_3\,(\text{胶体})+3\text{HCl}$$

书写时 $\text{Fe}(\text{OH})_3$ 后不加 $\downarrow$（胶体不是沉淀），且不可长时间煮沸（否则聚沉成沉淀）。

#### 物质的转化

- 单质、氧化物、酸、碱、盐之间可以相互转化，核心是四类反应：化合、分解、置换、复分解；
- **金属单质** $\to$ **碱性氧化物** $\to$ **碱** $\to$ **盐**；**非金属单质** $\to$ **酸性氧化物** $\to$ **酸** $\to$ **盐**。

四种基本反应类型：

|    类型    |                  形式                  |                                  举例                                  |
| :--------: | :------------------------------------: | :--------------------------------------------------------------------: |
|  **化合**  | 多变一 $\text{A}+\text{B}\to\text{AB}$ |    $2\text{Na}+\text{O}_2\xrightarrow{\Delta}\text{Na}_2\text{O}_2$    |
|  **分解**  | 一变多 $\text{AB}\to\text{A}+\text{B}$ | $\text{CaCO}_3\xrightarrow{\text{高温}}\text{CaO}+\text{CO}_2\uparrow$ |
|  **置换**  |               单质换单质               |          $\text{Fe}+\text{CuSO}_4\to\text{FeSO}_4+\text{Cu}$           |
| **复分解** |              相互交换成分              |       $\text{HCl}+\text{NaOH}\to\text{NaCl}+\text{H}_2\text{O}$        |

置换反应必有单质参与、化合价必变，所以 **一定是氧化还原反应**；复分解反应 **一定不是**。

### 第二节 离子反应

#### 电解质与非电解质

- **电解质**：在水溶液中 **或** 熔融状态下能导电的化合物（酸、碱、盐、金属氧化物、水）；
- **非电解质**：在两种状态下都不能导电的化合物（如 $\text{CO}_2$、$\text{SO}_2$、$\text{NH}_3$、蔗糖、酒精）。

判断要点：

- 电解质、非电解质研究对象都是 **化合物**；单质、混合物既不是电解质也不是非电解质；
- $\text{CO}_2$、$\text{SO}_2$、$\text{NH}_3$ 溶于水虽导电，但导电的是它们与水反应的产物（$\text{H}_2\text{CO}_3$、$\text{NH}_3\cdot\text{H}_2\text{O}$ 等），本身是 **非电解质**；
- **能否导电** 与 **是不是电解质** 无必然联系：$\text{BaSO}_4$ 难溶，水溶液几乎不导电，但熔融时导电，仍是电解质；液态 $\text{HCl}$（纯净）不电离、不导电，溶于水才导电，仍是电解质。

**强电解质与弱电解质**

- **强电解质**：在水中完全电离（强酸如 $\text{HCl}$、$\text{H}_2\text{SO}_4$、$\text{HNO}_3$，强碱如 $\text{NaOH}$、$\text{KOH}$，大多数盐）；
- **弱电解质**：在水中部分电离（弱酸如 $\text{CH}_3\text{COOH}$，弱碱如 $\text{NH}_3\cdot\text{H}_2\text{O}$，以及水）。

强弱之分看 **电离程度**，与溶解性、导电能力都无关：$\text{BaSO}_4$ 难溶却是强电解质，浓醋酸导电强却是弱电解质。

#### 离子方程式

- **电离方程式**：强电解质用 $\to$，弱电解质用 $\rightleftharpoons$，如 $\text{HCl}\to\text{H}^++\text{Cl}^-$、$\text{CH}_3\text{COOH}\rightleftharpoons\text{CH}_3\text{COO}^-+\text{H}^+$；
- **离子方程式** 表示反应的实质，书写按「写、拆、删、查」四步：
  - 写出正确的化学方程式；
  - 把易溶的强电解质 **拆成离子**，其余（单质、氧化物、气体、沉淀、弱电解质、水）写化学式；
  - 删去两边不参加反应的离子；
  - 检查原子守恒与 **电荷守恒**（两边离子所带电荷代数和相等）。

|          拆          |            不拆            |
| :------------------: | :------------------------: |
| 强酸、强碱、可溶性盐 |     单质、氧化物、气体     |
|                      | 沉淀、弱酸弱碱、水、微溶物 |

微溶物是关键易错点：作反应物且澄清时（如石灰水）拆，作生成物或浑浊时（如石灰乳）写化学式。$\text{NaHCO}_3$ 等弱酸的酸式盐，$\text{HCO}_3^-$ **不能再拆**（$\text{HCO}_3^-$ 是弱酸酸根，进一步电离很弱）。

例如稀盐酸与氢氧化钠、碳酸钙的反应：

$$\text{H}^++\text{OH}^-\to\text{H}_2\text{O}$$

$$\text{CaCO}_3+2\text{H}^+\to\text{Ca}^{2+}+\text{H}_2\text{O}+\text{CO}_2\uparrow$$

**与「量」有关的离子方程式**：反应物用量不同，产物可能不同，需按少量物质配平。如 $\text{Ca}(\text{OH})_2$ 与少量 $\text{CO}_2$ 生成 $\text{CaCO}_3$，过量 $\text{CO}_2$ 则生成 $\text{Ca}(\text{HCO}_3)_2$：

$$\text{Ca}^{2+}+2\text{OH}^-+\text{CO}_2\to\text{CaCO}_3\downarrow+\text{H}_2\text{O}\quad(\text{少量}\ \text{CO}_2)$$

$$\text{OH}^-+\text{CO}_2\to\text{HCO}_3^-\quad(\text{过量}\ \text{CO}_2)$$

#### 离子反应发生的条件

复分解型离子反应发生，需生成 **沉淀、气体或弱电解质（水）** 之一，即溶液中某些离子浓度显著减小：

- 生成沉淀：$\text{Ba}^{2+}+\text{SO}_4^{2-}\to\text{BaSO}_4\downarrow$；
- 生成气体：$\text{CO}_3^{2-}+2\text{H}^+\to\text{H}_2\text{O}+\text{CO}_2\uparrow$；
- 生成水（弱电解质）：$\text{H}^++\text{OH}^-\to\text{H}_2\text{O}$。

**离子共存** 判断：能相互反应（生成沉淀、气体或水，或发生氧化还原、双水解）的离子不能大量共存。常考互斥：

- 生成沉淀：$\text{Ba}^{2+}$ 与 $\text{SO}_4^{2-}$、$\text{CO}_3^{2-}$；$\text{Ag}^+$ 与 $\text{Cl}^-$；$\text{OH}^-$ 与 $\text{Cu}^{2+}$、$\text{Fe}^{3+}$、$\text{Mg}^{2+}$；
- 生成气体 / 水：$\text{H}^+$ 与 $\text{OH}^-$、$\text{CO}_3^{2-}$、$\text{HCO}_3^-$；
- 发生氧化还原：$\text{Fe}^{3+}$ 与 $\text{I}^-$、$\text{S}^{2-}$；$\text{H}^+$、$\text{NO}_3^-$ 共存时能氧化 $\text{Fe}^{2+}$；
- 附加隐含条件也要留意：「无色溶液」排除 $\text{Cu}^{2+}$（蓝）、$\text{Fe}^{3+}$（黄）、$\text{Fe}^{2+}$（浅绿）、$\text{MnO}_4^-$（紫）；「强酸性」补 $\text{H}^+$、「强碱性」补 $\text{OH}^-$ 后再判断。

### 第三节 氧化还原反应

#### 基本概念

- **氧化还原反应** 的本质是 **电子的转移**（得失或偏移），特征是 **元素化合价的升降**；
- 一句口诀：**升失氧、降得还**——化合价 **升** 高、**失** 电子、被 **氧** 化；化合价 **降** 低、**得** 电子、被 **还** 原。

|    概念    | 化合价 |  电子  |     过程     |
| :--------: | :----: | :----: | :----------: |
| **氧化剂** |  降低  | 得电子 | 发生还原反应 |
| **还原剂** |  升高  | 失电子 | 发生氧化反应 |

- **氧化剂** 具有氧化性，得电子，被还原，生成 **还原产物**；
- **还原剂** 具有还原性，失电子，被氧化，生成 **氧化产物**。

以 $\text{Fe}+\text{CuSO}_4\to\text{FeSO}_4+\text{Cu}$ 为例：$\text{Fe}$ 化合价由 $0$ 升到 $+2$，是还原剂；$\text{Cu}^{2+}$ 由 $+2$ 降到 $0$，是氧化剂。

同种元素不同价态间的两类特殊反应：

- **歧化反应**：同一元素一部分化合价升高、另一部分降低，如 $\text{Cl}_2$ 与 $\text{NaOH}$、$\text{Na}_2\text{O}_2$ 与 $\text{H}_2\text{O}$；
- **归中反应**：不同价态的同种元素向中间价态靠拢，如 $2\text{H}_2\text{S}+\text{SO}_2\to 3\text{S}\downarrow+2\text{H}_2\text{O}$。归中时化合价 **只靠拢不交叉**，相邻价态之间不反应。

#### 常见氧化剂与还原剂

- 常见 **氧化剂**：$\text{O}_2$、$\text{Cl}_2$、浓硫酸、$\text{HNO}_3$、$\text{KMnO}_4$、$\text{FeCl}_3$，还原产物依次为 $\text{H}_2\text{O}$、$\text{Cl}^-$、$\text{SO}_2$、$\text{NO}_2$ 或 $\text{NO}$、$\text{Mn}^{2+}$、$\text{Fe}^{2+}$；
- 常见 **还原剂**：活泼金属（$\text{Na}$、$\text{Al}$、$\text{Fe}$）、$\text{C}$、$\text{H}_2$、$\text{CO}$、$\text{I}^-$、$\text{S}^{2-}$；
- 中间价态元素（如 $\text{Fe}^{2+}$、$\text{SO}_2$）既可作氧化剂又可作还原剂；最高价态（$\text{Fe}^{3+}$、$\text{S}$ 中 $+6$ 价）只有氧化性，最低价态（$\text{S}^{2-}$、$\text{Cl}^-$）只有还原性。

**强弱比较** 的常用依据：

- 由方程式判断——氧化性：氧化剂 $>$ 氧化产物；还原性：还原剂 $>$ 还原产物（「两强制两弱」）；
- 金属活动性顺序里，金属越活泼还原性越强，对应阳离子氧化性越弱；
- 常见顺序：氧化性 $\text{MnO}_4^->\text{Cl}_2>\text{Br}_2>\text{Fe}^{3+}>\text{I}_2>\text{SO}_2$，还原性反之。据此 $\text{Cl}_2$ 通入 $\text{FeI}_2$ 溶液先氧化 $\text{I}^-$ 再氧化 $\text{Fe}^{2+}$。

#### 电子转移的表示

- **双线桥法**：两条桥都从反应物指向生成物中的 **同种元素**，分别标「失去 / 得到 $n\times m$ 个电子」，得失电子总数必须相等；
- **单线桥法**：一条桥从 **还原剂失电子的元素** 指向 **氧化剂得电子的元素**，箭头上只标电子转移总数，不写「得」「失」。

配平的核心是 **化合价升降守恒**，即得电子总数等于失电子总数，配平步骤：标价态 $\to$ 列升降 $\to$ 求最小公倍数定系数 $\to$ 用原子守恒、电荷守恒配平其余物质。

**十字交叉法** 用来快速求变价元素的系数：把氧化剂化合价降低的总数、还原剂升高的总数交叉过来，作对方的系数。例如 $\text{Na}_2\text{O}_2$ 与 $\text{H}_2\text{O}$ 的反应中氧由 $-1$ 歧化为 $-2$ 和 $0$，升 $1$ 降 $1$，故每 $2\ \text{mol}\ \text{Na}_2\text{O}_2$ 转移 $2\ \text{mol}$ 电子。该法同样用于按平均值反推混合物的组成比例（如求同位素丰度、混合气体各组分比例）。

## 第二章 海水中的重要元素——钠和氯

### 第一节 钠及其化合物

#### 钠单质

- **物理性质**：银白色、质软（可用小刀切）、密度小（比水小、比煤油大）、熔点低（约 $98\ ^\circ\text{C}$）、导电导热；
- 保存在 **煤油** 中，隔绝空气和水；取用用镊子，不可用手接触。

**化学性质**（钠是强还原剂，很活泼）

- 与 $\text{O}_2$ 反应，条件不同产物不同：

$$4\text{Na}+\text{O}_2\to 2\text{Na}_2\text{O}\quad(\text{常温，白色})$$

$$2\text{Na}+\text{O}_2\xrightarrow{\Delta}\text{Na}_2\text{O}_2\quad(\text{点燃 / 加热，淡黄色})$$

- 与水反应，**浮、熔、游、响、红**（浮在水面、熔成小球、四处游动、发出嘶响、酚酞变红），分别对应密度比水小、熔点低且放热、生成气体、反应剧烈、生成 $\text{NaOH}$：

$$2\text{Na}+2\text{H}_2\text{O}\to 2\text{NaOH}+\text{H}_2\uparrow$$

$$2\text{Na}+2\text{H}_2\text{O}\to 2\text{Na}^++2\text{OH}^-+\text{H}_2\uparrow$$

- 与酸溶液反应比与水更剧烈，钠直接与 $\text{H}^+$ 反应：$2\text{Na}+2\text{H}^+\to 2\text{Na}^++\text{H}_2\uparrow$；
- 与盐溶液反应，**先与水反应** 生成 $\text{NaOH}$，再由 $\text{NaOH}$ 与盐复分解，**不能** 置换出盐中的金属。投入 $\text{CuSO}_4$ 溶液出现蓝色沉淀而非红色铜：

$$2\text{Na}+2\text{H}_2\text{O}+\text{CuSO}_4\to\text{Cu}(\text{OH})_2\downarrow+\text{Na}_2\text{SO}_4+\text{H}_2\uparrow$$

钠着火不能用水或泡沫灭火器（会与水反应放 $\text{H}_2$、在 $\text{CO}_2$ 中也能燃烧），要用干燥沙土盖灭。

#### 氧化钠与过氧化钠

|            |                  $\text{Na}_2\text{O}$                   |                             $\text{Na}_2\text{O}_2$                             |
| :--------: | :------------------------------------------------------: | :-----------------------------------------------------------------------------: |
|    颜色    |                         白色固体                         |                                   淡黄色固体                                    |
| 氧的化合价 |                           $-2$                           |                                      $-1$                                       |
|    与水    | $\text{Na}_2\text{O}+\text{H}_2\text{O}\to 2\text{NaOH}$ | $2\text{Na}_2\text{O}_2+2\text{H}_2\text{O}\to 4\text{NaOH}+\text{O}_2\uparrow$ |
|    类别    |                        碱性氧化物                        |                            过氧化物（非碱性氧化物）                             |

$\text{Na}_2\text{O}_2$ 中含离子键和 $\text{O}\text{—}\text{O}$ 非极性共价键，阴阳离子（$\text{O}_2^{2-}:\text{Na}^+$）个数比为 $1:2$。它与 $\text{CO}_2$ 反应可用于呼吸面具和潜水艇供氧：

$$2\text{Na}_2\text{O}_2+2\text{CO}_2\to 2\text{Na}_2\text{CO}_3+\text{O}_2$$

该反应中 $\text{Na}_2\text{O}_2$ 既是氧化剂又是还原剂（氧由 $-1$ 歧化为 $-2$ 和 $0$）。$\text{Na}_2\text{O}_2$ 强氧化性还能把 $\text{Fe}^{2+}$ 氧化成 $\text{Fe}^{3+}$、使滴入水中的酚酞 **先变红后褪色**（中间产物 $\text{H}_2\text{O}_2$ 漂白）。

三条定量规律很好用：

- **气体减半**：$\text{CO}_2$ 或 $\text{H}_2\text{O}(\text{g})$ 通过足量 $\text{Na}_2\text{O}_2$，反应气体与生成 $\text{O}_2$ 的物质的量之比恒为 $2:1$，气体体积减小一半；
- **电子转移**：每生成 $1\ \text{mol}\ \text{O}_2$ 转移 $2\ \text{mol}$ 电子（$1\ \text{mol}\ \text{Na}_2\text{O}_2$ 转移 $1\ \text{mol}$ 电子）；
- **固体增重**：$\text{Na}_2\text{O}_2$ 吸收 $\text{CO}_2$ 增重相当于等物质的量的 $\text{CO}$，吸收 $\text{H}_2\text{O}$ 增重相当于等物质的量的 $\text{H}_2$。凡组成符合 $(\text{CO})_m(\text{H}_2)_n$ 的物质（如 $\text{HCHO}$、$\text{CH}_3\text{COOH}$、葡萄糖）完全燃烧的产物通过足量 $\text{Na}_2\text{O}_2$，固体增重等于该物质的原质量。

#### 碳酸钠与碳酸氢钠

|          | $\text{Na}_2\text{CO}_3$（纯碱、苏打） | $\text{NaHCO}_3$（小苏打） |
| :------: | :------------------------------------: | :------------------------: |
|   俗名   |               纯碱、苏打               |           小苏打           |
|  溶解性  |        易溶（同温下溶解度更大）        |        易溶（较小）        |
| 热稳定性 |                  稳定                  |          受热分解          |
|   与酸   |                  较慢                  |          剧烈、快          |
|   用途   |            玻璃、造纸、洗涤            |     发酵、灭火、制胃药     |

$\text{NaHCO}_3$ 受热分解：

$$2\text{NaHCO}_3\xrightarrow{\Delta}\text{Na}_2\text{CO}_3+\text{H}_2\text{O}+\text{CO}_2\uparrow$$

**鉴别与除杂**：

- 加热法——只有 $\text{NaHCO}_3$ 分解放出使石灰水变浑的 $\text{CO}_2$。「套管实验」把 $\text{NaHCO}_3$ 放内管、$\text{Na}_2\text{CO}_3$ 放外管，温度较低的内管先分解，更有说服力；
- 滴加相同浓度盐酸，$\text{NaHCO}_3$ 产气更快；加 $\text{BaCl}_2$（或 $\text{CaCl}_2$）只有 $\text{Na}_2\text{CO}_3$ 产生白色沉淀（不能用 $\text{Ba}(\text{OH})_2$、$\text{Ca}(\text{OH})_2$，两者都产生沉淀）；
- 除杂：$\text{Na}_2\text{CO}_3$ 固体中混 $\text{NaHCO}_3$ 直接加热即可；$\text{Na}_2\text{CO}_3$ 溶液中混 $\text{NaHCO}_3$ 加适量 $\text{NaOH}$，**不能** 用加热（会带出 $\text{Na}_2\text{CO}_3$ 且溶液中 $\text{NaHCO}_3$ 分解不彻底）。

二者可相互转化，通入足量 $\text{CO}_2$ 或加 $\text{NaOH}$：

$$\text{Na}_2\text{CO}_3+\text{CO}_2+\text{H}_2\text{O}\to 2\text{NaHCO}_3$$

$$\text{NaHCO}_3+\text{NaOH}\to\text{Na}_2\text{CO}_3+\text{H}_2\text{O}$$

**侯氏制碱法**（联合制碱法）由侯德榜创立，向饱和食盐水中先通 $\text{NH}_3$ 再通 $\text{CO}_2$，析出溶解度较小的 $\text{NaHCO}_3$，灼烧得纯碱，母液冷却盐析出副产品氯化铵（氮肥）：

$$\text{NaCl}+\text{NH}_3+\text{CO}_2+\text{H}_2\text{O}\to\text{NaHCO}_3\downarrow+\text{NH}_4\text{Cl}$$

先通溶解度大的 $\text{NH}_3$ 使溶液显碱性，才能吸收更多 $\text{CO}_2$。

#### 焰色试验

- 焰色试验是 **物理变化**，属元素的性质，是原子核外电子跃迁放出特定波长的光；
- 用铂丝（或洁净铁丝）蘸取样品，先用 **稀盐酸** 洗净灼烧至无色（不用稀硫酸，其盐熔沸点高、难挥发）；
- **钠** 焰色为 **黄色**；**钾** 焰色为 **紫色**，需 **透过蓝色钴玻璃** 观察（滤去钠的黄光）。

### 第二节 氯及其化合物

#### 氯气

- **物理性质**：黄绿色、有刺激性气味、有毒、密度比空气大、能溶于水（$1$ 体积水溶约 $2$ 体积氯气）；
- **氯气是强氧化剂**，化学性质活泼。

**与金属反应**（把金属氧化到高价）：

$$2\text{Na}+\text{Cl}_2\xrightarrow{\text{点燃}}2\text{NaCl}$$

$$2\text{Fe}+3\text{Cl}_2\xrightarrow{\text{点燃}}2\text{FeCl}_3$$

**与非金属反应**（$\text{H}_2$ 在 $\text{Cl}_2$ 中安静燃烧，苍白色火焰）：

$$\text{H}_2+\text{Cl}_2\xrightarrow{\text{点燃}}2\text{HCl}$$

**与水反应**（部分反应，是可逆反应）：

$$\text{Cl}_2+\text{H}_2\text{O}\rightleftharpoons\text{HCl}+\text{HClO}$$

**次氯酸**（$\text{HClO}$）是弱酸（比碳酸还弱）、强氧化性，有漂白和杀菌作用，见光分解：

$$2\text{HClO}\xrightarrow{\text{光照}}2\text{HCl}+\text{O}_2\uparrow$$

新制氯水成分复杂，含 $\text{Cl}_2$、$\text{H}_2\text{O}$、$\text{HClO}$ 三分子和 $\text{H}^+$、$\text{Cl}^-$、$\text{ClO}^-$、$\text{OH}^-$ 四离子。久置后 $\text{HClO}$ 分解、$\text{Cl}_2$ 耗尽，最终变为稀盐酸。据此氯水既显酸性（$\text{H}^+$）、又有强氧化漂白性（$\text{HClO}$）、还能与 $\text{AgNO}_3$ 生成沉淀（$\text{Cl}^-$）。

**与碱反应**（工业制漂白粉，有效成分 $\text{Ca}(\text{ClO})_2$）：

$$\text{Cl}_2+2\text{NaOH}\to\text{NaCl}+\text{NaClO}+\text{H}_2\text{O}$$

$$2\text{Cl}_2+2\text{Ca}(\text{OH})_2\to\text{Ca}(\text{ClO})_2+\text{CaCl}_2+2\text{H}_2\text{O}$$

**漂白原理对比**：$\text{HClO}$、$\text{Na}_2\text{O}_2$、$\text{O}_3$ 靠强氧化性漂白，把有色物质氧化为无色，褪色 **不可逆**；$\text{SO}_2$ 靠与有色物质结合成不稳定无色物，加热又复原，是 **可逆** 漂白；活性炭靠 **吸附** 脱色，是物理过程。

#### 氯离子的检验

取待测液，先滴加稀硝酸酸化，再加 $\text{AgNO}_3$ 溶液，生成 **不溶于稀硝酸的白色沉淀**，即含 $\text{Cl}^-$：

$$\text{Ag}^++\text{Cl}^-\to\text{AgCl}\downarrow$$

加稀硝酸是为了排除 $\text{CO}_3^{2-}$ 等离子的干扰（$\text{Ag}_2\text{CO}_3$ 溶于硝酸而 $\text{AgCl}$ 不溶）。$\text{Ag}^+$ 与 $\text{Cl}^-$、$\text{Br}^-$、$\text{I}^-$ 分别生成白、淡黄、黄色沉淀，均不溶于稀硝酸，可用于区分卤离子。

### 第三节 物质的量

#### 物质的量及其单位

- **物质的量**（$n$）是表示 **含有一定数目粒子集合体** 的物理量，单位是 **摩尔**（mol）；这四个字是一个整体的专用名词，不能拆改；
- **阿伏加德罗常数**（$N_\text{A}$）：$1\ \text{mol}$ 任何粒子含 $N_\text{A}$ 个粒子，$N_\text{A}\approx 6.02\times 10^{23}\ \text{mol}^{-1}$；
- 使用摩尔时必须指明粒子种类（分子、原子、离子、电子等），不能说「$1\ \text{mol}$ 氧」。

$$n=\frac{N}{N_\text{A}}$$

**$N_\text{A}$ 常见易错**：

- 「标准状况」才能用 $22.4\ \text{L/mol}$，且要确认标况下是气体——$\text{H}_2\text{O}$、$\text{SO}_3$、$\text{Br}_2$ 标况下不是气体；
- $\text{NaCl}$、$\text{Na}_2\text{O}_2$ 是离子化合物，无分子，问「分子数」是陷阱；$1\ \text{mol}\ \text{Na}_2\text{O}_2$ 含 $2\ \text{mol}\ \text{Na}^+$ 和 $1\ \text{mol}\ \text{O}_2^{2-}$；
- 未告知温度、压强时不能由体积算物质的量。

#### 摩尔质量

- **摩尔质量**（$M$）：单位物质的量的物质所具有的质量，单位 $\text{g/mol}$；
- 以 $\text{g/mol}$ 为单位时，摩尔质量在 **数值上等于该粒子的相对原子（分子）质量**。摩尔质量有单位，相对原子质量无单位，二者不能混为一谈。

$$n=\frac{m}{M}$$

#### 气体摩尔体积

- **气体摩尔体积**（$V_\text{m}$）：单位物质的量的气体所占的体积，单位 $\text{L/mol}$；
- **标准状况**（$0\ ^\circ\text{C}$、$101\ \text{kPa}$）下，$V_\text{m}\approx 22.4\ \text{L/mol}$；
- 该值只适用于气体，且必须是标准状况；非标况气体、任何状况的固液体都不能套用。

$$n=\frac{V}{V_\text{m}}$$

**阿伏加德罗定律**：同温同压下，相同体积的任何气体含有相同数目的分子。由此推出：

- 同温同压下，气体的 **体积比 $=$ 物质的量比 $=$ 分子数比**；
- 同温同压下，气体的 **密度比 $=$ 摩尔质量比**（$\rho=\frac{m}{V}=\frac{nM}{V}$，同温同压时 $\frac{n}{V}$ 相同）。

#### 物质的量浓度

- **物质的量浓度**（$c$）：单位体积溶液中所含溶质的物质的量，单位 $\text{mol/L}$；

$$c=\frac{n}{V}$$

- **配制一定物质的量浓度溶液** 用 **容量瓶**，步骤：计算 $\to$ 称量（或量取）$\to$ 溶解（冷却）$\to$ 转移 $\to$ 洗涤 $\to$ 定容 $\to$ 摇匀；
- 定容时要平视刻度线，滴加至凹液面最低处与刻度相切。

**误差分析** 都归到 $c=\frac{n}{V}$：溶质 $n$ 偏少或体积 $V$ 偏大都使 $c$ 偏低。常考几例：

- 定容时 **俯视** 刻度，视线下压使实际加水偏少、$V$ 偏小，$c$ 偏 **高**；**仰视** 则 $c$ 偏 **低**；
- 未冷却就转移，热溶液体积偏大，冷却后液面下降、定容补水使 $V$ 偏大，$c$ 偏 **低**；
- 转移后 **未洗涤** 烧杯和玻璃棒，$n$ 偏少，$c$ 偏 **低**；
- 称量时药品与砝码放反（用游码）会使 $n$ 偏小，$c$ 偏 **低**。

各量互相换算的桥梁都是物质的量 $n$：

$$n=\frac{m}{M}=\frac{V(\text{气})}{V_\text{m}}=\frac{N}{N_\text{A}}=c\cdot V(\text{液})$$

## 第三章 铁 金属材料

### 第一节 铁及其化合物

#### 铁单质

- 铁是较活泼金属，通常显 $+2$、$+3$ 价；纯铁抗腐蚀，含杂质易生锈；
- 与非金属：$3\text{Fe}+2\text{O}_2\xrightarrow{\text{点燃}}\text{Fe}_3\text{O}_4$（黑色），$2\text{Fe}+3\text{Cl}_2\xrightarrow{\text{点燃}}2\text{FeCl}_3$（生成 $+3$ 价）；
- 与水蒸气（高温）：

$$3\text{Fe}+4\text{H}_2\text{O}\,(\text{g})\xrightarrow{\text{高温}}\text{Fe}_3\text{O}_4+4\text{H}_2$$

- 与非氧化性酸（盐酸、稀硫酸）生成 **$+2$ 价**：$\text{Fe}+2\text{H}^+\to\text{Fe}^{2+}+\text{H}_2\uparrow$；
- 与盐溶液（置换较不活泼金属）：$\text{Fe}+\text{Cu}^{2+}\to\text{Fe}^{2+}+\text{Cu}$。

价态口诀「遇强变三、遇弱变二」：与足量 $\text{Cl}_2$、稀 $\text{HNO}_3$ 等强氧化剂生成 $\text{Fe}^{3+}$，与盐酸、稀硫酸、$\text{S}$、$\text{Cu}^{2+}$ 等生成 $\text{Fe}^{2+}$。铁与稀硝酸的产物还跟用量有关：

$$\text{Fe}+4\text{H}^++\text{NO}_3^-\to\text{Fe}^{3+}+\text{NO}\uparrow+2\text{H}_2\text{O}\quad(\text{硝酸足量})$$

$$3\text{Fe}+8\text{H}^++2\text{NO}_3^-\to 3\text{Fe}^{2+}+2\text{NO}\uparrow+4\text{H}_2\text{O}\quad(\text{铁足量})$$

常温下铁、铝遇 **冷的浓硫酸或浓硝酸** 会 **钝化**（表面生成致密氧化膜），故可用铁、铝容器盛装冷浓硫酸 / 浓硝酸。

#### 铁的氧化物与氢氧化物

|            | $\text{FeO}$ | $\text{Fe}_2\text{O}_3$ | $\text{Fe}_3\text{O}_4$ |
| :--------: | :----------: | :---------------------: | :---------------------: |
| 铁的化合价 |     $+2$     |          $+3$           |       $+2$、$+3$        |
|    颜色    |     黑色     |         红棕色          |          黑色           |
|    俗名    |      —       |          铁红           |       磁性氧化铁        |

**氢氧化亚铁与氢氧化铁**

- $\text{Fe}(\text{OH})_2$ 是 **白色** 沉淀，在空气中迅速变灰绿，最终变 **红褐色**：

$$4\text{Fe}(\text{OH})_2+\text{O}_2+2\text{H}_2\text{O}\to 4\text{Fe}(\text{OH})_3$$

- $\text{Fe}(\text{OH})_3$ 是 **红褐色** 沉淀，受热分解：$2\text{Fe}(\text{OH})_3\xrightarrow{\Delta}\text{Fe}_2\text{O}_3+3\text{H}_2\text{O}$；
- 制备 $\text{Fe}(\text{OH})_2$ 要 **隔绝空气**：先煮沸溶液赶尽溶解氧，把吸取 $\text{NaOH}$ 的滴管 **伸入 $\text{FeSO}_4$ 液面下** 再挤出，必要时在液面覆盖一层煤油，否则很快被氧化。

#### 铁盐与亚铁盐的相互转化

- $\text{Fe}^{2+}$ 遇氧化剂（$\text{Cl}_2$、$\text{HNO}_3$、$\text{H}_2\text{O}_2$、酸性 $\text{KMnO}_4$ 等）被氧化为 $\text{Fe}^{3+}$：$2\text{Fe}^{2+}+\text{Cl}_2\to 2\text{Fe}^{3+}+2\text{Cl}^-$；
- $\text{Fe}^{3+}$ 遇还原剂（$\text{Fe}$、$\text{Cu}$ 等）被还原为 $\text{Fe}^{2+}$：$2\text{Fe}^{3+}+\text{Fe}\to 3\text{Fe}^{2+}$、$2\text{Fe}^{3+}+\text{Cu}\to 2\text{Fe}^{2+}+\text{Cu}^{2+}$（工业用 $\text{FeCl}_3$ 腐蚀铜制电路板）。

保存 $\text{Fe}^{2+}$ 溶液常加少量 **铁粉** 防氧化、加少量对应酸抑制水解。

|                  |      $\text{Fe}^{2+}$      | $\text{Fe}^{3+}$ |
| :--------------: | :------------------------: | :--------------: |
|     溶液颜色     |           浅绿色           |       黄色       |
| 加 $\text{NaOH}$ | 白色 $\to$ 灰绿 $\to$ 红褐 |    红褐色沉淀    |
| 加 $\text{KSCN}$ |           不变色           |    **变红色**    |

**离子检验**：

- 检验 $\text{Fe}^{3+}$：滴加 $\text{KSCN}$ 溶液变 **血红色**（$\text{Fe}^{3+}+3\text{SCN}^-\to\text{Fe}(\text{SCN})_3$），最灵敏；
- 检验 $\text{Fe}^{2+}$：先滴 $\text{KSCN}$ **不变色**，再滴氯水（或双氧水）**变红**，证明原溶液是 $\text{Fe}^{2+}$（被氧化成 $\text{Fe}^{3+}$）。顺序不能反，否则无法排除原溶液已含 $\text{Fe}^{3+}$。

#### 铁三角

$\text{Fe}$、$\text{Fe}^{2+}$、$\text{Fe}^{3+}$ 三者可相互转化，抓住 **化合价升降** 就能判断该加氧化剂还是还原剂：

- $\text{Fe}\to\text{Fe}^{2+}$：与非氧化性酸、$\text{Cu}^{2+}$ 或适量 $\text{Fe}^{3+}$ 反应；
- $\text{Fe}\to\text{Fe}^{3+}$：与足量 $\text{Cl}_2$、足量稀 $\text{HNO}_3$ 反应；
- $\text{Fe}^{2+}\to\text{Fe}^{3+}$：加氧化剂（$\text{Cl}_2$、$\text{H}_2\text{O}_2$……）；
- $\text{Fe}^{3+}\to\text{Fe}^{2+}$：加还原剂（$\text{Fe}$、$\text{Cu}$）；
- $\text{Fe}^{3+}\to\text{Fe}$、$\text{Fe}^{2+}\to\text{Fe}$：更强还原剂或电解。

其中 $\text{Fe}$ 与 $\text{Fe}^{3+}$ 归中生成 $\text{Fe}^{2+}$（$\text{Fe}+2\text{Fe}^{3+}\to 3\text{Fe}^{2+}$）是最典型的一步。

### 第二节 金属材料

#### 铝及其化合物

铝是地壳中含量最高的金属元素，性质活泼但表面易生成致密 $\text{Al}_2\text{O}_3$ 膜而耐腐蚀。铝、$\text{Al}_2\text{O}_3$、$\text{Al}(\text{OH})_3$ 都 **既能与强酸反应、又能与强碱反应**，是「两性」的典型。

- 铝与酸、碱都放出 $\text{H}_2$：

$$2\text{Al}+6\text{H}^+\to 2\text{Al}^{3+}+3\text{H}_2\uparrow$$

$$2\text{Al}+2\text{OH}^-+2\text{H}_2\text{O}\to 2\text{AlO}_2^-+3\text{H}_2\uparrow$$

- 两性氧化物 $\text{Al}_2\text{O}_3$：$\text{Al}_2\text{O}_3+6\text{H}^+\to 2\text{Al}^{3+}+3\text{H}_2\text{O}$，$\text{Al}_2\text{O}_3+2\text{OH}^-\to 2\text{AlO}_2^-+\text{H}_2\text{O}$；
- 两性氢氧化物 $\text{Al}(\text{OH})_3$：$\text{Al}(\text{OH})_3+3\text{H}^+\to\text{Al}^{3+}+3\text{H}_2\text{O}$，$\text{Al}(\text{OH})_3+\text{OH}^-\to\text{AlO}_2^-+2\text{H}_2\text{O}$。$\text{Al}(\text{OH})_3$ 只溶于强酸强碱，**不溶于弱酸弱碱**，故实验室用可溶铝盐加 **氨水**（而非 $\text{NaOH}$，以免过量溶解）制取。

**铝三角** 是 $\text{Al}^{3+}$、$\text{Al}(\text{OH})_3$、$\text{AlO}_2^-$ 三者的相互转化，$\text{Al}(\text{OH})_3$ 居中，加酸得 $\text{Al}^{3+}$、加碱得 $\text{AlO}_2^-$：

$$\text{Al}^{3+}+3\text{OH}^-\to\text{Al}(\text{OH})_3\downarrow,\quad\text{Al}(\text{OH})_3+\text{OH}^-\to\text{AlO}_2^-+2\text{H}_2\text{O}$$

$$\text{AlO}_2^-+\text{H}^++\text{H}_2\text{O}\to\text{Al}(\text{OH})_3\downarrow,\quad\text{Al}(\text{OH})_3+3\text{H}^+\to\text{Al}^{3+}+3\text{H}_2\text{O}$$

**滴加顺序影响现象**（易错）：

- 向 $\text{AlCl}_3$ 中 **滴入 $\text{NaOH}$**：先见白色沉淀，$\text{NaOH}$ 过量后沉淀溶解；
- 向 $\text{NaOH}$ 中 **滴入 $\text{AlCl}_3$**：开始无沉淀（$\text{Al}^{3+}$ 立即被过量碱溶为 $\text{AlO}_2^-$），后期才出现沉淀且不再溶解。

明矾 $\text{KAl}(\text{SO}_4)_2\cdot 12\text{H}_2\text{O}$ 溶于水，$\text{Al}^{3+}$ 水解生成 $\text{Al}(\text{OH})_3$ 胶体，吸附水中悬浮杂质而 **净水**。

#### 合金

- **合金**：由两种或两种以上的金属（或金属与非金属）熔合而成的、具有金属特性的物质；
- 合金一般 **硬度比成分金属大、熔点比成分金属低**（原子大小不同扰乱了规则排列），性能更优；
- 常见合金：生铁与钢（铁碳合金，靠含碳量区分）、铝合金、铜合金、储氢合金、钛合金。

生铁含碳量 $2\% - 4.3\%$，硬而脆；钢含碳量 $0.03\% - 2\%$，硬而韧。不锈钢因含铬、镍而抗腐蚀。

#### 正确使用金属材料

- 金属的活泼性与用途、抗腐蚀性相关；铝表面易形成致密氧化膜（$\text{Al}_2\text{O}_3$）而耐腐蚀；
- 金属回收利用可节约资源、减少污染；
- 物质的性质决定用途，选材要综合考虑性能、成本与环境。

## 第四章 物质结构 元素周期律

### 第一节 原子结构与元素周期表

#### 原子结构

- 原子由 **原子核**（质子、中子）和 **核外电子** 构成；

$$\text{质子数}(Z)=\text{核电荷数}=\text{核外电子数}=\text{原子序数}$$

$$\text{质量数}(A)=\text{质子数}(Z)+\text{中子数}(N)$$

- 表示为 $^{A}_{Z}\text{X}$，如 $^{12}_{6}\text{C}$；
- **元素**：质子数相同的一类原子的总称；**核素**：具有一定质子数和中子数的一种原子；**同位素**：质子数相同、中子数不同的同一元素的不同核素（如 $^{1}_{1}\text{H}$、$^{2}_{1}\text{H}$、$^{3}_{1}\text{H}$）。同位素之间物理性质不同、化学性质几乎相同，天然存在的同位素丰度基本不变。

#### 核外电子排布

- 电子分层排布，由内向外依次为 $\text{K}$、$\text{L}$、$\text{M}$、$\text{N}$……（$1$、$2$、$3$、$4$……层）；
- 排布规律（几条同时满足）：
  - 每层最多容纳 $2n^2$ 个电子（$n$ 为层序数）；
  - 最外层不超过 $8$ 个（$\text{K}$ 层为最外层时不超过 $2$ 个）；
  - 次外层不超过 $18$ 个，倒数第三层不超过 $32$ 个；
  - 电子总是尽先排布在能量最低的电子层。

**结构示意图** 以圆圈中标核电荷数、弧线上标各层电子数表示，如钠为 $2$、$8$、$1$。

#### 元素周期表

- 编排原则：按 **原子序数递增** 从左到右排列；同一 **横行（周期）** 电子层数相同；同一 **纵列（族）** 最外层电子数相同（多数）；
- 结构：**7 个周期、18 个纵列、16 个族**；

|  周期  |   一   |   二   |   三   |   四   |   五   |   六   |   七   |
| :----: | :----: | :----: | :----: | :----: | :----: | :----: | :----: |
|  类别  | 短周期 | 短周期 | 短周期 | 长周期 | 长周期 | 长周期 | 长周期 |
| 元素数 |  $2$   |  $8$   |  $8$   |  $18$  |  $18$  |  $32$  |  $32$  |

- 族分 **7 个主族（$\text{I A} - \text{VII A}$）、7 个副族（$\text{I B} - \text{VII B}$）、1 个第 $\text{VIII}$ 族、1 个 $0$ 族**；
- **主族序数 $=$ 最外层电子数 $=$ 最高正化合价**（$\text{F}$、$\text{O}$ 除外）；非金属的 **最高正价 $+$ 负价绝对值 $=8$**。

**碱金属**（$\text{Li}$、$\text{Na}$、$\text{K}$、$\text{Rb}$、$\text{Cs}$）：最外层都是 $1$ 个电子，从上到下电子层数增多、原子半径增大、金属性 **增强**，与水、氧气反应越来越剧烈（与 $\text{O}_2$ 的产物由 $\text{Li}_2\text{O}$ 到 $\text{Na}_2\text{O}_2$ 再到 $\text{KO}_2$ 越来越复杂）。密度总体增大，但 **钾反常**，比钠小。焰色：锂紫红、钠黄、钾紫。

**卤素**（$\text{F}$、$\text{Cl}$、$\text{Br}$、$\text{I}$）：最外层都是 $7$ 个电子，从上到下非金属性 **减弱**；单质氧化性 $\text{F}_2>\text{Cl}_2>\text{Br}_2>\text{I}_2$，活泼卤素可把不活泼卤素从其盐中置换出来，如 $\text{Cl}_2+2\text{NaBr}\to 2\text{NaCl}+\text{Br}_2$。

### 第二节 元素周期律

#### 元素周期律

- **元素周期律**：元素的性质随 **原子序数递增** 而 **呈周期性变化**；本质是核外电子排布的周期性变化。

同周期（从左到右，电子层数相同）：

- 原子半径 **逐渐减小**（稀有气体除外）；
- 失电子能力（金属性）**减弱**，得电子能力（非金属性）**增强**；
- 最高正价从 $+1$ 到 $+7$，负价从 $-4$ 到 $-1$。

同主族（从上到下，最外层电子数相同）：

- 原子半径 **逐渐增大**；
- 金属性 **增强**，非金属性 **减弱**。

#### 金属性与非金属性的判断

**金属性强弱** 依据：

- 单质与水（或酸）反应置换出 $\text{H}_2$ 的难易（越易越强）；
- 最高价氧化物对应水化物（碱）的碱性强弱（越强越强）；
- 相互置换：活泼金属能把较不活泼金属从其盐溶液中置换出来。

**非金属性强弱** 依据：

- 单质与 $\text{H}_2$ 化合的难易及气态氢化物的稳定性（越稳定越强）；
- 最高价氧化物对应水化物（酸）的酸性强弱（越强越强）；
- 相互置换：活泼非金属能把较不活泼非金属从其盐（或氢化物）中置换出来。

例如碱性 $\text{NaOH}>\text{Mg}(\text{OH})_2>\text{Al}(\text{OH})_3$，说明金属性 $\text{Na}>\text{Mg}>\text{Al}$；酸性 $\text{HClO}_4>\text{H}_2\text{SO}_4>\text{H}_3\text{PO}_4$，说明非金属性 $\text{Cl}>\text{S}>\text{P}$。注意判据要用 **最高价** 含氧酸，$\text{HClO}$、$\text{HF}$（无含氧酸最高价）不能作依据。

#### 微粒半径比较

- **电子层数** 越多，半径一般越大；
- 电子层数相同时，**核电荷数** 越大，半径越小；
- 电子层结构相同的离子，核电荷数越大半径越小（如 $r(\text{O}^{2-})>r(\text{F}^-)>r(\text{Na}^+)>r(\text{Mg}^{2+})>r(\text{Al}^{3+})$）；
- 同种元素：阳离子半径小于原子，阴离子半径大于原子（$r(\text{Na}^+)<r(\text{Na})$，$r(\text{Cl}^-)>r(\text{Cl})$）；同种元素价态越高、半径越小（$r(\text{Fe}^{3+})<r(\text{Fe}^{2+})<r(\text{Fe})$）。

注意 $r(\text{Al}^{3+})<r(\text{Na}^+)$：虽然铝在钠右边，但两者电子层结构相同（都是 $2$、$8$），$\text{Al}^{3+}$ 核电荷数更大，半径更小。

#### 位、构、性关系

结构决定性质、位置反映结构：由 **原子结构** 可推 **周期表位置**（电子层数定周期、最外层电子数定主族），由位置可推 **元素性质** 的递变规律。三者相互推断是元素推断题的核心。

### 第三节 化学键

#### 离子键

- **离子键**：带相反电荷离子之间的 **静电作用**；
- 成因：活泼金属（$\text{Na}$、$\text{K}$、$\text{Ca}$）与活泼非金属（$\text{Cl}$、$\text{O}$）之间通过 **得失电子** 形成阴、阳离子；
- **离子化合物**：含离子键的化合物（强碱、大多数盐、活泼金属氧化物、铵盐）；
- 用 **电子式** 表示，如 $\text{NaCl}$ 中 $\text{Na}^+$ 与带方括号和电荷的 $\text{Cl}^-$。

#### 共价键

- **共价键**：原子间通过 **共用电子对** 形成的相互作用；
- 分 **非极性键**（同种原子，电子对不偏移，如 $\text{H}_2$、$\text{Cl}_2$）与 **极性键**（不同原子，电子对偏移，如 $\text{HCl}$）；
- **共价化合物**：只含共价键的化合物（非金属氧化物、酸、多数有机物、$\text{H}_2\text{O}$、$\text{NH}_3$）。

|          |        离子键        |         共价键         |
| :------: | :------------------: | :--------------------: |
|   本质   | 阴阳离子间的静电作用 |    原子间共用电子对    |
| 成键微粒 |    阴离子、阳离子    |          原子          |
| 成键元素 | 活泼金属与活泼非金属 |       非金属之间       |
|   存在   |      离子化合物      | 共价化合物、非金属单质 |

#### 化学键与物质类别

- **化学键**：相邻原子（或离子）之间强烈的相互作用，包括离子键和共价键；
- 化学反应的本质：**旧化学键断裂、新化学键形成**；
- 判断要点：
  - 只含共价键的化合物是共价化合物（如 $\text{HCl}$、$\text{CO}_2$）；
  - 含离子键的化合物一定是离子化合物（可能同时含共价键，如 $\text{NaOH}$ 含离子键和 $\text{O}\text{—}\text{H}$ 共价键，$\text{Na}_2\text{O}_2$ 含离子键和 $\text{O}\text{—}\text{O}$ 非极性共价键，$\text{NH}_4\text{Cl}$ 含离子键和共价键）；
  - 稀有气体（如 $\text{He}$、$\text{Ne}$）单原子分子 **不含化学键**；
  - 判物质类别看键的种类：只要含离子键就是离子化合物，不含离子键、只含共价键的化合物才是共价化合物。
