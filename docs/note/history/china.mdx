## 第一单元 从中华文明起源到秦汉统一多民族封建国家的建立与巩固

### 我国早期人类分布的基本特点

分布广泛、多元一体、中原核心。

### 旧石器时代

距今 $170$ 万年 $\sim 1$ 万年。

**文化遗存代表**

- 距今 $170$ 万年的**元谋人**（中国已知最早人类）；
- 距今 $70\sim 20$ 万年的**北京人**。

**生产生活**

- 打制石器；
- 从事渔猎和采集；
- 过着群居生活；
- 学会用火。

**社会组织**：晚期进入**母系氏族社会**。

### 三皇五帝时代

- **禅让制**；
- **炎帝、黄帝**是华夏始祖；
- **陶寺遗址**（国家初始形态）。

### 夏朝

约公元前 $2070$ 年 $\sim$ 公元前 $1600$ 年。

- **建立**：禹建立夏，是我国**最早的奴隶制国家**；
- **文化遗存**：**二里头遗址**（最能证明夏文化存在的遗址）；
- **政治制度**：
  - **世袭制**代替**禅让制**，「家天下」代替「公天下」；
  - 夏王是最高统治者；
  - 中央设专职官员（行政、军事、司法、宗教）；
  - 地方管理：聚族而居，夏王直接统治夏部落，其他部族实行间接统治。

### 商朝

公元前 $1600$ 年 $\sim$ 公元前 $1046$ 年。

- **政治制度**：**内外服制**（内服：商王直接控制的王畿之地；外服：商王间接控制的方国和部族）；
- **文化**：青铜器；**甲骨文**。

### 西周

公元前 $1046$ 年 $\sim$ 公元前 $771$ 年。

**西周建立与灭亡**

- 公元前 $1046$ 年，周武王伐纣，**牧野之战**获胜，建立西周；
- 公元前 $841$ 年，周厉王时期，引发「国人暴动」，周厉王出逃，召公、周公共同执政，史称「**共和行政**」；
- 公元前 $771$ 年，游牧民族犬戎攻破镐京，杀死周幽王，西周灭亡。

**分封制**

- **目的**：「封建亲戚，以藩屏周」，拱卫周王室的统治；
- **内容**：王畿之外的土地和人民；
- **对象**：王族（同姓宗亲）、功臣、古代帝王后代；
- **受封者义务**：镇守疆土、随从作战、交纳贡赋、朝觐述职；
- **受封者权利**：职位世袭、设置官员、建立武装、征派赋役；
- **特点**：层层分封，等级森严，形成「天子—诸侯—卿大夫—士」金字塔型的等级结构；
- **影响**：
  - 积极：加强了周天子对地方的统治，扩大了统治区域；
  - 消极：诸侯国有相当大的独立性，埋下分裂割据的隐患。

**宗法制**

- **含义**：以父系血缘亲疏与嫡庶来确定继承关系和名分的制度；
- **特点 / 核心**：**嫡长子继承制**；
- **影响**：
  - 对古代社会：保证了贵族在政治上的垄断和特权地位，有利于统治集团内部的稳定和团结；
  - 对现代社会：注重家庭建设，提倡尊老爱幼，有利于形成民族凝聚力；但过分强调尊卑等级和血缘亲疏，与现在倡导的民主平等相违背。

**分封制与宗法制的关系**：互为表里，宗法制是里，分封制是表。

**礼乐制**。

**土地制度**：**井田制**（奴隶主土地国有制）。

### 中国古代早期政治制度的特点

夏、商、西周（奴隶社会）：

- 以血缘关系为纽带，形成「家国一体」的政治结构；
- 神权与王权相结合；
- 最高执政集团尚未实现权力的高度集中；
- 职官系统实行**世卿世禄**。

### 春秋战国时期时代特征

- **政治**：王室衰微、宗法分封制遭到严重破坏；
- **经济**：铁犁牛耕的出现和推广；
- **思想**：百家争鸣；
- **民族关系**：华夏认同观念增强；
- **国家**：由分裂走向统一；
- **社会性质**：奴隶社会走向封建社会。

### 春秋战国时期农业发展的表现

- **生产工具**：铁犁牛耕的出现和推广；
- **土地制度**：井田制瓦解，封建土地私有制逐步确立；
- **水利工程**：各国兴建水利灌溉工程，以**都江堰**为典型代表。

### 商鞅变法

公元前 $356$ 年。

**内容**

| 领域 | 措施                                                             |
| :--: | :--------------------------------------------------------------- |
| 政治 | 普遍推行**县制**，县的主要官员由君主任免；什伍连坐，互相纠察告发 |
| 经济 | 「废井田，开阡陌」；重农抑商，奖励耕织；小家庭政策               |
| 军事 | 奖励军功，剥夺和限制贵族特权                                     |

**评价**

- 积极：顺应历史潮流，集列国变法之长；是战国时期持续时间最长、涉及面最广、改革最为彻底的一次变法；使秦国国富兵强，为秦统一中国奠定了基础；
- 消极：轻视教化，鼓吹轻罪重罚，压迫和剥削百姓。

### 孔子

代表没落奴隶主贵族。春秋后期儒家学派的创始人。

**思想主张**

- 核心是「**仁**」；
- 政治主张是**为政以德**；
- 克己复礼；
- 教育：有教无类、因材施教、兴办私学（打破贵族阶层垄断教育的局面）。

### 老子

代表没落奴隶主贵族。春秋后期道家学派的创始人。

**思想主张**

- 万物本原是「**道**」；
- 追求天人合一；
- **辩证法**：事物存在着互相依存、相互转化、对立统一的矛盾，认为物极必反，柔能克刚；
- 政治：主张顺其自然，**无为而治**，小国寡民。

### 百家争鸣

春秋战国时期。

**战国时期诸子百家的代表人物及其主张**

| 学派 |        代表（阶级）        | 主张                               |
| :--: | :------------------------: | :--------------------------------- |
| 儒家 |  **孟子**（新兴地主阶级）  | 「仁政」「民贵君轻」「性本善」     |
| 儒家 |  **荀子**（新兴地主阶级）  | 人性恶；隆礼重法，君舟民水         |
| 道家 | **庄子**（没落奴隶主贵族） | 崇尚逍遥自由                       |
| 墨家 |    **墨子**（小生产者）    | 提倡节俭、「兼爱」「非攻」「尚贤」 |
| 法家 |  **韩非**（新兴地主阶级）  | 以法治国，中央集权                 |

**影响**

- 中国历史上**第一次思想解放运动**；
- 为新兴的地主阶级登上历史舞台奠定了思想理论基础；
- 成为后世中华思想文化的源头活水，影响十分深远。

### 秦朝

公元前 $221$ 年 $\sim$ 公元前 $207$ 年。

**统一的条件**

- 长期战乱，人民渴望统一；
- 经济发展，要求打破政治分裂所带来的阻碍；
- 秦地理位置优越，物质基础雄厚；
- 数代秦王励精图治，广纳贤才，吏治较为清明；
- **商鞅变法**，奖励耕战，国家日益强盛（主要条件）；
- 秦尊奉法家思想。

**秦巩固中央集权的措施**

- **加强皇权**：
  - **皇帝制度**（特点：皇位世袭、皇帝独尊、皇权至上）；
  - **三公九卿制**：丞相（百官之首，帮助皇帝处理全国政务）；御史大夫（下达诏令，监察百官）；太尉（掌军事，虚设）。
- **加强中央集权**：
  - **郡县制**——特点：郡县长官由皇帝直接任免；
  - 作用：加强了中央集权，有助于巩固国家统一，标志着**官僚政治取代贵族政治**。
- **其他**：统一车轨、文字、货币和度量衡；修驰道、直道；颁行法律，编制户籍；迁徙六国贵族豪强；整顿社会风俗等。

**秦朝的灭亡**

- **根本原因——秦的暴政**：赋税徭役繁重；严刑峻法；焚书坑儒。
- 公元前 $207$ 年**陈胜、吴广**领导的秦末农民起义（中国历史上第一次农民起义）和楚汉战争。

### 西汉

公元前 $202$ 年 $\sim$ 公元 $9$ 年。

**汉初**

- **政治**：「**汉承秦制**」（君主专制中央集权制度、皇帝制度、三公九卿制）；地方：**郡国并行制**——削弱中央集权，引发「**七国之乱**」；
- **经济**：休养生息的政策；
- **思想**：尊奉黄老无为思想。

**汉武帝时期**

- **政治**：
  - 地方：「**推恩令**」（削弱了诸侯王势力，加强了中央集权）；设**刺史**完善地方监察体系；
  - 中央：设立**中朝**（加强皇权，削弱丞相权力）；
  - 选官制度：**察举制**，以品德（孝廉）为选官标准；
  - 任用酷吏。
- **经济**：
  - 铸币权收归中央；
  - 盐铁官营；
  - 均输平准，平抑物价；
  - 抑制工商业者，征收财产税。
- **思想**：尊崇儒术 / 接受**董仲舒**「罢黜百家，独尊儒术」的建议——影响：儒学独尊地位确立，成为我国封建社会的主流意识形态。
- **拓疆**：
  - 任用**卫青、霍去病**大败匈奴；
  - 设立河西四郡（酒泉、武威、张掖、敦煌）；
  - 「凿空之旅」，**张骞**出使西域，开辟**丝绸之路**；
  - 对东南沿海和西南少数民族地区的控制更加稳定；
  - 设置**西域都护府**，作为管理西域的军政机构。

### 新朝与东汉

**新朝**：公元 $9$ 年，外戚**王莽**建立新朝；公元 $23$ 年，绿林军推翻王莽政权。

**东汉**

- $25$ 年**刘秀**建立东汉，出现了「**光武中兴**」；
- 东汉后期：
  - 外戚宦官交替专权；
  - 「清议」和「**党锢之祸**」；
  - 豪强地主势力发展迅速，土地兼并严重，阶级矛盾日益尖锐；
  - 农民起义——$184$ 年，**黄巾起义**（动摇了东汉王朝的统治基础，军阀割据局面出现）。

| 文学 | 汉赋、乐府诗（具有现实主义特点）                                                   |

## 第二单元 三国两晋南北朝的民族交融与隋唐统一多民族封建国家的发展

### 政权变迁时间线

魏晋南北朝时期（$220\sim 589$ 年）政权分立：

|     阶段     |       时间       |
| :----------: | :--------------: |
|   三国鼎立   | $220\sim 280$ 年 |
|   西晋统一   | $266\sim 316$ 年 |
| 十六国与东晋 | $317\sim 420$ 年 |
|  南朝与北朝  | $420\sim 589$ 年 |

### 时代特征

- **政治**：政权分立。
- **经济**：江南开发。
- **民族**：民族交融。表现：五胡内迁；大部分由少数民族建立的十六国采用中原模式的国号、年号，学习汉族的典章制度；**北魏孝文帝改革**。

### 北魏孝文帝改革

**内容**：

- 迁都洛阳。
- 移风易俗：穿汉服、说汉语、改汉姓、定籍贯、结汉亲。

**影响**：

- 顺应了北方民族交融的历史趋势，大大缓解了民族矛盾。
- 促进了北魏的经济发展和社会繁荣，为以后北方统一南方以及隋唐盛世的出现打下了基础。

### 隋朝的兴亡

**建立与统一**：$581$ 年，**隋文帝杨坚**建立隋朝；$589$ 年，隋朝统一。

**隋朝的建设**：

- 广设仓库。
- 兴建洛阳城。
- 开通**大运河**（作用：巩固统一、促进南北经济交流以及运河沿岸城市发展）。

### 唐朝的灭亡

- $755\sim 763$ 年**安史之乱**导致唐朝由盛转衰。
- $875$ 年**黄巢起义**加速了唐朝灭亡。
- $907$ 年朱温灭唐，藩镇割据，**五代十国**分裂局面出现。

### 选官制度的演变

| 时期 |        制度         |             选官标准             |
| :--: | :-----------------: | :------------------------------: |
| 先秦 | 世卿世禄制 / 世官制 |               血缘               |
|  秦  |      军功爵制       |               军功               |
| 汉朝 |       察举制        |           品行 / 孝廉            |
| 曹魏 |     九品中正制      | 家世、门第（维护士族特权的工具） |
| 隋唐 |       科举制        |               才学               |

**科举制影响**：使出身社会中下层的读书人通过相对公平的考试参与政权，扩大了统治基础，提高了官员的文化素质，加强了中央集权。

### 中央行政制度的演变

| 朝代 |        制度        |
| :--: | :----------------: |
| 秦朝 |     三公九卿制     |
| 汉朝 |     中外朝制度     |
| 魏晋 |       三省制       |
| 隋唐 |     三省六部制     |
| 宋朝 |     二府三司制     |
| 元朝 | 中书省（三省合一） |
| 明朝 |  废宰相，权分六部  |
| 清朝 |      设军机处      |

**隋唐三省六部制**：

- 职权及流程：中书省—起草政令；门下省—封驳审议；尚书省—执行政令（下设六部）。
- 影响：①相权一分为三，削弱了相权，加强了皇权；②有利于提高行政效率，减少决策失误；③是中国政治制度的重大变革，对此后历朝产生了深远影响。

**宋朝二府三司制**：设中书门下作为最高行政机构，行使宰相职权；同时增设参知政事（副宰相）、枢密使、三司使分割宰相的行政权、军权和财权。

### 赋税制度的演变

|   时期   |   制度   |               要点               |
| :------: | :------: | :------------------------------: |
|   魏晋   |  租调制  | 按户征收粮和绢帛（基础：均田制） |
|   唐初   | 租庸调制 |             以庸代役             |
| 唐中后期 |  两税法  | 按人丁、资产纳户税，按田亩纳地税 |

**租庸调制作用**：以庸代役保证农时，减轻了对农民的人身控制，有利于农业的发展，增加了政府财政收入。

**两税法**：

- 背景：土地兼并严重，均田制遭到破坏，租庸调制无法维持，国家财政收入减少。
- 内容：每户按人丁和资产缴纳户税，按田亩缴纳地税，取消租庸调和一切杂税杂役；一年分夏秋两次纳税（纳税标准：资产和田亩）。
- 影响：①简化税收名目，扩大收税对象，保证国家财政收入；②改变了自战国以来以人丁为主的赋税制度；③减轻了政府对农民的人身控制。

## 第三单元 辽宋夏金多民族政权的并立与元朝的统一

### 北宋加强中央集权措施

**地方——加强对地方的控制**：

- 军事：地方精锐部队编入禁军，定期更换驻地。
- 行政权：文官担任地方长官知州，设通判负责监督。
- 财政：设转运司统管地方财政，保证各州赋税大部分上缴朝廷。
- 监察：设「四监司」。

**中央——分散机构权力，削弱相权（二府三司制）**：设中书门下作为最高行政机构，行使宰相职权；同时增设参知政事（副宰相）、枢密使、三司使分割宰相的行政权、军权和财权。

**抑制武将——崇文抑武方针**：

- 抑武：罢免宿将兵权，用文官担任枢密院长官。
- 崇文：大力提倡文治，扩大科举规模，抬高文官和士人地位。

### 北宋加强中央集权的特点与影响

**特点**：①强干弱枝，守内虚外；②分化事权，内外相制；③重文轻武。

**影响**：

- 有效预防了内部动乱，巩固了国家的统一和安定，强化了中央集权。
- 制度过于僵化，权力分割过细，影响了行政效率，助长了因循守旧的政治风气。
- 形成了**冗官、冗兵、冗费**的局面，为北宋埋下了**积贫积弱**的祸根。

### 王安石变法

**目的**：富国强兵，巩固统治。

**措施**：

- 富国：青苗法、募役法、农田水利法、方田均税法、市易法、均输法。
- 强兵：精减军队、保甲法、保马法、将兵法。

**影响**：

- 达到了富国目的，增加了大笔收入，但强兵效果并不明显。
- 一些措施在执行过程中加重了人民的负担。
- 统治集团内部分裂日益严重，北宋逐渐走向衰亡。

### 元朝

**建立和统一**：$1271$ 年，**忽必烈**定国号为大元，定都大都；$1279$ 年完成统一。

**巩固统一的措施**：

- 修筑驿道，设立驿站，分设急递铺。
- 实行**行省制度**：
  - 双重性质：既是地方行政机构，又是中央的派出机构。
  - 特点：以犬牙交错为划分原则；中央集权与地方分权相结合。
  - 影响：有利于加强中央集权，有利于多民族国家的巩固和统一，有利于促进民族交融；是中国古代地方行政制度的重大变革，是中国省制的开端。
- 吐蕃地区：由中央政府的**宣政院**进行管理。
- 西域：设北庭都元帅府、宣慰司等管理军政事务。
- 台湾：设**澎湖巡检司**。
- 民族关系：①民族交融，形成回族前身——回回；②**四等人制**，民族歧视和压迫政策。

### 宋元经济

**农业**：

- 耕作制度：一年两熟到三熟的稻麦复种制。
- 固定种植某种经济作物，对自然经济结构有一定突破。
- 棉花的种植和推广（元代黄道婆）。

**手工业**：

- 制瓷业：宋代出现五大名窑；元青花瓷和釉里红。
- 矿冶业：北宋普遍使用煤作燃料。
- 印刷业：北宋毕昇发明**活字印刷术**。

**商业**：

- 面向大众的基层市场蓬勃涌现。
- 城市兴盛，时空限制被打破，经济功能增强。
- 各民族间经济往来频繁，榷场贸易活跃。
- 北宋开始出现纸币（**交子**）。
- 海外贸易繁荣，外贸税收成为政府重要财源，出现广州、泉州、明州等港口城市。

### 经济重心南移

**过程**：开始于安史之乱，完成于两宋时期。

**原因**：

- 北方人民南迁，带来了先进的生产技术和丰富的劳动力。
- 南方社会环境相对安定。
- 南方自然条件优越。
- 统治者的重视和支持。

**影响**：文化重心的南移（南北分卷制度，江浙人才集中）。

### 辽宋夏金元时期社会的变化

- 门第观念淡化。
- 社会成员身份趋于平等。表现：①世袭奴婢变为雇佣；②农民签订契约，租种土地，租佃关系发展。
- 国家对社会的控制相对松弛。

### 儒学的复兴 / 程朱理学

**背景（儒学危机）**：①社会现实的需要；②儒学日益僵化；③道教、佛教的冲击。

**目的**：挖掘儒家经书的思想内涵，强调学术为现实服务；发挥儒学在强化社会伦理道德秩序、树立基本价值观方面的作用。

**代表人物**：北宋的程颢、程颐，南宋的**朱熹**（理学的集大成者）。

**主张**：

- 「理」是自然界和社会的根本原则。
- 「理」是社会伦理道德，即三纲五常。
- 「存天理，灭人欲」。
- 「格物致知」。

**影响**：从南宋后期起，程朱理学受到官方尊崇，成为官方哲学。

### 文学艺术

**宋词**：

- 兴起原因：商品经济发展，市民阶层壮大。
- 代表：婉约派柳永、李清照；豪放派苏轼、辛弃疾。

**迎合市民阶层文化需求的成就**：宋词、元曲、话本、风俗画（张择端《清明上河图》）。

|  第二次鸦片战争  | $1856\sim 1860$ 年 | 《天津条约》《北京条约》 |         半殖民地半封建化程度加深         |
|   甲午中日战争   | $1894\sim 1895$ 年 |       《马关条约》       | 半殖民地化程度大大加深，列强掀起瓜分狂潮 |
| 八国联军侵华战争 |     $1900$ 年      |       《辛丑条约》       |        完全沦为半殖民地半封建社会        |

### 鸦片战争

爆发的**必然性**：

- 政治：中国封建君主专制 VS 英国资产阶级君主立宪制。
- 经济：中国小农经济占主导 VS 英国率先完成工业革命。
- 军事：中国装备陈旧、军纪败坏 VS 英国船坚炮利。
- 外交：中国闭关锁国 VS 英国疯狂扩张。

**根本原因**：完成工业革命的英国要打开中国市场，倾销商品、掠夺原料。

**直接原因（借口）**：$1839$ 年 $6$ 月，**虎门销烟**。

结果——签订一系列不平等条约：

- 《南京条约》：开放广州、厦门、福州、宁波、上海五处通商口岸；割让**香港岛**；赔款 $2100$ 万银元；关税要与英国商定。
- 《五口通商章程》——**领事裁判权**（破坏司法主权）。
- 《虎门条约》——**片面最惠国待遇**。
- 中美《望厦条约》和中法《黄埔条约》——通商口岸传教权。

**影响**：

- 社会性质：中国**开始沦为半殖民地半封建社会**，鸦片战争是**中国近代史的开端**。
- 思想：中国人**开眼看世界**（林则徐《四洲志》，魏源《海国图志》「师夷长技以制夷」，徐继畲《瀛寰志略》）。
- 经济：被卷入资本主义世界市场，自然经济开始解体。
- 革命任务：进入**旧民主主义革命**时期，任务是反封建、反侵略。

### 第二次鸦片战争

**原因**：

- 根本原因：进一步打开中国市场。
- 直接原因：修约要求遭到拒绝。

**结果**：$1858$ 年签订《天津条约》，$1860$ 年签订《北京条约》；共开 $11$ 处通商口岸、内河航行权、鸦片贸易合法化。

**罪行**：

- 英法联军**火烧圆明园**。
- 俄国趁机割占中国北方 $100$ 多万平方千米的土地。

**影响**：

- 中国半殖民地半封建化程度加深。
- 中外反动势力公开勾结，共同镇压人民反抗。

### 甲午中日战争

**原因**：

- 根本原因：日本制定「大陆政策」，侵华野心蓄谋已久。
- 直接原因：朝鲜东学党起义。

**结果**：$1895$ 年签订《马关条约》：

- 割地：割**辽东半岛、台湾及附属岛屿、澎湖列岛**给日本。
- 赔款：赔款**二亿两白银**。
- 通商：开放沙市、重庆、苏州、杭州为商埠。
- 办厂：允许日本在通商口岸开设工厂（**商品输出到资本输出**）。

**影响**：

- 中国半殖民地化程度**大大加深**。
- 刺激列强掀起**瓜分中国的狂潮**。
- 标志着**洋务运动的破产**。
- 促进了民族觉醒。

### 八国联军侵华战争

**原因**：

- 根本原因：进一步侵略瓜分中国，维护在华利益。
- 直接原因：镇压义和团运动。

**结果**：$1901$ 年签订《辛丑条约》：

- 惩办「首祸诸臣」。
- 向各国赔款白银 $4.5$ 亿两，分 $39$ 年还清，本息共约 $9.82$ 亿两。
- 北京东交民巷设立使馆区，中国人不得居住，各国可派兵驻守。
- 拆毁北京至大沽口炮台，准许各国派兵驻守北京至山海关铁路沿线要地。
- 禁止华北科举考试 $5$ 年，禁止中国人成立或加入反帝组织。

**影响**：标志着中国**完全沦为半殖民地半封建社会**。

**东南互保**：南方各省督抚与列强洽商「东南互保」协议——反映中央权威式微、地方势力扩张，严重动摇清政府统治根基。

### 太平天国运动

农民阶级，$1851\sim 1864$ 年。

**过程**：

- $1851$ 年金田起义。
- $1853$ 年定都天京（与清对峙）。
- $1856$ 年军事全盛（东征西征、北伐失败），**天京事变**（由盛转衰）。
- $1864$ 年天京沦陷。

**纲领**：

- 前期：$1853$ 年《天朝田亩制度》——主张平均分配土地和产品。评价：否定了封建地主土地所有制（革命性），反映农民追求社会财富平均的理想，但绝对平均主义带有**空想性**。
- 后期：$1859$ 年《资政新篇》——主张发展资本主义，是先进的中国人最早提出的在中国发展资本主义的方案。

**失败原因**：

- 主观 / 根本：农民阶级的历史局限性。
- 客观 / 直接：中外反动势力的联合绞杀。

**意义**：沉重打击了清王朝的统治，引起政治和权力结构的变化；湘淮系官僚集团崛起，中央权力下移。

### 洋务运动

地主阶级，$19$ 世纪 $60\sim 90$ 年代。

- **背景**：内忧外患（第二次鸦片战争和太平天国运动）。
- **根本目的**：维护清王朝统治。
- **指导思想**：「中体西用」。
- **代表人物**：奕䜣、曾国藩、李鸿章、左宗棠、张之洞。

**内容**：

- $60\sim 70$ 年代以「自强」为口号创办**军用工业**（江南机器制造总局、福州船政局、天津机器局）。
- $70\sim 90$ 年代以「求富」为口号创办**民用企业**（上海轮船招商局、开平煤矿）。
- 创办近代教育，培养翻译和军事人才。
- 创办以**北洋舰队**为代表的新式海军。

**评价**：引进资本主义国家的机器生产技术，是**中国早期现代化的尝试**，刺激了民族资本主义的产生（积极）；只引进军事和生产技术，不改变封建统治（局限）。

### 戊戌维新运动

资产阶级维新派，$1898$ 年。

**背景**：

- 政治：甲午战败，民族危机加深。
- 经济：$19$ 世纪末民族资本主义初步发展，民族资产阶级力量壮大。
- 思想：康梁维新思想（康有为《新学伪经考》《孔子改制考》；梁启超《变法通议》）。

**过程**：

- 序幕：$1895$ 年**公车上书**。
- 高潮：$1898$ 年百日维新，标志为光绪帝颁布「明定国是」诏书。

**结果**：失败。

**失败原因**：

- 根本：民族资本主义发展不充分，资产阶级力量弱小。
- 守旧势力强大。
- 缺乏可靠的社会基础，没有严密的组织。
- 把希望寄托于没有实权的皇帝。

**意义**：推动了民族资本主义的发展和新思想的传播，冲击了旧式官僚体制。

### 义和团运动

农民阶级，$1898$ 年。

- **口号**：「扶清灭洋」。
- 积极：具有强烈的反帝爱国倾向，展现了中国人民不畏强暴的牺牲精神，粉碎了帝国主义瓜分中国的企图。
- 局限：盲目排外，无法阻止中国滑向半殖民地的深渊。

## 第六单元 辛亥革命与中华民国的建立

### 辛亥革命

**时间**：$1911\sim 1912$ 年。

**背景**：

- 社会背景：《辛丑条约》签订后民族危机严重，清政府「新政」和「预备立宪」没有解决危机。
- 经济和阶级基础：资本主义的发展和民族资产阶级的壮大。
- 思想基础：资产阶级民主革命思想的传播和**三民主义**的提出。
- 组织基础：资产阶级革命团体和政党的建立（兴中会、同盟会等）。
- 军事准备：黄花岗起义等武装起义。
- 有利时机：四川保路运动。

#### 三民主义

局限：**没有明确提出反帝反封建**。

|   名称   |   别称   |        内容        |                      说明                      |
| :------: | :------: | :----------------: | :--------------------------------------------: |
| 民族主义 | 民族革命 | 驱除鞑虏，恢复中华 |        推翻清王朝封建统治，没有明确反帝        |
| 民权主义 | 政治革命 |      创立民国      |   **核心**；推翻封建帝制，建立资产阶级共和国   |
| 民生主义 | 社会革命 |      平均地权      | 资本主义土地纲领，不是彻底的反封建土地革命纲领 |

#### 过程

- 开始标志：$1911.10.10$ **武昌起义**。
- 高潮：$1912$ 年元旦中华民国临时政府在南京建立。
- 清帝退位：$1912.2.12$ 溥仪正式下诏退位。

#### 《中华民国临时约法》

时间：$1912$ 年 $3$ 月。

- 目的：防止袁世凯专权，维护共和制度。
- 内容：主权在民、自由平等、三权分立、责任内阁制。
- 意义：**资产阶级共和国宪法**性质，具有反对封建专制的进步意义，推动中国的民主化进程。

#### 历史意义

- 性质：比较完全意义上的反帝反封建的民族民主革命。
- 政治：推翻了清王朝统治，**结束了君主专制制度**，建立起共和政体。
- 经济：打击了帝国主义在华势力，为民族资本主义发展创造了条件。
- 思想：传播了民主共和理念，推动了思想解放。
- 社会：革除封建社会风俗恶习，社会生活近代化。

#### 失败

- 表现：没有完成反帝反封建的革命任务；没有改变半殖民地半封建社会性质；革命果实落入**袁世凯**手中。
- 原因：
  - 客观 / 直接：中外反动势力的联合阻挠。
  - 主观 / 根本：民族资本主义发展不充分，资产阶级的软弱性和妥协性。

### 北洋军阀统治时期

时间：$1912\sim 1928$ 年（$1928$ 年「东北易帜」）。

特点：军阀混战和割据，政治上分崩离析。

### 民族资本主义的发展

|        阶段         |            时间             | 原因                                                                                                                                                             |
| :-----------------: | :-------------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|        产生         |     $19$ 世纪六七十年代     | 外商企业的刺激；洋务运动的诱导；自然经济逐步瓦解                                                                                                                 |
|      初步发展       |  $19$ 世纪末（甲午战争后）  | 甲午战后帝国主义加紧侵略，自然经济进一步瓦解；清政府放宽对民间设厂的限制                                                                                         |
| 黄金时期 / 短暂春天 | $1912\sim 1919$（一战期间） | 中华民国建立扫除部分政治束缚；南京临时政府鼓励兴办实业；群众性反帝爱国斗争推动；**一战期间西方列强忙于欧战，暂时放松对华经济侵略（主要原因）**；「实业救国」思潮 |
|      较快发展       |       $1927\sim 1937$       | 民族资产阶级兴办实业热情提高；反帝爱国运动蓬勃开展；国民政府倡导「国民经济建设运动」（$1935$ 年）；南京国民政府基本统一全国                                      |

补充：

- 初步发展的影响：**民族资产阶级作为独立的政治力量登上历史舞台**。
- 黄金时期的表现：面粉、纺织等轻工业发展迅速。
- 较快发展期：以四大家族为代表的**官僚资本**膨胀，阻碍了民族资本主义的发展。

#### 民族资本主义的特点

- 资金少、规模小、技术力量薄弱。
- 受外国资本和本国封建主义的双重压迫。
- 行业、地域分布不均匀。

#### 影响民族资本主义发展的因素

- 有利因素：自然经济逐渐解体；清末和民国政府鼓励发展实业；反帝爱国运动的推动；实业家自强不息的爱国精神。
- 不利因素：**半殖民地半封建的社会性质（根本）**；政局动荡；自然经济的抵制；帝国主义、封建主义、官僚资本主义的压迫。

### 新文化运动

**时间**：$1915$ 年。

**概况**：

- 开始标志：$1915$ 年陈独秀创办《青年杂志》（后改名《新青年》）。
- 主要阵地：北京大学和《新青年》杂志。
- 代表人物：陈独秀、李大钊、胡适、鲁迅、钱玄同等。

**原因**：

- 根本原因：缺乏对旧思想旧文化旧礼教的彻底批判。
- 直接原因：袁世凯复辟帝制，掀起「尊孔复古」逆流。

**内容**：

- 提倡民主，反对专制。
- 提倡科学，反对迷信愚昧。
- 提倡新道德，反对旧道德。
- 提倡新文学（白话文，胡适、鲁迅），反对旧文学。

> 前期以民主与科学为核心，后期宣传马克思主义。

**评价**：

- 进步性：猛烈冲击了封建思想，推动了思想解放，为马克思主义传播创造了有利条件。
- 局限性：没有同广大群众相结合，对待东西方文化采取绝对肯定或绝对否定的态度。

## 第七单元 中国共产党成立与新民主主义革命兴起

### 五四运动

**时间**：$1919$ 年。

- 导火线：巴黎和会上中国外交的失败。
- 经过：
  - $5$ 月 $4$ 日到 $6$ 月初：中心在**北京**，以学生游行为主要斗争方式。
  - $6$ 月份：中心在**上海**，出现三罢（罢工、罢市、罢课），**工人阶级登上政治舞台**。

**意义**：

- 性质：彻底反帝反封建的伟大爱国革命运动、伟大社会革命运动、伟大思想启蒙运动。
- 促进了马克思主义在中国的传播，促进了马克思主义同中国工人运动的结合。
- 为中国共产党的成立打下了基础。
- **新民主主义革命的开端**（新领导阶级：无产阶级；新指导思想：马克思主义）。

**五四精神**：爱国、进步、民主、科学，核心是爱国主义精神。

### 中国共产党的诞生

**时间**：$1921$ 年。

**诞生条件**：

- 思想基础：马克思主义的广泛传播。
- 政治基础：中国工人运动的持续发展。
- 外部因素：共产国际的帮助。
- 组织基础：共产党早期组织的建立。
- 经济基础：民族资本主义的发展，无产阶级力量壮大。

**标志**：中共一大召开（$1921$ 年），地点从上海转移到浙江嘉兴南湖。内容：

- 确定党的名称「中国共产党」。
- 奋斗目标：推翻资产阶级，建立无产阶级专政，实现社会主义和共产主义。
- 选举产生党的领导机构，**陈独秀**任书记。

**意义**：

- 开天辟地的大事变，给灾难深重的中国人民带来了光明和希望。
- 使中国革命有了坚强的领导力量和正确的前进方向，中国革命面貌焕然一新。

### 中共二大

**时间**：$1922$ 年。

内容：提出党的**最低纲领**（民主革命纲领，反帝反军阀）和**最高纲领**（共产主义社会）。

### 国共合作和国民革命

**时间**：$1924\sim 1927$ 年。

**国共合作**：

- 标志：$1924$ 年 $1$ 月国民党一大在广州召开。
- 提出了**新三民主义**（明确反帝反封建），新三民主义是国共合作的政治基础。
- 通过了联俄、联共、扶助农工三大政策。

**国民革命**：

- 目标：打倒列强，除军阀。
- 高潮：$1926$ 年北伐战争。
- 影响：基本推翻北洋军阀的反动统治。
- 结果：失败。
  - 标志：$1927$ 年「四一二」反革命政变（蒋介石）、「七一五」反革命政变（汪精卫）。
  - 原因：根本 / 主观——中国共产党犯了右倾错误，放弃了对革命和革命军队的领导权；客观——国民党右派叛变革命，中外反动势力联合绞杀。
  - 教训：坚持无产阶级对革命的领导权，掌握革命武装和坚持武装斗争。

### 南京国民政府的统治

- $1927$ 年「宁汉合流」，标志国民党专制统治的确立。
- $1928$ 年底张学良「东北易帜」，国民政府在形式上基本统一全国。

### 国共十年对峙

**时间**：$1927\sim 1937$ 年。中国共产党开辟革命新道路。

- $1927.8.1$ **南昌起义**——意义：
  - 打响了武装反抗国民党反动统治的第一枪。
  - 标志着中国共产党独立领导武装斗争、创建人民军队和武装夺取政权的开始。
- $1927.8.7$ **八七会议**（汉口）——内容：
  - 总结国民革命失败的教训。
  - 纠正陈独秀的右倾错误。
  - 确定实行土地革命和武装反抗国民党反动派的总方针。
  - 毛泽东提出「政权是由枪杆子中取得的」。
- $1927.9$ **秋收起义**（长沙——井冈山），毛泽东领导；$1927.10$ 建立第一个农村革命根据地——**井冈山革命根据地**。
- 井冈山革命根据地的意义：点燃了「工农武装割据」的星星之火，中国革命走上了农村包围城市、武装夺取政权的道路。
- $1931$ 年建立红色政权：中华苏维埃共和国（江西瑞金）。
- 土地革命：打土豪，分田地。

### 红军长征

**时间**：$1934.10\sim 1936.10$，甘肃会宁会师。

**原因**：

- 直接原因：第五次反「围剿」失利。
- 根本原因：左倾错误。

**遵义会议**（$1935$ 年 $1$ 月）——意义：

- 结束了军事、组织上的「左」倾错误。
- **开始确立以毛泽东为核心的党中央的领导地位**。
- 挽救了党、红军和中国革命，成为党的历史上**生死攸关的转折点**。
- 是中共由幼稚走向成熟的标志。

**长征胜利的意义**：

- 实现了红军的战略大转移。
- 宣传了党的政治主张，沿途播下革命的种子。
- 鼓舞广大人民群众，铸就了长征精神，打开了中国革命的新局面。

## 第八单元 中华民族的抗日战争和人民解放战争

### 抗日战争

时间：$1931\sim 1945$ 年。

#### 局部抗战（$1931\sim 1937$）

日本局部侵华：

- $1931$ 年**九一八事变**，占领东北三省，是中国**局部抗战的开始**。
- $1932$ 年**一·二八事变**，上海十九路军奋起抵抗。
- $1932$ 年 $3$ 月，扶植溥仪建立伪满洲国。
- $1935$ 年**华北事变**：中日民族矛盾上升为国内主要矛盾。

各方反应：

- 国民政府：「攘外必先安内」方针，对日本侵略实行不抵抗政策。
- 民众：东北人民革命军和东北抗日联军（$1936$ 年，杨靖宇、赵尚志、赵一曼）。
- 爱国学生：$1935$ 年「一二·九」运动——宣传了中国共产党「停止内战、一致抗日」的主张，促进了中华民族的新觉醒。
- 中共：
  - $1935$ 年发表《八一宣言》，主张「停止内战，一致抗日」。
  - $1935$ 年 $12$ 月**瓦窑堡会议**，确定建立**抗日民族统一战线**的方针。

**西安事变**（$1936$ 年 $12$ 月）：

- 和平解决，成为**扭转时局的枢纽**。
- 促进了中共中央联蒋抗日方针的实现。
- 十年内战的局面基本结束。
- 标志着抗日民族统一战线**初步形成**。

#### 日军的侵华暴行

- **南京大屠杀**。
- 「以华制华」，扶植傀儡政权。
- 「以战养战」，经济掠夺。
- 对敌后抗日根据地实施「三光」政策。
- 重庆大轰炸。
- 细菌战。
- 「慰安妇」制度。

#### 全面抗战的开始

- $1937$ 年 $7$ 月 $7$ 日**卢沟桥事变**（七七事变），**全面抗战开始**。
- 第二次国共合作：$1937.9.22$ 国民党发表国共合作抗战宣言，标志**第二次国共合作的实现**和**抗日民族统一战线的正式形成**。

#### 抗日战争的四个阶段

**战略防御阶段**（$1937.7\sim 1938.10$）

正面战场（国民党，主战场，**片面抗战路线**）四次会战：

|   会战   |                            地位                            |
| :------: | :--------------------------------------------------------: |
| 淞沪会战 |                 粉碎日军「三个月亡华」企图                 |
| 太原会战 |            平型关大捷是抗战以来的**第一次胜利**            |
| 徐州会战 |          台儿庄大捷是抗战以来**正面战场最大胜利**          |
| 武汉会战 | 抗战以来规模最大的战役，$1938.10$ 抗战进入**战略相持阶段** |

敌后战场：

- 中共建立敌后抗日根据地，开展游击战争。
- $1938.6$ 毛泽东发表《论持久战》。

**战略相持阶段**（$1938.10\sim 1944$）

- 日本对华政策：对国民党**政治诱降为主、军事打击为辅**，对共产党「囚笼政策」。
- 国民党：
  - 第三次长沙会战（$1941.12\sim 1942.1$，胜利）。
  - 消极抗战、积极反共（$1941$ 年**皖南事变**）。
  - 中国远征军入缅作战。
- 共产党（**敌后战场成主战场**）：
  - 发动**百团大战**（彭德怀，$1940$ 年下半年，破袭日军华北交通线）。
  - 坚持抗战、团结、进步方针。
  - 巩固抗日民主根据地：经济上减租减息、大生产运动；政治上精兵简政、「三三制」原则。
- 世界反法西斯统一战线正式形成：$1942$ 年在华盛顿签署《联合国家宣言》。

**战略反攻阶段**（$1944\sim 1945$）

- $1945$ 年中共**七大**召开（延安）：《论联合政府》，**毛泽东思想被确立为党的指导思想**。
- 抗日战争胜利：$1945$ 年 $8$ 月 $15$ 日日本天皇宣布无条件投降，$9$ 月 $2$ 日签订投降书，$10$ 月 $25$ 日**台湾光复**。

#### 抗战胜利的原因

- **抗日民族统一战线**的建立，全民族抗战（**根本原因**）。
- 中国共产党发挥了**中流砥柱**的作用。
- 世界反法西斯力量的相互配合。
- 正面战场和敌后战场的相互配合。
- 抗日战争是一场正义的、反侵略的战争。

#### 抗战胜利的意义

- 是近代以来中国抗击外敌入侵所取得的**第一次完全胜利**。
- 中国战场是世界反法西斯战争的**东方主战场**，重新确立了中国在世界上的大国地位。
- 开辟了中华民族伟大复兴的光明前景。

### 解放战争

时间：$1946\sim 1949$ 年。

#### 争取和平民主的斗争

- $1945.8\sim 10$ **重庆谈判**，签订《双十协定》。
- $1946.1$ 在重庆召开政治协商会议（旧政协）。

#### 人民解放战争的进程

**战略防御阶段**（$1946.6$ 国民党围攻中原解放区 $\sim 1947.6$）

粉碎国民党全面进攻和重点进攻：

- 陕北解放区：青化砭、沙家店战役。
- 山东解放区：孟良崮战役。

**战略进攻阶段**（$1947.6\sim 1948.9$）

$1947.6$ 刘伯承、邓小平千里跃进大别山，拉开战略进攻序幕。

**战略决战 / 三大战役**（$1948.9\sim 1949.1$）

|   战役   |             要点             |
| :------: | :--------------------------: |
| 辽沈战役 |              ——              |
| 淮海战役 |  人民群众用小车推出来的胜利  |
| 平津战役 | 和平改编傅作义，北平和平解放 |

意义：基本摧毁了国民党的主要军事力量。

**最后胜利**（$1949.1\sim 1949.4$）

- $1949.4$ **七届二中全会**（西柏坡，「进京赶考」）：工作重心由乡村转移到城市。
- $1949.4$ 国共北平和平谈判破裂。
- $1949.4.21\sim 23$ **渡江战役**，人民解放军占领南京，**国民党政权被推翻**。

#### 国统区统治

- 经济：无限制发行纸币，恶性通货膨胀；官僚资本巧取豪夺，挤压民族资本主义。
- 政治：坚持独裁内战，国统区形成**第二条战线**。

#### 解放区土地改革（$1947$ 年）

《中国土地法大纲》，废除封建半封建的土地剥削制度。

## 第九单元 中华人民共和国成立和社会主义革命与建设

### 新政协的召开

时间：$1949$ 年 $9$ 月。

内容：

- 通过《共同纲领》，具有**临时宪法**的作用，规定国家性质为**新民主主义国家**；
- 选举产生中央人民政府委员会；
- 决定首都、国旗、代国歌、纪年。

意义：中国人民政治协商会议暂时代行**全国人大职能**，初步建立了中国共产党领导的**多党合作和政治协商制度**。

### 新中国成立的历史意义

- 结束了帝国主义、封建主义和官僚资本主义长期压迫剥削中国各族人民的历史，人民真正成为国家的主人；
- 从根本上改变了中国社会的发展方向，为由新民主主义向社会主义过渡创造了前提条件；
- 中华民族开始以崭新的姿态自立于世界民族之林，中国历史进入新纪元。

### 人民政权的巩固

- **农村**：土地改革（$1950\sim 1953$ 年），《中华人民共和国土地改革法》，变封建土地私有制为**农民土地私有制**；
- **城市**：「银元之战」和「米粮之战」，稳定物价，统一国家财经；
- 肃清土匪和镇压反革命运动；
- **抗美援朝**（$1950\sim 1953$ 年）。

### $20$ 世纪 $50$ 年代外交

政策：独立自主的和平外交政策。

三大方针：

- 一边倒；
- 另起炉灶；
- 打扫干净屋子再请客。

成就：

|            事件            |   时间    |                                 意义                                 |
| :------------------------: | :-------: | :------------------------------------------------------------------: |
| **和平共处五项原则**的提出 | $1953$ 年 |            标志着新中国外交的成熟，成为国际关系的基本准则            |
|       **日内瓦会议**       | $1954$ 年 |               中国首次以五大国之一的地位参加的重要会议               |
|        **万隆会议**        | $1955$ 年 | 二战后第一次没有西方殖民国家参加的亚非会议，中国提出「求同存异」方针 |

### 社会主义经济制度的基本建立

- $1953$ 年**过渡时期总路线**：「一化」（社会主义工业化）「三改」（三大改造）。
- **一五计划**（$1953\sim 1957$ 年）：优先发展**重工业**，开始改变我国工业落后的面貌。
- **三大改造**（$1953\sim 1956$ 年）——农业、手工业、资本主义工商业：
  - 实质：生产资料**私有制**改造为社会主义**公有制**；
  - 意义：到 $1956$ 年年底基本完成，标志着**社会主义经济制度在我国初步建立**。

### 三大政治制度

- **人民代表大会制度**：$1954$ 年 $9$ 月召开第一届全国人民代表大会，通过《中华人民共和国宪法》，是一部**社会主义类型**的宪法，体现了**人民民主原则**和**社会主义原则**。
- 中国共产党领导的多党合作和政治协商制度。
- 民族区域自治制度。

### 社会主义建设的探索

时间：$1956\sim 1976$ 年。

**成功探索**：

|       时间        |            事件            |                        要点                        |
| :---------------: | :------------------------: | :------------------------------------------------: |
|     $1956$ 年     |        **中共八大**        | 正确分析国内主要矛盾，提出主要任务，是一次成功探索 |
|     $1960$ 年     |          八字方针          |               调整、巩固、充实、提高               |
|     $1962$ 年     |         七千人大会         |                         ——                         |
|     $1964$ 年     |   提出建设「四个现代化」   |                         ——                         |
| $1972$、$1975$ 年 | 周恩来、邓小平的调整、整顿 |                    「文革」时期                    |

中共八大主要矛盾：人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾；主要任务：把我国尽快从落后的农业国变成先进的工业国。

**探索失误**：

- $1958$ 年提出「鼓足干劲、力争上游、多快好省地建设社会主义」总路线；
- $1958$ 年「大跃进」（生产力领域）：特点为高速度、高指标、瞎指挥、浮夸风；错误夸大了人的主观能动性，违背经济发展客观规律；危害是造成三年（$1959\sim 1961$ 年）经济困难；
- $1958$ 年**人民公社化运动**（生产关系领域）：特点「一大二公」；错误是生产关系超越了生产力发展阶段；
- $1966\sim 1976$ 年「文化大革命」——教训：加强民主与法制建设。

失误的根本原因：「左」倾错误。

教训：

- 从国情出发，正确分析国内主要矛盾；
- 必须实事求是，尊重客观规律；
- 生产关系的变革必须与生产力水平相适应。

### $20$ 世纪 $50\sim 70$ 年代建设成就

- **工业**：逐步建成一批门类比较齐全的基础工业项目；大规模的**三线建设**增强了国防力量，改善了工业布局。
- **科技和国防**：成功爆炸原子弹（$1964$ 年）、氢弹（$1967$ 年），试制并成功发射中远程导弹（$1966$ 年）和人造卫星（$1970$ 年，**东方红一号**）。

### $70$ 年代外交

- $1971$ 年中国**恢复在联合国的一切合法权利**；
- $1972$ 年尼克松访华，**中美关系开始走向正常化**；
- $1972$ 年田中角荣访华，**中日建交**。

## 第十单元 改革开放与社会主义现代化建设新时期

### 十一届三中全会

时间：$1978$ 年 $12$ 月。

内容：

- **思想**：重新确立解放思想、实事求是的马克思主义思想路线；
- **政治**：停止使用「以阶级斗争为纲」的口号；
- **经济**：工作中心转移到**经济建设**上来，实行**改革开放**的战略决策；
- **组织**：恢复民主集中制，拨乱反正，形成以**邓小平**为核心的第二代领导集体。

意义：实现了新中国成立以来党和国家历史上的**伟大转折**，开启了改革开放和社会主义现代化建设新时期。

### 社会主义民主法制建设

- 十一届三中全会后平反冤假错案；
- $1981$ 年通过《关于建国以来党的若干历史问题的决议》；
- $1982$ 年《中华人民共和国宪法》。

### 改革开放

**对内改革**：

- **农村**（$1978$ 年）：**家庭联产承包责任制**（所有权仍是公有制，农民拥有使用权和经营权）——调动了农民生产积极性，解放和发展了生产力；
- $1984$ 年**城市经济体制改革** / 国有企业改革全面展开（扩大企业经营自主权、实行政企分开）；
- $1992$ 年中共十四大，提出经济体制改革的目标是建立**社会主义市场经济体制**。

**对外开放**：

- $1980$ 年设立深圳、珠海、汕头、厦门四个**经济特区**；
- $1984$ 年开放 $14$ 个沿海港口城市；
- $1990$ 年浦东的开发开放；
- $2000$ 年实施「引进来」和「走出去」相结合的开放战略；
- $2001$ 年中国加入世贸组织。

### 「一国两制」构想

内涵：在一个中国前提下，国家主体坚持社会主义制度，香港、澳门、台湾保持原有的**资本主义制度**长期不变。

实践：

- $1997$ 年 $7$ 月 $1$ 日中国对**香港**恢复行使主权；
- $1999$ 年 $12$ 月 $20$ 日中国对**澳门**恢复行使主权。
