历史选考笔记，对应统编版《经济与社会生活》（选择性必修二），按单元整理。

## 第一单元 食物生产与社会生活

### 从食物采集到食物生产

#### 农业出现前的采集渔猎

- 时间：农业出现前，人类经历了约二三百万年的采集渔猎阶段；
- 食物来源：采集植物果实、茎叶，猎取动物，捕捞鱼虾；
- 生活方式：使用木、骨、石等工具，依赖自然环境，**迁徙不定**，用火取暖、烧烤食物。

采集渔猎的根本局限：人类只是被动地从自然界获取现成食物，食物来源不稳定，无法长期定居，也难以形成剩余产品。

#### 农业的起源

距今约 1 万年，人类从食物采集者转变为食物生产者，这一转变史称 **农业革命**（Agricultural Revolution），又称「**新石器革命**」。

**含义**：人类开始 **栽培植物、驯化动物**，自己生产食物。

**世界主要农业起源地**

|         地区         |    主要作物 / 家畜     |             说明             |
| :------------------: | :--------------------: | :--------------------------: |
| **西亚**（两河流域） | 小麦、大麦；绵羊、山羊 |     最早培育出小麦和大麦     |
|   **东亚**（中国）   |    南稻北粟；猪、狗    | 世界上最早种植水稻和粟的国家 |
| **中美洲**（墨西哥） |   玉米、甘薯、马铃薯   |         玉米的原产地         |

**意义（对社会生活的影响）**

- **人类生活方式的改变**：从迁徙走向 **定居**，人口增长，聚落形成；
- **社会分工的出现**：一部分人从事农业，另一部分人专门从事手工业、商业等，出现 **社会分工**；
- **私有制、阶级和国家的产生**：农业带来 **剩余产品**，出现私有财产和贫富分化，产生阶级；为管理公共事务和维护统治，逐渐形成 **国家**；
- **科学技术的发展**：为观测天象、安排农时，天文历法、数学等科学得以发展。

> 农业的出现是人类经济和社会生活的第一次革命，是文明产生的物质基础。

#### 不同地区的食物生产与社会生活

各大古代文明依托各自的农业基础，形成了不同的社会形态。

|    文明区域    |       食物生产       |                      社会生活                      |
| :------------: | :------------------: | :------------------------------------------------: |
|  两河、尼罗河  | 大麦、小麦、水利灌溉 |  古巴比伦：王室、神庙控制土地；古埃及：土地归国家  |
|    古代中国    |  南稻北粟；小农经济  | 战国后 **铁犁牛耕**；商周井田制；秦以后自耕农为主  |
| 古希腊、古罗马 |   谷物、葡萄、橄榄   | 古希腊城邦公民占有土地；古罗马长期实行奴隶制大庄园 |
|      美洲      |     玉米、马铃薯     |       阿兹特克人的土地贵族私有与村社公有并存       |

**古代中国的食物生产要点**

- 主要模式：以 **家庭为单位**、男耕女织、自给自足的 **小农经济**；
- 生产工具：春秋战国铁犁牛耕出现并推广，此后成为传统农业的主要耕作方式；
- 水利：都江堰、郑国渠、龙首渠等灌溉工程；
- 特点：精耕细作，重视农时，强调「不违农时」。

#### 生产关系的变化

- 男子在生产中逐渐占据主导地位，妇女地位下降；
- **私有制** 产生，土地和大部分产品归 **家庭** 所有；
- 氏族社会逐步瓦解，阶级、国家相继出现。

### 新航路开辟后的食物物种交流

#### 背景

- 15 世纪末 – 16 世纪初，**新航路开辟**，世界各大洲之间的联系日益紧密；
- 伴随殖民扩张，各大洲的物种被带到世界各地，出现了 **全球范围** 的物种大交流。

这场交流以美洲与旧大陆之间的物种互换最为典型，史称「**哥伦布大交换**」（Columbian Exchange）。

#### 美洲物种的外传

美洲原产、后传入其他大洲的作物：**玉米、马铃薯、甘薯、番茄、辣椒、烟草、可可** 等。

|    作物    |                      传播与影响                      |
| :--------: | :--------------------------------------------------: |
|  **玉米**  | 传入欧洲、亚洲、非洲，成为重要粮食作物；明朝传入中国 |
| **马铃薯** |       传入欧洲后缓解粮食短缺，推动欧洲人口增长       |
|  **甘薯**  |     明朝传入中国，产量高、适应性强，助推人口增长     |
|  **番茄**  |          起初被当作观赏植物，后成为重要蔬菜          |

**玉米、甘薯在中国的意义**：作为高产作物在明清广泛种植，扩大了粮食供给，**促进了人口的迅速增长**，也使不适宜种稻麦的山地得到开发。

#### 其他地区物种传入美洲

- 农作物：小麦、大麦、水稻、燕麦，以及苹果、葡萄、甘蔗、柑橘等；
- 禽畜：马、牛、羊、猪、鸡等，极大改变了美洲的生产与生活；其中 **马** 的传入深刻影响了印第安人的生产和作战方式。

#### 食物物种交流的影响

| 方面 |                                           影响                                           |
| :--: | :--------------------------------------------------------------------------------------: |
| 积极 | 提高全球粮食产量，**促进世界人口增长**；丰富人们的食物结构与饮食种类；推动经济与贸易发展 |
| 消极 |          新作物过度开发导致 **水土流失、环境破坏**；美洲种植园经济依赖奴隶劳动           |

> 物种交流是新航路开辟后全球联系加强的重要表现，深刻改变了世界的物质生活。

### 现代食物的生产、储备与食品安全

#### 食物生产的现代化

工业革命后，食物生产逐步走向 **机械化、集约化、商品化**。

- **农业机械化**：20 世纪以来，拖拉机、联合收割机等广泛使用，美国率先实现农业机械化；
- **优良品种培育**：杂交育种、良种推广，大幅提高单产。中国 **袁隆平** 培育的 **杂交水稻** 为解决世界粮食问题作出重要贡献；
- **设施农业与生物技术**：大棚种植、无土栽培、转基因技术等，突破了自然条件的限制。

#### 食物储备技术的进步

- 古代：**粮仓** 储粮，中国有含嘉仓等大型官仓；
- 近代以来：出现 **机械通风、低温、气调** 等储粮技术；
- **冷链**：冷冻冷藏技术和冷链物流的发展，使生鲜食品可以远距离运输、长时间保存。

#### 消除饥饿与食品安全

- **粮食安全**：粮食是人类生存的基础，各国重视粮食生产与储备；1945 年成立的 **联合国粮食及农业组织**（FAO）致力于消除饥饿；
- **中国的贡献**：以约占世界 9% 的耕地，养活了约占世界 20% 的人口，实现了「谷物基本自给、口粮绝对安全」；
- **食品安全问题**：农药、化肥、添加剂的滥用，转基因食品的争议等，成为现代社会面对的新问题；各国加强食品安全立法与监管。

## 第二单元 生产工具与劳作方式

### 古代的生产工具与劳作

#### 农业工具的演进

生产工具是生产力发展水平的重要标志。农业工具经历了从石、木到金属的演变。

|   时期   |          工具 / 材质          |                     说明                     |
| :------: | :---------------------------: | :------------------------------------------: |
| 原始社会 |   打制、磨制 **石器**；耒耜   |                   刀耕火种                   |
| 青铜时代 |      **青铜农具**（少）       | 青铜贵重，多用于礼器、兵器，农具仍以木石为主 |
| 春秋战国 | **铁农具**、**牛耕** 出现推广 |         铁犁牛耕成为传统农业主要方式         |
|   汉代   |    **耦犁**、耧车（播种）     |                 耕作效率提高                 |
|   唐代   |     **曲辕犁**（江东犁）      |   可调节耕地深浅，标志着中国传统步犁的定型   |
|   唐代   |     **筒车**（灌溉工具）      |               利用水力自动灌溉               |

**灌溉工具的发展**：从人力的桔槔、辘轳，到东汉的翻车（龙骨水车），再到唐代的筒车，灌溉逐渐 **省力、自动化**。

#### 手工业工具的演进

- **纺织**：从原始的纺轮，到汉代提花机，再到能织出精美丝织品的花楼机；
- **陶瓷**：从手制到 **轮制**，从露天堆烧到 **窑炉** 烧制；南朝出现匣钵，唐宋出现支钉，工艺不断精进；
- **冶金**：从锻打到 **铸造**，出现坩埚、水排（东汉杜诗发明的水力鼓风装置）等，提高了冶炼效率。

#### 劳作方式的演变

|      劳作方式      |              特征              |             时期             |
| :----------------: | :----------------------------: | :--------------------------: |
| **农业家庭式劳作** |     以家庭为单位，男耕女织     |   战国以后，随小农经济形成   |
| **农业庄园式劳作** | 众多奴隶或农奴在庄园内集中劳作 | 中国魏晋田庄；西欧中世纪庄园 |
| **手工业家庭劳作** |     以家庭为单位的个体生产     |             历代             |
| **手工业作坊劳作** |   拥有某种技艺的工匠世代传承   |        官营、民营作坊        |

**中国古代手工业的三种经营形态**

- **官营手工业**：政府经营，规模大、分工细、产品供官府和贵族，长期占主导，明中叶后衰落；
- **民营手工业**：民间私人经营，明中叶后 **超过官营** 占据主导；
- **家庭手工业**：农户副业，与小农经济结合，「男耕女织」。

### 工业革命与工厂制度

#### 工业革命前的手工工场

工业革命前，西欧出现了 **手工工场**：工场主雇佣工人，进行 **集中生产** 和 **分工协作**，但仍以 **手工劳动** 为主，是从手工业向机器大生产过渡的中间环节。

#### 第一次工业革命

18 世纪 60 年代 – 19 世纪中期，率先在 **英国** 兴起。

**核心标志**：**机器大生产** 取代手工劳动；瓦特改良的 **蒸汽机** 提供了强大而稳定的动力，人类进入「**蒸汽时代**」。

**主要发明**

|  领域  |           发明           |           人物            |
| :----: | :----------------------: | :-----------------------: |
| 棉纺织 |  珍妮纺纱机、水力织布机  |  哈格里夫斯、阿克莱特等   |
|  动力  |      **改良蒸汽机**      |    **瓦特**（1785 年）    |
|  交通  | 蒸汽机车、**火车**；汽船 | 史蒂芬孙（火车，1814 年） |

#### 工厂制度的确立

工业革命最重要的组织成果是 **工厂制度** 的建立。

**工厂制度的特点**

- **集中生产**：工人被集中到工厂，使用机器进行 **大规模生产**；
- **规章制度**：制定严格的 **纪律和规章**，工人必须遵守规定的工作时间；
- **分工协作**：生产过程 **细化分工**，各工序流水协作；
- **标准化**：使用标准化零件，提高效率和产品质量。

**中国近代的工厂**

- **洋务运动** 引入机器生产，创办近代军用、民用企业，如江南机器制造总局；
- 民族资本家 **张謇** 等兴办实业（大生纱厂），推动了中国民族工业的发展。

#### 工厂制度的影响

| 方面 |                                 影响                                 |
| :--: | :------------------------------------------------------------------: |
| 积极 | 大幅提高生产效率，**推动生产力飞跃**；带来管理制度的革新；促进城市化 |
| 消极 |   工人劳动强度大、劳动条件恶劣；**贫富分化加剧**；环境污染问题出现   |

### 现代科技进步与人类社会发展

#### 第二次工业革命

19 世纪中后期 – 20 世纪初，以 **电力** 的广泛应用为标志，人类进入「**电气时代**」。

- 主要成就：发电机、电动机、电灯（爱迪生）、电话、内燃机、汽车、飞机；
- 特点：科学与技术 **紧密结合**，多国同时展开，出现垄断组织。

#### 第三次科技革命

20 世纪四五十年代以来，以 **原子能、电子计算机、航天技术、生物工程** 等的突破为标志，又称 **信息革命**。

**核心：计算机与互联网**

- 1946 年，第一台电子计算机 **ENIAC** 在美国诞生；
- 20 世纪后期，**互联网**（Internet）迅速普及，人类进入 **信息时代**；
- 人工智能、大数据、云计算等新技术不断涌现。

#### 现代科技对社会生活的影响

- **生产力**：极大提高劳动生产率，改变了劳作方式，出现 **自动化、智能化** 生产；
- **生活方式**：网络购物、移动支付、远程办公改变人们的日常生活；
- **社会结构**：脑力劳动者比重上升，产业结构升级；
- **新问题**：网络安全、信息鸿沟、个人隐私、人工智能伦理等。

> 科技是一把双刃剑：它极大地推动了社会发展，也带来了必须正视的新挑战。

## 第三单元 商业贸易与日常生活

### 古代的商业贸易

#### 商业的起源与发展

- **起源**：原始社会末期，随着 **剩余产品** 和 **社会分工** 的出现，人们进行 **以物易物** 的交换，商业由此产生；
- **发展**：随生产力提高，交换范围扩大，出现专门从事交换的 **商人** 和固定的交易场所——**市**。

#### 中国古代商业的演变

|   朝代   |                           商业概况                            |
| :------: | :-----------------------------------------------------------: |
|   商周   |      商人出现，货币产生（贝）；「工商食官」（官府垄断）       |
| 春秋战国 |           官府控制被打破，出现许多商业中心和大商人            |
|   秦汉   |         统一货币度量衡；「市」有专门管理，时空受限制          |
|   隋唐   |       商业繁荣，长安、洛阳等大都市兴盛；出现柜坊、飞钱        |
|   宋元   | **打破坊市界限**，出现 **交子**（世界最早纸币）；海外贸易繁荣 |
|   明清   |   白银广泛使用；出现 **商帮**（徽商、晋商）；工商业市镇兴起   |

**发展趋势**：商业活动的 **时间、空间限制逐渐被打破**，交易媒介从实物到货币再到纸币，市场从城市延伸到农村。

#### 货币、信贷与商业契约

- **货币**：从实物货币（贝、布帛）到 **金属货币**（铜钱、白银），再到 **纸币**（交子）。货币的出现便利了交换，是商业发展的产物；
- **信贷**：中国古代有借贷、典当；近代出现票号（如晋商票号）；
- **契约**：买卖、租佃、借贷多订立契约，是维护交易秩序的重要方式。

#### 世界古代的商贸

- **丝绸之路**：连接中国与中亚、西亚乃至欧洲的陆上通道，**丝绸、瓷器、茶叶** 西传，西方物产东来；
- **海上丝绸之路**：从东南沿海经南海通往印度洋，宋元时期尤为繁荣；
- 古代埃及、两河流域、希腊、罗马之间的地中海贸易频繁。

#### 商业贸易的影响

- 丰富了人们的物质生活；
- 促进了 **不同地区、不同文明之间的经济文化交流**；
- 推动了城市的兴起和货币金融的发展。

### 世界市场与商业贸易

#### 世界市场的形成过程

**世界市场**（World Market）指世界各国、各地区之间进行 **商品交换和贸易** 的领域。它随资本主义发展而逐步形成。

|     阶段     |         时间          |             推动事件             |                标志                |
| :----------: | :-------------------: | :------------------------------: | :--------------------------------: |
| **开始形成** |     15 – 16 世纪      |          **新航路开辟**          | 世界各地联系加强，世界市场雏形出现 |
| **初步形成** |   18 – 19 世纪中期    |   **第一次工业革命**、殖民扩张   |   以英国为中心的世界市场初步形成   |
| **最终形成** | 19 世纪末 – 20 世纪初 | **第二次工业革命**、瓜分世界狂潮 |    **资本主义世界市场最终形成**    |

#### 各阶段的具体表现

**新航路开辟**（世界市场雏形）

- 欧洲的商路和贸易中心从地中海转移到 **大西洋沿岸**，史称「**商业革命**」；
- 美洲白银大量流入欧洲，引发 **价格革命**（物价上涨，封建地主衰落，资产阶级壮大）；
- 出现 **三角贸易**（欧洲—非洲—美洲），罪恶的 **黑奴贸易** 盛行。

**工业革命**（世界市场初步、最终形成）

- 机器大生产要求更广阔的原料产地和商品市场，列强加紧殖民扩张；
- 亚非拉逐渐沦为 **殖民地、半殖民地**，成为资本主义世界市场的组成部分；
- 交通运输（轮船、铁路）和通讯（电报）的进步，把世界更紧密地联系起来。

#### 商业贸易的新变化

近代以来，商业经营方式发生重大变化：

- **股份制公司**：荷兰、英国的东印度公司是早期典型；
- **证券交易所**：1602 年荷兰阿姆斯特丹出现最早的证券交易所；
- **银行**：近代银行业兴起，1694 年英格兰银行成立；
- **股票、期货** 等金融工具出现；
- **百货公司、连锁店** 等新型零售业态兴起。

### 20 世纪以来人类的经济与生活

#### 世界经济的发展与曲折

|        事件        |    时间     |                             影响                              |
| :----------------: | :---------: | :-----------------------------------------------------------: |
|   第一次世界大战   | 1914 – 1918 |                      破坏了世界经济秩序                       |
|   **经济大危机**   | 1929 – 1933 | 波及整个资本主义世界，破坏性巨大；**罗斯福新政** 加强国家干预 |
|   第二次世界大战   | 1939 – 1945 |                       再次重创世界经济                        |
| **布雷顿森林体系** |   1944 年   |    确立以 **美元** 为中心的国际货币体系（美元与黄金挂钩）     |

**布雷顿森林体系** 下建立了 **国际货币基金组织**（IMF）和 **世界银行**（World Bank），美元取得了等同黄金的地位。20 世纪 70 年代该体系瓦解。

#### 战后经济的新变化

- **国家宏观调控**：普遍加强国家对经济的干预（凯恩斯主义）；
- **福利国家**：西方国家建立社会保障与福利制度；
- **经济全球化**：跨国公司发展，世界贸易组织（WTO）成立（1995 年），各国经济联系空前紧密。

#### 中国经济与生活的变化

- **计划经济到市场经济**：改革开放后，逐步建立 **社会主义市场经济体制**；
- **对外开放**：设立经济特区，加入 WTO（2001 年），深度融入世界市场；
- **消费与生活**：商品日益丰富，**电子商务、移动支付** 改变了人们的消费方式，中国在数字经济领域走在世界前列。

## 第四单元 村落、城镇与居住环境

### 古代的村落、集镇和城市

#### 村落的产生

- **前提**：农业的出现使人类走向 **定居**；
- **含义**：**村落** 是人类 **定居** 后形成的相对稳定的聚居点；
- **功能**：为居民提供居住、储存、集体活动的场所，是社会管理的基层单位。

#### 集镇的形成

- **成因**：手工业者和商人聚集在交通便利处进行交易，逐渐形成 **集镇**；
- **性质**：介于 **乡村与城市之间**，兼具生产、交换和居住功能；
- **中国的镇**：原为军事驻防之地，后逐渐发展为工商业中心，如四大名镇（景德镇、汉口镇、佛山镇、朱仙镇）。

#### 城市的产生与功能

**城市**（City）是一定区域内政治、经济、文化的中心。

| 功能 |           说明           |
| :--: | :----------------------: |
| 政治 | 统治中心，官府、宫殿所在 |
| 军事 |  「城」以城墙防御为核心  |
| 经济 |  「市」是商品交换的场所  |
| 文化 |  宗教、教育、礼仪的中心  |

**中国古代城市布局**：讲究 **规整对称**，宫城居中，坊市分区。唐长安是典型的坊市分离的棋盘式布局，北宋以后坊市界限被打破。

**西方古代城市**：古希腊、古罗马城市以广场、神庙、剧场、公共浴场为中心，注重公共空间。

#### 世界各地的民居

民居因 **自然环境、生产方式、文化传统** 不同而形态各异。

|    地区    |            民居特点             |        成因        |
| :--------: | :-----------------------------: | :----------------: |
|  两河流域  |    黏土和芦苇建房，墙厚屋密     | 缺石少木，气候炎热 |
|   古埃及   |       以泥砖、木材为材料        |  尼罗河沿岸多泥沙  |
|  古代中国  | **木构架** 为主，讲究对称、等级 | 木材易得，重视礼制 |
| 古希腊罗马 |    带天井、列柱的院落式住宅     | 石材丰富，气候温和 |
|    北美    |      印第安人的圆锥形帐篷       |    游猎迁徙生活    |

**中国民居**：北方 **四合院**、南方民居、福建 **土楼**、陕北 **窑洞** 等，体现了因地制宜和 **家国同构** 的等级观念。

### 近代以来的城市化进程

#### 城市化的兴起

**城市化**（Urbanization）指农村人口向城市转移、城市规模不断扩大、城市生活方式向农村扩散的过程。

- **动因**：**工业革命** 是城市化的根本推动力。机器大生产使工厂和人口向城市集中；
- **进程**：英国最早开始城市化，19 世纪中期城市人口首次超过农村人口；此后欧美各国相继城市化。

#### 中国的城市化

- **近代**：鸦片战争后，通商口岸（上海、天津等）率先近代化，出现了近代城市，但整体水平低、发展畸形；
- **现代**：新中国成立后，特别是 **改革开放** 以来，城市化 **迅速推进**，城镇人口比重大幅上升。

#### 城市化带来的生活变化

- **居住**：出现公寓、高层住宅；水、电、燃气、暖气进入家庭；
- **基础设施**：自来水、排水系统、电力网、电报电话、地铁等公共设施完善；
- **社会生活**：公园、图书馆、影剧院等公共空间增多，生活方式趋于现代化。

#### 城市化中的问题

|    问题    |           表现           |
| :--------: | :----------------------: |
|  环境污染  | 工业废气、废水、垃圾污染 |
|  交通拥堵  |    人口、车辆过度集中    |
|  贫富分化  | 出现贫民窟，社会矛盾突出 |
| 犯罪与治安 | 城市人口密集，管理难度大 |

**应对**：加强城市规划，治理污染，发展公共交通，推进可持续发展与生态城市建设。

## 第五单元 交通与社会变迁

### 水陆交通的变迁

#### 古代的陆路交通

- **道路**：随着人类活动范围扩大，道路逐渐形成。古罗马修建了以罗马城为中心、四通八达的道路网，「**条条大路通罗马**」；
- **中国的道路**：
  - 秦朝修筑 **驰道、直道**，「车同轨」，形成以咸阳为中心的道路网；
  - 汉代开通 **丝绸之路**，连接中西；
  - 元朝建立发达的 **驿站** 制度，遍布全国。

#### 古代的水路交通

- **运河**：中国 **大运河**（隋唐大运河、京杭大运河）沟通南北，促进南北经济交流；
- **海运**：宋元时期海上丝绸之路繁荣，指南针应用于航海；
- 世界：古代腓尼基人、希腊人、罗马人的地中海航运发达。

#### 近代交通的变革

工业革命带来 **交通运输的革命**，以蒸汽为动力的火车和轮船取代了传统的畜力和风力。

|   工具   |       时间 / 人物        |         意义         |
| :------: | :----------------------: | :------------------: |
| **轮船** |  1807 年 富尔顿（美国）  | 水上交通进入蒸汽时代 |
| **火车** | 1814 年 史蒂芬孙（英国） | 开创了 **铁路时代**  |

**中国近代交通**

- **铁路**：19 世纪 70 年代，出现淞沪铁路（后被拆除）；1881 年 **唐胥铁路** 是中国自建的第一条铁路；詹天佑主持修建 **京张铁路**（1909 年建成），是中国人自行设计施工的第一条铁路；
- **轮船**：洋务运动创办 **轮船招商局**（1872 年），打破了外国航运垄断。

#### 交通变迁的影响

- 大大 **缩短了时空距离**，加速了人员、物资、信息的流通；
- 促进了 **商品经济和城市的发展**；
- 密切了各地乃至各国之间的联系，推动了 **世界市场的形成**；
- 改变了人们的生活方式和时间观念。

### 现代交通运输的新变化

#### 陆海空立体交通网

| 领域 |                  新变化                   |
| :--: | :---------------------------------------: |
| 陆路 | 汽车普及，**高速公路**、**高速铁路** 发展 |
| 水路 |      巨型轮船、集装箱运输、航运网络       |
| 航空 |       飞机成为重要的远距离交通工具        |
| 城市 |       地铁、轻轨等 **城市轨道交通**       |

#### 中国现代交通的成就

- **高铁**：中国高速铁路里程居世界第一，「**复兴号**」成为国家名片；
- **公路**：高速公路网四通八达；
- **航空、港口**：民航和港口吞吐量位居世界前列；
- **桥梁隧道**：港珠澳大桥等超级工程展现了中国的工程实力。

#### 现代交通对社会生活的影响

- **出行便捷**：极大缩短了出行时间，「同城化」「一日生活圈」成为可能；
- **经济发展**：带动了沿线地区和城市群的发展；
- **生活观念**：改变了人们的时空观念和生活节奏；
- **新问题**：交通拥堵、能源消耗、环境污染、交通安全等。

## 第六单元 医疗与公共卫生

### 历史上的疫病与医学成就

#### 历史上的疫病

**疫病** 指由 **细菌、病毒** 等引起、可大规模流行的严重传染病，如天花、鼠疫、疟疾、流感等。

|      疫病      |                         危害                          |
| :------------: | :---------------------------------------------------: |
|    **天花**    |          古埃及、古代中国均有流行，致死率高           |
|    **鼠疫**    | 6 世纪、14 世纪（欧洲「**黑死病**」）大流行，人口锐减 |
| **疟疾、伤寒** |                     长期困扰人类                      |

**疫病的影响**

- 造成大量人口死亡，冲击社会经济；
- 引发社会恐慌，甚至改变历史进程（如黑死病动摇了欧洲封建统治，冲击了教会权威）；
- 推动了医学和公共卫生的发展。

**古代的防疫措施**：重视 **环境卫生**，清理污水垃圾；建立 **隔离** 制度；设置 **收容、救治机构**（如中国古代的病坊、养病坊）。

#### 中医药的成就

中国传统医学历史悠久，形成了独特的理论和实践体系。

|          时期 / 人物          |                  成就                  |
| :---------------------------: | :------------------------------------: |
|    战国 – 西汉《黄帝内经》    |          **奠定中医理论基础**          |
|      东汉《神农本草经》       |          现存最早的药物学专著          |
| 东汉 **张仲景**《伤寒杂病论》 | **奠定中医临床学基础**，被尊为「医圣」 |
|         东汉 **华佗**         |        发明 **麻沸散**、五禽戏         |
|    唐 **孙思邈**《千金方》    |              临床百科全书              |
|   明 **李时珍**《本草纲目》   |  「**东方药物巨典**」，总结药物学成就  |

**现代中医药的成就**：屠呦呦从青蒿中提取 **青蒿素**，为治疗疟疾作出重大贡献，获得 **诺贝尔生理学或医学奖**（2015 年）。

#### 西医的发展与传入

- **西医的发展**：近代以来，西方医学以 **解剖学、生理学、细菌学** 为基础迅速发展；显微镜的发明、病原菌的发现、麻醉与消毒技术、疫苗（如牛痘接种预防天花）等成就显著；
- **西医传入中国**：明清之际随传教士传入；鸦片战争后，西医通过 **医院、医学院校** 在中国广泛传播，与中医并存互补。

### 现代医疗卫生体系与社会生活

#### 现代医疗卫生体系的建立

现代 **医疗卫生体系** 包括医疗服务、公共卫生、药品供应、医疗保障等方面。

- **公共卫生体系**：建立疾病预防控制、卫生监督、突发公共卫生事件应急等机制；
- **医疗保障制度**：许多国家建立了 **医疗保险** 和 **社会保障** 制度；
- **世界卫生组织**（WHO，1948 年成立）：协调国际卫生事务，推动全球疾病防治。

#### 中国的医疗卫生成就

- 新中国成立后，逐步建立起覆盖城乡的医疗卫生网络；
- 开展 **爱国卫生运动**，消灭了天花、血吸虫病等疫病；
- 建立 **城乡基本医疗保险** 制度，推进 **分级诊疗**；
- 抗击 **非典**（SARS）、新冠疫情，展现了中国公共卫生体系的应对能力。

#### 医疗卫生对社会生活的影响

- **健康水平提高**：人均预期寿命显著延长，婴儿死亡率大幅下降；
- **卫生观念普及**：讲卫生、勤锻炼、重预防成为社会共识；
- **生活方式改善**：城乡卫生环境改善，公共卫生意识增强；
- **推动社会进步**：医疗卫生的进步是社会文明程度的重要标志。

> 从疫病肆虐到体系完善，医疗与公共卫生的进步，深刻改变了人类的生存状况和社会生活。
