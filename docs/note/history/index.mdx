## 说明

这份笔记的 **必修** 部分以一份学考知识点汇总为底稿，对照教材校对、补全；**选择性必修** 部分对照教材整理，都按单元归类，重考点：重结论、重时间线、重对比，省略大段史实叙述。加粗的多是关键概念、标志性事件和必记时间。

浙江历史 **学考** 只考 **必修**《中外历史纲要》上下两册；**选考** 在必修的基础上，加试 **选择性必修** 三册（《国家制度与社会治理》《经济与社会生活》《文化交流与传播》）。

凡与教材冲突，以学考汇总为准；教材只用来补漏和核对。
