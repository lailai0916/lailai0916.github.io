历史选考笔记，对应统编版《国家制度与社会治理》（选择性必修一），按单元整理。

## 第一单元 政治制度

### 中国古代政治制度的形成与发展

#### 先秦的政治制度

**夏商**

- **夏朝**：约公元前 2070 年禹建立，我国最早的奴隶制王朝。**世袭制** 代替禅让制，「家天下」代替「公天下」。
- **商朝**：**内外服制**——内服是商王直接控制的王畿之地，外服是间接控制的方国和部族；商王通过 **占卜** 决策，神权与王权结合。

**西周**

- **分封制**：「封建亲戚，以藩屏周」，形成「天子—诸侯—卿大夫—士」的等级结构；
- **宗法制**：以血缘亲疏定继承，核心是 **嫡长子继承制**，与分封制互为表里；
- **礼乐制**：维护分封制、宗法制的工具。

**早期政治制度的特点**：以血缘关系为纽带，家国同构；神权与王权结合；**最高执政集团尚未实现权力的高度集中**；原始民主传统对君主的权力有制约作用。

#### 秦朝的政治制度

公元前 221 年秦统一，建立 **专制主义中央集权制度**。

- **皇帝制度**：皇位世袭、皇帝独尊、皇权至上，是这一制度的核心；
- **三公九卿制**：丞相（百官之首）、太尉（掌军事，虚设）、御史大夫（副丞相，掌监察）；国家重大事务先经 **朝议**，再由皇帝裁断；
- **郡县制**：郡县长官由皇帝直接任免，标志着 **官僚政治取代贵族政治**；
- **文书行政**：以邮传为中心的文书传送系统，保障政令通达。

#### 两汉至明清的政治制度演变

**中枢（皇权与相权）的演变**

|  朝代  |       中枢机构       |                       演变要点                       |
| :----: | :------------------: | :--------------------------------------------------: |
|   秦   |      三公九卿制      |                       丞相权重                       |
| 汉武帝 |     **中外朝制**     |        设中朝架空外朝丞相，削弱相权、加强皇权        |
|  东汉  |      **尚书台**      |                尚书台成为新的行政中枢                |
|  隋唐  |    **三省六部制**    | 中书起草、门下封驳、尚书执行，相权一分为三，皇权加强 |
|   宋   |    **二府三司制**    |  中书门下掌行政，枢密院掌军政，三司掌财政，分割相权  |
|   元   | **中书省**（一省制） |               三省合一，中书省权力较大               |
|   明   |  **废宰相、设内阁**  |    明太祖废丞相，明成祖设内阁，内阁职责为「票拟」    |
|   清   |     **设军机处**     |    雍正设立，跪受笔录、上传下达，君主专制达到顶峰    |

**隋唐三省六部制**：中书省草拟政令，门下省封驳审议，尚书省执行（下设六部）。相权一分为三，既加强皇权，又提高行政效率、减少决策失误，是中国政治制度史上的重大变革。

**地方（中央与地方）的演变**

|  朝代  |     地方制度     |                      演变要点                      |
| :----: | :--------------: | :------------------------------------------------: |
|   秦   |      郡县制      |                     郡—县两级                      |
|  汉初  |   **郡国并行**   |            分封诸侯王，酿成「七国之乱」            |
| 汉武帝 |   推恩令、刺史   |  推恩令削弱诸侯，设刺史监察地方；后州成为一级政区  |
|  隋唐  | 州县二级、节度使 |  隋废郡，唐设道为监察区；安史之乱后 **藩镇割据**   |
|   宋   |      路州县      |      设路作监察区，收兵权、财权、行政权于中央      |
|   元   |    **行省制**    | 行省是中央派出机构，行省—路—府—州—县，中国省制开端 |
|  明清  |   省府县、督抚   |   明设三司分权，后设督抚；清承明制，总督巡抚统辖   |

**元朝行省制**：既是地方最高行政机构，又是中央的派出机构；以「犬牙交错」为划界原则，使中央集权与地方分权相结合，是中国省制的开端，影响深远。

**演变趋势**：皇权不断加强、相权不断削弱，直至宰相制度被废除；中央权力不断加强、地方权力不断削弱。

### 西方国家古代和近代政治制度的演变

#### 古希腊罗马的政治制度

|     政体     |    代表    |                              要点                              |
| :----------: | :--------: | :------------------------------------------------------------: |
| **民主政治** |    雅典    | 公民直接参政；公民大会、议事会、陪审法庭；仅成年男性公民享权利 |
| **贵族寡头** |   斯巴达   |                 公民大会形式民主，实为寡头统治                 |
|  **共和制**  | 罗马共和国 |               执政官、元老院、公民大会，贵族主导               |
|   **帝制**   |  罗马帝国  |                元首制、君主制，皇帝掌握最高权力                |

#### 中古西欧的封建制度

- **封君封臣制度**：以土地为纽带，「我的附庸的附庸，不是我的附庸」，王权衰微、政治分裂；
- **基督教会**：神权凌驾于王权之上，教权与王权长期并立又斗争；
- **城市自治**：中世纪后期，城市通过金钱赎买或武装斗争取得自治权，出现 **等级君主制**（英国议会、法国三级会议）。

#### 近代西方资本主义政治制度

| 国家 |         确立标志 / 文件         |            政体            |               特征               |
| :--: | :-----------------------------: | :------------------------: | :------------------------------: |
| 英国 |       1689 年《权利法案》       |  **君主立宪制**（议会制）  |    议会至上，「君主统而不治」    |
| 美国 |   1787 年《美利坚合众国宪法》   | **总统制共和制（联邦制）** | 三权分立、分权制衡、联邦与州分权 |
| 法国 | 1875 年《法兰西第三共和国宪法》 |      **议会制共和制**      |         共和最终战胜帝制         |

**英国责任内阁制**：18 世纪逐渐形成，内阁由议会多数党组建，对议会负责；首相是政府首脑与多数党领袖，掌握行政大权。

**美国 1787 年宪法**：确立联邦制、三权分立与制衡原则；立法（国会）、行政（总统）、司法（最高法院）相互独立又彼此牵制。

### 中国近代至当代政治制度的演变

#### 民国时期的政治制度

|     时期     |          政治探索           |                         结果                         |
| :----------: | :-------------------------: | :--------------------------------------------------: |
| 南京临时政府 | 1912 年《中华民国临时约法》 | 中国第一部资产阶级共和国宪法性质的文件；未能真正实行 |
| 北洋政府时期 |        政党政治尝试         |             军阀操纵，「宋教仁案」后破产             |
| 南京国民政府 | 1928 年后「训政」，一党专政 |       以「训政」之名行独裁之实，「宪政」是幌子       |

#### 中国共产党在根据地和解放区的探索

- **土地革命时期**：中华苏维埃共和国（1931 年，江西瑞金）；
- **抗日战争时期**：抗日根据地设 **边区政府**，实行「**三三制**」原则（共产党员、党外进步人士、中间派各占三分之一）；
- **解放战争时期**：设置行政区，为新中国政权建设奠定基础。

#### 中华人民共和国的政治制度

- **根本政治制度**：**人民代表大会制度**——1954 年第一届全国人大召开，通过 1954 年《宪法》，确立人民代表大会制度；
- **基本政治制度**：
  - 中国共产党领导的多党合作和政治协商制度；
  - 民族区域自治制度；
  - 基层群众自治制度。

**中国特色社会主义制度的最大优势**：中国共产党的领导。

| :----------------: | :-----------: | :------------------------------------------------: |

**近现代改革**：新中国成立后的社会主义改造；1978 年 **改革开放** 是决定当代中国命运的关键抉择。

## 第二单元 官员的选拔与管理

### 中国古代官员的选拔与管理

#### 选官制度的演变

| 时期 |      制度      |   选官标准   |                  方式                  |
| :--: | :------------: | :----------: | :------------------------------------: |
| 西周 | **世卿世禄制** |     血缘     |                  世袭                  |
|  秦  |  **军功爵制**  |     军功     |               按军功授爵               |
| 汉朝 |   **察举制**   | 品行（孝廉） |            地方长官考察推举            |
| 魏晋 | **九品中正制** |  家世、门第  | 中正官评定等级；后成维护士族特权的工具 |
| 隋唐 |   **科举制**   |     才学     |           分科考试、自由报考           |

**科举制的影响**：使中下层读书人凭考试参与政权，扩大统治基础，提高官员文化素质，加强中央集权；打破世家大族对政权的垄断，促进社会阶层流动。

#### 官员的考核与监察

**考核**

- 秦汉：**上计制**，郡县长官定期向中央呈报户口、垦田、财政等，作为奖惩依据；
- 隋唐：吏部考核，以品德、才能为标准；
- 明清：**考满**（任满考核）与 **考察**（京察、大计）并行。

**监察**

|  朝代  |           监察机构 / 官员           |
| :----: | :---------------------------------: |
|   秦   |              御史大夫               |
| 汉武帝 |    **刺史**（监察地方）、御史台     |
|  隋唐  |               御史台                |
|   宋   | 台谏合一（御史台 + 谏院）、通判监州 |
|   元   |               御史台                |
|  明清  | **都察院、六科给事中**（合称科道）  |

### 西方的文官制度

#### 背景与建立

- **背景**：中世纪的 **恩赐官职制**、近代早期的 **政党分肥制** 弊病丛生，行政效率低、腐败严重；工业革命后国家管理事务扩展，急需专业化、稳定的行政队伍；启蒙思想、中国科举制的影响。
- **建立**：**英国** 是文官制度的发源地——1854 年《诺思科特—屈维廉报告》，1870 年枢密院令确立 **公开竞争考试** 择优录用，标志英国文官制度确立。美国 1883 年《彭德尔顿法》确立文官制度。

#### 特点与影响

- **特点**：**公开考试、择优录用**；**政治中立**（文官不随政党更替而进退）；**常任制**、职务常任；「两官分途」，政务官与事务官分开。
- **影响**：
  - 积极：规范了政府行政，提高了管理水平与效率，有利于政治与行政的连续、稳定；
  - 局限：容易滋生 **官僚习气** 和僵化，层次繁多、人浮于事，增加行政成本。

### 近代以来中国的官员选拔与管理

#### 晚清与民国

- **晚清**：1905 年 **废除科举制**；学堂选官、留学毕业生选官成为主要方式；
- **南京临时政府**：孙中山提出 **文官考试** 思想，主张考试权独立（**五权分立**）；
- **北洋政府**：以 **考试** 和 **甄别** 相结合选拔官员；
- **南京国民政府**：建立公务员制度，以 **考试院** 为最高考试机关；实际执行中任人唯亲、党派干预严重。

#### 中华人民共和国

- 新中国成立后逐步建立 **干部制度**，坚持党管干部原则；
- **1993 年** 推行 **公务员制度**，2005 年《中华人民共和国公务员法》颁布，公务员考试录用制度正式形成；
- 特点：公开考试、公平竞争、择优录取，党管干部与依法管理相结合。

## 第三单元 法律与教化

### 中国古代的法治与教化

#### 法律的演变

|   时期   |         成就         |                                 要点                                 |
| :------: | :------------------: | :------------------------------------------------------------------: |
|   战国   |     李悝《法经》     |                     中国历史上最早成文法典的代表                     |
|    秦    | 以法家治国，严刑峻法 |                              律、令并行                              |
|   汉朝   |     **引礼入法**     |                 以经注律，「春秋决狱」，礼法开始结合                 |
|   魏晋   |      律令儒家化      |                   「准五服以制罪」，礼法进一步融合                   |
|   唐朝   |   **《唐律疏议》**   | 中国现存最早、最完整的封建法典，礼法结合的典范；是中华法系确立的标志 |
| 宋元明清 |       沿袭唐律       |               《宋刑统》《大明律》《大清律例》一脉相承               |

**中华法系的特点**：**礼法结合**，德主刑辅；以《唐律疏议》为代表，影响东亚诸国。

#### 教化

- **儒家教化**：以礼入法，以德化民；
- **基层教化载体**：宋代以后 **乡约** 兴起（北宋《吕氏乡约》是最早的乡约），明清乡约与法律逐渐合流，宣讲皇帝「圣谕」；
- **家训、族规、乡绅**：辅助基层治理，维护社会秩序。

### 近代西方的法律与教化

#### 法律传统

|     法系     |    代表    |                            特点                            |
| :----------: | :--------: | :--------------------------------------------------------: |
| **大陆法系** | 法国、德国 | 以罗马法为基础，成文法典（《拿破仑法典》），法官依法典判案 |
| **英美法系** | 英国、美国 |           以「判例法」为主，遵循先例，法官作用大           |

**渊源**：罗马法（《十二铜表法》《民法大全》）是近代西方法律的重要渊源；中世纪日耳曼法、教会法、罗马法共同奠定基础。

**近代法律的基本原则**：**法律面前人人平等**、保护 **私有财产**、**罪刑法定**、**无罪推定**、陪审团制度、律师制度。

**局限**：确认了财产权与个人权利，但也维护资产阶级利益；种族、性别、财产等歧视长期存在。

#### 宗教伦理与教化

- 基督教会控制社会精神生活，垄断教育与道德解释；
- 宗教伦理成为西方社会重要的教化力量，也束缚人们的思想。

### 当代中国的法治与精神文明建设

#### 社会主义法治建设

- **初创**：1954 年《中华人民共和国宪法》，我国第一部社会主义类型的宪法；
- **恢复发展**：改革开放后，1982 年《宪法》，逐步形成中国特色社会主义法律体系；
- **全面依法治国**：**2010 年** 中国特色社会主义法律体系形成；**2020 年** 颁布《中华人民共和国民法典》——新中国第一部以「法典」命名的法律，被称为「社会生活的百科全书」。

#### 精神文明建设

- 20 世纪 80 年代 「五讲四美三热爱」；
- 21 世纪初提出社会主义荣辱观；
- 2006 年后倡导建设社会主义核心价值体系，凝练 **社会主义核心价值观**（富强、民主、文明、和谐等）。

## 第四单元 民族关系与国家关系

### 中国古代的民族关系与对外交往

#### 历代民族政策与治理

| 朝代 |                                 民族治理机构 / 政策                                 |
| :--: | :---------------------------------------------------------------------------------: |
|  秦  |                             设典客、典属国管理民族事务                              |
|  汉  |                       设 **西域都护府**（前 60 年），管理西域                       |
| 隋唐 |                  唐设都护府、羁縻府州；和亲（文成公主入藏）、会盟                   |
|  元  |                    **宣政院** 管理西藏；澎湖巡检司管辖澎湖、台湾                    |

**处理方式**：战争、和亲、会盟、册封、设置机构、朝贡贸易等。

#### 对外交往

- **汉朝**：**丝绸之路**，张骞通西域；
- **唐朝**：兼容并包，鉴真东渡、玄奘西行；
- **宋元**：海上丝绸之路繁荣，设 **市舶司** 管理对外贸易；
- **明清**：郑和下西洋（朝贡贸易，政治目的大于经济）；后期实行 **海禁** 与「闭关自守」，逐渐落后于世界。

### 近代西方民族国家与国际法的发展

#### 近代民族国家的形成

- 中世纪西欧王权、教权并立，国家意识淡薄；
- 宗教改革冲击教会权威，专制王权加强；
- 法国大革命、拿破仑战争传播民族主义，**民族国家** 逐渐形成，国家主权观念确立。

| 1648 年 | **《威斯特伐利亚和约》** | 确立主权、领土、独立等原则，形成威斯特伐利亚体系 |

### 当代中国的民族政策

- **民族区域自治制度**：1947 年建立第一个省级自治区——**内蒙古自治区**；1954 年《宪法》确立为基本政治制度；全国已建立五个省级自治区；
- **内涵**：在国家统一领导下，各少数民族聚居地方设立自治机关，行使自治权；
- **意义**：维护国家统一和民族团结，保障少数民族当家作主，促进各民族共同繁荣；
- **新时代**：铸牢 **中华民族共同体意识**。

## 第五单元 货币与赋税制度

### 货币的使用与世界货币体系的形成

#### 中国货币的演变

| 时期 |        货币形态        |              要点              |
| :--: | :--------------------: | :----------------------------: |
| 先秦 |   贝币、布币、刀币等   |       形制多样，各国不一       |
|  秦  | **半两钱**（圆形方孔） |  统一货币，圆形方孔沿用两千年  |
|  汉  |         五铢钱         |      汉武帝铸币权收归中央      |
|  唐  |      **开元通宝**      |  不再以重量命名，通宝钱制确立  |
|  宋  |        **交子**        |    北宋出现世界上最早的纸币    |
|  元  |     纸币（钞）为主     |         推行不兑换纸币         |
| 明清 |    **白银** 货币化     | 白银成为主要货币，赋役折银征收 |

#### 世界货币体系

- **金本位制**：19 世纪确立，以英镑为中心；
- **布雷顿森林体系**：1944 年确立，美元与黄金挂钩、各国货币与美元挂钩，美元成为国际主要货币；1973 年前后瓦解；
- 此后形成多种货币并存、汇率浮动的国际货币格局。

### 中国赋税制度的演变

#### 演变脉络

|   时期   |          赋税制度          |                         要点与影响                         |
| :------: | :------------------------: | :--------------------------------------------------------: |
|   先秦   |         贡赋、田租         |                      按田亩、人丁征收                      |
|   魏晋   |         **租调制**         |               按户征粮和绢帛，以均田制为基础               |
|   唐初   |        **租庸调制**        |            「以庸代役」，保证农时，减轻人身控制            |
| 唐中后期 |    **两税法**（780 年）    | 按人丁、资产纳户税，按田亩纳地税；改变以人丁为主的赋税传统 |
|   北宋   |          募役法等          |                     王安石变法调整赋役                     |
|   明朝   |        **一条鞭法**        |          赋役合并、折银征收；简化名目，赋役货币化          |
|   清朝   | **摊丁入亩**（「地丁银」） |     将丁银摊入田赋，**废除人头税**，人身控制进一步松弛     |

**两税法**：改变了自战国以来以人丁为主的赋税制度，扩大收税对象，减轻政府对农民的人身控制；是中国赋税制度史上的重大变革。

**摊丁入亩**：彻底废除延续两千年的人头税，赋税征收以土地为主要依据，一定程度上减轻了无地、少地农民的负担，也促进了人口增长。

**演变趋势**：征税标准 **由人丁为主转向以土地财产为主**；征收方式 **由实物、力役转向货币**；国家对农民的人身控制逐渐松弛。

#### 关税与个人所得税

- **关税**：中国古代很早就有关税；近代 **协定关税**（《南京条约》）破坏了关税主权，直到 1949 年后中国才真正收回关税自主权；
- **个人所得税**：1980 年《中华人民共和国个人所得税法》颁布，个人所得税制度正式确立。

## 第六单元 基层治理与社会保障

### 中国古代的户籍制度与社会治理

#### 户籍制度

| 朝代 |              户籍制度              |                要点                |
| :--: | :--------------------------------: | :--------------------------------: |
| 战国 |            国家编排民户            |        「编户」用于赋役征发        |
|  秦  |          **分类登记制度**          |        户籍是征发赋役的依据        |
|  汉  |            **编户齐民**            |    百姓编入户籍，依户籍纳税服役    |
| 隋唐 |     「大索貌阅」、户籍三年一造     |         核查户口，防止隐匿         |
|  宋  |            分主户、客户            |           按有无田产划分           |
|  明  | **黄册**（户籍）、鱼鳞图册（田亩） |       黄册管人、鱼鳞图册管地       |
|  清  |            户籍管理松弛            | 摊丁入亩后，户籍的赋役功能大为削弱 |

#### 基层组织与社会治理

- **基层组织**：秦汉的 **乡里制**（乡、里、亭）；唐代 **邻保制**；北宋 **保甲制**；明代 **里甲制**；均寓户口管理、征发赋役、维护治安于一体；
- **社会救济**：政府主导 **常平仓**（调节粮价）、**义仓**（备荒赈灾）；宗族、慈善组织（如宋代义田、明清善堂）辅助救济；
- **特点**：政府主导、宗族与社会力量辅助；重点在优抚鳏寡孤独、备荒赈灾。

### 世界主要国家的基层治理与社会保障

#### 西方基层治理

|   时期   |                     基层治理形式                     |
| :------: | :--------------------------------------------------: |
|  古希腊  |                       村社自治                       |
| 中古西欧 |                 庄园、行会、城市自治                 |
|   近代   |    地方自治（英国教区、法国自治市、美国乡镇自治）    |
|   现代   | 社区成为基层治理的重要单元，政府、社会、居民共同参与 |

#### 现代社会保障制度

- **建立**：19 世纪 80 年代 **德国** 率先建立 **社会保险制度**（俾斯麦政府），是现代社会保障制度的开端；
- **发展**：1935 年美国《社会保障法》；二战后 **英国** 率先宣布建成「**福利国家**」，西方国家普遍推行；
- **评价**：
  - 积极：一定程度上保障了公民基本生活，缓和了社会矛盾，稳定了社会秩序；
  - 局限：加重国家财政负担，容易助长部分人的依赖心理，降低劳动积极性。

**中国的社会保障**：新中国成立后逐步建立；改革开放后不断完善，形成覆盖养老、医疗、失业、工伤、生育等的社会保障体系，是世界上规模最大的社会保障体系。
