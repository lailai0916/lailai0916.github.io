历史选考笔记，对应统编版《文化交流与传播》（选择性必修三），按单元整理。

## 第一单元 源远流长的中华文化

### 中华文化的发展历程

#### 起源与奠基

- **多元起源**：黄河流域、长江流域、珠江流域、辽河流域、北方草原、四川盆地、青藏高原、天山南北，都是孕育中华文明的摇篮，中华文化 **多元一体**，源远流长，博大精深；
- **华夏认同**：春秋战国，内迁的戎狄蛮夷逐渐融入华夏族，各民族产生 **华夏认同** 观念，成为多元一体中华文化的重要基础；
- **百家争鸣**：春秋战国「学在官府」的局面被打破，形成 **百家争鸣** 的局面，是中华文化的 **第一次思想解放**，成为后世思想文化的源头。

#### 儒学的演进

|   时期   |         演进         |                     内容                     |
| :------: | :------------------: | :------------------------------------------: |
| 春秋战国 |     儒学创立发展     |         孔子创立儒学，孟子、荀子发展         |
|    秦    |       遭到打击       |                 「焚书坑儒」                 |
|  汉武帝  |     确立正统地位     | 「罢黜百家，独尊儒术」，儒学成为主流意识形态 |
| 魏晋隋唐 | 受佛道冲击，三教并行 |              儒、佛、道交汇融通              |
|   宋元   |       理学形成       |   吸收佛道思想，形成 **理学**，朱熹集大成    |
| 明清之际 |  进步思想家批判理学  |         李贽、黄宗羲、顾炎武、王夫之         |

#### 传承与转折

- **文字发展**：从甲骨文到金文、篆书、隶书、楷书，汉字延续不断，成为中华文化传承的重要载体；
- **文化载体**：造纸术、印刷术推动典籍的编纂与流传，历代修史、编书使文化代代相传；
- **明清批判**：明清之际，提倡个性自由的思想出现，进步思想家批判理学、抨击专制，但未能突破封建藩篱；
- **近代冲击**：鸦片战争后，向西方学习成为潮流；新文化运动 **抨击封建思想**，科学与民主成为潮流；五四运动后，**马克思主义** 在中国广泛传播，并与中国实际相结合。

### 中华优秀传统文化的内涵与特点

#### 内涵

- **以人为本**：周公「敬天保民」，孔子「仁」，主张 **民本** 思想，「民贵君轻」「君舟民水」；
- **崇尚天人合一，道法自然**：老子提出「道」，追求天人合一；荀子主张「制天命而用之」；
- **提倡爱国，追求家国情怀**：「修身齐家治国平天下」，范仲淹「先天下之忧而忧」，顾炎武「天下兴亡，匹夫有责」；
- **崇德尚贤，推崇天下为公**：西周「明德」，孔子「为政以德」，墨子「尚贤」；
- **崇尚自强不息，厚德载物**：「天行健，君子以自强不息」；
- **主张和而不同**：西周太史伯提出「和实生物，同则不继」，孔子、孟子视「和」为最高境界。

#### 特点与价值

- **特点**：本土性、多样性、包容性、凝聚性、连续性；
- **价值**：是中华民族发展的 **内在思想源泉和精神动力**；蕴含着丰富的道德伦理，为治国理政、道德建设提供借鉴；维护着中国团结统一的政治局面，维系着多民族大家庭，推动着中华文明的发展。

### 中华文化的世界意义

#### 中华文化在交流中发展

外来文化传入中国，反过来又推动中华文化的更新，交流是 **双向** 的。

|   外来文化   |       路径与时间       |                                          影响                                          |
| :----------: | :--------------------: | :------------------------------------------------------------------------------------: |
|   **佛教**   | 两汉之际经丝绸之路传入 | 魏晋渐趋本土化，隋唐完成，形成 **禅宗**；影响文学、艺术、建筑，与儒道融合催生 **理学** |
| **西学东渐** |        明末清初        |          传教士 **利玛窦** 与徐光启合译《几何原本》；带来天文、地理、数学知识          |
| **近代西学** |       鸦片战争后       | 林则徐、魏源「师夷长技以制夷」；洋务运动办新式学堂、派留学生；新文化运动传入民主与科学 |

- 佛教传入的影响需两面看：丰富了中华文化，但其消极因素也产生了一定的负面作用；
- 明末西学传入范围有限，仅在少数士大夫中传播；近代西学的传入则伴随民族危机，规模、深度远超以往。

#### 中华文化对世界的影响

以中国为中心，中华文化向外辐射，形成以 **儒家文化** 为核心的 **东亚文化圈**（汉字文化圈）。

|  方向  |                                                     传播内容与影响                                                     |
| :----: | :--------------------------------------------------------------------------------------------------------------------: |
|  东亚  | 汉字外传，朝鲜、日本、越南创制本国文字；儒学、佛教、律令、科技、生活方式传入，日本大化改新、朝鲜典章制度均以唐制为蓝本 |
| 东南亚 |                                        郑和下西洋后，大批华侨移居，传播中华文化                                        |
|  欧洲  |     四大发明经阿拉伯人西传，推动欧洲社会转型；16 – 18 世纪儒家经典传入欧洲，掀起「中国热」，启蒙思想家推崇儒家思想     |

**四大发明的世界意义**：造纸术促进欧洲文化发展；火药推动欧洲封建制度的瓦解；指南针促成新航路开辟；印刷术推动文艺复兴和宗教改革。

## 第二单元 丰富多样的世界文化

### 古代西亚、非洲文化

#### 古代西亚文化

诞生于两河流域（美索不达米亚），公元前 3500 年左右出现最初的文明。

- **文字**：苏美尔人发明 **楔形文字**，是世界上最古老的文字之一；
- **文学**：《吉尔伽美什》是目前所知世界上最早的史诗；
- **法律**：《汉谟拉比法典》是世界上现存最早的较为完整的成文法典，宣扬「君权神授」，维护奴隶主利益；
- **建筑与艺术**：苏美尔人的泥偶雕塑、亚述帝国的壁画、波斯的琉璃砖装饰。

#### 古代埃及文化

诞生于尼罗河流域，公元前 3500 年左右形成若干小国。

- **文字**：**象形文字**，几乎和楔形文字一样古老；
- **历法**：制定世界上第一部 **太阳历**；
- **建筑**：**金字塔** 体现了对法老的崇拜，是古代埃及人智慧的结晶；
- **文学**：有神话、诗歌、哲理故事和散文故事；
- **科技**：书写用的 **莎草纸** 曾大量出口，是古代埃及主要的文化用品和出口物资。

#### 阿拉伯文化

- **宗教**：伊斯兰教，穆罕默德创立；
- **文学**：《天方夜谭》（《一千零一夜》）；
- **科学**：改造印度数字形成 **阿拉伯数字**；白塔尼修订历法；伊本·西那《医典》长期用作医学教科书；
- **桥梁作用**：阿拉伯人翻译保存了大量古希腊、罗马典籍，并将中国、印度文化传入欧洲，是东西方文化交流的 **桥梁**。

### 欧洲文化的形成

#### 古典希腊与罗马文化

|      |                      古代希腊                      |                           古代罗马                           |
| :--: | :------------------------------------------------: | :----------------------------------------------------------: |
| 哲学 | **苏格拉底、柏拉图、亚里士多德**，奠定西方哲学基础 |                       深受希腊哲学影响                       |
| 文学 |              《荷马史诗》、悲剧与喜剧              |                   西塞罗的散文、维吉尔的诗                   |
| 法律 |                         —                          | **《十二铜表法》**，是古罗马第一部成文法；罗马法体系影响深远 |
| 建筑 |                     帕特农神庙                     |                       万神殿、大竞技场                       |

**「人是万物的尺度」**（普罗塔戈拉）体现了希腊人文精神。希腊文化崇尚 **人文主义**、理性与自由。

#### 中古西欧与拜占庭、俄罗斯文化

- **中古西欧**：**基督教** 会控制着人们的精神生活，教士是唯一有文化的阶层；出现骑士文学和市民文学；
- **拜占庭文化**：继承古希腊罗马文化，融汇基督教文化，**《查士丁尼法典》** 是罗马法的集大成之作；圣索菲亚大教堂宏伟壮观；
- **俄罗斯文化**：深受拜占庭文化影响，接受 **东正教**；创造以 **西里尔字母** 为基础的俄文；圆顶多塔风格的教堂别具特色。

### 南亚、东亚与美洲的文化

#### 古代印度文化

- **宗教**：**佛教**（公元前 6 世纪产生）、**印度教**（4 世纪兴起）；
- **等级**：**种姓制度**，把人分为婆罗门、刹帝利、吠舍、首陀罗四个等级；
- **文学**：史诗《摩诃婆罗多》和《罗摩衍那》；
- **数学**：发明了包括「0」在内的 **十个数字符号**，后经阿拉伯人传播，称为「阿拉伯数字」；
- **外传**：以佛教为载体，广泛影响东南亚、东亚。

#### 古代朝鲜与日本文化

- **朝鲜**：7 世纪修建 **瞻星台** 观测天象；15 世纪创制本民族文字；史书《三国史记》；深受中华文化影响；
- **日本**：神道逐渐演变，武士道形成；《源氏物语》是杰出的文学作品；建筑、绘画、法隆寺深受唐文化影响；大和绘、浮世绘是极具特色的美术作品。

#### 美洲印第安文化

美洲印第安人独立创造了灿烂的文化，因与其他大陆几乎隔绝，独立发展。

|     文明     |    区域    |                                    成就                                     |
| :----------: | :--------: | :-------------------------------------------------------------------------: |
|   **玛雅**   |   中美洲   | 城市建筑精美；发明独特文字；创造 **玛雅历**；采用 **20 进位制**，知道「零」 |
| **阿兹特克** |   墨西哥   |               太阳金字塔；使用图画文字；「浮动园地」扩大耕地                |
|   **印加**   | 南美安第斯 |           修建完善的道路系统；使用 **结绳记事**（奇普）；建太阳庙           |

新航路开辟后，欧洲殖民者的入侵使印第安文化遭到毁灭性打击。

## 第三单元 人口迁徙、文化交融与认同

### 古代人类的迁徙和区域文化的形成

#### 印欧人的迁徙

约公元前 3000 年，印欧人以马和马拉战车为工具，进行了长达数千年的迁徙。

|   分支   |   去向   |                结果                |
| :------: | :------: | :--------------------------------: |
|  赫梯人  | 小亚细亚 |   建立赫梯帝国，率先掌握冶铁技术   |
|  波斯人  |   伊朗   |   建立波斯帝国，融合西亚北非文明   |
|  希腊人  |  巴尔干  |      成为古代希腊文化的开创者      |
| 雅利安人 |   南亚   | 建立一系列国家，创立种姓制度和佛教 |

#### 亚欧游牧民族大迁徙

3 – 6 世纪，游牧民族大迁徙深刻改变了亚欧大陆的政治与文化格局。

| 区域 |                               表现                                |
| :--: | :---------------------------------------------------------------: |
| 中国 | 匈奴、鲜卑、羯、氐、羌 **五胡内迁**，推动民族交融，北魏孝文帝改革 |
| 欧洲 |      **日耳曼人** 灭亡西罗马帝国，在其废墟上建立起一系列王国      |

**共同影响**：区域文化不断交汇、发展；游牧世界与农耕世界碰撞，各个区域的文化在碰撞、交流与交融中形成和发展。

### 近代殖民活动和人口的跨地域转移

#### 殖民扩张与美洲族群变化

新航路开辟后，殖民扩张造成美洲人口结构的剧变。

- **印第安人锐减**：屠杀、奴役、传染病使印第安人大量死亡；
- **黑人奴隶输入**：**黑奴贸易** 把大批非洲黑人贩卖到美洲，成为奴隶；
- **白人移入**：大量欧洲人移居美洲；
- **族群混合**：白人、黑人、印第安人及混血种人共处，形成新的族群和多元文化，**拉丁美洲** 成为世界上族群混合程度最高的地区之一。

#### 华工与亚洲移民

- **背景**：19 世纪黑奴贸易受到限制，殖民者转而招募华工、印度人等作为廉价劳动力；
- **华工出洋**：大批中国人被贩卖到美洲、大洋洲等地，从事开矿、修路、种植等艰苦劳动；
- **影响**：华工为当地开发作出巨大贡献，形成 **唐人街**，保留和传播中华文化，促进了当地多元文化的发展。

### 现代社会的移民和多元文化

#### 经济全球化下的移民

- **劳动力流动**：全球化推动劳动力大规模跨国流动，从 **发展中国家** 流向 **发达国家**；
- **移民类型变化**：20 世纪下半叶起，**知识精英** 的跨国流动加强，「难民」问题也日益突出；
- **移民社会**：不同文化背景的移民聚居，形成 **多元一体** 的移民社会，也带来文化认同的困扰。

#### 文化认同

- **美国**：以欧洲文化为主流，吸收其他民族文化，形成移民文化，被称为「大熔炉」，但族群间的社会问题始终存在；
- **新加坡**：马来人、华人、印度人、欧洲人共同组成多元文化社会，尊重不同民族的文化和宗教信仰，形成和谐共处的典范；
- **难题**：移民文化认同问题与本土居民和移民之间的矛盾，成为许多国家面临的挑战。

## 第四单元 商路、贸易与文化交流

### 古代的商路、贸易与文化交流

#### 丝绸之路

**丝绸之路** 是经中亚沟通中国与欧洲、贯通亚欧大陆的贸易与文化交流通道，因大量中国 **丝绸** 经此西运而得名。

- **开辟**：张骞出使西域，被称为「**凿空**」，大大推进了丝绸之路的畅通和贸易的繁荣；
- **繁荣**：汉唐时期最为繁荣，唐中期以后因战乱及经济重心南移而渐趋衰落。

#### 其他商路

|       商路       |               路线               |                                    内容                                    |
| :--------------: | :------------------------------: | :------------------------------------------------------------------------: |
| **草原丝绸之路** |   经蒙古草原、中亚北部通往欧洲   |                       匈奴、突厥等游牧民族沟通东西方                       |
| **西南丝绸之路** | 从关中平原入蜀，经云南出境入印度 |                         沟通中国西南与南亚、东南亚                         |
| **海上丝绸之路** |     从东南沿海经南海、印度洋     | 唐宋以来因造船、航海技术进步而繁荣，宋元时空前繁荣，明初郑和下西洋达到顶峰 |

#### 商路上的东西交流

商路是双向的，物质、技术、思想、艺术往来传播。

|     方向     |                                            内容                                            |
| :----------: | :----------------------------------------------------------------------------------------: |
| **中国传出** |                丝绸、瓷器、茶叶、漆器；养蚕缫丝、冶铁、造纸、印刷、火药技术                |
| **传入中国** | 汗血马、苜蓿、葡萄、胡瓜等物种；佛教、祆教、摩尼教等宗教；西亚、中亚的乐舞、杂技、绘画艺术 |

物质技术的西传，深刻影响了世界文明的进程；异域文化的东来，也丰富了中华文化的内涵。

### 近代以来的世界贸易与文化交流的扩展

#### 世界贸易与文化的全球化

- **形成**：新航路开辟后，全球贸易网络逐步形成；工业革命后，形成以 **欧美工业国** 为主导的资本主义世界市场；
- **深化**：两次世界大战使世界贸易一度受挫；二战后，**关贸总协定** 及后来的 **世界贸易组织** 推动贸易自由化，世界贸易突飞猛进。

#### 商品流动与文化交流

商品的世界性流动，把文化元素带到世界各地，促进了文化的交流与传播。

|     商品     |                          文化交流                           |
| :----------: | :---------------------------------------------------------: |
|   **茶叶**   |       中国茶及茶文化传向世界，英国形成「下午茶」习俗        |
|   **服饰**   | 中国服装元素传入法国等地；**中山装** 融汇中西，体现民族审美 |
|   **钟表**   |            明朝经澳门传入内地，融入中国审美情趣             |
| **电影电视** |  20 世纪以来，成为文化传播的重要载体，促成文化的全球性传播  |

## 第五单元 战争与文化交锋

战争是人类文明的浩劫，客观上又造成了不同文化的碰撞、交流与重构。

### 古代战争与地域文化的演变

#### 亚历山大远征与「希腊化时代」

- **远征**：公元前 4 世纪，**亚历山大** 灭亡波斯帝国，建立起地跨欧亚非三洲的大帝国；
- **希腊化时代**：从亚历山大远征到罗马征服托勒密埃及的约 300 年，被称为「**希腊化时代**」；
- **文化交融**：希腊文化与埃及、西亚等地文化交汇，形成新的文化局面，如亚历山大城的缪斯宫成为学术中心，欧几里得、阿基米德等作出重大贡献。

#### 蒙古西征与东西交流

13 世纪蒙古的三次西征，客观上推动了东西方的交流。

- 大批移民迁徙，如契丹人耶律楚材等；
- 东西方的贸易、技术、文化交流因驿道畅通而加强，中国的 **火药、印刷术、指南针** 西传欧洲；
- 阿拉伯、波斯的天文、历法、医药知识传入中国。

### 近代战争与西方文化的扩张

#### 拿破仑战争与欧洲文化重构

- **传播**：拿破仑战争把 **法国大革命** 的成果和资产阶级思想传播到欧洲各地；
- **《拿破仑法典》**：确立了资本主义社会的立法规范，成为许多国家立法的蓝本；
- **民族意识**：客观上促进了欧洲各国民族意识的觉醒，为民族国家的形成奠定基础。

#### 殖民战争与文化侵略

- **文化侵略**：欧洲列强凭借殖民战争，把西方文化强行移植到殖民地半殖民地，进行 **文化侵略**；
- **本土回应**：殖民地半殖民地人民既学习西方文化中的先进成分，又努力维护本民族文化，形成 **文化认同** 与 **文化重构**；
- 印度的甘地把印度传统文化与西方思想结合，中国的近代先进分子把马克思主义与中华文化结合。

### 现代战争与不同文化的碰撞和交流

#### 两次世界大战的影响

- **一战**：削弱了西方文化的影响力，促进了殖民地半殖民地的 **民族觉醒**；十月革命后，**社会主义** 文化广泛传播；
- **二战**：**世界殖民体系** 走向瓦解，战后新兴民族国家纷纷独立。

#### 战后新兴民族国家的文化发展

二战后独立的新兴民族国家，努力将 **本土文化** 与 **外来文化** 结合，探索独立自主的文化发展道路。

|       国家       |                                    文化特点                                    |
| :--------------: | :----------------------------------------------------------------------------: |
|     **印度**     | 政治体制学习英国；官方语言为印地语和英语；崇尚甘地思想，尊崇传统宗教与民族语言 |
| **新加坡、韩国** |     发扬 **儒家文化** 中的宽容、和谐等价值观，同时吸收西方的自由、平等观念     |
|     **埃及**     |                具有阿拉伯、伊斯兰特点，兼有欧洲和西亚的文化元素                |

## 第六单元 文化的传承与保护

### 文化传承的多种载体及其发展

#### 学校教育

- **中国古代**：**官学** 与 **私学** 并行；汉朝设 **太学**，西晋设 **国子监**；私学自 **孔子** 开始，宋代 **书院** 兴起，如白鹿洞书院；
- **西方**：古希腊产生欧洲最初的学校教育形式；中世纪出现 **大学**，如巴黎大学、牛津大学；
- **近代**：北京大学、清华大学等成为中国教育、科研的重镇；
- **作用**：学校教育在文化传承、科学研究、人才培养等方面发挥了重要作用。

#### 印刷书

- **纸的发明**：**东汉蔡伦** 改进造纸术，纸成为主要书写材料；
- **印刷术**：唐朝有 **雕版印刷** 品，北宋 **毕昇** 发明 **活字印刷术**；
- **西方**：约 15 世纪中叶，德国人 **谷登堡** 用金属活字印刷，推动了报纸、杂志的出现；
- **意义**：印刷书的普及大大有助于文化的传承与传播，堪称影响人类文明进程的伟大发明。

#### 图书馆与博物馆

|    载体    |                                                                               发展                                                                               |
| :--------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| **图书馆** |                古代西亚的泥板文书是最早的档案；中国古代官府、私家、书院藏书丰富；近代逐步发展为向公众开放，兼具保存 **文化遗产** 与服务公众的职能                |
| **博物馆** | 亚历山大城的缪斯宫被视为博物馆的开端；近代博物馆向公众开放，中国的 **南通博物苑**、**国立历史博物馆** 等相继建立，是征集、收藏、展示文物，进行科普教育的重要场所 |

### 文化遗产：全人类共同的财富

#### 文化遗产的保护

- **意义**：文化遗产是人类历史文化的载体，保护文化遗产有利于文化的传承与发展，有利于研究人类文明的演进；
- **中国保护**：古代通过 **金石学** 研究文物；现代颁布《中华人民共和国文物保护法》，建立起较完整的保护体系；
- **国际保护**：1972 年，联合国教科文组织通过 **《保护世界文化和自然遗产公约》**，建立 **《世界遗产名录》**，对世界文化和自然遗产进行保护。

#### 世界遗产

《保护世界文化和自然遗产公约》将世界遗产分为文化遗产、自然遗产、文化与自然双重遗产。此外还有 **非物质文化遗产**。

|        类型        |                        代表                        |
| :----------------: | :------------------------------------------------: |
|    **文化遗产**    | 中国的长城、莫高窟、明清故宫、秦始皇陵及兵马俑坑等 |
|    **自然遗产**    |                中国的黄龙、九寨沟等                |
|    **双重遗产**    |    中国的泰山、黄山、峨眉山—乐山大佛、武夷山等     |
| **非物质文化遗产** |           中国的昆曲、京剧、二十四节气等           |

- 中国是较早加入《保护世界文化和自然遗产公约》的国家之一，是世界遗产大国；
- 保护文化遗产的同时，要处理好 **保护** 与 **开发利用** 的关系，坚持保护第一。
