## 第一课 我国的生产资料所有制

### 公有制主体地位及其体现

#### 生产资料所有制的地位

**生产资料所有制** 是生产关系的核心，是区分不同社会形态的根本标志。谁占有生产资料，谁就在生产中处于支配地位。

我国的 **社会主义初级阶段**，与之相适应的 **基本经济制度** 包括三方面内容：

- **公有制为主体、多种所有制经济共同发展**；
- **按劳分配为主体、多种分配方式并存**；
- **社会主义市场经济体制**。

三者从所有制、分配、资源配置三个层面，共同构成社会主义经济制度的基础。所有制是分配和资源配置的前提。

#### 公有制经济的构成

**公有制经济** 包括 **国有经济**、**集体经济**，以及混合所有制经济中的 **国有成分和集体成分**。

|          类别          |              含义              |                 地位与作用                 |
| :--------------------: | :----------------------------: | :----------------------------------------: |
|      **国有经济**      | 生产资料归全体人民（全民）所有 | 国民经济的 **主导** 力量，控制国民经济命脉 |
|      **集体经济**      |  生产资料归部分劳动者共同所有  |   促进共同富裕，广泛吸纳社会资金和劳动力   |
| 混合所有制中的公有成分 |   国有资本、集体资本参股其中   |   放大国有资本功能，提高其竞争力和影响力   |

- **国有经济** 是国有企业和国有资本的统称。国有资产属于国家所有即全民所有，是全体人民共同的宝贵财富。它在关系国家安全、国民经济命脉的重要行业和关键领域占支配地位；
- **集体经济** 由部分劳动者共同占有生产资料，是公有制经济的重要组成部分；
- 发展 **混合所有制经济**，有利于国有资本放大功能、保值增值、提高竞争力。

#### 公有制的主体地位

公有制的 **主体地位** 主要体现在两方面：

- **公有资产在社会总资产中占优势**（就全国而言，不是每个地方、每个行业都占优势，是量的优势与质的提高的统一）；
- **国有经济控制国民经济命脉，对经济发展起主导作用**（主导作用主要体现在控制力上）。

:::tip

要分清「公有制的主体地位」与「国有经济的主导作用」：主体地位讲的是公有制整体在社会总资产中的优势，主导作用讲的是国有经济对国民经济的控制力。国有经济起主导作用，公有制才能主体地位稳固。

:::

#### 巩固和发展公有制经济

- 毫不动摇巩固和发展公有制经济；
- 推动 **国有资本和国有企业做强做优做大**，增强其竞争力、创新力、控制力、影响力、抗风险能力；
- 深化 **国有企业改革**，发展混合所有制经济，培育具有全球竞争力的世界一流企业；
- 保证 **国有资产保值增值**，防止国有资产流失。

### 多种所有制经济共同发展

#### 非公有制经济的构成

**非公有制经济** 包括 **个体经济**、**私营经济**、**外资经济** 等。

|     类别     |                            特点                            |
| :----------: | :--------------------------------------------------------: |
| **个体经济** | 以劳动者个人及家庭占有生产资料为基础，规模小，个体劳动为主 |
| **私营经济** |             以生产资料私人占有和雇佣劳动为基础             |
| **外资经济** |           外国投资者及港澳台投资者在我国境内投资           |

#### 非公有制经济的地位与作用

非公有制经济是 **社会主义市场经济的重要组成部分**，是 **推动经济社会发展的重要力量**。作用可概括为：

- 支撑经济增长、促进创新、扩大就业、增加税收的重要力量；
- 具有稳定预期、激发活力的重要作用。

#### 两个毫不动摇

**「两个毫不动摇」** 是基本经济制度的关键论断：

- **毫不动摇巩固和发展公有制经济**；
- **毫不动摇鼓励、支持、引导非公有制经济发展**。

|    对比    |                                   公有制经济                                   |          非公有制经济          |
| :--------: | :----------------------------------------------------------------------------: | :----------------------------: |
|  **地位**  |                                    主体地位                                    | 社会主义市场经济的重要组成部分 |
|  **态度**  |                             巩固和发展（毫不动摇）                             |  鼓励、支持、引导（毫不动摇）  |
| **共同点** | 都是社会主义市场经济的重要组成部分，都受法律保护，地位平等、公平竞争、相互促进 |

- 公有制经济和非公有制经济 **都是社会主义市场经济的重要组成部分，都是我国经济社会发展的重要基础**；
- 权利平等、机会平等、规则平等，各种所有制经济依法平等使用生产要素、公开公平公正参与市场竞争、同等受到法律保护。

> 坚持「两个毫不动摇」，把公有制经济巩固好、发展好，把非公有制经济引导好、发展好。

## 第二课 我国的社会主义市场经济体制

### 使市场在资源配置中起决定性作用

#### 合理配置资源的必要性

资源具有 **稀缺性**，人的需要无限，可用资源有限，所以要合理配置资源。**资源配置** 的两种基本手段：

|   手段   |              含义              |            局限            |
| :------: | :----------------------------: | :------------------------: |
| **市场** | 通过价格、供求、竞争等机制配置 | 存在自发性、盲目性、滞后性 |
| **计划** |   通过行政指令、计划安排配置   |  难以及时反映供求、成本高  |

社会主义市场经济，既发挥市场作用，又发挥计划（政府）作用，两种手段结合。

#### 市场配置资源的机制

在市场经济中，**价格、供求、竞争** 等因素形成 **市场机制**，引导资源流动。

- **价格是市场机制的核心**。价格随供求变化：供不应求，价格上涨；供过于求，价格下跌；
- 价格变动引导生产者和消费者调节，起到「无形的手」的作用——**价格波动传递信息、调节生产、实现资源配置**；
- 市场决定资源配置是市场经济的 **一般规律**。

#### 市场调节的优点与局限

|   角度   |                        表现                        |
| :------: | :------------------------------------------------: |
| **优点** | 灵敏反映供求，实现资源合理配置，激发活力、提高效率 |
| **弊端** |               自发性、盲目性、滞后性               |

- **自发性**：为追求自身利益，可能损害他人、社会利益，如假冒伪劣、坑蒙拐骗；
- **盲目性**：单个生产者无法掌握全社会信息，一哄而上或一哄而下；
- **滞后性**：市场调节是事后调节，从价格涨落到扩缩生产存在时间差。

弊端会导致 **资源浪费、经济波动、收入差距扩大** 等后果，仅靠市场解决不了，需要 **国家宏观调控**。

#### 规范市场秩序

- 良好的 **市场秩序** 依赖 **公平开放透明的市场规则** 来维护，包括市场准入规则、市场竞争规则、市场交易规则；
- 建立以 **道德** 为支撑、**法律** 为保障的 **社会信用制度**，是规范市场秩序的治本之策；
- 市场调节要有效发挥作用，市场主体须自觉遵守市场道德和诚信原则。

### 更好发挥政府作用

#### 社会主义市场经济体制的特征

社会主义市场经济，既有市场经济的 **一般特征**，又有 **自己的特点**。基本特征：

- 坚持 **公有制的主体地位**，是社会主义市场经济的基本标志；
- 坚持 **党的领导**，是中国特色社会主义制度的最大优势；
- 以 **人民为中心**、促进 **共同富裕**，是社会主义市场经济的根本目标；
- 能够实施 **科学的宏观调控**，是社会主义市场经济体制的内在要求。

社会主义市场经济体制能把 **社会主义制度的优越性** 同 **市场经济的长处** 有机结合起来。

#### 科学的宏观调控

**宏观调控**：国家运用各种手段对国民经济进行调节和控制。

- **主要目标**：促进经济增长、稳定物价、增加就业、保持国际收支平衡；
- **主要手段**：

|     手段     |                   内容                   |           特点           |
| :----------: | :--------------------------------------: | :----------------------: |
| **经济手段** | 经济政策（财政政策、货币政策）、经济计划 |  **主要手段**，间接调节  |
| **法律手段** |            经济立法、经济司法            |       规范经济行为       |
| **行政手段** |           行政命令、指示、规定           | 直接、见效快，但不能滥用 |

- 财政政策与货币政策是宏观调控最常用的经济手段，须综合运用、协调配合；
- 行政手段应符合经济规律，不能片面强调、以行政命令替代市场调节。

#### 更好发挥政府作用

- 科学的宏观调控和有效的政府治理，是发挥社会主义市场经济体制优势的内在要求；
- 政府的经济职能和作用主要是：保持宏观经济稳定，加强和优化公共服务，保障公平竞争，加强市场监管，维护市场秩序，推动可持续发展，促进共同富裕，弥补市场失灵。

#### 市场与政府的关系

|   对比   |        市场（市场机制）        |    政府（宏观调控）    |
| :------: | :----------------------------: | :--------------------: |
| **作用** | 在资源配置中起 **决定性** 作用 | **更好发挥** 政府作用  |
| **手段** |        价格、供求、竞争        |  经济、法律、行政手段  |
| **别称** |          「无形的手」          |      「有形的手」      |
| **定位** |      一般规律，基础性配置      | 弥补市场失灵，保驾护航 |

**必记结论**：使市场在资源配置中起决定性作用，**并不是起全部作用**；更好发挥政府作用，**不是取代市场、不是政府作用越大越好**。二者有机统一、相辅相成，不是相互否定。让 **看不见的手** 与 **看得见的手** 都用好，才能实现资源配置最优化和效益最大化。

## 第三课 我国的经济发展

### 坚持新发展理念

#### 以人民为中心的发展思想

- 发展是解决我国一切问题的基础和关键；
- **以人民为中心的发展思想**，是马克思主义政治经济学的根本立场，要坚持发展为了人民、发展依靠人民、发展成果由人民共享。

#### 新发展理念的内涵

**新发展理念**：**创新、协调、绿色、开放、共享**，是针对发展中的突出问题提出的，回答了「实现什么样的发展、怎样实现发展」。

|   理念   |       解决的问题       |                  着力点                  |
| :------: | :--------------------: | :--------------------------------------: |
| **创新** |   发展 **动力** 问题   | 把创新摆在核心位置，实施创新驱动发展战略 |
| **协调** |  发展 **不平衡** 问题  |      区域、城乡、经济与社会协调发展      |
| **绿色** | 人与自然 **和谐** 问题 |     绿水青山就是金山银山，可持续发展     |
| **开放** | 发展 **内外联动** 问题 | 主动参与全球经济合作，形成对外开放新格局 |
| **共享** | 社会 **公平正义** 问题 |  全民共享、全面共享、共建共享、渐进共享  |

- 新发展理念是一个 **整体**，相互贯通、相互促进，不能顾此失彼、相互替代；
- 贯彻新发展理念，必须坚持问题导向、系统观念。

### 建设现代化经济体系

#### 建设现代化经济体系的必要性

- 国家强，经济体系必须强。建设现代化经济体系，是跨越关口的迫切要求和我国发展的战略目标；
- 现代化经济体系是由 **社会经济活动各个环节、各个层面、各个领域** 相互关系和内在联系构成的一个有机整体。

#### 现代化经济体系的内容

统编教材从两方面把握——「一个体系、六个体系」与「一个市场体系」：

- 建设创新引领、协同发展的 **产业体系**；
- 建设统一开放、竞争有序的 **市场体系**；
- 建设体现效率、促进公平的 **收入分配体系**；
- 建设彰显优势、协调联动的 **城乡区域发展体系**；
- 建设资源节约、环境友好的 **绿色发展体系**；
- 建设多元平衡、安全高效的 **全面开放体系**；
- 建设充分发挥市场作用、更好发挥政府作用的 **经济体制**。

#### 建设现代化经济体系的着力点

- 大力发展 **实体经济**，筑牢现代化经济体系的坚实基础（把发展经济的着力点放在实体经济上）；
- 加快实施 **创新驱动发展战略**，强化现代化经济体系的战略支撑；
- 积极推动 **城乡区域协调发展**，优化现代化经济体系的空间布局；
- 着力发展 **开放型经济**，提高现代化经济体系的国际竞争力；
- 深化 **经济体制改革**，完善现代化经济体系的制度保障。

#### 推动高质量发展

- 我国经济已由 **高速增长阶段** 转向 **高质量发展阶段**；
- 贯彻新发展理念、建设现代化经济体系，是实现高质量发展的必然要求；
- 坚持 **质量第一、效益优先**，推动经济发展 **质量变革、效率变革、动力变革**。

|   对比   |   高速增长阶段   |     高质量发展阶段     |
| :------: | :--------------: | :--------------------: |
| **追求** |    速度、规模    |       质量、效益       |
| **方式** | 主要依靠要素投入 | 依靠创新、全要素生产率 |
| **导向** |      有没有      |         好不好         |

## 第四课 我国的个人收入分配与社会保障

### 我国的个人收入分配

#### 按劳分配为主体

**按劳分配** 是 **社会主义公有制** 经济中个人消费品分配的 **基本原则**，在所有分配方式中占 **主体地位**。

- **前提**：生产资料公有制；
- **物质基础**：社会主义公有制条件下生产力的发展水平；
- **直接原因**：社会主义条件下人们劳动的性质和特点（还存在旧的社会分工，劳动还是谋生手段）；
- **基本内容**：在公有制经济中，以劳动者向社会提供的劳动（数量和质量）为尺度分配个人消费品，多劳多得、少劳少得。

**意义**：

- 有利于充分调动劳动者的积极性和创造性，激励劳动者努力学习技术、提高劳动技能；
- 体现劳动者共同劳动、平等分配的社会地位，是消灭剥削和消除两极分化的重要条件。

:::tip

按劳分配只存在于 **公有制经济** 中。公有制经济内，工资、奖金、津贴属于按劳分配。而 **公务员、事业单位人员** 的工资也属于按劳分配。

:::

#### 多种分配方式并存

在社会主义初级阶段，除按劳分配外，还存在 **按生产要素分配**。生产要素包括 **劳动、资本、土地、知识、技术、管理、数据** 等。

|         分配方式         |                      范围与举例                      |
| :----------------------: | :--------------------------------------------------: |
|       **按劳分配**       | 公有制经济中，按劳动数量质量分配（工资、奖金、津贴） |
|    **按劳动要素分配**    |     非公有制经济中，劳动者获得的工资、奖金、津贴     |
|    **按资本要素分配**    |    存款利息、股息、债券收益、私营企业主的税后利润    |
|    **按土地要素分配**    |           出租土地、房屋获得的租金、转让金           |
| **按技术、知识要素分配** |             技术入股、专利转让获得的收入             |
|    **按管理要素分配**    |            管理者凭管理才能参与分配的收入            |
|    **按数据要素分配**    |            凭借数据这一生产要素获得的收入            |

- 让 **一切劳动、知识、技术、管理、资本、数据** 的活力竞相迸发；
- 让 **一切创造社会财富的源泉** 充分涌流。

**注意区分**：公有制经济里劳动者的工资是按劳分配；非公有制（私营、外资）企业里劳动者的工资是 **按劳动要素分配**，两者不能混淆。

#### 完善个人收入分配

**处理好效率与公平的关系**——效率与公平各有侧重，二者辩证统一：

- **效率** 是公平的物质前提，公平是提高效率的保证；
- 初次分配和再分配都要 **兼顾效率和公平**。

完善个人收入分配的措施：

- **理顺分配关系**：坚持按劳分配为主体、多种分配方式并存，完善各类生产要素按贡献参与分配的机制；
- **在经济增长的同时实现居民收入同步增长**，在劳动生产率提高的同时实现劳动报酬同步提高（「两个同步」）；
- **初次分配**：着重保护劳动所得，增加劳动者特别是一线劳动者劳动报酬，提高劳动报酬在初次分配中的比重；
- **再分配**：健全以 **税收、社会保障、转移支付** 为主要手段的再分配调节机制，加大调节力度；
- **第三次分配**：重视发挥 **慈善** 等第三次分配作用，发展慈善事业；
- 规范收入分配秩序，鼓励勤劳致富，保护合法收入，增加低收入者收入，扩大中等收入群体，调节过高收入，取缔非法收入。

|      环节      |            主体 / 手段             |          目标          |
| :------------: | :--------------------------------: | :--------------------: |
|  **初次分配**  | 市场（企业、个人、政府按要素分配） |   效率优先，兼顾公平   |
|   **再分配**   |    政府（税收、社保、转移支付）    | 更加注重公平，缩小差距 |
| **第三次分配** |      社会（慈善、志愿、捐赠）      | 道德力量推动，补充调节 |

### 我国的社会保障

#### 社会保障的功能

**社会保障** 通过国民收入的再分配，为社会成员的基本生活提供保障，是保障和改善民生、维护社会公平、增进人民福祉的基本制度保障。功能可概括为：

- **「减震器」**：化解社会成员因年老、失业、疾病等面临的生存危机，充当社会运行的稳定器；
- **调节收入分配**：通过再分配缩小收入差距，维护社会公平；
- **推动经济发展**：既是「安全网」也是「助推器」，通过合理引导消费预期、扩大内需、优化资源配置，促进经济社会持续健康发展。

#### 社会保障的形式

我国 **多层次的社会保障体系** 主要包括：

|     形式     |                               含义与举例                               |   特点   |
| :----------: | :--------------------------------------------------------------------: | :------: |
| **社会保险** | 保障范围最广、地位最重要的形式，包括养老、医疗、失业、工伤、生育五大险 |   核心   |
| **社会救助** |    对生活陷入困境者给予物质帮助（最低生活保障、特困供养、灾害救助）    |   底线   |
| **社会福利** |   政府和社会为改善生活提供的公共服务（残疾人、老年人、妇女儿童福利）   | 最高层次 |
| **社会优抚** |            对军人及其家属予以优待、抚恤（军烈属、伤残军人）            | 特定对象 |

- **社会保险** 是社会保障体系的 **核心**，覆盖面最广；
- **社会救助** 是最先形成、历史最悠久的社会保障形式，是 **最基础** 的一道防线。

#### 完善社会保障体系

- **公平** 是社会保障体系建设的 **价值取向**：既要 **尽力而为**，又要 **量力而行**（既不「养懒汉」，也要覆盖到需要保障的人）；
- 建立起 **覆盖全民、城乡统筹、权责清晰、保障适度、可持续** 的多层次社会保障体系；
- 要 **公平对待每个社会成员的社会保障权益**，同时 **恪守权利与义务相统一** 的原则；
- 既要防止社会保障不足（不能保障社会成员基本生活），又要防止过度保障（脱离经济发展水平、超越承受能力）。

> 社会保障要坚持人人尽责、人人享有，既不「越位」也不「缺位」。
