## 第一课 历史和人民的选择

### 中华人民共和国成立前各种政治力量

近代中国的两大历史任务是 **争取民族独立、人民解放** 和 **实现国家富强、人民幸福**。前者是后者的前提。

鸦片战争后，为完成这两大任务，不同政治力量给出不同方案：

|    政治力量    |        方案        |       结果       |
| :------------: | :----------------: | :--------------: |
| 地主阶级洋务派 |      自强求富      |   甲午战败破产   |
|    农民阶级    |  太平天国、义和团  | 无法摆脱失败结局 |
| 资产阶级改良派 | 维新变法，君主立宪 |   戊戌政变失败   |
| 资产阶级革命派 | 辛亥革命，民主共和 | 果实被袁世凯窃取 |

结论：**资本主义道路在中国走不通**。地主阶级、农民阶级、资产阶级都不能领导中国革命走向胜利。历史证明，中国需要新的领导力量和新的思想指引。

### 中国共产党的诞生

- **十月革命** 送来了马克思列宁主义，为中国送来了科学的思想武器；
- **五四运动** 促进了马克思主义在中国的传播，工人阶级登上历史舞台；
- 1921 年，中国共产党成立。**这是开天辟地的大事变**，中国革命的面貌焕然一新。

「新」在何处：

- 新的领导核心：中国共产党；
- 新的指导思想：马克思列宁主义；
- 新的奋斗目标：实现共产主义。

### 没有共产党就没有新中国

**中国共产党的领导是历史和人民的选择。**

- 完成新民主主义革命，建立中华人民共和国，实现了民族独立、人民解放；
- 完成社会主义革命，确立社会主义基本制度，为发展奠定根本政治前提和制度基础；
- 进行改革开放，开创、坚持、发展中国特色社会主义。

近代以来，各种政治力量轮番登场，唯有中国共产党团结带领人民完成了历史任务。**没有共产党，就没有新中国，就没有中华民族伟大复兴。**

## 第二课 中国共产党的先进性

### 始终坚持以人民为中心

党的性质：中国共产党是 **中国工人阶级的先锋队**，同时是 **中国人民和中华民族的先锋队**。这一「两个先锋队」的定位，决定了党没有自己的特殊利益。

- **党的根本立场**：人民立场；
- **党的初心和使命**：为中国人民谋幸福，为中华民族谋复兴；
- **党的根本宗旨**：全心全意为人民服务；
- **群众路线**：一切为了群众，一切依靠群众，从群众中来，到群众中去。这是党的生命线和根本工作路线。

「以人民为中心」不是抽象口号，落在 **发展为了人民、发展依靠人民、发展成果由人民共享** 三层上。

### 始终走在时代前列

党的先进性还体现在 **指导思想与时俱进**。马克思主义中国化时代化产生了系列理论成果：

|             理论成果             |                      解决的核心问题                      |
| :------------------------------: | :------------------------------------------------------: |
|            毛泽东思想            |        什么是新民主主义革命、怎样进行社会主义革命        |
|            邓小平理论            |             什么是社会主义、怎样建设社会主义             |
|       「三个代表」重要思想       |                建设什么样的党、怎样建设党                |
|            科学发展观            |                实现什么样的发展、怎样发展                |
| 习近平新时代中国特色社会主义思想 | 新时代坚持和发展什么样的中国特色社会主义、怎样坚持和发展 |

习近平新时代中国特色社会主义思想是 **当代中国马克思主义、二十一世纪马克思主义**，是全党全国人民为实现中华民族伟大复兴而奋斗的行动指南。

党要管党、全面从严治党，是保持先进性和纯洁性的必然要求。**打铁必须自身硬。**

## 第三课 坚持和加强党的全面领导

### 党的领导是最本质的特征和最大优势

**中国共产党领导是中国特色社会主义最本质的特征，是中国特色社会主义制度的最大优势，党是最高政治领导力量。**

- 最本质的特征：从「本质」层面界定中国特色社会主义姓什么；
- 最大优势：从「优势」层面说明制度为何管用；
- 最高政治领导力量：从「地位」层面明确党在国家治理中的位置。

「党政军民学，东西南北中，党是领导一切的。」这里的「领导一切」指 **总揽全局、协调各方**，不是包办代替具体事务。

### 坚持党的领导

党的领导是 **全面的、系统的、整体的**，涵盖各个领域、各个方面、各个环节。

坚持党对一切工作的领导，首先要坚持 **党中央权威和集中统一领导**，做到「两个维护」：维护习近平总书记核心地位、维护党中央权威和集中统一领导。

- **党的领导方式**：政治领导、思想领导、组织领导；
- 三种领导方式分别管方向、管理论、管队伍，不可偏废。

### 全面从严治党

不忘初心、牢记使命，必须以 **自我革命** 精神推进党的建设。

新时代党的建设总要求以 **政治建设为统领**，全面推进政治建设、思想建设、组织建设、作风建设、纪律建设，把 **制度建设** 贯穿其中，深入推进反腐败斗争。

:::tip

区分党的领导地位的三重表述：**最本质特征**（谈本质）、**最大优势**（谈优势）、**最高政治领导力量**（谈地位）。三者角度不同，不能混用。

## 第四课 人民民主专政的社会主义国家

### 国家的本质

**国家是经济上占统治地位的阶级进行阶级统治的工具，本质是阶级统治的工具。**

- **民主** 与 **专政** 相互依存：民主是在一定阶级范围内、按平等原则和少数服从多数原则管理国家；专政是统治阶级凭借国家机器实行的强制统治；
- 民主具有 **鲜明的阶级性**，从来没有超阶级的、全民的民主。

### 我国的国体

**中华人民共和国是工人阶级领导的、以工农联盟为基础的人民民主专政的社会主义国家。**

- **国体**：即国家性质，回答「谁掌握国家政权」；
- **领导阶级**：工人阶级；
- **阶级基础**：工农联盟；
- **本质**：**人民当家作主**。

人民民主专政的最大特点是对占人口绝大多数的人民实行 **民主**，对少数敌对势力实行 **专政**。人民民主具有 **广泛性**（主体广、权利广）和 **真实性**（有制度、法律、物质保障）。

|   概念   |          内涵          |
| :------: | :--------------------: |
|   国体   |    国家性质，谁掌权    |
|   政体   | 政权组织形式，怎样掌权 |
| 国家职能 |  政治职能、经济职能等  |

### 坚持人民民主专政

坚持人民民主专政作为 **四项基本原则** 之一，被写入宪法，是我国的立国之本。

新时代坚持人民民主专政，重点在 **发展社会主义民主、健全社会主义法治**，扩大人民有序政治参与，同时依法打击各种危害国家安全的行为。**民主和专政统一于人民当家作主的实践。**

## 第五课 我国的根本政治制度

### 人民代表大会

**人民行使国家权力的机关是全国人民代表大会和地方各级人民代表大会。**

- **全国人大** 是最高国家权力机关，在国家机构中居于 **最高地位**；
- 地方各级人大是地方国家权力机关。

人大代表由民主选举产生，代表人民的利益和意志行使权力。人大代表的权利与义务：

- **权利**：审议权、表决权、提案权、质询权；
- **义务**：协助宪法和法律的实施，与人民群众保持密切联系，听取和反映意见和要求，接受人民监督。

全国人大及其常委会行使 **立法权、决定权、任免权、监督权** 四项职权。

### 人民代表大会制度

**人民代表大会制度是我国的根本政治制度，是我国的政体，是实现人民当家作主的根本途径和最高实现形式。**

组织和活动的最重要特点是 **民主集中制**：

|        关系        |                     民主集中制的体现                     |
| :----------------: | :------------------------------------------------------: |
|     人大与人民     |      人大代表由民主选举产生，对人民负责，受人民监督      |
| 人大与其他国家机关 | 行政、监察、审判、检察机关由人大产生，对它负责，受它监督 |
|     中央与地方     |        在中央统一领导下，充分发挥地方主动性积极性        |

人民代表大会制度的优越性：保障人民当家作主，动员全体人民投身建设，保证国家机关协调高效运转，维护国家统一和民族团结。**坚持好、完善好、运行好人民代表大会制度**，把制度优势转化为治理效能。

## 第六课 我国的基本政治制度

我国有三项基本政治制度：中国共产党领导的多党合作和政治协商制度、民族区域自治制度、基层群众自治制度。

### 中国共产党领导的多党合作和政治协商制度

我国的政党制度不是一党制，也不是多党制，而是 **中国共产党领导的多党合作和政治协商制度**。

- **首要前提和根本保证**：坚持中国共产党的领导；
- **通力合作的关系**：中国共产党是执政党，各民主党派是 **参政党**，二者是 **亲密友党**，不是在野党、反对党；
- **共同思想政治基础**：坚持中国特色社会主义。

参政党的「参政」内容：参加国家政权，参与大政方针和国家领导人选的协商，参与国家事务管理，参与方针政策、法律法规的制定执行。

**人民政协** 是发扬社会主义民主的重要形式，主题是 **团结** 和 **民主**，主要职能是 **政治协商、民主监督、参政议政**。政协不是国家机关，也不是权力机关，只有建议权，没有决定权。

### 民族区域自治制度

我国处理民族关系的基本原则是 **民族平等、民族团结、各民族共同繁荣**。社会主义民族关系是 **平等团结互助和谐**。

**民族区域自治制度** 是在国家统一领导下，各少数民族聚居的地方实行区域自治，设立自治机关，行使自治权。

- **前提和基础**：国家统一领导，自治地方是国家不可分离的部分；
- **自治机关**：自治地方的人民代表大会和人民政府（不含法院、检察院）；
- **核心内容**：**自治权**；
- **三级自治地方**：自治区、自治州、自治县。

实行民族区域自治，是由 **统一多民族国家的历史传统** 和 **「大杂居、小聚居」的民族分布特点** 决定的。它把国家统一与民族自治、民族因素与区域因素结合起来，是符合国情的制度安排。

### 基层群众自治制度

**基层群众自治制度是我国的一项基本政治制度，是社会主义民主政治的基础。**

- **城市**：**居民委员会**；
- **农村**：**村民委员会**。

二者都是 **基层群众性自治组织**，不是国家机关，也不是一级政府。群众依法直接行使 **民主选举、民主协商、民主决策、民主管理、民主监督** 的权利，自我管理、自我服务、自我教育、自我监督。

发展基层民主是社会主义民主 **最广泛的实践**，是人民当家作主最有效、最广泛的途径。

|    组织    |    范围    |        性质        |
| :--------: | :--------: | :----------------: |
| 村民委员会 |    农村    | 基层群众性自治组织 |
| 居民委员会 |    城市    | 基层群众性自治组织 |
|  人民政府  | 各级行政区 |    国家行政机关    |

## 第七课 治国理政的基本方式

### 法的产生与本质

- **法** 是人类社会发展到一定历史阶段的产物，随国家的产生而产生；
- **法的本质**：由社会 **物质生活条件** 决定，反映 **统治阶级的根本利益和共同意志**，是维护、巩固和发展有利于统治阶级的社会关系和社会秩序的工具。

法的两大基本特征：

- **由国家制定或认可**：这是法区别于其他行为规范的重要标志；
- **由国家强制力保证实施**：具有强制性，任何人违法都要受到追究。

法的职能：**政治职能**（维护阶级统治）和 **社会职能**（管理公共事务）。

### 我国法治建设的成就

- 形成以 **宪法** 为核心的中国特色社会主义法律体系；
- 依法行政和公正司法水平不断提高；
- 人权的法治保障取得巨大成就。

**全面依法治国** 是国家治理的一场深刻革命，关系党执政兴国，关系人民幸福安康，关系党和国家长治久安。**法治是治国理政的基本方式。**

我国走的是 **中国特色社会主义法治道路**，核心要义是坚持党的领导、坚持中国特色社会主义制度、贯彻中国特色社会主义法治理论。**党的领导是社会主义法治最根本的保证。**

## 第八课 法治中国建设

全面依法治国是一个系统工程，总目标是 **建设中国特色社会主义法治体系、建设社会主义法治国家**，抓手是 **科学立法、严格执法、公正司法、全民守法**。

### 法治国家

**法治国家** 就是实行 **依法治国、依宪治国**，用法律治理国家，法律在国家和社会治理中具有权威地位。

- **前提**：坚持宪法法律至上，法律面前人人平等；
- **要求**：良法之治，尊重和保障公民权利，规范国家权力运行。

建设法治国家，要 **推进宪法实施、建立完备的法律体系、完善法律实施机制**。法治国家意味着国家权力受到法律约束，公民权利得到有效保障。

### 法治政府

**法治政府** 就是职能科学、权责法定、执法严明、公开公正、智能高效、廉洁诚信、人民满意的政府。

- **核心**：依法行政，把权力关进制度的笼子；
- **底线**：**法定职责必须为，法无授权不可为**；
- 措施：转变政府职能，健全权责统一的行政决策机制，全面推进政务公开。

### 法治社会

**法治社会** 是指法律得到 **普遍公认和遵从**、社会治理依法开展、公共生活和谐有序的社会。

- 建设要求：**信仰法律、遵守法律、维护法律、运用法律**；
- 弘扬社会主义法治精神，建设社会主义法治文化，增强全社会法治观念。

|   层面   |        核心        |            关键表述            |
| :------: | :----------------: | :----------------------------: |
| 法治国家 | 依法治国、依宪治国 |     宪法法律至上，良法之治     |
| 法治政府 |      依法行政      | 法定职责必须为，法无授权不可为 |
| 法治社会 |      全民守法      |      法律被普遍公认和遵从      |

三者相互联系、相互支撑：**法治国家是目标，法治政府是主体和关键，法治社会是基础。** 三位一体，共同推进。

## 第九课 全面推进依法治国的基本要求

全面推进依法治国要抓住 **科学立法、严格执法、公正司法、全民守法** 四个环节。四者贯穿立法、执法、司法、守法全过程，形成完整链条。

### 科学立法

**科学立法** 的要求是 **尊重和体现客观规律**，核心在于 **使法律准确反映经济社会发展要求，正确处理各种利益关系**。

- **科学**：符合国情和实际，遵循客观规律；
- **民主**：坚持人民主体地位，广泛听取意见，恰当处理各种利益关系；
- 立法要与改革发展决策相衔接，做到 **重大改革于法有据**。

### 严格执法

**严格执法** 要求 **严格规范公正文明** 执法。行政机关是执法主体，是实施法律法规的重要力量。

- **严格**：有法必依、执法必严、违法必究；
- **规范**：健全执法程序，规范执法行为；
- **公正**：平等对待行政相对人；
- **文明**：坚持理性、平和、文明执法。

### 公正司法

**公正司法** 是维护社会公平正义的最后一道防线。

- **公正是司法的生命线**；
- 要求：确保审判机关、检察机关 **依法独立公正行使审判权、检察权**，任何组织和个人都不得干预；
- 深化司法体制改革，让人民群众在每一个司法案件中感受到公平正义。

### 全民守法

**全民守法** 是指所有社会成员 **依法行使权利、履行义务**，成为法治的忠实崇尚者、自觉遵守者、坚定捍卫者。

- 关键：调动人民群众投身依法治国的积极性和主动性；
- 途径：加强宣传教育，建设法治文化，使尊法学法守法用法成为自觉。

|   环节   |   主体   |          核心要求          |
| :------: | :------: | :------------------------: |
| 科学立法 | 立法机关 | 尊重客观规律，反映人民意志 |

## 三者有机统一

**党的领导、人民当家作主、依法治国有机统一，是社会主义政治发展道路的必然要求。**

三者各有定位，缺一不可：

|     要素     |             地位             |            核心逻辑            |
| :----------: | :--------------------------: | :----------------------------: |
|   党的领导   |  最高政治领导力量，根本保证  | 把方向、谋大局、定政策、促改革 |
| 人民当家作主 | 社会主义民主政治的本质和核心 |        一切权力属于人民        |
|   依法治国   | 党领导人民治理国家的基本方式 |       依宪治国，依法办事       |

三者的内在关系：

- **党的领导是根本保证**：只有坚持党的领导，人民当家作主才能充分实现，依法治国才能有序推进；
- **人民当家作主是本质要求**：党的领导和依法治国，都是为了实现人民当家作主；
- **依法治国是基本方式**：党领导人民、人民当家作主，都要通过法治来落实和保障。

结论：**坚持党的领导，是人民当家作主和依法治国的根本保证；人民当家作主，是社会主义民主政治的本质特征；依法治国，是党领导人民治理国家的基本方式。三者统一于中国特色社会主义伟大实践。**
