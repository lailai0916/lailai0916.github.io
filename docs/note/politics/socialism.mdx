## 第一课 社会主义从空想到科学、从理论到实践的发展

这一课讲人类社会形态更替的历史逻辑：为什么原始社会会解体，为什么资本主义终将被社会主义取代，以及社会主义如何从空想变为科学、从理论变为现实。核心是把握 **社会基本矛盾** 是社会发展的根本动力。

### 原始社会到资本主义的演进

#### 人类社会的基本矛盾

- **生产力与生产关系** 的矛盾；
- **经济基础与上层建筑** 的矛盾。

这两对矛盾贯穿人类社会始终，是社会发展的根本动力。**生产关系一定要适应生产力状况、上层建筑一定要适应经济基础状况**，是社会发展的普遍规律。生产关系包括三方面：生产资料归谁所有（决定性）、人们在生产中的地位和相互关系、产品如何分配。

#### 五种社会形态

人类社会依次经历五种社会形态，其更替由生产力发展推动，而非任何人的意志：

|     社会形态     |    生产资料所有制    |      阶级关系      |        社会性质        |
| :--------------: | :------------------: | :----------------: | :--------------------: |
|   **原始社会**   |   共同占有生产资料   |       无阶级       |     人类社会的童年     |
|   **奴隶社会**   |  奴隶主占有生产资料  |    奴隶主与奴隶    | 人类进入文明时代的门槛 |
|   **封建社会**   | 地主占有绝大部分土地 |     地主与农民     |    人身依附关系松弛    |
| **资本主义社会** |  资本家占有生产资料  | 资产阶级与无产阶级 |      雇佣劳动制度      |
| **共产主义社会** |    生产资料公有制    |       无阶级       |     人类最理想社会     |

#### 原始社会的解体

- 原始社会是 **无阶级、无剥削、无压迫** 的社会，实行 **生产资料氏族公有、共同劳动、平均分配**；
- 生产工具的改进（金属工具出现）使 **生产力发展**，个体劳动逐渐取代共同劳动，生产资料由氏族公有转向 **家庭私有**；
- **私有制** 的确立是原始社会解体的标志，土地变成私有财产是其最重要标志；
- 随之出现 **贫富分化**，产生 **阶级**，最终形成 **奴隶主阶级与奴隶阶级**，原始社会瓦解。

> 私有制、阶级和国家不是从来就有的，也不会永远存在。

#### 奴隶社会与封建社会

**奴隶社会**

- 主要矛盾：奴隶阶级与奴隶主阶级的矛盾；
- 奴隶毫无人身自由，劳动成果被奴隶主完全占有；
- **国家** 是阶级矛盾不可调和的产物，是阶级统治的工具，随奴隶制国家的产生，人类摆脱蒙昧野蛮、进入 **文明时代**；
- 尽管奴隶制度残酷，相对原始社会仍是历史的进步——推动了生产力发展。

**封建社会**

- 主要矛盾：农民阶级与地主阶级的矛盾；
- 地主阶级凭借占有土地、通过 **收取地租** 剥削农民，地租是主要剥削方式；
- 农民有一定人身自由，能相对自主地劳动，生产积极性高于奴隶，推动生产力进一步发展；
- 上层建筑上，地主阶级借助 **君主专制、等级制度和封建思想** 维护统治。

#### 资本主义社会

**资本主义生产关系的建立需要两个基本条件**

- 大批失去生产资料、有人身自由、能自由出卖劳动力的人；
- 开办资本主义企业所必需的大量货币当作资本。

**资本家剥削工人的秘密**

- 工人的劳动分为两部分：**必要劳动**（创造相当于劳动力价值的部分）与 **剩余劳动**（创造 **剩余价值** 的部分）；
- **剩余价值** 是由工人 **无偿** 创造、被资本家占有的那部分价值，是资本家剥削工人的秘密所在；
- 生产资料私有制是资本主义一切罪恶的总根源。

**经济危机**

- 基本特征：**生产过剩**——相对过剩，相对于劳动人民有支付能力的需求而言的过剩，并非绝对过剩；
- 直接原因：生产无限扩大的趋势与劳动人民有支付能力的需求相对缩小的矛盾；生产的社会化与生产资料资本主义私人占有之间的矛盾（**根本原因**）；
- 资本主义的 **基本矛盾** 是生产社会化与生产资料资本主义私人占有之间的矛盾，是资本主义社会 **一切矛盾和冲突的总根源**，在阶级关系上表现为无产阶级与资产阶级的对立。

> 资本主义基本矛盾是资本主义社会的固有矛盾，它的存在和发展决定了资本主义必然被社会主义所取代，但这一过程是漫长而曲折的。

### 科学社会主义的创立

#### 空想社会主义

- 空想社会主义是科学社会主义的 **思想来源**；
- 代表人物：圣西门、傅立叶、欧文；
- 进步性：对资本主义进行了揭露和批判，表达了对未来理想社会的诉求；
- 局限性：仅从 **理性、正义** 等原则出发揭露批判资本主义，没有揭示资本主义运行的特殊规律，没有看到 **广大人民群众** 特别是 **无产阶级** 的力量，没有找到 **进行社会变革的正确途径**，因而是 **空想**。

#### 科学社会主义的理论基石

- **两大发现**：**唯物史观** 和 **剩余价值学说**；
- 唯物史观揭示了人类社会发展的一般规律，剩余价值学说揭示了资本主义运行的特殊规律和资本家剥削工人的秘密；
- 这两大发现，使社会主义 **从空想变成科学**。

#### 《共产党宣言》

- 时间：1848 年；
- 意义：**科学社会主义诞生的标志**；
- 主要内容：
  - 分析了资本主义生产方式的内在矛盾和人类社会的发展规律，科学论证了资本主义必然灭亡、社会主义必然胜利的历史必然性；
  - 阐述了 **无产阶级政党** 的性质、特点、目的和策略原则，第一次系统论述了无产阶级政党的学说；
  - 阐明了未来共产主义社会的理想目标——每个人的自由发展是一切人自由发展的条件。

|       对比       |   空想社会主义   |      科学社会主义      |
| :--------------: | :--------------: | :--------------------: |
|     理论依据     | 理性、正义等原则 | 唯物史观、剩余价值学说 |
| 对社会变革的认识 | 寄希望于开明人士 |    依靠无产阶级斗争    |
|     是否科学     |       空想       |          科学          |

### 社会主义从一国到多国

#### 从理论到实践

- **十月革命**（1917 年）：建立了世界上第一个社会主义国家，实现了社会主义 **从理论到实践** 的伟大飞跃；
- 十月革命的胜利，为世界上其他国家的无产阶级革命树立了榜样，开创了世界历史的新纪元。

#### 从一国到多国

- 第二次世界大战后，一批国家走上社会主义道路，社会主义 **从一国到多国**，形成了世界社会主义阵营；
- 20 世纪 80 年代末 90 年代初，东欧剧变、苏联解体，世界社会主义遭受严重曲折，但这只是 **社会主义发展进程中的曲折**，没有改变社会主义终将代替资本主义的历史大趋势。

#### 社会主义的发展历程

|       阶段       |         标志性事件         |     时间     |
| :--------------: | :------------------------: | :----------: |
|   空想社会主义   | 圣西门、傅立叶、欧文的学说 | 16 – 19 世纪 |
| **从空想到科学** |     《共产党宣言》发表     |   1848 年    |
| **从理论到实践** |          十月革命          |   1917 年    |
| **从一国到多国** |      社会主义阵营形成      |    二战后    |

> 社会主义经历了从空想到科学、从理论到实践、从一国到多国的发展进程，历史和现实告诉我们，社会主义必将取代资本主义，这是历史发展的总趋势，尽管这是一个漫长而曲折的过程。

## 第二课 只有社会主义才能救中国

这一课回答一个历史结论：**近代中国的各种救国方案都失败了，只有社会主义才能救中国**。中国走上社会主义道路，是历史和人民的选择，是历史发展的必然。

### 新民主主义革命的胜利

#### 近代中国的社会性质与主要矛盾

- **1840 年鸦片战争** 后，中国逐步沦为 **半殖民地半封建社会**，这是近代中国的基本国情；
- **两对主要矛盾**：帝国主义与中华民族的矛盾、封建主义与人民大众的矛盾。前者是最主要的矛盾；
- **两大历史任务**：**争取民族独立、人民解放**（前提），**实现国家富强、人民幸福**。

#### 各种救国方案的探索与失败

近代以来，为改变中华民族的命运，各阶级、各种政治力量反复探索，但都未能改变旧中国的社会性质和人民的悲惨命运：

|  力量 / 方案   |       代表       | 结果 |
| :------------: | :--------------: | :--: |
|    农民阶级    | 太平天国、义和团 | 失败 |
|    地主阶级    |     洋务运动     | 失败 |
| 资产阶级改良派 |     戊戌变法     | 失败 |
| 资产阶级革命派 |     辛亥革命     | 失败 |

**根本原因**：这些方案没有触及封建土地制度、没有依靠广大人民群众，资本主义道路在半殖民地半封建的中国走不通。

> 资本主义道路在中国走不通，不是中国人民不想走，而是走不通。帝国主义决不允许中国通过发展资本主义强大起来，成为它的竞争对手。

#### 新民主主义革命

- **五四运动**（1919 年）标志着中国 **新民主主义革命的开端**，工人阶级登上政治舞台；
- **中国共产党成立**（1921 年）是开天辟地的大事变，中国革命的面貌焕然一新；
- 新旧民主主义革命的区别，关键在 **领导阶级**：新民主主义革命由 **无产阶级** 领导，指导思想是 **马克思主义**；
- 经过 28 年浴血奋战，1949 年 **中华人民共和国成立**，标志着新民主主义革命取得基本胜利。

**新中国成立的意义**

- 实现了 **民族独立、人民解放**，完成了近代以来第一大历史任务；
- 彻底结束了旧中国半殖民地半封建社会的历史，结束了极少数剥削者统治广大劳动人民的历史；
- 实现了 **中国从几千年封建专制政治向人民民主的伟大飞跃**；
- 为实现由 **新民主主义向社会主义过渡** 创造了前提条件。

### 社会主义制度在中国的确立

#### 过渡时期与总路线

- **过渡时期**：从 1949 年新中国成立到 1956 年社会主义改造基本完成；
- **过渡时期总路线**：「**一化三改**」——逐步实现国家的社会主义 **工业化**（主体），逐步实现国家对 **农业、手工业、资本主义工商业** 的社会主义 **改造**（两翼）；
- 特征：**发展生产力** 与 **变革生产关系** 有机统一。

#### 社会主义改造的完成

- **1956 年**，社会主义改造 **基本完成**，标志着 **社会主义制度在中国确立**，我国进入 **社会主义社会**；
- 这是中国历史上 **最深刻最伟大的社会变革**，是中华民族发展史上开天辟地的大事变；
- 社会主义制度的确立，为当代中国一切发展进步 **奠定了根本政治前提和制度基础**。

#### 社会主义建设的探索

- 完成社会主义改造后，我国开始 **全面的大规模的社会主义建设**；
- 取得独创性理论成果和巨大成就，也经历过严重曲折（如「大跃进」、人民公社化运动、「文化大革命」等失误）；
- 意义：在一穷二白的基础上建立起独立的比较完整的 **工业体系和国民经济体系**，为新时期开创中国特色社会主义提供了 **物质基础、宝贵经验和理论准备**。

> 只有社会主义才能救中国——这是近代以来中国历史发展的必然结论，是中国人民经过反复比较作出的郑重选择。

## 第三课 只有中国特色社会主义才能发展中国

这一课讲改革开放这场「新的伟大革命」如何深刻改变中国，以及中国特色社会主义道路如何被创立、发展和完善。

### 伟大的改革开放

#### 历史转折

- **1978 年 12 月，党的十一届三中全会** 召开，实现了 **伟大的历史转折**：
  - 重新确立 **解放思想、实事求是** 的思想路线；
  - 停止使用「以阶级斗争为纲」的口号，把党和国家的工作中心 **转移到经济建设上来**；
  - 作出 **实行改革开放** 的历史性决策；
- 十一届三中全会是新中国成立以来党的历史上具有 **深远意义的伟大转折**，开启了 **改革开放和社会主义现代化建设新时期**。

#### 改革开放的进程

|     阶段     |                       标志性节点                        |
| :----------: | :-----------------------------------------------------: |
|     起步     |      1978 年十一届三中全会；农村家庭联产承包责任制      |
| 对外开放突破 |              1980 年设立深圳等四个经济特区              |
|   目标确立   | 1992 年南方谈话、党的十四大提出建立社会主义市场经济体制 |
|   全面深化   |       2013 年十八届三中全会作出全面深化改革的决定       |

- 改革从 **农村** 到 **城市**，从经济体制改革到各方面体制改革；
- 对外开放从 **经济特区** 到 **沿海开放城市**，形成 **全方位、多层次、宽领域** 的对外开放格局；
- 2001 年加入世界贸易组织（WTO），对外开放进入新阶段。

#### 改革开放的意义

- 改革开放是决定当代中国命运的 **关键一招**，是当代中国最鲜明的特色；
- 是党和人民 **大踏步赶上时代** 的重要法宝；
- 极大解放和发展了社会生产力，使中国大踏步赶上时代，实现了 **从站起来到富起来** 的伟大飞跃。

### 中国特色社会主义的创立、发展和完善

#### 中国特色社会主义的开创

- **1982 年，党的十二大** 上，邓小平明确提出「**建设有中国特色的社会主义**」的重大命题；
- 中国特色社会主义是改革开放以来党的全部理论和实践的 **主题**。

#### 中国特色社会主义理论体系

改革开放以来，党在实践中不断推进理论创新，形成了 **中国特色社会主义理论体系**：

|               理论成果               |                         核心命题                         |            解决的问题            |
| :----------------------------------: | :------------------------------------------------------: | :------------------------------: |
|            **邓小平理论**            |             什么是社会主义、怎样建设社会主义             | 走自己的路，建设中国特色社会主义 |
|       **「三个代表」重要思想**       |                建设什么样的党、怎样建设党                |        加强和改进党的建设        |
|            **科学发展观**            |                实现什么样的发展、怎样发展                |     以人为本，全面协调可持续     |
| **习近平新时代中国特色社会主义思想** | 新时代坚持和发展什么样的中国特色社会主义、怎样坚持和发展 |       实现中华民族伟大复兴       |

- 这些理论成果 **一脉相承又与时俱进**，是马克思主义中国化的重大理论成果；
- 中国特色社会主义 **道路、理论、制度、文化** 四位一体，构成中国特色社会主义的完整内容——道路是实现途径，理论是行动指南，制度是根本保障，文化是精神力量。

> 只有中国特色社会主义才能发展中国。中国特色社会主义道路是实现社会主义现代化、创造人民美好生活的必由之路。

## 第四课 只有坚持和发展中国特色社会主义才能实现中华民族伟大复兴

这一课立足新时代，讲清习近平新时代中国特色社会主义思想的核心内容，以及实现中华民族伟大复兴的中国梦。

### 习近平新时代中国特色社会主义思想

#### 中国特色社会主义进入新时代

- **2017 年，党的十九大** 作出重大判断：**中国特色社会主义进入了新时代**；
- 「新时代」是我国发展新的历史方位，意味着中华民族迎来了从站起来、富起来到 **强起来** 的伟大飞跃。

#### 新时代我国社会主要矛盾

- **人民日益增长的美好生活需要** 和 **不平衡不充分的发展** 之间的矛盾；
- 变化的原因：我国稳定解决了十几亿人的温饱问题，人民需要从「物质文化需要」升级为「美好生活需要」，制约满足需要的主要因素从落后的社会生产变为 **不平衡不充分的发展**；
- 需要把握两个「没有变」：我国仍处于并将长期处于 **社会主义初级阶段** 的基本国情没有变，我国是 **世界最大发展中国家** 的国际地位没有变。

#### 习近平新时代中国特色社会主义思想

- 地位：是 **马克思主义中国化的最新成果**，是当代中国马克思主义、二十一世纪马克思主义，是全党全国人民为实现中华民族伟大复兴而奋斗的 **行动指南**；
- **「四个自信」**：道路自信、理论自信、制度自信、文化自信；
