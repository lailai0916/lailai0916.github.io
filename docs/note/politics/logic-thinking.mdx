## 第一单元 树立科学思维观念

### 走进思维世界

#### 思维的含义

**思维**有广义和狭义之分。

- **广义**：与意识同义，泛指人的一切精神活动；
- **狭义**：指认识过程的理性阶段，即 **理性认识**。

本课在狭义上使用「思维」，它是对客观事物 **概括**、**间接** 的反映，是认识的高级阶段。感觉、知觉、表象属于感性认识；概念、判断、推理属于理性认识，是思维的基本形式。

#### 思维的特征

|   特征   |                     含义                     |
| :------: | :------------------------------------------: |
| **间接性** |  借助已有知识，从已知推断未知、把握本质与规律  |
| **概括性** |     抛开个别、非本质属性，抽取共同、本质属性     |
| **能动性** |    不是消极的镜像反映，能动地反映并指导实践    |

三者相互联系：概括性是间接性的前提，能动性贯穿始终。**正确的思维**如实反映对象，**错误的思维**则歪曲反映对象——思维本身有正误之分，这是可以训练、可以纠正的。

#### 思维的形态

按 **思维运行的基本单元** 划分，思维分为抽象思维和形象思维。

|   形态   |     基本单元     |         特征          |          示例          |
| :------: | :--------------: | :-------------------: | :--------------------: |
| **抽象思维** | 概念、判断、推理 | 基本单元的 **概念性** | 数学证明、法律条文推演 |
| **形象思维** | 感觉、知觉、表象 | 基本单元的 **形象性** |   文学创作、绘画构思   |

两者是同一思维过程中相互补充的两个方面，不是互不相干的两种思维。抽象思维侧重把握事物的 **共同属性和本质**，形象思维侧重把握事物的 **整体形象和外部特征**。

#### 科学思维

**科学思维** 泛指符合认识规律、遵循逻辑规则、达到正确认识结果的思维。它的三个基本条件——

- **内容真实**：从实际出发，反映事物的本来面目；
- **形式正确**：遵循逻辑思维的规则；
- **结论可靠**：能够正确认识事物、指导实践。

科学思维的特征是 **追求认识的客观性**、**结果具有预见性**、**结果具有可检验性**。学习科学思维，有助于把握事物的本质和规律、纠正错误认识、提升思维素养。

### 逻辑思维的基本要求

「逻辑」一词有多种用法：可指 **规律**（合乎逻辑）、**逻辑规律与规则**（合乎逻辑思维规则）、**认识问题的思维方法**、**逻辑学** 这门学问。本课讲的是 **狭义逻辑**，即研究思维形式结构及其规律的形式逻辑。

形式逻辑的基本规律有三条：同一律、矛盾律、排中律。三者是形式逻辑对思维一致性、确定性的最低要求。

|   规律   |     公式     |             要求             |            违反后的逻辑错误            |
| :------: | :----------: | :--------------------------: | :------------------------------------: |
| **同一律** |  $A\to A$  | 同一思维过程中，概念、判断保持同一 | **偷换概念**、**偷换论题**（转移论题） |
| **矛盾律** | $\lnot(A\land\lnot A)$ |  互相矛盾的思想不能同真，必有一假  |         **自相矛盾**（两可）          |
| **排中律** |  $A\lor\lnot A$  |   互相矛盾的思想不能同假，必有一真   |         **两不可**（骑墙居中）         |

三条规律辨析要点：

- **同一律** 反对在同一时间、同一方面、对同一对象的自由变换，不否认思维在不同条件下的发展变化；
- **矛盾律** 只针对 **互相矛盾**（或反对）的判断，如「这朵花是红的」与「这朵花不是红的」，不能同真；
- **排中律** 只针对 **互相矛盾** 的判断（一真一假），不针对 **反对关系**（可同假），如「全是」与「全不是」可以都错。

:::tip

矛盾律解决「不能同真」，排中律解决「不能同假」。反对关系（如「所有 S 是 P」与「所有 S 不是 P」）受矛盾律约束（不同真），但不受排中律约束（可同假），这是最易错的一处。

:::

### 正确运用概念

#### 概念的含义

**概念** 是反映事物 **本质属性** 的思维形式。属性有本质属性与非本质属性之分，概念反映的是决定事物之所以成为该事物的本质属性。

概念要用 **语词** 表达。同一概念可用不同语词表达（近义、方言、外语），同一语词也可表达不同概念（多义词）。语词是概念的语言形式，概念是语词的思想内容。

#### 概念的内涵与外延

|      |          含义          |            回答            |
| :--: | :--------------------: | :------------------------: |
| **内涵** |    概念所反映的本质属性    | 「是什么」（对象的性质）  |
| **外延** | 具有这些本质属性的对象范围 | 「有哪些」（对象的数量） |

以「商品」为例：内涵是「用于交换的劳动产品」，外延是「一切用于交换的劳动产品的总和」。

概念间的外延关系（$A$、$B$ 为两概念的外延）——

|    关系    |             外延情况              |       示例        |
| :--------: | :-------------------------------: | :---------------: |
| **全同关系** |      $A$ 与 $B$ 完全重合       |   等边三角形、等角三角形   |
| **属种关系** |    $A$ 真包含 $B$（$B\subsetneq A$）    |    水果、苹果     |
| **交叉关系** |    $A$ 与 $B$ 有部分重合    |     青年、党员     |
| **矛盾关系** | $A$、$B$ 互斥，且外延之和等于属概念 |     男人、女人     |
| **反对关系** | $A$、$B$ 互斥，但外延之和小于属概念 |     红色、白色     |

矛盾关系与反对关系都属于 **全异关系**，区别在于二者外延之和是否穷尽属概念：穷尽为矛盾，不穷尽为反对。

## 第二单元 遵循逻辑思维规则

### 准确把握概念

#### 明确内涵——定义

**定义** 是明确概念内涵的逻辑方法，最常用的是 **属加种差** 定义法：

$$
\text{被定义项}=\text{种差}+\text{邻近的属}
$$

例如「商品是用于交换的劳动产品」：邻近的属是「劳动产品」，种差是「用于交换」。

下定义的逻辑规则——

|         规则         |          违反后的错误           |
| :------------------: | :-----------------------------: |
| 定义项外延与被定义项外延 **全同** | 定义过宽 / 定义过窄 |
|   定义项不得直接或间接包含被定义项   |    同语反复 / 循环定义    |
|   定义一般不用否定形式    |          否定定义          |
|   定义要用清楚确切的科学术语   |    比喻定义（含混、比喻）    |

#### 明确外延——划分

**划分** 是明确概念外延的逻辑方法，把一个属概念分成若干种概念。划分由 **母项**（被划分的属概念）、**子项**（划分得到的种概念）、**划分标准** 组成。

划分的逻辑规则——

|         规则         |          违反后的错误           |
| :------------------: | :-----------------------------: |
| 子项外延之和必须等于母项外延 | 划分不全 / 多出子项 |
|   每次划分只能用同一标准   |        划分标准不一        |
|    子项外延互相排斥     |     子项相容（交叉划分）     |

例如把「三角形」按角分为锐角、直角、钝角三角形，标准统一、子项不重不漏；若再加「等腰三角形」，则标准不一且子项相容。

### 正确运用判断

#### 判断的含义

**判断** 是对认识对象有所 **断定** 的思维形式。判断有两个基本特征：**有所肯定或否定**、**有真假之分**。判断用语句表达，但并非所有语句都是判断——疑问句、祈使句、感叹句一般不表达判断。

按是否包含其他判断，判断分为 **简单判断** 和 **复合判断**。

#### 简单判断

简单判断包括 **性质判断**（直言判断）和 **关系判断**。

**性质判断** 断定对象具有或不具有某种性质，由 **量项 + 主项 + 联项 + 谓项** 构成。按质（联项肯定 / 否定）和量（量项全称 / 特称 / 单称）组合成六种基本类型，其中最重要的是四种直言判断：

|  类型  |    逻辑形式    |         示例         | 符号 |
| :----: | :------------: | :------------------: | :--: |
| 全称肯定 |  所有 S 是 P   |   所有金属都导电   | $A$ |
| 全称否定 | 所有 S 不是 P  | 所有恒星都不是行星 | $E$ |
| 特称肯定 |  有的 S 是 P   |   有的学生是团员   | $I$ |
| 特称否定 | 有的 S 不是 P  | 有的学生不是团员 | $O$ |

运用性质判断要注意：不能缺少主项和谓项，要准确使用量项和联项，避免主项与谓项配合不当。

**关系判断** 断定对象之间的关系，由 **关系者项 + 关系项 + 量项** 构成。关系的性质有——

|   性质   |             含义             |     示例      |
| :------: | :--------------------------: | :-----------: |
| **对称性** | $aRb$ 真，$bRa$ 也真 |    等于、相邻    |
| **反对称性** | $aRb$ 真，$bRa$ 必假 |    大于、小于    |
| **传递性** | $aRb$、$bRc$ 真，$aRc$ 真 |  大于、平行、包含  |
| **反传递性** | $aRb$、$bRc$ 真，$aRc$ 必假 |    比……大一岁    |

#### 复合判断

复合判断由 **联结词** 联结简单判断而成，主要有联言、选言、假言三种。真假由支判断的真假和联结词的性质共同决定。

**联言判断**：断定几种情况 **同时存在**，形式为「$p$ 并且 $q$」。**全真才真，一假即假**。

| $p$ | $q$ | $p\land q$ |
| :-: | :-: | :--------: |
| 真 | 真 |    真     |
| 真 | 假 |    假     |
| 假 | 真 |    假     |
| 假 | 假 |    假     |

**选言判断** 断定几种情况 **至少有一存在**，分相容与不相容——

- **相容选言**「$p$ 或者 $q$」：支判断可同真，**一真即真，全假才假**；
- **不相容选言**「要么 $p$，要么 $q$」：支判断有且仅有一真，**有且只有一个真时才真**。

| $p$ | $q$ | 相容 $p\lor q$ | 不相容 |
| :-: | :-: | :------------: | :----: |
| 真 | 真 |      真       |   假   |
| 真 | 假 |      真       |   真   |
| 假 | 真 |      真       |   真   |
| 假 | 假 |      假       |   假   |

**假言判断**（条件判断）断定事物情况之间的条件关系，形式为「若 $p$，则 $q$」，$p$ 是前件，$q$ 是后件。分充分、必要、充要三种——

|   类型   |       联结词        |       条件关系       |    唯一为假的情形    |
| :------: | :-----------------: | :------------------: | :------------------: |
| **充分条件** |    若……则…… ($p\to q$)    | 有前件必有后件，无前件未必无后件 |    前件真、后件假    |
| **必要条件** | 只有……才…… |  无前件必无后件，有前件未必有后件  | 前件假、后件真 |
| **充要条件** | 当且仅当…… ($p\leftrightarrow q$) | 有前件必有后件，无前件必无后件 |  前件与后件真假不同  |

充分条件假言判断 $p\to q$ 的真值表——

| $p$ | $q$ | $p\to q$ |
| :-: | :-: | :------: |
| 真 | 真 |   真    |
| 真 | 假 |   假    |
| 假 | 真 |   真    |
| 假 | 假 |   真    |

:::tip

假言判断只断定前件与后件之间的 **条件联系**，不断定前件、后件本身的真假。因此「若太阳从西边升起，则我请你吃饭」这个充分条件假言判断为真——前件假时整个判断都真。

:::

### 掌握演绎推理方法

**推理** 从已知判断（前提）推出新判断（结论）。按前提与结论断定范围的关系，推理分为——

- **演绎推理**：从一般到个别，前提蕴含结论，**前提真且形式有效则结论必然真**（保真）；
- **归纳推理**：从个别到一般；
- **类比推理**：从个别到个别（或一般到一般）。

后两者是 **或然推理**，结论超出前提断定范围，只是可能为真。

#### 推理的有效性

演绎推理的结论可靠，须同时满足两个条件：**前提真实**、**形式有效**。二者缺一，结论都不可靠。逻辑只保证「形式有效」，内容真假交由实践检验。

#### 换质推理与换位推理

对直言判断作 **直接推理**——

- **换质法**：改变判断的质（肯定变否定），谓项换成其矛盾概念。如「所有金属是导体」$\to$「所有金属不是非导体」；
- **换位法**：交换主项与谓项的位置，质不变。换位要遵守「前提中不周延的项，结论中不得周延」，故「所有 S 是 P」只能换成「有的 P 是 S」。

**周延性**：判断对主项或谓项的全部外延作了断定，就说该项周延。规律是——全称判断主项周延，特称判断主项不周延；肯定判断谓项不周延，否定判断谓项周延。

#### 三段论

**三段论** 由两个包含 **共同项** 的性质判断作前提，推出一个新性质判断作结论。

- **大项**（$P$）：结论的谓项，在大前提中；
- **小项**（$S$）：结论的主项，在小前提中；
- **中项**（$M$）：两前提共有、结论中不出现，起联结桥梁作用。

经典示例——

> 大前提：所有金属都是导体（$M$ 是 $P$）；
> 小前提：铁是金属（$S$ 是 $M$）；
> 结论：所以铁是导体（$S$ 是 $P$）。

三段论一般规则（必记）——

|             规则             |            违反后的错误            |
| :--------------------------: | :-------------------------------: |
|      有且只有三个不同的项      |    四概念（多为中项歧义）    |
|    中项在前提中至少周延一次    |        中项两次不周延        |
| 前提中不周延的项，结论中不得周延 | 大项 / 小项不当扩大 |
|   两个否定前提推不出结论   |          双否定前提          |
| 前提一否定，结论必否定；反之亦然 |        前提结论质不一致        |
|    两个特称前提推不出结论    |          双特称前提          |

由这些规则可进一步导出：两前提中必有一个全称、必有一个肯定。若前提有一特称，结论必特称。

:::example

判断「有的花是红的，有的花是香的，所以有的红的东西是香的」是否有效。

中项「花」在两个前提中都是特称的主项，两次都不周延，违反「中项至少周延一次」，推理无效。看似都真，形式却不成立——这正是演绎推理只保「形式有效」的意义。

:::

#### 联言、选言、假言推理

**联言推理**——

- **合成式**：由 $p$、$q$ 分别为真，推出 $p\land q$ 为真；
- **分解式**：由 $p\land q$ 为真，推出 $p$（或 $q$）为真。

**选言推理**（以「否定肯定式」有效）——

- **相容选言** 只能用 **否定肯定式**：$p\lor q$ 真，$\lnot p$，则 $q$。不能用肯定否定式（因可同真）；
- **不相容选言** 两式皆可：否定其一则肯定另一，肯定其一则否定另一。

**假言推理**（以充分条件 $p\to q$ 为例，两条有效、两条无效，必记）——

|         推理形式          | 结构 |  有效性  |
| :-----------------------: | :--: | :------: |
| **肯定前件式** | $p\to q$，$p$，则 $q$ |  有效  |
| **否定后件式** | $p\to q$，$\lnot q$，则 $\lnot p$ |  有效  |
|    肯定后件式    | $p\to q$，$q$，则 $p$ | 无效 |
|    否定前件式    | $p\to q$，$\lnot p$，则 $\lnot q$ | 无效 |

必要条件假言推理正相反：**否定前件式** 和 **肯定后件式** 有效，肯定前件、否定后件无效。充要条件假言推理四式全有效。

:::tip

充分条件记「肯前、否后」有效；必要条件记「否前、肯后」有效。混淆有效式是最常见的失分点。

:::

### 学会归纳与类比推理

#### 归纳推理

**归纳推理** 从个别性前提推出一般性结论，分完全归纳与不完全归纳。

- **完全归纳推理**：考察某类的 **全部** 对象，结论 **必然为真**，但认识范围没有扩大（属演绎性质的保真推理）；
- **不完全归纳推理**：只考察 **部分** 对象，结论 **或然为真**，能扩展认识，但可能因反例而被推翻。

提高不完全归纳结论可靠性的办法：**尽可能多考察对象**、**分析现象间的因果联系**。

**探求因果联系的方法**（穆勒五法）——

|   方法   |               思路                |
| :------: | :-------------------------------: |
| **求同法** |    异中求同：共同的先行情况是原因    |
| **求异法** |  同中求异：唯一不同的情况是原因  |
| **共变法** |    一现象变化，另一现象随之变化    |
| **求同求异并用法** |    正反两组对照，兼用求同与求异    |
| **剩余法** | 从复合因果中减去已知，剩余即剩余原因 |

#### 类比推理

**类比推理** 根据两个（类）对象在一系列属性上相同或相似，推出它们在另一属性上也相同或相似。它是 **从特殊到特殊**（或一般到一般）的或然推理。

$$
A \text{ 具有属性 } a,b,c,d;\quad B \text{ 具有属性 } a,b,c;\quad \therefore B \text{ 可能具有 } d
$$

提高类比结论可靠性的办法——

- **相同属性尽可能多**；
- **相同属性与推出属性之间的相关程度尽可能高（本质关联）**；
- 前提中不能出现与结论相反的情况。

类比与比喻不同：类比是一种 **推理**（得出新知识），比喻只是 **修辞**（使表达生动）。类比若忽视本质差异，会导致 **机械类比** 的错误。

## 第三单元 运用辩证思维方法

### 领会辩证思维

#### 辩证思维的含义

**辩证思维** 是用联系、发展、全面的观点看待事物，把握其内在矛盾运动的思维方式。它以 **实践** 为基础，如实反映事物的辩证本性。与之相对的是孤立、静止、片面看问题的 **形而上学思维**。

#### 辩证思维的特征

|   特征   |                     含义                     |
| :------: | :------------------------------------------: |
| **整体性** | 从整体出发认识对象，把握部分与整体、要素间的联系 |
| **动态性** |    用发展变化的眼光把握对象的过去、现在与未来    |

整体性强调 **联系与全面**，动态性强调 **发展与过程**。二者体现了辩证思维不同于形而上学思维的根本之处。

### 把握逻辑与历史的统一

- **历史** 有两义：客观事物自身发展的过程，以及人类对客观事物的认识过程；
- **逻辑** 指理论思维反映历史发展的 **逻辑进程**（概念、原理的推演顺序）。

**逻辑与历史的统一**：科学的理论体系的逻辑起点与终点，应当与研究对象的历史起点与终点相一致；逻辑是「修正过的历史」，它反映历史的主流与本质，而非照搬历史的一切偶然与曲折。逻辑的东西以历史的东西为基础，历史的东西以逻辑的东西为内在根据。

### 分析与综合

|      |             含义             |          原则          |
| :--: | :--------------------------: | :--------------------: |
| **分析** | 把对象分解为各要素分别研究 | 在整体的引导下把握部分 |
| **综合** | 把各要素结合成整体来把握 | 立足事物内在联系，不是机械拼凑 |

分析与综合是 **辩证统一** 的：分析是综合的基础，综合是分析的先导；没有分析的综合是空洞的，没有综合的分析是零碎的。二者相互渗透、相互转化，其内在的实质是 **矛盾分析法**。

### 质量互变

- **质** 是事物成为自身、区别于他物的规定性；**量** 是事物可以用数量表示的规定性；**度** 是保持事物质的稳定性的数量界限，度的两端是 **关节点**；
- **量变** 是数量的增减和场所的变更，是渐进、连续的；**质变** 是根本性质的改变，是飞跃、间断的；
- **量变** 是质变的必要准备，**质变** 是量变的必然结果；质变又为新的量变开辟道路。

方法论：既要 **积累量变**（不积跬步无以至千里），又要 **抓住时机促成质变**，还要 **坚持适度原则**——在度的范围内维持事物的质，防止「过」与「不及」。

### 否定之否定

- 辩证否定是事物 **自身的否定**，是发展和联系的环节，实质是 **扬弃**——既克服又保留，既批判又继承；
- 事物的发展要经过「肯定—否定—否定之否定」两次否定、三个阶段，仿佛回到出发点，但已是更高阶段的复归；
- 事物发展的 **总趋势是前进上升的**，**道路是曲折迂回的**，是前进性与曲折性的统一，表现为 **螺旋式上升、波浪式前进**。

方法论：对待传统文化、外来文化要 **辩证否定**，取其精华、去其糟粕；既要对前途充满信心，又要准备走曲折的路。

## 第四单元 提高创新思维能力

### 创新思维的含义与特征

**创新思维** 是指以新颖独特的方式解决问题的思维活动。它的根基在于 **实践**，其思维过程有 **多向性**、**跨越性**、**独特性**、**综合性** 等特点。培养创新思维，要突破思维定势，打破经验、权威、书本的束缚。

### 联想思维

**联想思维** 由一事物想到另一事物，是创新思维的基础。它有两个特征——

- **跨越的联结性**：跨越时空、跨越领域建立联系；
- **非逻辑制约的畅想性**：不受既有逻辑框架束缚，自由联想。

**迁移** 与 **想象** 是联想思维的两种基本形式。迁移是已有知识、经验向新情境的推广；想象是对已有形象材料的加工改造，分再造想象与创造想象。

### 发散思维与聚合思维

|   类型   |            方向            |          作用          |
| :------: | :------------------------: | :--------------------: |
| **发散思维** | 由一到多，多角度、多方向展开 | 提供尽可能多的方案（求异） |
| **聚合思维** | 由多到一，向核心目标收敛 | 从众多方案中筛选最优（求同） |

二者相辅相成：发散思维为聚合思维提供素材，聚合思维为发散思维确定方向。完整的创新过程往往是「发散—聚合—再发散—再聚合」的循环。发散思维的方法有 **检核表法**、**信息交合法** 等。

### 逆向思维

**逆向思维** 是对已成定论的事物或观点 **反过来思考** 的思维方法。可从事物的 **功能、结构、因果** 等方面进行反向探索，如「有用变无用、无用变有用」「正向变逆向」。

逆向思维要以 **科学性** 为前提，与正向思维 **辩证统一**：不是为反而反，而是在遵循客观规律的基础上另辟蹊径。运用时要具体分析对象是否具备可逆的条件，防止陷入「诡辩」。

### 超前思维

**超前思维** 是指人们在实践中对事物发展趋势和未来图景的 **预测和构想**。它具有 **探索性**、**预测性**、**不确定性** 等特点。

科学的超前思维要——

- 以 **辩证思维的方法论** 为指导，把握事物的普遍联系与变化发展；
- 注重 **调查研究**，立足客观实际，掌握充分的信息；
- 运用 **矛盾分析法**，分析事物的现状与趋势；
- 发挥 **想象力**，但预测须以事实和规律为依据，不是主观臆断。
