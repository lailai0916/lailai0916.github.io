## 第一单元 探索世界与把握规律

### 追求智慧的学问

#### 哲学的起源

- 哲学的 **本义** 是「爱智慧」或「智慧之学」；
- 哲学源于人们在实践中对世界的 **追问** 和 **思考**；
- 哲学的 **智慧产生于人类的实践活动**，是社会实践的产物。

#### 哲学与世界观、方法论

- **世界观**：人们对整个世界以及人与世界关系的 **总的看法和根本观点**；
- 世界观人人都有，但往往是 **自发的、零散的**；
- **哲学是系统化、理论化的世界观**；
- **方法论**：人们认识世界和改造世界的根本原则和根本方法。

|   概念   |          含义          |            关系            |
| :------: | :--------------------: | :------------------------: |
| 世界观 | 对世界的总看法、根本观点 | 世界观决定方法论 |
| 方法论 | 认识、改造世界的根本方法 | 方法论体现世界观 |
| 哲学 | 世界观和方法论的统一 | 是二者的统一 |

**哲学是世界观和方法论的统一**：有什么样的世界观，就有什么样的方法论。

#### 哲学与具体科学

- 具体科学揭示某一 **具体领域** 的规律，哲学揭示 **整个世界** 最一般的本质和规律；
- **具体科学是哲学的基础**，具体科学的进步推动哲学的发展；
- **哲学为具体科学提供世界观和方法论的指导**；
- 二者是 **共性与个性、一般与个别** 的关系，不是整体与部分、多数与少数的关系。

### 哲学的基本问题

#### 基本问题的内容

**哲学的基本问题是思维和存在的关系问题**，也就是意识和物质的关系问题。它包括两方面：

|      方面      |             内容             |          回答的分野          |
| :------------: | :--------------------------: | :--------------------------: |
| 何者为本原 | 思维和存在谁是本原、谁决定谁 | 划分 **唯物主义** 与 **唯心主义** |
| 有无同一性 | 思维能否正确认识存在 | 划分 **可知论** 与 **不可知论** |

- **第一方面** 是划分唯物主义和唯心主义的 **唯一标准**；
- 凡认为 **存在决定思维** 的是唯物主义，凡认为 **思维决定存在** 的是唯心主义。

#### 为什么是基本问题

- 它是 **一切哲学都不能回避、必须回答** 的问题；
- 它贯穿哲学发展的始终，是研究和解决其他哲学问题的前提和基础；
- 它也是 **人们在实践中首先遇到并无法回避** 的基本问题。

#### 唯物主义与唯心主义

|    基本派别    |     基本观点     |            历史形态            |
| :------------: | :--------------: | :----------------------------: |
| **唯物主义** | 物质是本原，物质决定意识 | 古代朴素唯物主义、近代形而上学唯物主义、辩证唯物主义和历史唯物主义 |
| **唯心主义** | 意识是本原，意识决定物质 | 主观唯心主义、客观唯心主义 |

- **古代朴素唯物主义**：把物质归结为一种或几种具体形态（如水、火、气）；
- **近代形而上学唯物主义**：把物质归结为原子，具有机械性、形而上学性、历史观上的唯心主义等局限；
- **主观唯心主义**：把人的主观精神（感觉、经验、心灵）当作本原；
- **客观唯心主义**：把客观精神（理念、绝对精神、上帝）当作本原。

### 科学的世界观和方法论

#### 马克思主义哲学的产生

- **阶级基础**：无产阶级的产生和发展；
- **自然科学基础**：细胞学说、能量守恒和转化定律、生物进化论三大发现；
- **直接理论来源**：德国古典哲学，主要是黑格尔的辩证法和费尔巴哈的唯物主义。

#### 马克思主义哲学的基本特征

- 第一次实现了 **唯物主义与辩证法** 的有机统一，实现了 **唯物辩证的自然观与历史观** 的有机统一；
- **实践观点是马克思主义哲学的核心观点**；
- 马克思主义哲学的 **显著特征** 是实践性，它是「改变世界」的科学、指导人类解放的科学。

|   组成部分   |             基本内容             |
| :----------: | :------------------------------: |
| **辩证唯物主义** | 辩证唯物论、唯物辩证法、认识论 |
| **历史唯物主义** | 关于社会历史发展一般规律的科学 |

### 世界的物质性

以下进入 **辩证唯物论** 部分，逐条列「原理 + 方法论」。

#### 物质的概念

- **物质是不依赖于人的意识、并能为人的意识所反映的客观实在**；
- 物质的 **唯一特性是客观实在性**；
- 物质是对具体物质形态 **共同本质的概括**，不等于具体的物质形态。

#### 世界的物质统一性原理

|          |                       内容                       |
| :------: | :----------------------------------------------: |
| **原理** | 自然界是物质的；人类社会在本质上是一个客观的物质体系；意识是物质世界长期发展的产物。**世界的真正统一性在于它的物质性** |
| **方法论** | 想问题、办事情要 **一切从实际出发，实事求是**，使主观符合客观 |

- 人类社会的 **物质性** 体现在：地理环境、人口因素、生产方式都是客观的物质要素，其中 **生产方式** 是社会存在和发展的基础。

### 运动的规律性

#### 运动与物质

|          |                        内容                        |
| :------: | :------------------------------------------------: |
| **原理** | 运动是物质固有的根本属性和存在方式；物质是运动的物质，运动是物质的运动，物质是运动的 **承担者** |
| **方法论** | 要用运动、变化、发展的观点看问题；离开物质谈运动是唯心主义，离开运动谈物质是形而上学 |

#### 运动与静止

- **运动是绝对的、无条件的、永恒的**；
- **静止是相对的、有条件的、暂时的**；
- 静止是运动的一种 **特殊状态**；
- 物质世界是 **绝对运动与相对静止的统一**；
- 只承认静止而否认运动是形而上学的不变论，只承认运动而否认静止会导致相对主义和诡辩论。

#### 规律的客观性与普遍性

|          |                       内容                       |
| :------: | :----------------------------------------------: |
| **原理** | 规律是事物运动过程中 **固有的、本质的、必然的、稳定的** 联系。规律是客观的，不以人的意志为转移，既不能被创造也不能被消灭；规律是普遍的 |
| **方法论** | 必须 **遵循规律，按客观规律办事**，而不能违背规律；一旦违背就会受到规律的惩罚 |

#### 尊重规律与发挥主观能动性

|          |                        内容                        |
| :------: | :------------------------------------------------: |
| **原理** | 尊重客观规律是正确发挥主观能动性的 **前提和基础**；发挥主观能动性是认识和利用规律的 **必要条件** |
| **方法论** | 把 **尊重客观规律与发挥主观能动性** 结合起来，把 **高度的革命热情与严谨踏实的科学态度** 结合起来 |

### 正确发挥主观能动性

以下为 **意识** 的原理，仍属辩证唯物论。

#### 意识的本质

- **从起源看**：意识是物质世界长期发展的产物；
- **从生理基础看**：意识是人脑的机能，人脑是意识活动的物质器官；
- **从内容看**：意识是客观存在的 **主观映象**；
- 结论：**物质决定意识**，意识是对物质的反映。

#### 物质与意识的辩证关系原理

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理一** | **物质决定意识**，意识是物质世界长期发展的产物、是人脑的机能、是客观存在的反映 |
| **方法论一** | 坚持 **一切从实际出发，实事求是** |
| **原理二** | **意识对物质具有能动作用**：能动地认识世界，能动地改造世界 |
| **方法论二** | 重视意识的作用，重视精神的力量，树立正确的思想意识 |

#### 意识能动作用的表现

- **能动地认识世界**：意识活动具有 **目的性、自觉选择性、能动创造性**，能透过现象把握本质；
- **能动地改造世界**：意识可以指导人们改造客观世界，正确的意识促进事物发展，错误的意识阻碍事物发展；意识还能调节和控制人体的生理活动。

#### 一切从实际出发，实事求是

|          |                       内容                       |
| :------: | :----------------------------------------------: |
| **含义** | 从客观存在的事物出发，经过调查研究，找出事物本身固有的规律作为行动的依据 |
| **要求** | 尊重物质运动的客观规律，从客观实际出发；充分发挥主观能动性，不断解放思想、与时俱进；把发挥主观能动性和尊重客观规律结合起来；反对夸大意识作用的 **唯意志主义**，反对片面强调客观条件、安于现状的 **保守思想** |

### 世界是普遍联系的

以下进入 **唯物辩证法** 部分。唯物辩证法的 **总特征是联系的观点和发展的观点**，**实质与核心是矛盾规律**（对立统一规律）。

#### 联系的普遍性

|          |                     内容                     |
| :------: | :------------------------------------------: |
| **原理** | 联系是事物之间以及事物内部诸要素之间的相互依赖、相互影响、相互制约、相互作用。世界是普遍联系的有机整体，任何事物都处在联系之中 |
| **方法论** | 用 **联系的观点** 看问题，反对孤立地看问题 |

普遍性不等于任意两个事物都有联系，联系是有条件的。

#### 联系的客观性

|          |                       内容                       |
| :------: | :----------------------------------------------: |
| **原理** | 联系是事物本身 **固有的**，不以人的意志为转移。联系分为 **自在事物** 的联系和 **人为事物** 的联系，人为事物的联系也是客观的 |
| **方法论** | 从事物固有的联系中把握事物，切忌主观随意性；人可以根据事物固有的联系，改变事物的状态，建立新的具体联系 |

#### 联系的多样性

|          |                     内容                     |
| :------: | :------------------------------------------: |
| **原理** | 事物的联系是多种多样的：直接联系与间接联系、内部联系与外部联系、本质联系与非本质联系、必然联系与偶然联系 |
| **方法论** | 注意分析和把握事物存在和发展的各种条件；一切以时间、地点和条件为转移 |

#### 整体与部分

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 整体居于主导地位，统率着部分；部分影响整体，关键部分的功能及其变化甚至对整体的功能起决定作用；二者相互联系、密不可分 |
| **方法论** | 树立 **全局观念**，立足整体，实现整体的最优目标；重视部分的作用，用局部的发展推动整体的发展 |

#### 系统优化的方法

- **系统** 的基本特征：整体性、有序性、内部结构的优化趋向；
- 掌握系统优化的方法，要着眼于事物的整体性，遵循系统内部结构的有序性，注重系统内部结构的优化趋向；
- 要用 **综合的思维方式** 认识事物。

### 世界是永恒发展的

#### 发展的普遍性与实质

|          |                     内容                     |
| :------: | :------------------------------------------: |
| **原理** | 世界是永恒发展的。发展的 **实质是事物的前进和上升，是新事物的产生和旧事物的灭亡** |
| **方法论** | 用 **发展的观点** 看问题；反对用静止的观点看问题 |

- **新事物** 是符合客观规律、具有强大生命力和远大前途的事物；**旧事物** 是违背规律、丧失存在必然性的事物。

#### 发展的前进性与曲折性

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 事物发展的前途是光明的，道路是曲折的。新事物必然战胜旧事物，但新事物的成长要经历艰难曲折的过程 |
| **方法论** | 对未来充满信心，热情支持新事物；做好充分的思想准备，勇敢地接受挑战，不断克服前进道路上的困难 |

#### 量变与质变

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | **量变是质变的必要准备**，质变是量变的必然结果；质变又为新的量变开辟道路，事物的发展是量变与质变的统一 |
| **方法论** | 重视量的积累，为质变创造条件；抓住时机，促成质变，实现事物的飞跃；坚持 **适度原则** |

- **量变**：事物数量的增减和场所的变更，是渐进的、不显著的变化；
- **质变**：事物根本性质的变化，是根本的、显著的变化。

### 用对立统一的观点看问题

以下为 **矛盾规律**，是唯物辩证法的实质与核心。

#### 矛盾的同一性与斗争性

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 矛盾即对立统一。矛盾的 **同一性** 指双方相互依赖、相互贯通、在一定条件下相互转化；矛盾的 **斗争性** 指双方相互排斥、相互对立。同一性以斗争性为前提，斗争性寓于同一性之中 |
| **方法论** | 在对立中把握统一，在统一中把握对立，坚持一分为二、全面地看问题 |

#### 矛盾的普遍性与特殊性

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **普遍性** | 矛盾存在于一切事物中，且贯穿每一事物发展的始终，即 **事事有矛盾、时时有矛盾** |
| **方法论** | 承认矛盾、分析矛盾，敢于揭露和解决矛盾 |
| **特殊性** | 不同事物的矛盾各有特点，同一事物在不同阶段的矛盾各有特点，同一矛盾的两个方面也各有特点 |
| **方法论** | 坚持 **具体问题具体分析** |

**普遍性与特殊性的关系**：

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 二者相互联结、相互转化：普遍性寓于特殊性之中，并通过特殊性表现出来，特殊性也离不开普遍性 |
| **方法论** | 坚持共性与个性、一般与个别的具体的历史的统一；是马克思主义普遍原理与中国具体实际相结合的哲学基础 |

#### 具体问题具体分析

- **含义**：在矛盾普遍性原理指导下，具体分析矛盾的特殊性，找出解决矛盾的正确方法；
- **地位**：是 **马克思主义活的灵魂**；
- **意义**：是正确认识事物的基础，是正确解决矛盾的关键。

#### 主要矛盾与次要矛盾

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | **主要矛盾** 在事物发展过程中处于支配地位、起决定作用；次要矛盾也影响事物发展；二者相互依赖、相互影响，在一定条件下相互转化 |
| **方法论** | 善于抓 **重点**，集中力量解决主要矛盾；统筹兼顾，恰当处理次要矛盾 |

#### 矛盾的主要方面与次要方面

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 事物的性质主要由 **主要矛盾的主要方面** 决定；二者相互依赖、相互排斥，在一定条件下相互转化 |
| **方法论** | 着重把握矛盾的主要方面，抓 **主流**；不忽视支流 |

- 主要矛盾与次要矛盾讲的是 **办事情**，矛盾的主次方面讲的是 **看问题**；
- 二者结合，构成 **坚持两点论与重点论相统一** 的认识方法。

#### 两点论与重点论

- **两点论**：既看到主要矛盾，又看到次要矛盾；既看到矛盾的主要方面，又看到次要方面；
- **重点论**：着重把握主要矛盾和矛盾的主要方面；
- 二者是 **辩证统一** 的：两点是有重点的两点，重点是两点中的重点。

### 辩证否定观与创新意识

#### 辩证否定观

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 辩证否定是 **事物自身的否定**，是发展的环节，是联系的环节，其实质是 **「扬弃」**——既肯定又否定，既克服又保留 |
| **方法论** | 树立 **创新意识**，做到不唯上、不唯书、只唯实；既尊重书本和权威，又立足实践解放思想 |

- 辩证否定不是简单地肯定一切或否定一切；
- 与辩证否定观相统一的是 **辩证法的革命批判精神**。

#### 创新的作用

- **创新推动社会生产力的发展**：更新生产工具和生产技术，提高劳动者素质，开辟更广阔的劳动对象；
- **创新推动生产关系和社会制度的变革**；
- **创新推动人类思维和文化的发展**。

## 第二单元 认识社会与价值选择

### 社会存在与社会意识

以下进入 **历史唯物主义** 部分。

#### 两种历史观

|      历史观      |               核心观点               |
| :--------------: | :----------------------------------: |
| **历史唯物主义** | 社会存在决定社会意识；承认人民群众的历史作用 |
| **历史唯心主义** | 把精神当作社会历史发展的最终原因；夸大英雄人物的作用 |

**社会存在** 是社会生活的物质方面，最主要的是物质资料的生产方式；**社会意识** 是社会生活的精神方面。

#### 社会存在与社会意识的辩证关系

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理一** | **社会存在决定社会意识**，社会意识是对社会存在的反映，随社会存在的变化而变化 |
| **原理二** | 社会意识具有 **相对独立性**：可能落后于或先于社会存在；社会意识对社会存在具有 **能动的反作用**，先进的社会意识促进社会发展 |
| **方法论** | 一切从社会实际出发；树立正确的、先进的社会意识 |

### 社会历史的发展

#### 社会基本矛盾运动

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **规律一** | **生产关系一定要适合生产力状况的规律**：生产力决定生产关系，生产关系对生产力具有反作用 |
| **规律二** | **上层建筑一定要适合经济基础状况的规律**：经济基础决定上层建筑，上层建筑对经济基础具有反作用 |

- **生产力与生产关系** 的矛盾、**经济基础与上层建筑** 的矛盾，是贯穿人类社会始终的 **基本矛盾**；
- 这两对基本矛盾运动，推动着人类社会的发展。

#### 社会历史发展的总趋势

- 社会历史发展的总趋势是 **前进的、上升的**，发展的过程是 **曲折的**；
- 在 **阶级社会** 里，社会基本矛盾的解决主要通过 **阶级斗争** 实现；
- 在 **社会主义社会**，基本矛盾是 **非对抗性** 的，通过 **改革** 即社会主义的自我完善和发展来解决。**改革是发展中国特色社会主义的强大动力**。

### 社会历史的主体

#### 人民群众是历史的创造者

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 人民群众是社会 **物质财富** 的创造者，是社会 **精神财富** 的创造者，是社会 **变革** 的决定力量 |
| **方法论** | 坚持 **群众观点** 和 **群众路线** |

- **群众观点**：相信人民群众自己解放自己，全心全意为人民服务，一切向人民群众负责，虚心向人民群众学习；
- **群众路线**：一切为了群众，一切依靠群众，从群众中来，到群众中去；
- 群众观点和群众路线是我们党的 **根本立场** 和 **工作路线**。

### 价值与价值观

#### 人的价值

- **价值** 是一事物对主体的积极意义，即一事物所具有的能够满足主体需要的属性和功能；
- 人的价值在于 **创造价值**，在于对社会的责任和贡献，即通过自己的活动满足社会和他人的需要；
- 对一个人价值的评价，归根到底是看他的 **贡献**，主要是对社会发展和人类进步事业的贡献。

#### 价值观的导向作用

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | **价值观** 对人们认识世界和改造世界的活动、对人生道路的选择具有重要的 **导向作用**，是人生的重要向导 |
| **方法论** | 树立和践行 **社会主义核心价值观**，作出正确的价值判断和价值选择 |

### 价值判断与价值选择

#### 特征

- **社会历史性**：价值判断和价值选择随时间、地点和条件的变化而变化；
- **阶级性**：在阶级社会中，价值判断和价值选择因人们的社会地位、阶级立场不同而不同；
- **主体差异性**：立场不同、需要不同，价值判断和价值选择也不同。

#### 正确标准

作出正确的价值判断和价值选择，必须：

- **自觉遵循社会发展的客观规律**；
- **自觉站在最广大人民的立场上**，把人民群众的利益作为最高的价值标准；
- 正确处理个人利益与他人、集体、社会利益的关系。

### 价值的创造和实现

#### 在劳动和奉献中创造价值

- 劳动是人的存在方式，是创造和实现人的价值的根本途径；
- 积极投身为人民服务的实践，是实现人生价值的必由之路；
- 在 **个人与社会的统一** 中实现价值，社会提供的客观条件是实现价值的基础。

#### 在砥砺自我中走向成功

|                      |                       内容                       |
| :------------------: | :----------------------------------------------: |
| **发挥主观能动性** | 需要顽强拼搏、自强不息的精神 |
| **提高个人素质** | 需要努力发展自己的才能，全面提高个人素质 |
| **坚定理想信念** | 需要有坚定的理想信念，坚持正确价值观的指引 |

## 第三单元 文化传承与文化创新

### 文化的内涵与功能

#### 什么是文化

- **文化** 是相对于经济、政治而言的人类全部 **精神活动** 及其 **产品**；
- 文化是人类社会实践的产物，纯粹自然的东西不能称为文化；
- 每个人的文化素养是通过对社会生活的 **体验**，尤其是通过参与文化活动、接受文化知识教育逐步培养出来的。

|            关系            |                     内容                     |
| :------------------------: | :------------------------------------------: |
| **文化与经济、政治** | 经济、政治决定文化，文化反作用于经济、政治，给予政治、经济以重大影响 |
| **文化与文明** | 文化有先进与落后之分，文明是文化中的积极成果，与野蛮相对 |
| **文化载体** | 文化通过载体呈现出来，如器物、行为、制度等 |

#### 文化的功能

- **引领风尚**：教育引导人们向善向上；
- **教育人民**：提高人们的知识水平和道德修养；
- **服务社会**：满足人民的精神文化需求，推动经济社会发展；
- 优秀文化能够 **增强精神力量**，促进人的全面发展。

### 正确认识中华传统文化

#### 中华优秀传统文化的主要内容与特点

- **主要内容**：核心思想理念、中华传统美德、中华人文精神；
- **特点**：**源远流长、博大精深**；
- **中华文化的力量** 集中表现为民族精神的力量，**爱国主义** 是中华民族精神的核心。

#### 对待传统文化的态度

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **原理** | 中华优秀传统文化具有 **当代价值**，能够为治国理政、道德建设提供有益启发，是中华民族的精神命脉 |
| **方法论** | **取其精华、去其糟粕，批判继承、古为今用**；坚持 **创造性转化和创新性发展** |

- **创造性转化**：按照时代特点和要求，改造传统文化中有借鉴意义的内涵和陈旧的表现形式；
- **创新性发展**：按照时代的新进步新进展，补充、拓展、完善传统文化的内涵。

### 学习借鉴外来文化的有益成果

#### 文化的民族性与多样性

- 文化 **民族性** 是一个民族区别于其他民族的独特标识，**民族节日** 和 **文化遗产** 是民族文化的集中展示；
- 文化多样性表征文化存在的丰富程度，是人类社会的基本特征，也是文化魅力和活力的重要来源。

#### 文化交流与文化交融

|          |                          内容                          |
| :------: | :----------------------------------------------------: |
| **交流** | 文化交流构成文化发展的重要动力，要 **尊重文化多样性**，坚持各民族文化 **一律平等** |
| **交融** | 文化交融推动文化的发展，学习和借鉴其他民族优秀文化成果，促进本民族文化发展 |

#### 正确对待外来文化

- 反对两种错误倾向：**彻底拒斥外来文化的文化复古主义**，**盲目照搬西方文化的全盘西化论**；
- 坚持 **面向世界、博采众长**，坚持 **以我为主、为我所用**；
- 立足国情、交流互鉴，实现外来文化的 **本土化**。
