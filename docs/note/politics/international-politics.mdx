## 第一课 国体与政体

### 国家的本质

#### 国家的产生

- **国家** 是阶级矛盾不可调和的产物，是阶级统治的工具；
- 国家不是从来就有的，随 **私有制、阶级** 的出现而产生，也将随阶级的消亡而消亡；
- 国家的 **根本属性是阶级性**，此外还有社会管理等社会属性。

#### 国家的构成要素

**人口、领土、政权、主权** 是构成国家的四个基本要素，其中 **主权** 是国家的生命和灵魂。

- **主权** 对内表现为国家最高权力的至高无上性，对外表现为国家的独立性和自主性；
- 主权是国家统一而不可分割的最高权力，具有 **对内最高、对外独立** 两重属性。

四个要素中，人口是国家的基础，领土是国家行使主权的空间，政权是国家的政治核心，主权则是国家最重要的构成要素。**没有主权就没有国家**，一个国家一旦丧失主权，就会沦为别国的殖民地或附庸。

#### 民主与专制

- **专政** 指统治阶级凭借国家机器对被统治阶级实行的强力统治，是国家阶级性的直接体现；
- 一个国家的政权，对统治阶级内部实行 **民主**，对被统治阶级实行 **专政**，二者相辅相成；
- 社会主义国家实行 **人民民主专政**，即对人民内部实行民主、对敌对势力实行专政。

### 国家性质

#### 国体

**国体**（国家性质）指社会各阶级在国家中的地位，即哪个阶级掌握国家政权。

- 国体由 **占统治地位的阶级** 的性质决定；
- **民主** 与 **专制** 相对：民主是一种国家制度，指在统治阶级内部按多数人意志管理国家；专制指最高统治者独自掌握政权。

#### 民主的阶级性

- **民主具有鲜明的阶级性**，从来都是 **统治阶级的民主**，是特定阶级享有的权利；
- 任何民主都不是抽象的、超阶级的，世界上不存在适用于一切国家的统一民主模式；
- **资本主义民主** 本质是资产阶级的民主，是维护资本利益的工具；**社会主义民主** 是绝大多数人的民主，本质是人民当家作主。

|          |          资本主义民主          |                社会主义民主                |
| :------: | :----------------------------: | :----------------------------------------: |
| **主体** |      少数资产阶级享有实权      |             最广大人民当家作主             |
| **本质** |     维护资本利益、金钱政治     |           人民当家作主、真实广泛           |
| **实现** | 三权分立、多党竞争、周期性选举 | 民主集中制、根本政治制度是人民代表大会制度 |

### 政体

#### 政体的含义

**政体**（国家管理形式）指一个国家 **政权机关的组织形式**，即统治阶级采取何种形式去组织自己的政权机关。

- 国体决定政体，政体反映并服务于国体；
- 相同国体可采取 **不同政体**，相同政体可服务于 **不同国体**；
- 一国采取何种政体，与该国的 **历史传统、经济发展、阶级力量对比** 等因素有关。

#### 代议制

**代议制**（Representative System）是现代国家普遍采用的政权组织形式，即由选举产生的代表组成代议机关，行使国家权力。

- 是间接民主的实现形式，弥补直接民主在大范围内难以实施的不足；
- 资本主义国家代议机关是 **议会**，故资本主义国家又称 **议会制国家**；
- 我国的代议机关是 **人民代表大会**，实行 **人民代表大会制度**，与西方议会制有本质区别。

#### 君主立宪制与民主共和制

现代资本主义国家的政体主要有两类：

|              |         君主立宪制         |         民主共和制         |
| :----------: | :------------------------: | :------------------------: |
| **国家元首** |  世袭君主（国王 / 女王）   |     选举产生（总统等）     |
| **产生方式** |         世袭、终身         |        选举、有任期        |
| **典型国家** |    英国、日本、西班牙等    |     美国、法国、德国等     |
|   **实质**   | 资产阶级专政的政权组织形式 | 资产阶级专政的政权组织形式 |

- **君主立宪制** 保留君主，但君主 **统而不治**，权力受宪法限制，实权在议会和内阁；
- **民主共和制** 又分 **总统制**（美国，总统既是国家元首又是政府首脑，行政与立法相互独立制衡）和 **议会制**（如德国、意大利，政府由议会产生并对议会负责）。

#### 美国的三权分立

以美国为例，资本主义国家标榜 **分权制衡**，把国家权力分为立法、行政、司法三部分：

- **立法权** 归 **国会**（参议院、众议院），负责制定法律；
- **行政权** 归 **总统**，负责执行法律、管理国家事务；
- **司法权** 归 **联邦法院**，负责解释法律、审理案件。

三权相互独立、相互制衡，其实质是资产阶级内部不同利益集团之间的分权，是维护资产阶级统治的手段，掩盖不了资本主义民主的阶级本质。

### 国家结构形式

#### 单一制与联邦制

**国家结构形式** 指国家的整体与部分、中央与地方之间的关系。现代国家主要有 **单一制** 和 **联邦制** 两种。

|          |             单一制             |               联邦制               |
| :------: | :----------------------------: | :--------------------------------: |
| **构成** | 若干普通行政区域构成的统一国家 |  若干成员单位（州、邦）联合的国家  |
| **主权** |     主权属于中央，国家统一     | 联邦享有主权，成员单位有部分自主权 |
| **宪法** |       一部宪法、一套法律       |   联邦宪法之外，成员单位另有宪法   |
| **国籍** |            单一国籍            |         联邦与成员双重身份         |
| **典型** |     中国、法国、日本、英国     |      美国、德国、俄罗斯、印度      |

#### 影响国家结构形式的因素

- 国家结构形式由 **国家性质** 决定，同时受历史传统、民族状况、地理条件等因素影响；
- 单一制下地方权力来自中央授予，联邦制下成员单位权力由联邦宪法规定。

#### 我国是单一制国家

- 我国是 **统一的多民族国家**，实行 **单一制** 的国家结构形式；
- 在中央统一领导下，实行 **民族区域自治** 和 **特别行政区** 等制度，既维护国家统一，又照顾地方和民族的特殊性；
- 单一制有利于国家的统一、社会的稳定和各民族的团结，符合我国的国情。

## 第二课 世界多极化

### 时代主题

#### 和平与发展

**和平与发展** 是当今时代的主题。

- **和平问题** 指维护世界和平、防止新的世界大战的问题；
- **发展问题** 指世界经济特别是发展中国家经济发展的问题，是核心问题；
- 两者相互联系：和平是发展的前提，发展是和平的保障。

#### 威胁和平与发展的因素

- **霸权主义** 和 **强权政治** 是威胁世界和平与发展的主要根源；
- 局部战争与冲突、恐怖主义、南北差距扩大、气候变化等问题依然突出；
- 解决和平与发展问题，必须建立 **国际政治经济新秩序**，反对霸权主义和强权政治。

### 世界多极化趋势

#### 多极化的表现

**世界多极化** 指世界向多个力量中心发展的趋势，是不可逆转的历史潮流。

- 冷战结束后，美国成为唯一超级大国，但 **单极世界** 无法维系；
- 欧盟、日本、中国、俄罗斯以及广大发展中国家的力量不断增长；
- 世界正处于 **一超多强、走向多极** 的过渡时期。

多极化不会一帆风顺，将是一个 **长期、曲折、复杂** 的过程，其间充满各种力量的较量与斗争。

#### 各力量中心

|    力量中心    |              特点              |
| :------------: | :----------------------------: |
|    **美国**    |   唯一超级大国，力图维持单极   |
|    **欧盟**    | 区域一体化程度最高的国家联合体 |
|    **日本**    |   经济大国，谋求政治大国地位   |
|    **中国**    |   最大的发展中国家，快速崛起   |
|   **俄罗斯**   |   军事大国，力图恢复大国地位   |
| **发展中国家** |  整体力量上升，多极化重要力量  |

#### 多极化的意义

- 有利于抑制霸权主义和强权政治，维护世界和平与稳定；
- 有利于推动建立公正合理的国际政治经济新秩序；
- 有利于世界的多样性发展，符合各国人民的共同利益。

#### 综合国力

- **综合国力** 是衡量一个国家总体实力的重要标志，是多极化格局中各力量中心竞争的核心；
- 经济和科技实力是综合国力的基础，**科技创新** 越来越成为竞争的关键；
- 我国增强综合国力的根本途径是 **发展经济、集中力量搞好现代化建设**。

### 国际关系

#### 国际关系的含义

**国际关系** 指国家之间、国际组织之间以及它们相互之间的关系，其中最主要的是 **国家与国家之间的关系**。

- 基本形式是 **竞争、合作与冲突**；
- 主要内容有政治关系、经济关系、文化关系、军事关系等。

#### 决定国际关系的因素

- **国家利益** 是国际关系的 **决定性因素**，是国家对外活动的出发点和落脚点；
- **国家力量**（综合国力）是国家生存和发展的基础，是国际关系变化的重要因素；
- 国家间 **利益的对立** 是引起冲突的根源，**利益的共同** 是合作的基础。

> 没有永远的朋友，也没有永远的敌人，只有永远的利益。

#### 我国的国家利益

- 我国的国家利益是 **广大人民的根本利益**，国家利益与人民利益相一致；
- 我国的核心利益包括：国家主权、国家安全、领土完整、国家统一、宪法确立的国家政治制度和社会大局稳定、经济社会可持续发展的基本保障；
- 任何外国不要指望我国拿自己的核心利益做交易，不要指望我国吞下损害我国主权、安全、发展利益的苦果。

#### 维护国家利益的态度

- 维护国家利益要 **坚定不移**，同时尊重其他国家的正当利益，反对损人利己、以邻为壑；
- 我国在维护自身利益的同时，兼顾他国合理关切，谋求本国发展中促进各国共同发展；
- 各国应在追求本国利益时兼顾人类共同利益，走 **合作共赢** 之路。

## 第三课 经济全球化

### 经济全球化的表现

#### 含义与表现

**经济全球化**（Economic Globalization）指商品、服务、生产要素以及信息的跨国跨地区流动，使世界经济日益成为紧密联系的整体。

- **生产全球化**：跨国公司在全球范围内组织生产，形成国际分工；
- **贸易全球化**：商品与服务的国际交换空前扩大；
- **资本全球化**：资本在国际间自由流动，国际金融市场迅速扩张。

|      表现      |             含义             |        突出标志        |
| :------------: | :--------------------------: | :--------------------: |
| **生产全球化** |  各国生产相互联系、相互依赖  | 国际分工深化、跨国生产 |
| **贸易全球化** | 商品与服务的国际交换空前扩大 |   世界贸易额迅速增长   |
| **资本全球化** |     资本在国际间自由流动     |   国际金融市场一体化   |

#### 载体与推动力量

- **跨国公司** 是经济全球化的强有力载体和主要推动者；
- **科技进步**（尤其是信息技术）是经济全球化的根本动力；
- 生产社会化和国际分工的深化是经济全球化的客观基础。

### 经济全球化的影响

#### 双重影响

经济全球化是一把 **双刃剑**，既带来机遇，也带来挑战。

|          |           积极影响           |               消极影响               |
| :------: | :--------------------------: | :----------------------------------: |
| **资源** |  促进生产要素在全球优化配置  |       加剧世界经济发展的不平衡       |
| **发展** | 推动生产力发展、加快经济增长 | 发达国家主导，发展中国家处于不利地位 |
| **风险** | 使各国经济相互依存、优势互补 |    局部风险易演变为全球性经济危机    |

- 经济全球化的规则主要由 **发达国家** 主导，实质上是 **发达资本主义国家主导** 的全球化；
- 发展中国家应趋利避害，积极参与并推动经济全球化朝更加公正合理的方向发展。

### 经济全球化与中国

#### 中国的机遇与挑战

- 经济全球化为我国发展带来 **资金、技术、市场、管理经验** 等机遇；
- 同时也带来产业竞争加剧、经济安全受冲击等挑战；
- 我国坚持 **对外开放** 的基本国策，实行更高水平的对外开放。

#### 新发展格局

- 我国加快构建以 **国内大循环为主体、国内国际双循环相互促进** 的新发展格局；
- 坚持 **共商共建共享** 原则，推动 **共建「一带一路」**；
- 推动建设开放型世界经济，反对 **保护主义、单边主义**。

#### 一带一路

- **「一带一路」** 指丝绸之路经济带和 21 世纪海上丝绸之路，是我国扩大对外开放的重大举措；
- 坚持 **共商共建共享** 原则，以政策沟通、设施联通、贸易畅通、资金融通、民心相通为主要内容；
- 是构建人类命运共同体的重要实践平台，为世界经济增长注入新动力。

### 人类命运共同体

#### 内涵

**人类命运共同体** 指在追求本国利益时兼顾他国合理关切，在谋求本国发展中促进各国共同发展。

- 建设 **持久和平、普遍安全、共同繁荣、开放包容、清洁美丽** 的世界；
- 核心是 **合作共赢**，坚持对话协商、共建共享、合作共赢、交流互鉴、绿色低碳。

#### 中国方案

- 我国倡导构建人类命运共同体，是为解决全球性问题贡献的 **中国智慧、中国方案**；
- 顺应了和平、发展、合作、共赢的时代潮流，反映了各国人民的共同愿望。

#### 全球治理

- **全球治理** 指各国政府、国际组织等共同管理全球公共事务；
- 全球性问题（气候变化、恐怖主义、公共卫生、金融危机等）单靠一国无法解决，必须加强国际合作；
- 现有全球治理体系存在不公正、不合理之处，我国主张推动全球治理体系朝着更加 **公正合理** 的方向发展，增强新兴市场国家和发展中国家的代表性和发言权。

## 第四课 国际组织

### 国际组织概述

#### 含义与分类

**国际组织** 指两个以上国家（政府或民间团体）为实现共同目标，依据条约或协议建立的常设机构。

- 按成员分为 **政府间国际组织**（如联合国）和 **非政府间国际组织**（如国际奥委会）；
- 按地理范围分为 **世界性国际组织**（如联合国）和 **区域性国际组织**（如欧盟）；
- 国际组织是当今 **国际社会的重要成员**，在国际事务中发挥日益重要的作用。

#### 国际组织的作用

- 促进国家间的政治、经济、文化交流与合作；
- 协调国际政治、经济关系，维护成员国的共同利益；
- 调解国际争端、缓和国家间矛盾，维护世界和平；
- 但也存在受少数大国操纵、作用有限等局限性。

对国际组织的作用应 **一分为二** 地看待：既要看到它在协调国际关系、促进合作方面的积极作用，也要看到它常被少数大国利用、难以从根本上解决国际不公问题的局限。

### 联合国

#### 宗旨与原则

**联合国**（United Nations，UN）是当今世界最具普遍性、代表性和权威性的政府间国际组织，1945 年成立，总部设在纽约。

- **宗旨**：维护国际和平与安全；发展国际间友好关系；促进国际合作；协调各国行动；
- **原则**：各会员国 **主权平等**；履行宪章义务；以和平方式解决国际争端；不干涉别国内政；集体协作。

#### 主要机构

- **联合国大会**：由全体会员国组成，一国一票；
- **安全理事会**：负有维护国际和平与安全的 **主要责任**，其决议对会员国有约束力；
- 安理会 **五个常任理事国**（中、美、俄、英、法）对实质问题拥有 **否决权**，实行 **大国一致原则**。

#### 中国与联合国

- 中国是联合国的 **创始会员国** 和安理会 **常任理事国** 之一；
- 1971 年第 26 届联大恢复中华人民共和国在联合国的一切合法权利；
- 中国始终维护联合国宪章的宗旨和原则，是维护世界和平、促进共同发展的坚定力量。

#### 中国对联合国的贡献

- 中国是安理会常任理事国中派遣 **维和人员最多** 的国家，也是联合国维和摊款的重要出资国；
- 中国全面参与联合国在裁军、反恐、气候变化、可持续发展等领域的工作；
- 中国坚定维护以联合国为核心的国际体系、以国际法为基础的国际秩序。

### 世界贸易组织

#### 地位与作用

**世界贸易组织**（World Trade Organization，WTO）是当今世界最重要的国际经济组织之一，1995 年成立，总部设在日内瓦。

- 与 **世界银行**、**国际货币基金组织** 并称为当今世界经济体制的 **三大支柱**；
- 是多边贸易体制的组织基础和法律基础，被称为「经济联合国」。

#### 基本原则

- **非歧视原则**，包括 **最惠国待遇** 和 **国民待遇** 原则；
- **市场准入原则**、**促进公平竞争原则**、**贸易政策法规透明原则** 等；
- **非歧视原则** 是世界贸易组织最重要的原则。

#### 中国与世界贸易组织

- 中国于 2001 年正式加入世界贸易组织；
- 加入 WTO 是我国对外开放进入新阶段的标志，权利与义务对等；
- 中国坚定维护多边贸易体制，是经济全球化的重要参与者、贡献者。

#### 世界经济三大支柱

|                      |                  主要职能                  |
| :------------------: | :----------------------------------------: |
|   **世界贸易组织**   |      促进贸易自由化，管理多边贸易规则      |
|     **世界银行**     |     向成员国提供长期贷款，促进经济发展     |
| **国际货币基金组织** | 稳定国际货币体系，提供短期资金缓解收支困难 |

三者共同构成当今世界经济体制的支柱，其规则主要由发达国家主导，发展中国家应积极参与规则的制定与改革。

### 区域性国际组织

#### 主要区域组织对比

|          |          亚太经合组织          |         金砖国家         |      上海合作组织      |
| :------: | :----------------------------: | :----------------------: | :--------------------: |
| **性质** |      亚太地区经济合作论坛      |   新兴市场国家合作机制   |   区域安全与合作组织   |
| **成立** |            1989 年             |         2006 年          |        2001 年         |
| **成员** | 中、美、日、俄等环太平洋经济体 | 中、俄、印、巴西、南非等 |   中、俄及中亚国家等   |
| **宗旨** |    促进区域贸易与投资自由化    |   加强新兴国家团结合作   | 维护地区安全、促进合作 |

- **亚太经合组织**（APEC）遵循 **相互尊重、平等互利、协商一致、自主自愿** 的原则，采取 **非约束性** 承诺；
- **金砖国家**（BRICS）代表新兴市场和发展中国家，推动全球治理体系变革；
- 此外还有 **欧盟**（区域一体化程度最高）、**东盟**、**二十国集团**（G20）等。

#### 欧盟

- **欧盟**（European Union，EU）是当今区域一体化程度 **最高** 的国际组织，1993 年成立；
- 从经济一体化走向政治一体化，拥有统一货币 **欧元**、共同的外交与安全政策；
- 有自己的旗帜、盟歌等，具有超国家的性质，是世界多极化中的重要一极。

#### 中国与国际组织

- 中国积极参与全球治理，是众多国际组织的重要成员，发挥着 **建设性作用**；
- 中国推动创建 **亚洲基础设施投资银行**（亚投行）、**金砖国家新开发银行** 等新型国际组织；
- 中国倡导 **共商共建共享** 的全球治理观，为完善全球治理贡献中国力量。

## 第五课 中国的外交

### 独立自主的和平外交政策

#### 外交政策的决定因素

- 我国的 **国家性质**（人民民主专政的社会主义国家）和 **国家利益** 决定了我国奉行独立自主的和平外交政策；
- 国家利益是国家对外活动的出发点和落脚点。

#### 外交政策的内容

- **基本立场**：独立自主；
- **宗旨**：维护世界和平、促进共同发展；
- **基本目标**：维护我国的主权、安全和发展利益，促进世界的和平与发展；
- **基本准则**：**和平共处五项原则**——互相尊重主权和领土完整、互不侵犯、互不干涉内政、平等互利、和平共处。

#### 和平共处五项原则

- 由中国、印度、缅甸于 1954 年共同倡导，是我国处理对外关系的 **基本准则**；
- 成为 **国际社会公认** 的国际关系基本准则和国际法基本原则；
- 其中 **互相尊重主权和领土完整** 是核心和基础。

#### 独立自主的表现

- 我国对国际问题独立地作出判断和决策，**不屈从于任何外来压力**；
- **不同任何大国或国家集团结盟**，不搞军事集团，不参加军备竞赛；
- 坚持把国家的主权和安全放在第一位，反对干涉别国内政。

### 走和平发展道路

#### 为什么走和平发展道路

- **和平发展道路** 是我国根据自身国情和时代发展潮流作出的战略抉择；
- 是由我国的社会主义性质决定的，是中华民族爱好和平的文化传统的传承；
- 顺应了和平与发展的时代主题，符合我国和世界人民的根本利益；
- 中国不走 **国强必霸** 的老路，中华民族的血液中没有侵略他人、称霸世界的基因。

#### 如何走和平发展道路

- 坚持 **对外开放**，在平等互利基础上发展同各国的合作；
- 坚持 **正确义利观**，做到义利兼顾、以义为先；
- 反对各种形式的 **霸权主义和强权政治**，永远不称霸、不搞扩张；
- 推动构建 **相互尊重、公平正义、合作共赢** 的新型国际关系。

### 日益重要的国际地位

#### 中国的国际地位与作用

- 中国是世界上最大的 **发展中国家**，是维护世界和平、促进共同发展的坚定力量；
- 中国是联合国安理会常任理事国，是全球治理的重要参与者、贡献者、引领者；
- 中国日益走近世界舞台中央，为解决人类问题贡献中国智慧和中国方案。

#### 中国特色大国外交

- **总目标**：推动构建新型国际关系、推动构建人类命运共同体；
- 秉持 **共商共建共享** 的全球治理观，践行真正的 **多边主义**；
- 坚持 **亲、诚、惠、容** 的周边外交理念和 **真、实、亲、诚** 的对非政策理念；
- 中国的发展是世界和平力量的增长，永远是发展中国家的可靠朋友和真诚伙伴。

#### 新型国际关系

- **新型国际关系** 以 **相互尊重、公平正义、合作共赢** 为核心；
- 与传统国际关系中的弱肉强食、零和博弈不同，强调 **合作而非对抗、共赢而非独占**；
- 构建新型国际关系与构建人类命运共同体相辅相成，是中国特色大国外交的努力方向。
