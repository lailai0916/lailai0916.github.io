## 说明

浙江高考的 **技术** 是一门独立选考科目，由 **信息技术** 与 **通用技术** 两部分合并而成，两部分各占一半。这份笔记按这两部分分别整理，重考点：重概念、重方法、重流程，配对比表与必记结论。

信息技术以 **数据、算法与信息系统** 为主线，代码用 Python；通用技术以 **技术设计思想与结构、流程、系统、控制** 为主线。下面列出两部分的主要内容。

## 信息技术

### 数据与计算

- 数据与信息、数据编码
- 数据处理与应用
- 算法与程序实现（Python）
- 人工智能初步

### 信息系统与社会

- 信息系统的组成与功能
- 网络与物联网
- 信息安全
- 信息技术与社会

### 数据与数据结构

- 线性表、栈与队列
- 树与图
- 查找与排序

## 通用技术

### 技术与设计 1

- 技术及其性质
- 设计的一般过程与原则
- 发现与明确问题、方案构思
- 设计图样的绘制、模型制作

### 技术与设计 2

- 结构与设计
- 流程与设计
- 系统与设计
- 控制与设计

### 电子控制技术

- 电子控制系统的组成与开环、闭环控制
- 传感器与信息的获取转换
- 模拟电路、数字电路与逻辑门
- 执行部件与直流电磁继电器
- 控制电路的安装、调试与设计
