## 第一章 人口与地理环境

### 人口分布

- 世界人口分布特征：主要分布在中低纬、沿江沿海平原区；
- 影响因素：自然因素（地形、气候、土壤、河流、自然资源）；人文因素（经济、社会、科技、文化、历史）。

### 人口容量

- 影响因素：资源状况（首要因素）、科技水平、开放程度、消费水平（负相关）；
- 特点：临界性、警戒性、相对性。

### 人口迁移

- 推拉理论：推力 + 拉力 + 阻力；
- 影响因素：自然因素（气候、地形、土壤、水源、生态恶化、自然灾害、自然资源）；经济因素（区域经济发展水平、城市化水平、大型工程项目、交通）；社会因素（政策、战争、政治中心变革、文化教育、婚姻、家庭、宗教信仰）；
- 我国人口迁移特点：历史上黄河流域向长江、珠江流域迁移（移民支边、战争和自然灾害）；新中国成立到 80 年代中期规模小、频率低，自发性迁移较少（计划经济和严格的户籍管理政策）；80 年代中期以来流动人口大量增加，以自发性为主（农村 $\to$ 城市，内地 $\to$ 沿海工矿地）。

**人口迁移的影响**

|        |                                     有利                                     |                         不利                         |
| :----: | :--------------------------------------------------------------------------: | :--------------------------------------------------: |
| 迁出区 |     缓解当地人地矛盾；更好开发利用土地资源；加强与外界交流联系；增加收入     | 人才外流，影响经济发展；留守儿童、空巢老人等社会问题 |
| 迁入区 | 获得廉价劳动力，促进经济发展；加快资源开发，改善当地综合环境；缓解人口老龄化 |  增加当地负担和管理难度；引起或加剧当地生态环境问题  |

## 第二章 城镇和乡村

### 城市区位因素

- 自然因素：气候、地形、河流等；
- 社会经济因素：自然资源、交通、政治、军事、宗教、旅游、科技；
- 河流与城市：功能（供水、运输、防卫）；位置（河口处、干支流交汇处、水运起讫点、过河点、河曲处）。

### 城乡土地利用

|   一级类型   |                                 二级类型                                 |        备注        |
| :----------: | :----------------------------------------------------------------------: | :----------------: |
|  **农用地**  |             耕地、园地、林地、草地、农田水利用地、养殖水面等             |  乡村以农用地为主  |
| **建设用地** | 住宅用地、公共设施用地、工矿用地、交通水利设施、旅游用地、军事设施用地等 | 城市以建设用地为主 |
| **未利用地** |                          沼泽地、盐碱地、沙地等                          |         —          |

### 城市功能区

- 居住区：占地最大，是城市中最基本的功能区，分高级、低级住宅区；
- 商业区：占地最小，位于城市几何中心、交通干线两侧或街角路口处；
- 工业区：不断向郊区外移，趋于沿交通干线分布（地价 + 环境污染、运输需求）；
- 中心商务区（CBD）特征：高楼林立，公司总部云集；城市经济枢纽和观光购物区域；商贸服务业发达；土地利用集约，地价房租高；人流汇集，交通繁忙。

### 城市空间结构

- 城市地租递减理论：影响地租水平的因素是距市中心的远近和交通通达度；
- 空间结构类型：同心圆状（成都、芝加哥）、扇状（沈阳）、条带状（兰州）、多核心状（重庆、杭州）、棋盘格状；
- 影响因素：自然地理条件、历史文化、经济水平、交通运输、政策；
- 乡村空间形态：密集型（街道式、团块式、棋盘格式）、分散型、半聚集型。

### 地域文化与城乡景观

- 民居特色：形态结构适应当地自然地理特征；建筑材料有鲜明的地域性；
- 城市格局：中国古代都城以宫殿为城市中心并确定中轴线，体现皇权至上、严格的等级制度；欧洲城市多围绕教堂、市政厅、城市广场布局，体现历史文化和现代文化的融合；美国城市发展历史短，中心多围绕摩天大楼布局，外围建筑高度逐渐下降；
- 建筑风格：中国砖墙木梁架结构、斗拱飞檐、雕梁画栋，体现天人合一思想；西方建筑柱廊、尖塔、喷泉、雕塑突出，受宗教思想影响。

### 城镇化

- 内涵：人口城市化 + 地域城市化 + 产业城市化；
- 表现：城市人口占总人口比重持续上升；人口从第一产业向第二、三产业逐渐转移；城市用地规模不断扩大；乡村景观逐渐转化为城市景观；人们生产方式、生活方式、文化、价值观念等发生显著变化；
- 动力：社会经济发展；
- 发达国家：起步早、水平高、速度放缓、出现逆城市化；发展中国家：起步晚、水平低、发展速度快、城市化不合理（虚假城市化和滞后城市化）；
- 城市化问题：中心区人口密集；环境质量下降；交通拥堵；住房困难；就业困难；社会秩序混乱；城市内涝严重；
- 解决：建立卫星城、开发新区；保护和治理城市环境；合理规划城市道路网；加快住房建设、改善居住条件；加快经济建设、扩大就业；
- 新型城市化方向：生态城市、田园城市、智慧城市、海绵城市。

## 第三章 产业区位选择

### 农业

- 分类：按对象（种植业、畜牧业、混合农业）；按投入（集约农业、粗放农业）；按用途（商品农业、自给农业）；
- 生产特点：地域性、季节性和周期性；
- 农业区位因素：自然条件（气候——光照、热量、降水、昼夜温差；地形、土壤、水源）；社会经济（市场、交通、政策、劳动力、农产品消费状况）；科学技术（技术装备、生产技术、种植方式、信息化、冷藏保鲜技术）；
- 亚洲水稻种植业（一大一小、一高三低）：区位条件（季风气候，水热充足；集中沿海平原，地形平坦；水稻土广布，深厚肥沃；人口稠密，劳动力丰富；粮食需求量大；种植历史悠久、经验丰富；一年两熟、两年三熟，产量高；亚洲人喜食大米）；不利因素（山地丘陵广布、平地面积狭小——小农机械；旱涝灾害多发——加强水利设施建设）；
- 世界农业生产发展变化特点：从自给自足到商品化生产；从「小而全」到专业化；从相对分散到适度集中；从「千篇一律」到地域分工；从粗放型到集约型；农业生产经营与现代信息技术结合。

### 工业

- 分类：按对象（采掘业、制造业）；按性质（重工业、轻工业）；按形成时间（传统工业、新兴工业）；按投入（资源、资金、劳动、技术密集型）；按指向类型（原料、市场、动力、劳动力、技术指向型）；
- 工业区位因素：自然条件（地质、地形、气候、水文、植被）；自然资源（矿产、土地、水、生物资源）；社会经济（市场、交通、政策、劳动力、基础设施、公共服务、投资环境、集聚）；科学技术；
- 环境因素：水污染——河流下游或远离居民区；大气污染——盛行风向的下风向、与季风风向垂直的郊外，或最小风频的上风向；固体废弃物污染——远离居民区、少占良田；
- 工业集聚的优势：共享基础设施，降低生产成本；节约运输费用；减少交易开支；利于技术协作和信息交流；增强对外竞争力；便于集中处理污染物；
- 美国东北部老工业区：兴起（矿产资源丰富——阿巴拉契亚山的煤炭、五大湖区的铁矿；市场广阔；交通便利）；衰落（以重化工业为主、产业结构单一、环境污染严重、交通拥挤、地价昂贵、税收高；日本西欧和发展中国家的竞争；美国西部和南部「阳光地带」高科技园区崛起）；振兴（调整产业结构，引进新兴产业；治理污染，美化环境；完善交通网络；加大科技投入）。

### 服务业

- 分类与特征：生产性服务业（专业化程度高、产业关联性强、服务范围广）；生活性服务业（服务对象以家庭和居民为主，行业种类繁多）；公共服务业（公益性、非营利性、遍在性）；
- 金融服务业（生产性）：高度流动性 + 地域集中性；区位因素（优越的地理位置、雄厚的经济基础、先进的交通通信设施、灵活创新的制度与环境、发达的科技水平与研发能力、高素质专业人才、广阔的市场需求）；
- 商业服务业（生活性）区位因素：市场、交通、集聚、中心地理论；
- 商业中心选址：零售类——市场最优原则（市中心、街道沿线、街角路口）；批发类——交通最优原则（城市边缘交通干线沿线）；
- 中心地理论：中心地服务范围为正六边形；中心地级别越高，服务职能越多、服务范围越大、距离越远、数量越少；高级中心地的服务范围覆盖低级中心地，形成层层嵌套的服务等级体系。

## 第四章 区域发展战略

### 交通运输方式

| 方式 |                            优点                            |                     缺点                     |              适合运输              |
| :--: | :--------------------------------------------------------: | :------------------------------------------: | :--------------------------------: |
| 铁路 |      运量大、运速快、运费较低、受自然影响小、连续性好      |        造价高、占地多、短途运输成本高        |          远程、大宗的货物          |
| 公路 |               运速快、自然适应性强、灵活性强               |             运量小、长途运费较高             |   短程、量小、易变质的货物和客运   |
| 水路 |                   运量大、运费低、投资少                   | 运速慢、灵活性差、连续性差、受自然条件影响大 |    远程、大宗笨重、不急需的货物    |
| 航空 |                       运速快、效率高                       |      运量小、运费高、投资大、技术要求高      | 客运为主；急需、贵重、量不大的货物 |
| 管道 | 运具和线路合二为一；运量大、运费低、连续性强、安全、损耗小 |        投资大、灵活性差、只能单向运输        |            大量流体货物            |

### 交通运输区位因素

- 自然因素（仍然重要）：地形、地质、河流、气候；
- 社会经济（决定性因素）：资源、城市、工农业基础；社会稳定、国防巩固、民族团结；
- 科学技术（保障性因素）；
- 交通运输与区域经济发展：交通促经济（建设时带动相关产业、促进就业，建成后增加运营里程、扩大客货运量）；经济促交通（增大运输需求，促进交通运输基础设施集聚）。

### 区域发展战略

- 三大自然区：东部季风区、西北干旱半干旱区、青藏高寒区；
- 四大经济地区：东部、中部、西部、东北（$10+6+12+3$）；
- 西部大开发的意义：缩小东西部发展差距，推动区域协调发展，实现共同富裕；加强民族团结，保持社会稳定与边疆安全；扩大国内市场需求；将西部资源优势转为经济优势。

### 长江经济带

- 战略定位：具有全球影响力的内河经济带；东中西互动合作的协调发展带；沿海沿江沿边全面推进的对内对外开放带；生态文明建设的先行示范带；
- 上游（成都平原等）：生态脆弱、水能丰富——梯级开发，三峡水利枢纽具有防洪、发电、航运综合效益；加强防护林和水源涵养林建设；
- 中游（两湖平原）：重要商品粮基地，洪涝威胁严重——退耕还湖、加固堤防、营造防护林；
- 下游（长三角）：沪宁杭综合性工业基地，大气和水体污染严重——调整优化产业结构、加强环境治理。

### 京津冀协同发展

- 北京：全国政治、文化、国际交往、科技创新中心；疏解非首都功能；
- 天津：先进制造研发基地、北方国际航运核心区、金融创新运营示范区、改革开放先行区；
- 河北：现代商贸物流基地、产业转型升级试验区、新型城镇化与城乡统筹示范区、京津冀生态环境支撑区；
- 雄安新区：集中疏解北京非首都职能；调整优化京津冀城市布局和空间结构；培育创新驱动发展新引擎；促进区域协调发展。

### 海洋权益与发展战略

- 海洋权益：12 海里领海；12 海里毗连区；200 海里专属经济区；公海；
- 我国海洋发展战略：坚持陆海统筹发展；推动海洋经济发展；创新海洋科技；保护海洋生态环境；维护海洋权益；全面参与全球海洋治理；
- 南海诸岛、钓鱼岛及其附属岛屿是中国固有领土，中国对其拥有无可争辩的主权。

## 第五章 人地关系与可持续发展

### 环境问题

- 成因：过度排放废弃物超过环境自净能力（环境污染）；过度索取资源超过环境供给能力（资源衰竭、生态破坏）；
- 分类：环境污染（大气、水、土壤、生物、海洋污染；酸雨、全球变暖、臭氧空洞）；生态破坏（水土流失、土地荒漠化、土地盐碱化、森林砍伐、生物多样性减少）；资源衰竭（森林、草原、生物、矿产等资源的减少和枯竭）；
- 人地关系思想演变：崇拜自然 $\to$ 改造自然 $\to$ 征服自然 $\to$ 善待自然。

### 可持续发展

- 概念：既满足当代人的需要，又不对后代人满足其自身需求的能力构成危害的发展；
- 内涵：生态可持续发展（基础）、经济可持续发展（条件）、社会可持续发展（目的）；
- 基本原则：公平性原则（代际、区际）、持续性原则、共同性原则；
- 途径：控制人口规模（根本措施）；转变发展观念；转变发展模式（工业、农业）；公众参与，从我做起（文明消费）。

### 循环经济与清洁生产

- 循环经济：以「资源 — 产品 — 再生资源」的循环流程组织经济活动，是清洁生产和废弃物综合利用的结合；
- 清洁生产：在「原料开采 — 生产制造 — 消费使用 — 废弃物处理」的全过程中，合理利用自然资源，把对环境的影响和对人类的危害减到最小；
- 传统发展模式（高投入、高消耗、高污染）与可持续发展模式（低投入、低消耗、低污染）对比。

### 自然资源可持续利用

|         利用问题         |                可持续利用途径                |
| :----------------------: | :------------------------------------------: |
|        土地荒漠化        |           退耕还草，防治土地荒漠化           |
|         水土流失         |            退耕还林，治理水土流失            |
|       海洋「赤潮」       |               减少废弃物的排放               |
|         过度捕捞         |            禁止滥捕，实行休渔制度            |
| 滥伐森林、森林覆盖率下降 |              植树造林，保护森林              |
|  水资源短缺、水污染严重  | 节约水资源，防治水污染，修建水库，跨流域调水 |
