地理选考笔记，对应湘教版《区域发展》（选择性必修二），按章节整理。

## 第一章 认识区域

### 区域的含义、类型与特征

#### 区域的含义

- **区域**（Region）：人们在地理差异的基础上，按一定指标和方法划分出来的地球表面的空间单位；
- 区域是主观划分、客观存在的：边界有的明确（行政区、流域），有的模糊、呈过渡带（气候区、干湿地区）；
- 划分指标可以是单一的（热量带），也可以是综合的（东部季风区）；同一地表可按不同指标划出不同区域，彼此重叠交叉。

#### 区域的类型

|   分类依据   |                类型                |               举例               |
| :----------: | :--------------------------------: | :------------------------------: |
|  自然与人文  |     自然区域、人文区域、综合区域      | 长江流域、京津冀城市群、东部季风区 |
|   尺度大小   |     大、中、小尺度区域        | 亚欧大陆、长三角、一个城市的社区  |
|  边界性质    |        均质区、功能区          |   干湿地区（均质）、城市经济区（功能） |

- **均质区**：内部某要素相对一致的区域，如地形区、方言区；
- **功能区**：由中心与周边通过某种功能联系构成的区域，中心地位突出，如城市经济圈、港口腹地。

#### 区域的特征

- **整体性**：区域内各要素相互联系、相互影响，构成统一整体，牵一发而动全身；
- **差异性**：区域之间、区域内部普遍存在差异，是因地制宜的前提；
- **开放性**：区域并非封闭，与外界存在物质、能量、信息的交流（人口、资源、产品、资金流动）；
- **动态性**：区域的边界、内涵、发展水平随时间变化。

### 区域差异与因地制宜

#### 区域差异的分析

比较两个区域，先分**自然**、后分**人文**，逐项对照：

|   要素   |                       分析内容                        |
| :------: | :---------------------------------------------------: |
| **自然** | 地理位置、地形、气候、水文、植被、土壤、自然资源  |
| **人文** | 人口、城市、农业、工业、交通、科技、历史文化、政策 |

分析要落到「差异是什么 — 差异成因 — 差异导致的发展方向不同」这条链上，而非罗列现象。

**南方与北方地区差异**

|   要素   |             北方地区              |             南方地区              |
| :------: | :-------------------------------: | :-------------------------------: |
|   气候   |    温带季风气候，雨热同期，较干     |  亚热带季风气候，水热充足，湿润   |
|   地形   |    以平原、高原为主，地势平坦      |  丘陵、山地广布，平原分散         |
|   耕地   |    以旱地为主，一年一熟或两熟      |  以水田为主，一年两熟至三熟       |
| 主要作物 | 小麦、玉米、大豆、甜菜             |  水稻、油菜、甘蔗                 |
|   水运   |    河流结冰期，航运受限            |  河网密布，航运发达               |

#### 因地制宜

- **因地制宜**：根据区域自身条件，扬长避短，选择适宜的发展方向与方式；
- 核心是把区域的**优势条件**转化为发展优势，同时规避**限制性因素**；
- 区域是发展变化的，因地制宜也要随条件变化而调整，不能一成不变。

## 第二章 区域发展

### 区域发展条件分析的一般方法

分析一个区域「能发展什么、如何发展」，从区位因素入手，先自然、后人文，再作辩证的**优势 — 限制**评价。

#### 农业区域发展条件

| 类别 |                             区位因素                             |
| :--: | :--------------------------------------------------------------: |
| **自然** | 气候（光照、热量、降水、昼夜温差）、地形、土壤、水源     |
| **社会经济** | 市场、交通、政策、劳动力、地价、农业技术、种植历史     |

评价时既看有利、也看不利：如东北平原地广土肥、机械化程度高（有利），但热量不足、只能一年一熟（不利）。

#### 工业区域发展条件

| 类别 |                              区位因素                               |
| :--: | :----------------------------------------------------------------: |
| **自然** | 土地、水源、原料、燃料（能源）                              |
| **社会经济** | 市场、交通、劳动力、科技、政策、资金、基础设施、集聚、协作 |
| **环境** | 主导风向、河流上下游、与居民区的距离                       |

一般方法可归纳为：**明确区域范围 $\to$ 逐项列出区位因素 $\to$ 判断主导因素 $\to$ 评价优势与限制 $\to$ 提出发展方向**。

### 不同类型区域的发展

#### 生态脆弱区：荒漠化

- **荒漠化**（Desertification）：由于气候变化和人类活动等因素，干旱、半干旱及部分半湿润地区的土地退化；
- 我国典型区：西北干旱半干旱区（新疆、内蒙古、甘肃、宁夏）。

**荒漠化的成因**

|   类别   |                                     成因                                     |
| :------: | :--------------------------------------------------------------------------: |
| **自然** | 深居内陆、降水少、气候干旱；大风日数多，风力强；地表多疏松沙质沉积物；植被稀疏 |
| **人为** |    过度开垦、过度放牧、过度樵采；水资源利用不当；工矿交通建设破坏植被     |

人为因素是荒漠化加速扩展的**主导原因**。

**荒漠化的治理**

- 生物措施：植树种草，营造防护林（如「三北」防护林）；
- 工程措施：设置草方格沙障固沙；
- 管理措施：合理利用水资源；调整农林牧结构、退耕还林还草；控制人口增长、解决农牧区能源问题（推广沼气、太阳能）；
- 治理原则：**以防为主，防治结合，因地制宜**。

#### 生态脆弱区：石漠化

- **石漠化**：在湿润的喀斯特地区，土层流失、基岩裸露、土地退化的过程；
- 典型区：云贵高原等西南喀斯特地区；
- 成因：自然上石灰岩广布、成土慢、土层薄、地形崎岖、降水集中冲刷强；人为上过度开垦、过度樵采、陡坡耕种破坏植被；
- 治理：退耕还林还草、封山育林、发展立体农业、坡改梯、解决农村能源、生态移民。

对照记忆：荒漠化在**干旱**区、终点是**沙**；石漠化在**湿润**喀斯特区、终点是**石**。二者都以植被破坏为诱因。

#### 资源枯竭型地区

- **资源枯竭型城市**：因矿产等自然资源开采殆尽、主导产业衰退而面临转型的城市，如东北的阜新（煤）、大庆（石油）；
- 面临的问题：资源趋于枯竭、产业结构单一、经济增长乏力；生态破坏、环境污染严重；就业岗位减少、失业与社会问题突出；
- 转型方向：
  - 调整产业结构，培育接续替代产业，改变「一业独大」；
  - 发展科技与第三产业，延长资源产业链、提高附加值；
  - 治理环境污染、修复生态，改善人居环境；
  - 完善基础设施，加强人才引进与再就业培训。

> 资源枯竭型地区转型的关键，是把对资源的单一依赖转为多元产业支撑。

#### 产业转型地区

- **产业转型**：区域因原有主导产业衰退，通过技术升级、结构调整培育新增长点的过程；
- 德国鲁尔区（老工业区）：
  - 兴起——煤炭丰富、离铁矿近、水源充足、水陆交通便利、市场广阔；
  - 衰落——煤炭能源地位下降、钢铁市场萎缩；产业结构单一、以重化工为主；环境污染严重、用地紧张；新技术革命冲击；
  - 整治——调整产业结构，发展新兴产业和第三产业；调整工业布局；发展科技教育；治理环境污染、美化环境；完善交通网络。

**中国东北老工业基地振兴**：改造传统产业、发展装备制造与高新技术；发展非公有制经济与第三产业；治理污染；发挥农业与资源优势，走新型工业化道路。

### 区域可持续发展的探索

- 区域发展要在**经济、社会、生态**三者间求平衡，不能以牺牲环境换一时增长；
- 生态脆弱区优先**生态修复**，资源型地区着力**产业接续**，发达地区侧重**转型升级**；
- 共同思路：因地制宜发挥优势、优化产业结构、保护生态环境、促进区域协调。

## 第三章 区域联系与区域协调发展

### 大都市的辐射功能

- **大都市**：规模大、功能强、在区域中居主导地位的城市，如上海、纽约；
- **辐射功能**：大都市凭借经济、科技、信息、交通优势，向周边输出资金、技术、人才、产品，带动区域发展；
- 上海对长三角的辐射：产业与技术扩散、金融与信息服务、交通枢纽联结，形成以上海为核心的城市群协作网络；
- 大都市与腹地是双向联系：都市辐射带动腹地，腹地为都市提供资源、劳动力和市场。

### 产业转移

- **产业转移**（Industrial Transfer）：企业将部分或全部生产由原区域迁至其他区域的经济行为，分区域内转移和国际转移；
- 转移的一般规律：由**劳动、资源密集型**产业先转移，从**发达地区向欠发达地区**、由**成本高向成本低**的区域转移。

**产业转移的影响因素**

|   因素   |                            说明                            |
| :------: | :--------------------------------------------------------: |
| 劳动力  | 转入区廉价、充足的劳动力吸引劳动密集型产业             |
|   市场   | 突破转入区的市场限制，就近生产、就近销售               |
|   内部交易成本 | 转入区完善基础设施、优惠政策可降低交易成本         |
|   地价、资源 | 转出区地价高、资源紧张，推动产业外移              |
|   环境   | 转出区环境压力大，将污染产业外迁               |

**产业转移的影响**

|        |                       转出区                        |                        转入区                        |
| :----: | :-------------------------------------------------: | :--------------------------------------------------: |
| **有利** | 腾出空间发展高新技术产业，优化产业结构；改善环境 | 承接产业，加快工业化、城镇化，增加就业，带动经济   |
| **不利** | 短期内可能造成失业、经济增速放缓             | 可能加重环境污染与生态破坏，消耗资源           |

我国产业转移的典型路径：珠三角、长三角的劳动密集型产业向内陆及中西部转移；沿海承接国际产业转移，内陆承接沿海产业转移。

### 资源跨区域调配

- **资源跨区域调配**：为解决资源空间分布与生产、消费需求不匹配，把资源从富余区调往短缺区的工程；
- 根本原因：自然资源分布**不均**，且与区域经济发展水平、消费需求**不匹配**。

**南水北调**

- 背景：我国水资源南多北少、东多西少，与北方人口、耕地、经济需求不匹配；
- 三线方案：
  - 东线——扬州引长江水，沿京杭运河北送，需逐级抽水提升，可利用已有河道；
  - 中线——由丹江口水库引水，沿途自流至京津，水质好；
  - 西线——从长江上游引水入黄河上游，工程难度最大；
- 意义：缓解北方缺水，改善供水与生态；促进受水区经济发展；但需防调水区生态影响、沿线水污染。

**西气东输**

- 线路：由塔里木盆地（新疆）向东输往长三角等东部地区；
- 对西部（输出区）：把资源优势转为经济优势，带动相关产业与就业，完善基础设施；但需防生态破坏；
- 对东部（输入区）：缓解能源紧张，优化能源消费结构，改善大气环境；
- 对整体：促进东西部协调发展，带动沿线经济。

跨区域调配的一般分析框架：**为什么调（供需矛盾）$\to$ 怎样调（线路、方式）$\to$ 有何影响（输出区、输入区、沿线与整体）**。

### 流域内部的协作开发

- **流域**：由分水岭所包围、汇集地表水与地下水的集水区域，具有整体性、关联性，上中下游一荣俱荣、一损俱损；
- 开发原则：**统筹上中下游、兼顾防洪与开发、协调经济与生态**。

**长江流域的协作开发**

| 河段 |         优势与问题          |                   开发方向                    |
| :--: | :-------------------------: | :-------------------------------------------: |
| 上游 | 水能丰富，生态脆弱、水土流失 | 梯级开发水能（三峡等）；保护植被、涵养水源  |
| 中游 |  重要商品粮基地，洪涝威胁大   | 退耕还湖、加固堤防、营造防护林；治理水患    |
| 下游 |  经济发达，水体、大气污染重   | 调整优化产业结构；加强环境治理；发展航运    |

- **三峡工程**综合效益：防洪（首要）、发电、航运、供水、灌溉、养殖、旅游；
- 长江「黄金水道」：航道整治、港口建设、多式联运，支撑长江经济带。

**黄河流域的协作开发**

- 上游：水能与光热资源丰富，梯级开发、发展灌溉农业（宁夏平原、河套平原「塞上江南」），但要防土地荒漠化、盐碱化；
- 中游：流经黄土高原，水土流失最严重，是治理重点——打坝淤地、退耕还林还草、小流域综合治理；
- 下游：「地上悬河」，防洪任务重——加固堤防、调水调沙、统一调度水量，缓解断流；
- 全流域推行**生态保护和高质量发展**，量水而行、节水优先。

### 区域协调发展

- **区域协调发展**：缩小区域差距，促进各区域在功能上互补、要素上流通、发展上共赢；
- **京津冀协同发展**：疏解北京非首都功能，建设雄安新区；北京定位政治、文化、国际交往、科技创新中心，天津定位先进制造与航运，河北承接产业转移、构筑生态支撑区；
- **长三角一体化**：以上海为龙头，推进基础设施、产业分工、公共服务、生态环境的一体化，建设世界级城市群；
- 我国区域协调总体格局：西部大开发、东北振兴、中部崛起、东部率先发展，四大板块协同并进。

## 第四章 区位分析通法

区位分析是本册的核心方法。无论农业、工业、城市还是交通，都按**自然 — 社会经济**两大类逐项列举，再判主导因素、作辩证评价。

### 农业区位因素

| 类别 |                          因素                          |                     影响                     |
| :--: | :----------------------------------------------------: | :------------------------------------------: |
| **气候** | 光照、热量、降水、昼夜温差             | 决定作物种类、熟制与品质（温差大则糖分高） |
| **地形** | 平原、丘陵、山地                       | 平原宜耕作，山地宜林牧、立体农业           |
| **土壤** | 肥力、厚度、酸碱                       | 影响作物长势与选择（如东北黑土肥沃）       |
| **水源** | 灌溉水源是否充足                       | 干旱区决定绿洲农业的分布                   |
| **市场** | 需求量与价格                           | 决定农业类型和规模（近郊多乳畜、园艺）     |
| **交通** | 运输便利程度、保鲜冷藏                 | 影响商品农业、鲜活农产品外销               |
| **政策、劳动力、技术** | 补贴、劳动力、良种与农机 | 影响生产规模、方式与效益               |

主导因素判断示例：城郊乳畜、园艺——**市场**；河西走廊绿洲农业——**水源**；横断山区立体农业——**地形（热量随海拔变化）**。

### 工业区位因素

| 类别 |                     因素                      |                指向型工业                 |
| :--: | :-------------------------------------------: | :---------------------------------------: |
| **原料** | 原料获取的难易与运费           | 原料指向型（制糖、水产加工）           |
| **市场** | 产品市场的远近与规模           | 市场指向型（家具、印刷、饮料）         |
| **动力** | 能源供应是否充足               | 动力指向型（电解铝、炼铝）             |
| **劳动力** | 数量、价格、素质             | 劳动力指向型（服装、电子装配）         |
| **技术** | 科技与人才水平                 | 技术指向型（集成电路、飞机、精密仪器） |
| **交通、政策、集聚、环境** | 便利、扶持、协作、污染 | 综合影响厂址选择             |

- 环境因素定厂址：大气污染企业布局在**盛行风下风向**或与季风垂直的郊外、最小风频的上风向；水污染企业布局在河流**下游**、远离水源地；
- 工业集聚利于共享基础设施、降低成本、加强协作、便于集中治污。

### 城市区位因素

| 类别 |                          因素                          |
| :--: | :----------------------------------------------------: |
| **自然** | 地形（平原利于建城）、气候（中低纬、湿润）、河流（供水、运输、军事） |
| **社会经济** | 自然资源、交通、政治、军事、宗教、科技、旅游   |

- 河流对城市的作用最典型：城市常分布于**河口、干支流交汇处、水运起讫点、过河点**；
- 早期城市多受自然因素支配，现代城市更受交通、科技、政策等社会经济因素影响。

### 交通区位因素

| 类别 |                            因素                            |
| :--: | :--------------------------------------------------------: |
| **自然** | 地形、地质、水文、气候——影响选线难度与造价（仍重要） |
| **社会经济** | 经济与资源开发、人口与城市分布、政策、国防、民族团结（决定性因素） |
| **科技** | 逢山开隧、遇水架桥，是交通建设的保障性因素     |

- 交通线（点）的区位分析，落到「自然条件是否允许、社会经济是否需要、科技能否实现」三问；
- 交通与区域发展相互促进：交通建设带动沿线产业、就业与城镇发展，区域经济增长又拉动交通需求。

区位分析的通用答题思路可统一为一句：**列因素、分自然人文、判主导、评优劣、提对策**。
