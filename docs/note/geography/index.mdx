这是我的地理学考笔记，目标是过浙江省的高中地理学考。

## 参考资料

- [湘教版高中地理教材 - 国家中小学智慧教育平台](https://basic.smartedu.cn/tchMaterial)
- [自然地理学 - 维基百科](https://zh.wikipedia.org/wiki/自然地理学)
- [人文地理学 - 维基百科](https://zh.wikipedia.org/wiki/人文地理学)

## 说明

这份笔记以一份地理学考考点导引讲义为底稿，对照教材整理，按教材章节归类。

笔记面向 **学考**，只保留考点：重结论、重区位、重对比，省略大段过程推导。讲义中的教材活动题答案和综合题答题模板未收录，只保留系统知识考点。

浙江地理学考用的是 **湘教版**，只考 **必修** 两册（《地理1》自然地理、《地理2》人文地理），下面列出这两册的教材目录。

## 必修

### 《地理 1》（必修一）

第 1 章 宇宙中的地球

- 第 1 节 地球的宇宙环境
- 第 2 节 太阳对地球的影响
- 第 3 节 地球的圈层结构
- 第 4 节 地球的演化

第 2 章 地球表面形态

- 第 1 节 流水地貌
- 第 2 节 风成地貌
- 第 3 节 喀斯特、海岸和冰川地貌

第 3 章 地球上的大气

- 第 1 节 大气的组成与垂直分层
- 第 2 节 大气受热过程
- 第 3 节 大气热力环流

第 4 章 地球上的水

- 第 1 节 水循环
- 第 2 节 海水的性质和运动
- 第 3 节 海洋与人类

第 5 章 地球上的植被与土壤

- 第 1 节 主要植被与自然环境
- 第 2 节 土壤的形成

### 《地理 2》（必修二）

第 1 章 人口与地理环境

- 第 1 节 人口分布
- 第 2 节 人口迁移
- 第 3 节 人口容量

第 2 章 城镇和乡村

- 第 1 节 城乡空间结构
- 第 2 节 地域文化与城乡景观
- 第 3 节 城镇化进程及其影响

第 3 章 产业区位选择

- 第 1 节 农业区位因素与农业布局
- 第 2 节 工业区位因素与工业布局
- 第 3 节 服务业的区位选择

第 4 章 区域发展战略

- 第 1 节 交通运输与区域发展
- 第 2 节 我国区域发展战略
- 第 3 节 海洋权益与我国海洋发展战略

第 5 章 人地关系与可持续发展

- 第 1 节 人类面临的主要环境问题
- 第 2 节 协调人地关系，实现可持续发展
