这是我的地理笔记，目标是过浙江省的高中地理学考与选考。

## 参考资料

- [湘教版高中地理教材 - 国家中小学智慧教育平台](https://basic.smartedu.cn/tchMaterial)
- [自然地理学 - 维基百科](https://zh.wikipedia.org/wiki/自然地理学)
- [人文地理学 - 维基百科](https://zh.wikipedia.org/wiki/人文地理学)

## 说明

这份笔记对照 **湘教版** 教材，按教材章节归类，重考点：重结论、重区位、重对比，省略大段过程推导。必修部分以一份地理学考考点导引讲义为底稿整理，选择性必修部分对照教材补全。

浙江地理 **学考** 只考 **必修** 两册（《地理1》自然地理、《地理2》人文地理）；**选考** 在必修的基础上，加试 **选择性必修** 三册（《自然地理基础》《区域发展》《资源、环境与国家安全》）。下面列出这五册的教材目录。
