地理选考笔记，对应湘教版《资源、环境与国家安全》（选择性必修三），按章节整理。

## 第一章 自然资源与国家安全

### 自然资源与人类活动

#### 自然资源的属性

- **自然资源**（Natural Resources）：从自然界中获取、能为人类所利用的物质与能量；
- 基本属性：稀缺性（相对于人类需求有限）、整体性（各资源相互联系）、地域性（分布不均）、多用性（同一资源多种用途）、社会性（利用受社会条件制约）；
- 按再生性分：可再生资源（土地、水、气候、生物）与非可再生资源（矿产、化石能源）。可再生资源过度使用同样会枯竭，可更新速度是关键。

#### 资源安全的内涵

- **资源安全**（Resource Security）：一个国家或地区可以持续、稳定、及时、足量、经济地获取所需自然资源的状态；
- 五个维度：数量（供给充足）、质量（品质合格）、结构（品种匹配需求）、均衡（时空分布协调）、经济（价格可承受）；
- 资源安全是国家安全的**物质基础**，资源短缺、供给中断或价格剧烈波动都会威胁经济运行与社会稳定。

#### 资源问题的成因

|   环节   |               机制                |               后果               |
| :------: | :-------------------------------: | :------------------------------: |
| 需求增长 | 人口增加、经济发展、消费水平提高  |    需求量持续攀升，供需缺口大    |
| 供给受限 |  资源总量有限、分布不均、开采难度大  |     可采储量下降，成本上升     |
| 利用粗放 |      技术落后、浪费严重、回收率低      |   加速消耗、加剧短缺与污染   |
|  贸易依赖  |     国内不足、依赖进口、来源单一      | 易受国际市场与地缘政治冲击 |

资源问题的根源是**需求与供给的失衡**，叠加利用效率低与对外依赖高，安全风险被放大。

### 耕地资源与粮食安全

#### 粮食安全的概念

- **粮食安全**（Food Security）：保证任何人在任何时候都能得到为了生存与健康所需的足够食物；
- 三层含义：数量安全（产量充足）、质量安全（营养卫生）、可获得（买得到、买得起）；
- 我国基本方略：**以我为主、立足国内、确保产能、适度进口、科技支撑**，坚守「谷物基本自给、口粮绝对安全」。

#### 耕地与粮食的关系

耕地是粮食生产的根基。我国人多地少，耕地资源约束突出：耕地总量大但人均少、优质耕地比重低、后备耕地有限且多分布在生态脆弱区。粮食安全的压力最终传导到耕地上。

- 影响粮食产量的因素：耕地数量（播种面积）、耕地质量（土壤肥力、灌溉条件）、单产水平（品种、技术、投入）、复种指数（热量条件）；
- 我国粮食生产格局变化：重心「南粮北运」转为「**北粮南运**」，主产区向东北、华北集中，水土资源错配加剧（北方水少地多、南方水多地少）。

#### 耕地退化与保护

| 成因（问题）  |             影响             |                 对策                 |
| :-----------: | :--------------------------: | :----------------------------------: |
|  建设占用耕地   |    耕地面积减少，良田流失    |   划定永久基本农田，占补平衡   |
|  水土流失、荒漠化   |   耕层变薄、肥力下降、弃耕   | 退耕还林还草，工程与生物措施结合 |
| 过度耕作、重用轻养 |     土壤板结、地力衰退     |   秸秆还田、轮作休耕、测土配方施肥   |
|   土壤污染、盐碱化   |  产量与品质下降，威胁食品安全  |  控制污染源，改良治理，排盐洗盐  |

保障粮食安全的对策链：**守住耕地红线（数量）$\to$ 提升耕地质量（产能）$\to$ 藏粮于地、藏粮于技（潜力）$\to$ 节约减损、适度进口（调节）**。我国划定 $18$ 亿亩耕地红线，作为不可逾越的底线。

### 石油资源与国家安全

#### 战略资源的地位

石油被称为「工业的血液」，是最重要的**战略性资源**。其安全牵动能源、交通、化工乃至军事，供给中断会造成全局性冲击。

- 特点：分布高度集中（中东、俄罗斯等）、需求普遍、消费与生产空间错位、运输依赖海上通道；
- 我国石油安全形势：需求大、增长快，对外依存度高，进口来源与运输通道集中，易受地缘政治与海上咽喉要道（如马六甲海峡）制约。

#### 石油安全的风险与对策

| 风险环节  |               表现               |                 对策                 |
| :-------: | :------------------------------: | :----------------------------------: |
|  供给风险   |   进口来源单一，产油国局势动荡   |  多元化进口，分散来源与供应商  |
|  运输风险   |   海运航线长，咽喉通道易受封锁   | 拓展陆上管道（中亚、中俄、中缅） |
|  价格风险   |     国际油价剧烈波动，成本失控     |  建立石油储备，平抑价格冲击  |
|  技术风险   |  勘探开采技术受制，深海油气难采  |   加大科技投入，攻关深海页岩  |

石油安全的核心是**降低脆弱性**：进口多元化、通道多元化、储备制度、节约替代（发展新能源与新能源汽车）四管齐下。

### 战略性矿产资源与国家安全

- **战略性矿产**：对国民经济、国防建设与新兴产业不可或缺、供给又存在风险的关键矿产，如稀土、铁、铜、铝、锂、钴、镍等；
- 安全逻辑：现代高端制造（芯片、新能源、航空航天）高度依赖特定矿产，一旦「卡脖子」，产业链即受制于人；
- 我国矿产特点：**总量大、种类全，但人均少、贫矿多、共伴生多、大宗矿产（铁、铜）对外依存度高**；稀土储量与产量占优，是重要的战略筹码；
- 对策：加强勘查增储、提高利用率与回收率、建立储备、开展国际合作与海外权益投资、发展循环利用与替代材料。

### 海洋空间资源与海洋权益

#### 海洋资源开发

- 海洋是重要的**资源接续基地与战略空间**，蕴含生物、矿产（石油、天然气、可燃冰、多金属结核）、化学、能源（潮汐、波浪）与空间资源；
- 海洋空间利用：交通运输（港口、航道）、生产（围填海、海上风电、海洋牧场）、储藏与倾废、旅游娱乐；
- 开发中的问题：过度捕捞使渔业资源衰退；围填海破坏滩涂湿地；海洋污染与生态退化。开发须与保护并重，坚持**陆海统筹、生态优先**。

#### 海洋权益的构成

| 海域类型 | 宽度（自领海基线起） |             主权与权利             |
| :------: | :------------------: | :--------------------------------: |
|   领海   |     $12$ 海里     |     沿海国享有完全主权     |
|  毗连区  |     $24$ 海里     |  在海关、财政、移民、卫生方面管辖  |
| 专属经济区 |    $200$ 海里     | 对资源勘探、开发、养护享有主权权利 |
|  大陆架  |  可延伸至 $350$ 海里  |    对海床与底土自然资源享有权利    |
|   公海   |     不属任何国家     |      各国共享航行、飞越等自由      |

- 维护海洋权益的意义：拓展生存与发展空间、保障资源与能源供给、维护国家安全与主权完整；
- 南海诸岛、钓鱼岛及其附属岛屿是中国固有领土，中国对其拥有无可争辩的主权。我国坚持通过谈判协商解决争端，同时坚定维护领土主权与海洋权益。

## 第二章 环境安全与国家安全

### 环境安全概述

#### 环境安全的概念

- **环境安全**（Environmental Security）：人类赖以生存发展的环境处于不受或少受破坏与威胁的状态；
- 相对性：环境安全是相对的，任何开发利用都有环境代价，关键在于将影响控制在可承受范围内；
- 整体性与长期性：环境要素相互关联，一处破坏可波及全局；许多环境损害滞后显现、难以逆转，威胁具有累积性。

#### 环境问题威胁国家安全的途径

环境问题可通过三条途径上升为国家安全问题：

- **突发环境事件**：污染事故、有毒物质泄漏等，短时间造成重大损失；
- **环境问题演化**：污染与生态退化长期累积，削弱资源环境承载力，动摇发展基础；
- **跨国传导**：污染物跨境转移、资源争夺、生态难民等，引发国家间矛盾甚至冲突。

判断环境事件是否威胁国家安全，看其是否**危及人的生命健康、破坏经济社会正常运行、损害国家重大利益**。

### 环境污染与国家安全

#### 污染事件的安全影响

环境污染达到一定程度即威胁国家安全：损害国民健康、破坏生产条件、造成经济损失、引发社会恐慌与不稳定。突发污染事故往往在短时间内集中爆发，应急处置难度大。

- 影响链：污染物排放 $\to$ 环境介质（大气、水、土壤）承载超限 $\to$ 人体暴露与生态受损 $\to$ 健康、经济、社会连锁反应；
- 应对：源头预防（清洁生产）、过程控制（达标排放）、末端治理（污染修复）、应急管理（预警与处置）相结合。

#### 跨境污染

**跨境污染**（Transboundary Pollution）：污染物借助大气环流、水流等自然介质跨越国界，使一国排放危害另一国环境。

| 类型 |             传播介质             |             典型案例             |
| :--: | :------------------------------: | :------------------------------: |
| 大气 |    盛行风、大气环流输送污染物    |    酸雨跨国沉降、放射性尘埃扩散    |
| 水体 |  国际河流、界湖、海洋洋流搬运  |  跨界河流污染、海洋石油泄漏  |
| 固废 |  越境转移倾倒（多为发达 $\to$ 发展中）  |  电子垃圾、危险废物非法出口  |

跨境污染的特点：责任难界定、受害方被动、协调成本高。往往「排污在此国、受害在彼国」，单靠一国难以解决，必须**国际合作**。我国既要防范境外污染输入，也要对外承担相应责任。

### 生态退化与国家安全

- **生态退化**：生态系统的结构与功能受损、服务能力下降的过程，如水土流失、土地荒漠化、森林与湿地萎缩、生物多样性减少；
- 生态系统提供**生态系统服务**（Ecosystem Services）：供给（食物、原料、水）、调节（气候、水文、净化）、支持（土壤形成、养分循环）、文化（美学、休闲）；退化即意味着这些服务的丧失；
- 威胁国家安全的机制：

| 生态退化类型 |            连锁影响            |               对策               |
| :----------: | :---------------------------: | :------------------------------: |
|   水土流失   | 土层流失、河湖淤积、洪涝加剧、耕地减少 |  植树种草、修梯田、小流域综合治理  |
|   土地荒漠化   |  可利用土地萎缩、沙尘暴、生态难民  | 退耕还林还草、防护林、控制载畜量 |
|  森林湿地萎缩   |  调蓄与净化功能下降、碳汇减少  |  保护与修复、退田还林还湿  |
| 生物多样性减少 |  生态链断裂、遗传资源丧失、系统失稳  |  就地与迁地保护、恢复栖息地  |

生态退化通过削弱**资源环境承载力**，从根本上动摇一个国家的可持续发展能力，是缓慢而深远的安全威胁。

## 第三章 生态环境保护与国家安全

### 生态保护红线

- **生态保护红线**（Ecological Conservation Redline）：在生态空间范围内具有特殊重要生态功能、必须强制严格保护的区域；
- 划定对象：具有水源涵养、水土保持、防风固沙、生物多样性维护等重要功能的区域，以及生态敏感脆弱区；
- 「三区三线」中的红线之一：生态保护红线、永久基本农田、城镇开发边界，是国土空间开发的刚性约束；
- 意义：以空间管控守住**生态安全底线**，做到「应保尽保、能划尽划」，确保生态功能不降低、面积不减少、性质不改变。

### 自然保护地体系

我国构建以**国家公园**为主体、自然保护区为基础、各类自然公园为补充的自然保护地体系。

| 类型 |               定位               |               特点               |
| :--: | :------------------------------: | :------------------------------: |
| 国家公园 | 保护最重要、最精华的自然生态系统 |  国家代表性强、面积大、全民公益性  |
| 自然保护区 |  保护典型生态系统与珍稀物种、遗迹  |  分核心区、缓冲区、实验区管理  |
|  自然公园  |  森林、湿地、地质、海洋等公园  |  兼顾保护与合理利用、休闲科普  |

- 分区管控：核心保护区原则上禁止人为活动，一般控制区限制开发；
- 意义：系统保护自然生态系统的原真性与完整性，为国家保留最宝贵的自然遗产与生态本底。

### 生态修复

- **生态修复**（Ecological Restoration）：借助自然恢复力，辅以人工措施，使受损、退化的生态系统恢复功能的过程；
- 两条路径：**自然恢复为主**（封育、休养，适用于退化较轻区域）、**人工修复为辅**（工程与生物措施，适用于严重退化区域），二者结合、宜自然则自然；
- 遵循「山水林田湖草沙**一体化保护和系统治理**」的理念，把生态系统作为有机整体统筹修复，而非单一要素孤立处理；
- 典型工程：三北防护林、退耕还林还草、天然林保护、湿地修复、矿山复绿。修复不是简单绿化，而是恢复生态系统的**结构、功能与稳定性**。

## 第四章 环境污染与防治

### 大气污染及其防治

#### 成因与危害

- 主要污染物：颗粒物（$\text{PM}_{2.5}$、$\text{PM}_{10}$）、二氧化硫（$\text{SO}_2$）、氮氧化物（$\text{NO}_x$）、臭氧、挥发性有机物；
- 主要来源：化石燃料燃烧、工业排放、机动车尾气、扬尘；
- 危害：损害呼吸系统健康、形成酸雨与雾霾、降低能见度、影响气候。

| 大气问题 |            成因            |             危害             |               对策               |
| :------: | :------------------------: | :--------------------------: | :------------------------------: |
|   酸雨   | 燃烧排放 $\text{SO}_2$、$\text{NO}_x$ | 危害水体、土壤、生物、建筑 |  脱硫脱硝，清洁能源，控制排放  |
|   雾霾   | 颗粒物大量排放叠加静稳天气 |   降低能见度，危害健康   |  减排 + 调整能源与产业结构  |
| 光化学烟雾 | 氮氧化物与碳氢化物光照反应  |   刺激眼睛呼吸道，损害植物   |  控制机动车尾气，限行减排  |

#### 防治措施

治理大气污染的对策链：**调整能源结构（减煤增清洁）$\to$ 优化产业与交通结构 $\to$ 末端治理（除尘脱硫脱硝）$\to$ 联防联控（区域协同）**。大气污染扩散范围广，跨区域联防联控尤为关键。

### 水污染及其防治

- 污染源分类：**点源污染**（工业废水、生活污水，集中排放，易监测治理）与**面源污染**（农田化肥农药随径流汇入，分散、难控制）；
- 典型问题——**水体富营养化**（Eutrophication）：氮、磷等营养物质过量输入 $\to$ 藻类暴发（水华、赤潮）$\to$ 水体缺氧、水质恶化、生物死亡；
- 危害：破坏水生态、威胁饮用水安全、损害渔业与旅游。

| 污染类型 |         成因         |            对策            |
| :------: | :------------------: | :------------------------: |
|  点源污染  |  工业与生活污水直排  |  污水集中处理，达标排放  |
|  面源污染  | 化肥农药、养殖废弃物随径流 | 测土施肥、生态拦截、控养殖 |
| 富营养化  |  氮磷过量、水体缓流  |  截污控源，生态修复调水  |

水污染防治的关键是**控源截污**：管住入河排污口，从源头削减污染负荷，再辅以生态修复与水系连通。

### 固体废物污染及其防治

- 分类：工业固废、生活垃圾、危险废物（有毒有害，需特殊处置）；
- 危害：占用土地、污染土壤与地下水（渗滤液）、污染大气（焚烧与恶臭）、传播疾病；
- 防治遵循「**减量化、资源化、无害化**」三原则：

| 原则 |            含义            |             措施             |
| :--: | :------------------------: | :--------------------------: |
| 减量化 |   从源头减少废物产生量   |  清洁生产、简化包装、循环使用  |
| 资源化 |   回收利用，变废为宝   |  分类回收、堆肥、垃圾发电  |
| 无害化 | 安全处置，消除或降低危害 | 卫生填埋、焚烧、危废专门处理 |

垃圾分类是资源化与无害化的前提。危险废物与电子垃圾的越境转移属于国际管控对象，须依法监管。

## 第五章 全球气候变化与国家安全

### 全球气候变化的影响

- **全球气候变化**主要表现为**全球变暖**，主因是化石燃料燃烧与毁林导致大气中二氧化碳等**温室气体**（Greenhouse Gases）浓度上升，温室效应增强；
- 影响的传导：气温升高 $\to$ 冰川融化、海平面上升、极端天气增多、水热格局改变 $\to$ 淹没沿海、加剧灾害、冲击农业与水资源 $\to$ 威胁生存空间、经济安全与社会稳定；

| 影响领域 |               表现               |               安全含义               |
| :------: | :------------------------------: | :----------------------------------: |
| 海平面上升 |  沿海低地与岛国被淹，海岸侵蚀  |  丧失国土与家园，产生气候难民  |
| 极端天气 |  高温、干旱、洪涝、台风增多  |  灾害损失加大，威胁生命财产  |
|  农业   |  产量与分布变化，病虫害北扩  |  威胁粮食安全  |
|  水资源  |  冰川退缩，径流与分布改变  |  加剧水资源短缺与争端  |
| 生态系统 |  物种迁移与灭绝，带谱移动  |  生物多样性下降，生态失衡  |

气候变化是典型的**全球性、跨代际**环境安全问题，没有一个国家能独善其身。

### 碳排放与碳中和

- **碳达峰**（Carbon Peaking）：二氧化碳排放量达到历史峰值后转入下降；
- **碳中和**（Carbon Neutrality）：通过节能减排与碳汇（森林、海洋）等抵消人为碳排放，实现净零排放；
- 我国目标：力争 $2030$ 年前实现碳达峰、$2060$ 年前实现碳中和（「双碳」目标）；
- 实现路径：

| 路径 |            措施            |
| :--: | :------------------------: |
| 减排 |  发展清洁能源，优化能源与产业结构，提高能效  |
| 增汇 |  植树造林、保护森林湿地，发展蓝碳  |
| 制度 |  碳交易市场、碳税、能耗与碳排放双控  |
| 技术 |  碳捕集利用与封存、低碳与零碳技术  |

减排是根本、增汇是补充、制度与技术是保障。「双碳」既是应对气候变化的责任担当，也是推动发展方式转型的契机。

### 应对气候变化的国际合作

全球气候变化的**外部性**决定了必须国际合作：排放在各国、影响在全球，任何单边行动都难以奏效。

- 基本原则：**共同但有区别的责任**——各国都有减排义务，但发达国家因历史累积排放多、能力强，应承担更多责任并向发展中国家提供资金与技术援助；
- 重要进程：《联合国气候变化框架公约》确立合作框架，《京都议定书》为发达国家设定减排指标，《巴黎协定》确立各国「国家自主贡献」并将温升控制在 $2\ ^\circ\text{C}$、力争 $1.5\ ^\circ\text{C}$ 以内；
- 我国立场：坚持多边主义，积极履行减排承诺，推动构建公平合理、合作共赢的全球气候治理体系。

## 第六章 污染物跨境转移与国际环境合作

### 污染物跨境转移

- **污染物跨境转移**：污染物或废物越过国界从一国转移到另一国，包括自然介质输送与人为转移两类；
- 人为转移多为**危险废物与电子垃圾**从发达国家流向发展中国家：转出方为规避处理成本与环境责任，转入方或为经济利益而接收，结果是污染与健康风险的转嫁；

| 转移方式 |             机制             |             治理             |
| :------: | :--------------------------: | :--------------------------: |
| 自然介质转移 | 大气、水流搬运污染物越境  |  国际监测预警，联防联控  |
| 废物贸易转移 | 危废、电子垃圾出口倾倒 | 立法禁止，海关查验，国际公约约束 |
| 产业转移带出 | 高污染产业向欠发达地区迁移 |  提高准入门槛，绿色标准  |

我国已立法禁止「洋垃圾」进口，从源头堵住污染物跨境输入。

### 国际环境公约与合作

跨境环境问题超越国界，须以**国际公约**为纽带开展合作。主要公约：

| 公约 |               针对问题               |
| :--: | :----------------------------------: |
| 《巴塞尔公约》 |  控制危险废物越境转移及其处置  |
| 《蒙特利尔议定书》 |  淘汰消耗臭氧层物质（氟氯烃）  |
| 《生物多样性公约》 |  保护生物多样性，公平分享遗传资源  |
| 《联合国气候变化框架公约》 |  应对全球气候变化  |
| 《联合国海洋法公约》 |  确立海洋权益与海洋环境保护规则  |

国际环境合作的难点在于各国利益与发展阶段不同，需要在**责任分担与利益协调**之间取得平衡。我国是全球环境治理的重要参与者、贡献者与引领者。

## 第七章 走向生态文明与可持续发展

### 生态文明建设

- **生态文明**（Ecological Civilization）：以尊重和保护自然为前提、以人与自然和谐共生为宗旨的文明形态，是继工业文明之后的新阶段；
- 核心理念：**「绿水青山就是金山银山」**——良好生态本身蕴含经济与社会价值，保护生态与发展经济并非对立；
- 「五位一体」总体布局：把生态文明建设融入经济、政治、文化、社会建设各方面和全过程；
- 建设路径：优化国土空间开发格局、全面促进资源节约、加大生态环境保护力度、健全生态文明制度体系。

### 可持续发展

- **可持续发展**（Sustainable Development）：既满足当代人的需要，又不对后代人满足其自身需要的能力构成危害的发展；
- 三个内涵：生态可持续（**基础**）、经济可持续（**条件**）、社会可持续（**目的**），三者相互联系、缺一不可；
- 三条基本原则：公平性（代际与代内、区际公平）、持续性（不超越资源环境承载力）、共同性（全球共同参与）；
- 实现途径——**循环经济**：以「资源 $\to$ 产品 $\to$ 再生资源」的闭环组织生产，对应工业上的清洁生产、农业上的生态农业，用**减量、再用、循环**取代「资源 $\to$ 产品 $\to$ 废物」的线性模式。

### 国家安全视角下的资源环境

贯穿全书的逻辑：**资源、环境与国家安全三者环环相扣**。

- 资源短缺威胁供给安全，环境污染与生态退化削弱承载力，全球气候变化冲击生存空间——三者最终都指向国家的可持续发展能力；
- 应对之道，是从「先污染后治理」的旧路，转向**节约优先、保护优先、自然恢复为主**的方针，走生产发展、生活富裕、生态良好的文明发展道路；
- 建设**人与自然生命共同体**与**人类命运共同体**：本国的资源环境安全，离不开全球的合作与共治。

> 资源是安全的物质基础，环境是安全的空间保障，可持续是安全的长远之道。
