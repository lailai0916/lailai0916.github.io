## 说明

这份笔记对照人教版教材，按专题归类，重考点：重概念、重公式、重模型、重方法，省略大段推导与实验细节；加粗的多是关键概念、必记公式和标志性结论。

浙江高中物理用的是 **人教版**。**学考** 考 **必修** 三册；**选考** 在必修的基础上，加试 **选择性必修** 三册。下面列出这六册的教材目录。

公式用 KaTeX 排版，如 $v=v_0+at$、$F=ma$、$E_k=\frac{1}{2}mv^2$，矢量记作 $\vec{F}$。
