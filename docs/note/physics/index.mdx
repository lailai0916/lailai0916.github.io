## 说明

这份笔记对照人教版教材，按知识板块归类，重概念、公式、模型与方法，省略大段推导与实验细节。加粗的多是关键概念、必记公式和标志性结论。

浙江高中物理学考考必修三册；选考在此基础上加试选择性必修三册。下面按知识对象与前置关系重新编排，不再沿用教材册次。

## 板块

- [运动与力](/docs/note/physics/motion-force)：运动学、受力分析、牛顿定律、抛体与圆周运动；
- [功、能量与动量](/docs/note/physics/energy-momentum)：功、功能关系、机械能与动量守恒；
- [万有引力与航天](/docs/note/physics/gravitation)：开普勒定律、天体计算、卫星与宇宙速度；
- [电场与电路](/docs/note/physics/electricity)：静电场、电势、电容、直流电路与电功率；
- [磁场与电磁感应](/docs/note/physics/electromagnetism)：安培力、洛伦兹力、电磁感应与交变电流；
- [振动、波与光](/docs/note/physics/wave-optics)：简谐运动、机械波、几何光学与波动光学；
- [热学](/docs/note/physics/thermology)：分子动理论、物态、气体定律与热力学；
- [近代物理](/docs/note/physics/modern-physics)：量子、波粒二象性、原子结构与原子核。
