# 对拍器

## 文件目录

```bash
checker/
├── main.bat                 # 批处理脚本
├── gen.exe                  # 数据生成器
├── std.exe                  # 标准程序
└── sol.exe                  # 被测试的程序
```

## 代码

### main.bat

```batch
@echo off
setlocal enabledelayedexpansion
set num=0

:loop
	set /a num+=1
	gen.exe > in
	std.exe < in > ans
	sol.exe < in > out
	fc out ans > nul 2>&1
	if errorlevel 1 (
		powershell -command "Write-Host '>>> Error #!num!' -ForegroundColor Red"
		fc out ans
		goto end
	) else (
		powershell -command "Write-Host '>>> Success #!num!' -ForegroundColor Green"
		goto loop
	)

:end
pause
```

### gen.cpp

```cpp
#include <bits/stdc++.h>

random_device rd;

{
	uniform_int_distribution<int> dist(x,y);
	return dist(gen);
```
