# 竞赛

信息学奥林匹克竞赛（Olympiad in Informatics）在中国起源于 1984 年，是五大高中学科竞赛之一。

国际大学生程序设计竞赛（International Collegiate Programming Contest）由 ICPC 基金会（ICPC Foundation）举办，是最具影响力的大学生计算机竞赛。
