# 康托展开

$$
\sum_{i=1}^n(n-i)!\sum_{j=i}^n[a_j<a_i]
$$

## 参考资料

- [置换和排列 - OI Wiki](https://oi.wiki/math/permutation/)

## 例题

### 洛谷 P5367 【模板】康托展开

:::info[[洛谷 P5367 【模板】康托展开](https://www.luogu.com.cn/problem/P5367)]

求 $1\sim N$ 的一个给定全排列在所有 $1\sim N$ 全排列中的排名。结果对 $998244353$ 取模。

:::

```cpp
#include <bits/stdc++.h>
#include <bits/extc++.h>
using namespace std;
using namespace __gnu_pbds;

using ll=long long;
const int mod=998244353;
const int N=1000005;
int a[N];
tree<int,null_type,less<int>,rb_tree_tag,tree_order_statistics_node_update> T;
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		int x;
		cin>>x;
		a[n-i+1]=x;
	}
	ll ans=1,fac=1;
	for(int i=1;i<=n;i++)
	{
		ans=(ans+fac*T.order_of_key(a[i])%mod)%mod;
		fac=fac*i%mod;
		T.insert(a[i]);
	}
	cout<<ans<<'\n';
	return 0;
}
```
