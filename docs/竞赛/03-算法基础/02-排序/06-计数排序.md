# 计数排序（Counting Sort）

## 参考资料

- [计数排序 - OI Wiki](https://oi.wiki/basic/counting-sort/)

## 实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int inf=0x3f3f3f3f;
const int N=10005;
int mn=inf,mx=0;
int a[N];
int main()
{
	int n;
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		int x;
		cin>>x;
		a[x]++;
		mn=min(mn,x);
		mx=max(mx,x);
	}
	for(int i=mn;i<=mx;i++)
	{
		for(int j=1;j<=a[i];j++)
		{
			cout<<i<<' ';
		}
	}
	return 0;
}
```
