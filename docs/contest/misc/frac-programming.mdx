## 简介

**分数规划** 求一个子集，使「选中元素的收益之和除以代价之和」最大或最小，最常见的是每个元素只能选或不选的 **01 分数规划**。二分答案 $mid$，将每个元素的权值改为 $\text{收益}-mid\times\text{代价}$，再判断是否存在子集使总权值非负，即把分式最优化转化为可行性判定。
