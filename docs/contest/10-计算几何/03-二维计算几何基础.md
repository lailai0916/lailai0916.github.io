### 洛谷 P1183 多边形的面积

:::info[[洛谷 P1183 多边形的面积](https://www.luogu.com.cn/problem/P1183)]

给出一个没有缺口的简单多边形，它的边是垂直或者水平的，要求计算多边形的面积。

:::

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N=105;
struct Point
{
	double x,y;
}a[N];
double Cross(Point A,Point B){return A.x*B.y-A.y*B.x;}
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i].x>>a[i].y;
	}
	double ans=0;
	for(int i=1;i<=n;i++)
	{
		ans+=Cross(a[i],a[i%n+1])/2;
	}
	cout<<fabs(ans)<<'\n';
	return 0;
}
```
