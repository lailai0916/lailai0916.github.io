## 简介

ST 表（Sparse Table，稀疏表）是用于解决 **可重复贡献问题** 的数据结构。它预处理出每个以 $2^k$ 为长度的区间的聚合值（如区间最值），查询 $[l,r]$ 时用两个可重叠的 $2^k$ 长区间覆盖，做到 $O(n\log n)$ 预处理、$O(1)$ 查询。它仅适用于满足结合律且可重复贡献的运算，如 $\max$、$\gcd$。

## 实现

```cpp
struct ST
{
	int a[30][N];
	void init(int n)
	{
		for(int i=1;i<=__lg(n);i++)
		{
			for(int j=1;j<=n-(1<<i)+1;j++)
			{
				a[i][j]=max(a[i-1][j],a[i-1][j+(1<<(i-1))]);
			}
		}
	}
	int query(int l,int r)
	{
		int k=__lg(r-l+1);
		return max(a[k][l],a[k][r-(1<<k)+1]);
	}
};
```
