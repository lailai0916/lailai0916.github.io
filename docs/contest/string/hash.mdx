## 简介

**字符串哈希**（String Hashing）把字符串映射成一个整数，使字符串比较降到 $O(1)$。常用多项式哈希：把串看作 $\mathrm{base}$ 进制数，$\operatorname{hash}(s)=\sum_i s_i\cdot\mathrm{base}^{|s|-1-i}\bmod p$。配合前缀哈希可 $O(1)$ 取出任意子串的哈希值。取大质数模数与较大底数可降低冲突概率。

## 实现

```cpp
using ull=unsigned long long;
const int base=131;
const ull mod=212370440130137957;
ull get_hash(string s)
{
	ull res=0;
	for(int i=0;i<s.size();i++)
	{
		res=(res*base+s[i])%mod;
	}
	return res;
}
```
