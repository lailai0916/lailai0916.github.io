## 简介

**字符串匹配**（String Matching）求模式串 $t$ 在文本串 $s$ 中的所有出现位置。朴素枚举为 $O(nm)$。**KMP** 预处理模式串的前缀函数 $\mathrm{nxt}$，失配时跳到最长相等前后缀处继续，做到 $O(n+m)$。**Rabin–Karp** 用滚动哈希逐位比较子串，平均 $O(n+m)$。

## KMP 算法

```cpp
int kmp(string s,string t)
{
	int n=s.size(),m=t.size();
	for(int i=1,j=0;i<m;i++)
	{
		while(j&&t[i]!=t[j])j=nxt[j-1];
		if(t[i]==t[j])j++;
		nxt[i]=j;
	}
	for(int i=0,j=0;i<n;i++)
	{
		while(j&&s[i]!=t[j])j=nxt[j-1];
		if(s[i]==t[j])j++;
		if(j==m)return i-m+1;
	}
	return -1;
}
```

## Rabin–Karp 算法

详见 [字符串哈希](/docs/contest/string/hash)。

```cpp
int RK(string s,string t)
{
	int n=s.size(),m=t.size();
	if(n<m)return -1;
	ull h1=0,h2=0,p=Pow(base,m-1);
	for(int i=0;i<m;i++)
	{
		h1=h1*base+s[i];
		h2=h2*base+t[i];
	}
	s+='$';
	for(int i=0;i<=n-m;i++)
	{
		if(h1==h2&&s.substr(i,m)==t)return i;
		h1=(h1-s[i]*p)*base+s[i+m];
	}
	return -1;
}
```
