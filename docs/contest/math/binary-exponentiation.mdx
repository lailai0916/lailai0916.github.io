## 快速幂

**快速幂**（Binary Exponentiation）把指数 $y$ 按二进制拆分，每一步将底数平方，遇到为 $1$ 的二进制位就把当前底数乘入答案，从而在 $O(\log y)$ 内求出 $x^y\bmod p$。

```cpp
ll Pow(ll x,ll y)
{
	x%=mod;
	ll res=1;
	while(y)
	{
		if(y&1)res=res*x%mod;
		x=x*x%mod;
		y>>=1;
	}
	return res;
}
```

## 龟速乘

龟速乘用于计算 $x\times y\bmod p$，可以防止直接计算 $x\times y$ 过大导致溢出。

```cpp
ll mul(ll x,ll y)
{
	x%=mod;
	ll res=0;
	while(y)
	{
		if(y&1)res=(res+x)%mod;
		x=(x+x)%mod;
		y>>=1;
	}
	return res;
}
```
