## 线性逆元

$$
i^{-1}\equiv-\left\lfloor\frac{p}{i}\right\rfloor(p\bmod i)^{-1}\pmod p
$$

详见 [模逆元](/docs/contest/math/number-theory/inverse)。

```cpp
ll inv[N],fac[N],jv[N];
void init()
{
	fac[0]=jv[0]=1;
	for(int i=1;i<N;i++)
	{
		inv[i]=i==1?1:(mod-mod/i)*inv[mod%i]%mod;
		fac[i]=fac[i-1]*i%mod;
		jv[i]=jv[i-1]*inv[i]%mod;
	}
}
```

## 组合

组合数 $C_n^m$（即 $\binom{n}{m}$）表示从 $n$ 个不同元素中取出 $m$ 个、不计顺序的方案数。预处理阶乘与阶乘逆元后可 $O(1)$ 求值。

$$
C_n^m=\frac{n!}{m!(n-m)!}
$$

```cpp
ll C(ll n,ll m)
{
	if(n<m||m<0)return 0;
	return fac[n]*jv[n-m]%mod*jv[m]%mod;
}
```

## 排列

排列数 $A_n^m$ 表示从 $n$ 个不同元素中取出 $m$ 个、按顺序排列的方案数。

$$
A_n^m=\frac{n!}{(n-m)!}
$$

```cpp
ll A(ll n,ll m)
{
	if(n<m||m<0)return 0;
	return fac[n]*jv[n-m]%mod;
}
```

## 卢卡斯定理

**卢卡斯定理**（Lucas）用于质数模 $p$ 下、$n$ 与 $m$ 很大时求 $\binom{n}{m}\bmod p$：把 $n$、$m$ 在 $p$ 进制下逐位取组合数再相乘，递归求解。

$$
C_n^m\equiv C_{n/p}^{m/p}\cdot C_{n\bmod p}^{m\bmod p}\pmod p
$$

```cpp
ll lucas(ll n,ll m)
{
	if(m==0)return 1;
	return C(n%mod,m%mod)*lucas(n/mod,m/mod)%mod;
}
```
