## 康托展开

**康托展开** 求一个排列在全部 $n!$ 个排列中按字典序的排名。第 $i$ 位的贡献为「其后比 $a_i$ 小的数的个数」乘以 $(n-i)!$，求和即得名次。用树状数组统计后缀小于个数可做到 $O(n\log n)$。逆康托展开则由名次还原出排列。

$$
x=\sum_{i=1}^n(n-i)!\sum_{j=i}^n[a_j<a_i]
$$

序号通常从 $1$ 开始，所以初始令 `ans=1`。

```cpp
ll ans=1;
for(int i=n;i>=1;i--)
{
	ans=(ans+fac[n-i]*sum(a[i]))%mod;
	add(a[i]);
}
```
