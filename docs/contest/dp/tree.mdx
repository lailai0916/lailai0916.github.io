## 简介

**树形 DP** 在树上做动态规划，通常设 $f_u$ 表示以 $u$ 为根的子树的某种最优值，自底向上由各子节点的状态合并而来，用一遍 DFS 实现。

## 实现

以「没有上司的舞会」为例：$f_{u,0}$、$f_{u,1}$ 分别表示不选、选节点 $u$ 时子树的最大权值。选 $u$ 则子节点都不能选，不选 $u$ 则子节点选或不选取较大者。

```cpp
void dfs(int u)
{
	for(auto v:G[u])
	{
		dfs(v);
		f[u][0]+=max(f[v][0],f[v][1]);
		f[u][1]+=f[v][0];
	}
}
```
