# 选择排序（Selection Sort）

## 参考资料

- [选择排序 - OI Wiki](https://oi-wiki.org/basic/selection-sort/)

## 实现

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N=10005;
int a[N];
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i];
	}
	for(int i=1;i<n;i++)
	{
		int k=i;
		for(int j=i+1;j<=n;j++)
		{
			if(a[j]<a[k])k=j;
		}
		swap(a[i],a[k]);
	}
	for(int i=1;i<=n;i++)
	{
		cout<<a[i]<<' ';
	}
	return 0;
}
```
