## 文件

- 文件名使用 **小写字母** 或 **数字**，使用 `-` 分割，避免使用中文或拼音、`_`。

> text-format.md

- 文件必须以 $1$ 个 **空行** 结束。

## 标题

- 每篇文本最多只能有 $1$ 个 **一级标题**。
