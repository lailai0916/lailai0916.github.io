## 文件

- 文件名使用 **小写字母** 或 **数字**，避免使用中文或拼音。
- 文件名使用 `-` 分割，避免使用 `_`，例如 `text-format.md`。
- 文件必须以 $1$ 个 **空行** 结束。

## 通用

- 每篇文本最多只能有 $1$ 个 **一级标题**。
- 页面开头应有一段简短的文字概述页面内容，例如「本页面将介绍……」。
- 对于尚未撰写或未完成的内容，应使用 `[TODO]` 标记。

## 标点

- 在每句话的末尾添加 **句号**。
- 注意区分各种不同的 **连接号**：`-`（U+002D，hyphen，连字符、减号）；`–`（U+2013，en dash，短破折号）；`—`（U+2014，em dash，长破折号）。
