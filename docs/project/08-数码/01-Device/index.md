## MacBook Pro 16 英寸

我之前从没接触过 macOS 系统，直到用了 MacBook 之后才发现，这才是真正的电脑该有的样子。

现在我可以毫不犹豫地说，我的下一台电脑也还会是 Mac。

虽然不是所有场景 macOS 都占优，比如某些 **专业软件** 或 **大型游戏** 会不太适配。

但对于日常轻办公（写文档、浏览网页、看视频）、写代码、剪视频这些工作，macOS 却特别顺手。

MacBook 不仅系统好用，设计出色，续航很长，触摸板用习惯后完全能取代鼠标。

我到现在几乎没碰过鼠标，出门只带笔记本就够了，不像 Windows 笔记本还得拖着充电器和鼠标。

从被人推荐用 Mac，到现在自己忍不住推荐给别人，这种对 macOS 相见恨晚的感觉，恐怕只有用过的人才懂。

如果你还没用过 Mac，那么你或许真的错过了很多。

### 配置

- 尺寸：16 英寸
- 外观：银色
- 芯片：M3 Max
- 内存：36GB
- 存储：1TB
- 规格：[技术规格](https://support.apple.com/zh-cn/117737)
- 售价：$3,499 / ¥27,999

### 配件

MacBook 比较贵重，又是铝合金外壳，所以我买了个 Apple Store 官网的 Incase **保护壳**。

- Incase 硬质保护壳

我之前试过贴 **屏幕膜** 和 **键盘膜**，但发现屏幕和键盘的间隔很小，贴了膜会让键盘印在屏幕上，所以最后还是放弃了。
