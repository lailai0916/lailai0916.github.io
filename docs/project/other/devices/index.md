## MacBook Pro (M3 Max)

- 类型：笔记本电脑
- 配置：16 英寸 / 银色 / M3 Max / 30+40 / 36GB / 1TB
- 配件：Incase 硬质保护壳
- 技术规格：[MacBook Pro - Apple 支持](https://support.apple.com/zh-cn/117737)

MacBook Pro 的铝合金外壳容易变形，所以我配了一个保护壳。

我以前试过贴屏幕膜和键盘膜，但因为 B 面（屏幕）和 C 面（键盘）间隙特别小，键盘会在屏幕上留下印子，最后就放弃了。
